# merchantIdentifier

**Framework**: Apple Pay on the Web  
**Kind**: property

The merchant identifier you registered with Apple for use with Apple Pay.

**Availability**:
- Safari Desktop 10.0+
- Safari Mobile 10.0+

## Declaration

```swift
required DOMString merchantIdentifier;
```

#### Discussion

This is the merchant ID you created and registered with Apple. See [`Configuring Your Environment`](https://developer.apple.com/documentation/apple_pay_on_the_web/configuring_your_environment) for more information.
