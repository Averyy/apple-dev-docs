# supportsCouponCode

**Framework**: Apple Pay on the Web  
**Kind**: property

A Boolean value that determines whether the payment sheet displays the coupon code field.

**Availability**:
- Safari Desktop 10.0+
- Safari Mobile 10.0+

## Declaration

```swift
boolean supportsCouponCode;
```

#### Discussion

Set the value to `true` to display the coupon code field.
