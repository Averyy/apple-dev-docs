# locale

**Framework**: Apple Pay on the Web  
**Kind**: property

The language and region used for the displayed Apple Pay button.

**Availability**:
- Safari Desktop 10.0+
- Safari Mobile 10.0+

## Declaration

```swift
attribute ApplePayButtonLocale locale;
```

#### Discussion

For a list of button locales, see [`ApplePayButtonLocale`](applepaybuttonlocale.md).
