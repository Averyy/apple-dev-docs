# STATUS_FAILURE

**Framework**: Apple Pay on the Web  
**Kind**: var

The requested action failed.

**Availability**:
- Safari Desktop 10.0+
- Safari Mobile 10.0+

## Declaration

```swift
const unsigned short STATUS_FAILURE;
```

#### Discussion

The payment is not completed and the payment sheet is dismissed.
