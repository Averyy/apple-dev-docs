# ApplePayButtonType

**Framework**: Apple Pay on the Web  
**Kind**: enum

A type that indicates the button types that you can display to initiate Apple Pay transactions.

**Availability**:
- Safari Desktop 10.0+
- Safari Mobile 10.0+

## Declaration
