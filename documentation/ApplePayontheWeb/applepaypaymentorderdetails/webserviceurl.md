# webServiceURL

**Framework**: Apple Pay on the Web  
**Kind**: property

The URL of your web service.

**Availability**:
- Safari Desktop 10.0+
- Safari Mobile 10.0+

## Declaration

```swift
required DOMString webServiceURL;
```

#### Discussion

The URL for your web service needs to begin with `https://`; for example, `https://example.com`.
