# Using the Apple Pay JS SDK for iOS 18

**Framework**: Apple Pay on the Web

Add the latest Apple Pay JS SDK.

#### Overview

Use Apple Pay JS features for iOS 18 and later by adding the following script tag to the head tags for your site:

```javascript
<head>
    <script crossorigin 
      src="https://applepay.cdn-apple.com/jsapi/1.latest/apple-pay-sdk.js"> 
    </script>
</head>

```
