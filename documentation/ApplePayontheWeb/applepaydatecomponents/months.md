# months

**Framework**: Apple Pay on the Web  
**Kind**: property

A number between 1 and 12 that represents a month.

**Availability**:
- Safari Desktop 10.0+
- Safari Mobile 10.0+

## Declaration

```swift
long months;
```

#### Discussion

Numbers representing months are 1-based. For example, to represent January, use `1` and to represent December, use `12`.
