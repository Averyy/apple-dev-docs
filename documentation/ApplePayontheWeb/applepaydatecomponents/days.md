# days

**Framework**: Apple Pay on the Web  
**Kind**: property

A number that represents a day.

**Availability**:
- Safari Desktop 10.0+
- Safari Mobile 10.0+

## Declaration

```swift
long days;
```

#### Discussion

Numbers representing days are 1-based. For example, to represent the third day of a month, use `3`.
