# init(bigEndian:)

**Framework**: Network  
**Kind**: init  
**Required**: Yes

Creates an integer from its big-endian representation, changing the byte order if necessary.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+
- watchOS 26.0+

## Declaration

```swift
override init(bigEndian: Self)
```

## Parameters

- `value`: A value to use as the big-endian representation of the   new integer.
