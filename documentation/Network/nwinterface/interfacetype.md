# NWInterface.InterfaceType

**Framework**: Network  
**Kind**: enum

Types of network interfaces, based on their link layer media types.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 12.0+
- macOS 10.14+
- tvOS 12.0+

- watchOS 5.0+
