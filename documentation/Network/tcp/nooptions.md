# noOptions(_:)

**Framework**: Network  
**Kind**: method

Enable no-options mode.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+
- watchOS 26.0+

## Declaration

```swift
func noOptions(_ noOptions: Bool) -> TCP
```

#### Discussion

A boolean indicating that TCP should use no-options mode (`TCP_NOOPT`).

## Parameters

- `noOptions`: True to use no-options mode, false otherwise.
