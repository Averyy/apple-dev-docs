# noPush(_:)

**Framework**: Network  
**Kind**: method

Enable no-push mode.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+
- watchOS 26.0+

## Declaration

```swift
func noPush(_ noPush: Bool) -> TCP
```

#### Discussion

A boolean indicating that TCP should use no-push mode (`TCP_NOPUSH`).

## Parameters

- `noPush`: True to use no-push mode, false otherwise.
