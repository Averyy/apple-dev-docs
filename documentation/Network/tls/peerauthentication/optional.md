# TLS.PeerAuthentication.optional

**Framework**: Network  
**Kind**: case

Requests the peer certificate, but if none is provided, proceed with the connection.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+
- watchOS 26.0+

## Declaration

```swift
case optional
```

#### Discussion

This is for servers. Clients will ignore this option.
