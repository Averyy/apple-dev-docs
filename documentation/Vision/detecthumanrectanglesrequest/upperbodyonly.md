# upperBodyOnly

**Framework**: Vision  
**Kind**: property

A Boolean value that indicates whether the request requires only detecting a human upper body to produce a result.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+

## Declaration

```swift
var upperBodyOnly: Bool
```

#### Discussion

The default value is `true.`
