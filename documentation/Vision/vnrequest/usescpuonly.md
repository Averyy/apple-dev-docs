# usesCPUOnly

**Framework**: Vision  
**Kind**: property

A Boolean signifying that the Vision request should execute exclusively on the CPU.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.13+
- tvOS 11.0+
- visionOS 1.0+

## Declaration

```swift
var usesCPUOnly: Bool { get set }
```

#### Discussion

This value defaults to [`false`](https://developer.apple.com/documentation/Swift/false) to signify that the Vision request is free to leverage the GPU to accelerate its processing.
