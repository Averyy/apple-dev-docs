# init(_:)

**Framework**: Vision  
**Kind**: init

Creates a video processor to perform framework requests against the video asset you specify.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+

## Declaration

```swift
init(_ videoURL: URL)
```

## Parameters

- `videoURL`: The video asset in a video format that   supports.
