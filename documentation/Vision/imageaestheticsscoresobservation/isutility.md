# isUtility

**Framework**: Vision  
**Kind**: property

A Boolean value that represents images that are not necessarily of poor image quality, but may not have memorable or exciting content.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
