# detectsDarkOnLight

**Framework**: Vision  
**Kind**: property

A Boolean value that indicates whether the request detects a dark object on a light background to aid in detection.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+

## Declaration

```swift
var detectsDarkOnLight: Bool
```

#### Discussion

The default value is `true`.
