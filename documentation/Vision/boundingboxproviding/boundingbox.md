# boundingBox

**Framework**: Vision  
**Kind**: property  
**Required**: Yes

The bounding box of the object.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+

## Declaration

```swift
var boundingBox: NormalizedRect { get }
```

#### Discussion

The coordinate system is normalized to the dimensions of the processed image, with the origin at the lower-left corner of the image.
