# confidence

**Framework**: Vision  
**Kind**: property

A confidence estimate for the detected landmarks.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.13+
- tvOS 11.0+
- visionOS 1.0+

## Declaration

```swift
var confidence: VNConfidence { get }
```

#### Discussion

A value of `0` indicates no confidence. A value of `1` indicates full confidence.
