# hash(into:)

**Framework**: Vision  
**Kind**: method  
**Required**: Yes

Hashes the essential components of the value by passing them into the hasher.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+

## Declaration

```swift
override func hash(into hasher: inout Hasher)
```

## Parameters

- `hasher`: The hasher to use when combining the components of this instance.
