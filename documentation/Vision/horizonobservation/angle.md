# angle

**Framework**: Vision  
**Kind**: property

The angle of the observed horizon.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+

## Declaration

```swift
let angle: Measurement<UnitAngle>
```

#### Discussion

Use the angle to orient the image in an upright position and make the detected horizon level.
