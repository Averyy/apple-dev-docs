# pointCount

**Framework**: Vision  
**Kind**: property

The number of points in the face region.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.13+
- tvOS 11.0+
- visionOS 1.0+

## Declaration

```swift
var pointCount: Int { get }
```

#### Discussion

The value is zero if no points for a region could be found.
