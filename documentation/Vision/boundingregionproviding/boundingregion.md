# boundingRegion

**Framework**: Vision  
**Kind**: property  
**Required**: Yes

A polygon that defines the boundary of an area in the image.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+

## Declaration

```swift
var boundingRegion: NormalizedRegion { get }
```

#### Discussion

Similar to a bounding box, however it can have more than 4 points comprising the boundary.
