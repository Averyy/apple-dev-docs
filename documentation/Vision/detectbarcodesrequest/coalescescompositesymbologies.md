# coalescesCompositeSymbologies

**Framework**: Vision  
**Kind**: property

A Boolean value that indicates whether the request coalesces multiple codes into one.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+

## Declaration

```swift
var coalescesCompositeSymbologies: Bool
```

#### Discussion

The default value for this property is `false`.
