# rightPupil

**Framework**: Vision  
**Kind**: property

The region containing the point where the right pupil is located.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.13+
- tvOS 11.0+
- visionOS 1.0+

## Declaration

```swift
var rightPupil: VNFaceLandmarkRegion2D? { get }
```

#### Discussion

This value may be inaccurate if the eye is blinking.
