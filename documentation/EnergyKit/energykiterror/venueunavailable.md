# EnergyKitError.venueUnavailable

**Framework**: EnergyKit  
**Kind**: case

The requested energy venue doesn’t exist.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst ?+
- macOS 26.0+ (Beta)

## Declaration

```swift
case venueUnavailable
```

#### Discussion

The venue ID is potentially invalid, nonexistent, or restricted by the person.
