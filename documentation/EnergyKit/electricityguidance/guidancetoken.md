# guidanceToken

**Framework**: EnergyKit  
**Kind**: property

A unique token for the guidance that you use to create an electrical load event.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst ?+
- macOS 26.0+

## Declaration

```swift
let guidanceToken: UUID
```

#### Discussion

Use the unique token to create an `ElectricalLoadEvent`.
