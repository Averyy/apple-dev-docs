# init()

**Framework**: ARKit  
**Kind**: init

Initializes a new orientation tracking configuration.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+

## Declaration

```swift
init()
```

#### Discussion

To use the configuration in an AR session, pass it to the [`ARSession`](arsession.md) [`run(_:options:)`](arsession/run(_:options:).md) method.
