# init(symbologies:)

**Framework**: ARKit  
**Kind**: init

Creates a barcode detection provider that looks for the specified symbologies.

**Availability**:
- visionOS 2.0+

## Declaration

```swift
init(symbologies: [BarcodeAnchor.Symbology])
```

## Parameters

- `symbologies`: The barcode symbologies to look for.
