# allowsContentScaling

**Framework**: ARKit  
**Kind**: property

A Boolean value that determines whether the user can scale your virtual content using pinch gestures.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
