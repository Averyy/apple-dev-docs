# targetAlignment

**Framework**: ARKit  
**Kind**: property

The target’s alignment with respect to gravity.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+

## Declaration

```swift
var targetAlignment: ARRaycastQuery.TargetAlignment { get }
```

#### Discussion

The available options are horizontal, vertical, or any.
