# ar_accessory_anchors_get_count

**Framework**: ARKit  
**Kind**: func

Get the count of accessory anchors in the collection.

**Availability**:
- visionOS 26.0+

## Declaration

```swift
extern size_t ar_accessory_anchors_get_count(ar_accessory_anchors_t accessory_anchors);
```

#### Return Value

The number of accessory anchors in the collection.

## Parameters

- `accessory_anchors`: The collection of accessory anchors.
