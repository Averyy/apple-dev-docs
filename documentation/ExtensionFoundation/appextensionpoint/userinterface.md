# AppExtensionPoint.UserInterface

**Framework**: ExtensionFoundation  
**Kind**: struct

Specifies that the extension point supports user interface

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
- visionOS 1.1+

## Declaration

```swift
struct UserInterface
```

#### Overview

When this attribute is omitted the extension point will default to `NO` user interface support.
