# PassFields

**Framework**: Walletpasses  
**Kind**: dictionary

An object that represents the groups of fields that display information on the front and back of a pass.

**Availability**:
- iOS 6.0+
- iPadOS 6.0+
- watchOS 2.0+

## Declaration

```swift
object PassFields
```
