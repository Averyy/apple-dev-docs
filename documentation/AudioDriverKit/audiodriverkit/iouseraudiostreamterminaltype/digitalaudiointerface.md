# DigitalAudioInterface

**Framework**: AudioDriverKit  
**Kind**: case

A stream to or from a digital audio interface, defined by ISO 60958.

**Availability**:
- DriverKit 21.0+

## Declaration

```swift
DigitalAudioInterface
```

#### Discussion

This type of interface is also known as SPDIF or AES/EBU. This terminal type applies to both input and output streams.
