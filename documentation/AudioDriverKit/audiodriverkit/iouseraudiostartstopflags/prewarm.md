# Prewarm

**Framework**: AudioDriverKit  
**Kind**: case

A flag that indicates starting or stopping for prewarming.

**Availability**:
- DriverKit 21.0+

## Declaration

```swift
Prewarm
```

#### Discussion

This option causes the minimal enabling or disabling of hardware, to minimize the transition to normal I/O operation.
