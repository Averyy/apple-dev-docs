# AES3

**Framework**: AudioDriverKit  
**Kind**: case

The AES3-2003 format.

**Availability**:
- DriverKit 21.0+

## Declaration

```swift
AES3
```

#### Discussion

MXF and MPEG-2 containers and SDTI transport streams adopt this format with SMPTE specs 302M-2002 and 331M-2000.

This format uses no flags.
