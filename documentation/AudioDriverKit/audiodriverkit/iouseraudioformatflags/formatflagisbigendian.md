# FormatFlagIsBigEndian

**Framework**: AudioDriverKit  
**Kind**: case

A flag to indicate whether samples use big-endian values.

**Availability**:
- DriverKit 21.0+

## Declaration

```swift
FormatFlagIsBigEndian
```

#### Discussion

Set this flag for big-endian values; clear it for little-endian.
