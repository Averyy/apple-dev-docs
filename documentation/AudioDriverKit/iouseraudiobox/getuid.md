# GetUID

**Framework**: AudioDriverKit  
**Kind**: method

Returns the UID of the audio box.

**Availability**:
- DriverKit 21.0+

## Declaration

```swift
OSSharedPtr<OSString> GetUID();
```

#### Return Value

The UID of the audio box.

#### Discussion

This method synchronizes by using the work queue created by the object.
