# copy(into:)

**Framework**: XPC  
**Kind**: method

Copies the keys and values of the dictionary to a different dictionary.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst ?+
- macOS 13.0+
- tvOS 16.0+
- watchOS 9.0+

## Declaration


## Parameters

- `destination`: The dictionary to copy the keys and values into.
