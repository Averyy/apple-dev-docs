# sessionIdentifier

**Framework**: SensorKit  
**Kind**: property

A unique identifier for the keyboard session.

**Availability**:
- iOS 16.4+
- iPadOS 16.4+
- Mac Catalyst 16.4+

## Declaration

```swift
var sessionIdentifier: String { get }
```

#### Discussion

Use this property to correlate the current stream with other streams that use the keyboard.
