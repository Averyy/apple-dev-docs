# init(x:y:)

**Framework**: SensorKit  
**Kind**: init

Creates a chromaticity instance from the argument coordinate pair.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+

## Declaration

```swift

```

## Parameters

- `x`: The chromaticity x-value.
