# init(sensor:)

**Framework**: SensorKit  
**Kind**: init

Initializes a new sensor reader object.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+

## Declaration

```swift
init(sensor: SRSensor)
```

## Parameters

- `sensor`: The sensor from which to read information.
