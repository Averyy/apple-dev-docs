# sensor

**Framework**: SensorKit  
**Kind**: property

The particular sensor that this object reads.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+

## Declaration

```swift
var sensor: SRSensor { get }
```

#### Discussion

The framework sets this property to the sensor you pass into [`init(sensor:)`](srsensorreader/init(sensor:).md).
