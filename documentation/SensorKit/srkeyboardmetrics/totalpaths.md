# totalPaths

**Framework**: SensorKit  
**Kind**: property

The total number of completed paths for the keyboard.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+

## Declaration

```swift
var totalPaths: Int { get }
```

#### Discussion

Each path refers to one word the user enters by using QuickPath.
