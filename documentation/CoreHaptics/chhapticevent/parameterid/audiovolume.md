# audioVolume

**Framework**: Core Haptics  
**Kind**: property

The volume of an audio event.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 14.0+
- visionOS 1.0+

## Declaration

```swift
static let audioVolume: CHHapticEvent.ParameterID
```

#### Discussion

This parameter value ranges from 0.0 (silent) to 1.0 (maximum volume).
