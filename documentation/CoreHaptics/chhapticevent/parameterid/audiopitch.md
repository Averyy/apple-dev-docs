# audioPitch

**Framework**: Core Haptics  
**Kind**: property

The pitch of an audio event.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 14.0+
- visionOS 1.0+

## Declaration

```swift
static let audioPitch: CHHapticEvent.ParameterID
```

#### Discussion

This parameter value ranges from -1.0 (lowest pitch) to 1.0 (highest pitch).
