# supportsHaptics

**Framework**: Core Haptics  
**Kind**: property  
**Required**: Yes

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 14.0+
- visionOS 1.0+
