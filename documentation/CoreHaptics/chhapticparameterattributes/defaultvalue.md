# defaultValue

**Framework**: Core Haptics  
**Kind**: property  
**Required**: Yes

The default value of the parameter value.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 14.0+
- visionOS 1.0+

## Declaration

```swift
var defaultValue: Float { get }
```

#### Discussion

The parameter assumes this default value when initialized.
