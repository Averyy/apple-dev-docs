# loopEnabled

**Framework**: Core Haptics  
**Kind**: property  
**Required**: Yes

A Boolean that determines whether the haptic repeats itself on completion.

**Availability**:

- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
