# CBConnectPeripheralOptionRequiresANCS

**Framework**: Core Bluetooth  
**Kind**: var

An option to require Apple Notification Center Service (ANCS) when connecting a device.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+
