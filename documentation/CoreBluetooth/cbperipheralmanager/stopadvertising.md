# stopAdvertising()

**Framework**: Core Bluetooth  
**Kind**: method

Stops advertising peripheral manager data.

**Availability**:
- iOS 6.0+
- iPadOS 6.0+
- Mac Catalyst 13.1+
- macOS 10.9+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
func stopAdvertising()
```

#### Discussion

Call this method when you no longer want to advertise peripheral manager data.
