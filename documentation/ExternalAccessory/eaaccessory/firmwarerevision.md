# firmwareRevision

**Framework**: External Accessory  
**Kind**: property

The current firmware version for the accessory.

**Availability**:
- iOS 3.0+
- iPadOS 3.0+
- Mac Catalyst 13.1+
- macOS 10.13+
- tvOS 10.0+
- visionOS 1.0+

## Declaration

```swift
var firmwareRevision: String { get }
```

#### Discussion

The format of this string is determined by the accessory manufacturer.
