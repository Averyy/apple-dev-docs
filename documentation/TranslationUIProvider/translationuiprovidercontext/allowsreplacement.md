# allowsReplacement

**Framework**: TranslationUIProvider  
**Kind**: property  
**Required**: Yes

A Boolean value that indicates whether the control can replace the source text.

**Availability**:
- iOS 18.4+
- iPadOS 18.4+

## Declaration

```swift
var allowsReplacement: Bool { get }
```

#### Discussion

Use this flag to indicate when a control replaces the source text with the translation.
