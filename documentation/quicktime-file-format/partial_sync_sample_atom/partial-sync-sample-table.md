# Partial sync sample table

**Framework**: QuickTime File Format  
**Kind**: property

A table of sample numbers.

#### Overview

The layout of a partial sync sample table is as follows.

| Sample number | Sample |
| --- | --- |
| Number | Sample 1 |
| Number | Sample 2 |
| Number | Sample 3 |
| Number | Sample 4 |
| Number | Sample 5 |
