# Language tag string

**Framework**: QuickTime File Format  
**Kind**: property

A string containing a language tag.

#### Overview

A NULL-terminated C string containing an RFC 4646 (BCP 47) compliant language tag string in ASCII encoding, such as “en-US”, “fr-FR”, or “zh-CN”.
