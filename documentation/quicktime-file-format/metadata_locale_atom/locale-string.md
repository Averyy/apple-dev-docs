# Locale string

**Framework**: QuickTime File Format  
**Kind**: property

A string holding a language tag.

#### Overview

A `NULL`-terminated string of UTF-8 characters holding a language tag complying with RFC 4646 (BCP 47).

Examples include `'en-US'`, `'fr-FR'`, or `'zh-CN'`.
