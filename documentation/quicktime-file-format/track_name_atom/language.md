# Language

**Framework**: QuickTime File Format  
**Kind**: property

A 16-bit integer holding a packed ISO 639-2/T code.

#### Overview

Language codes are described in User Data Text Strings and Language Codes in [`User data atoms`](user_data_atoms.md).
