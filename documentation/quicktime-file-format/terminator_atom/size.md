# Size

**Framework**: QuickTime File Format  
**Kind**: property

An unsigned 32-bit integer holding the size of the decompression parameters atom.

#### Overview

Always set to `8`.

- [Type](terminator_atom/type.md)
  An unsigned 32-bit integer.
