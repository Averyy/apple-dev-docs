# Sync sample table

**Framework**: QuickTime File Format  
**Kind**: property

A table of sample numbers.

#### Overview

Each sample number corresponds to a key frame.

The layout of a sync sample table is as follows.

| Entry number | Sample |
| --- | --- |
| Number | Sample 1 |
| Number | Sample 2 |
| Number | Sample 3 |
| Number | Sample 4 |
| Number | Sample 5 |
