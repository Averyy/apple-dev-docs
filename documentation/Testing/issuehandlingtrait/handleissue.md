# handleIssue(_:)

**Framework**: Swift Testing  
**Kind**: method

Handle a specified issue.

**Availability**:
- Swift 6.2+
- Xcode 26.0+

## Declaration

```swift
func handleIssue(_ issue: Issue) -> Issue?
```

#### Return Value

An issue to replace `issue`, or else `nil` if the issue should not be recorded.

## Parameters

- `issue`: The issue to handle.
