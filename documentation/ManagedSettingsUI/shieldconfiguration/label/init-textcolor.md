# init(text:color:)

**Framework**: ManagedSettingsUI  
**Kind**: init

Creates a shield label with the provided text and color.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+

## Declaration

```swift
init(text: String, color: UIColor)
```

## Parameters

- `text`: The text to display in the label.
- `color`: The color for the text in the label.
