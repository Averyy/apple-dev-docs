# accessoryView

**Framework**: Address Book  
**Kind**: property

**Availability**:
- macOS 10.3+

## Declaration

```swift

@MainActor var accessoryView: NSView! { get set }
```

#### Discussion

If `accessory` is `nil`, the accessory view is removed.
