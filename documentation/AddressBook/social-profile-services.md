# Social Profile Services

**Framework**: Address Book

The predefined constants are used to identify social networking services.

#### Overview

You may use these constants or any other string.

The value of these constants is undefined until one of the following functions has been called: [`ABAddressBookCreate()`](abaddressbookcreate().md), [`ABPersonCreate()`](abpersoncreate().md), [`ABGroupCreate()`](abgroupcreate().md). Neither developers nor users can add more keys.
