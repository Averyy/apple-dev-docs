# ABAuthorizationStatus.restricted

**Framework**: Address Book  
**Kind**: case

The app is not authorized to access address book data. The user cannot change this access, possibly due to restrictions such as parental controls.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
