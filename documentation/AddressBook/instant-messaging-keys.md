# Instant Messaging Keys

**Framework**: Address Book

The keys used to specify the different fields in a [`kABInstantMessageProperty`](kabinstantmessageproperty.md) property.

#### Overview

Neither developers nor users can add more keys.

## Topics

### Constants
- [let kABInstantMessageUsernameKey: String](kabinstantmessageusernamekey.md)
  The instant messaging username or handle.
- [let kABInstantMessageServiceKey: String](kabinstantmessageservicekey.md)
  The instant messaging service.
