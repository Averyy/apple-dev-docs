# CPAlertAction.Style.cancel

**Framework**: CarPlay  
**Kind**: case

The display style indicating that the action leaves data unchanged.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.1+

## Declaration

```swift
case cancel
```

#### Discussion

Use this style when providing a cancel action on the alert.
