# CPAlertAction.Style.destructive

**Framework**: CarPlay  
**Kind**: case

The style indicating that the action changes or deletes data.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.1+

## Declaration

```swift
case destructive
```

#### Discussion

Use this style when the action does something destructive, such as deleting data.
