# secondaryAction

**Framework**: CarPlay  
**Kind**: property

An optional secondary action, and button, for the navigation alert.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.1+

## Declaration

```swift
@MainActor
var secondaryAction: CPAlertAction? { get }
```

#### Discussion

The alert displays the button for a secondary action on the trailing edge of the alert.
