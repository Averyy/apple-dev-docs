# primaryAction

**Framework**: CarPlay  
**Kind**: property

The primary action, and button, for the navigation alert.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.1+

## Declaration

```swift
@MainActor
var primaryAction: CPAlertAction { get }
```

#### Discussion

Each navigation alert must have a primary action.
