# isEnabled

**Framework**: CarPlay  
**Kind**: property

A Boolean value that enables and disables the bar button.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.1+

## Declaration

```swift
var isEnabled: Bool { get set }
```

#### Discussion

The default value is [`true`](https://developer.apple.com/documentation/swift/true).
