# title

**Framework**: CarPlay  
**Kind**: property

The title displayed on the bar button.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.1+

## Declaration

```swift
var title: String? { get set }
```

#### Discussion

Setting this property has an effect only when the button type is [`CPBarButton.Type.text`](cpbarbutton/type/text.md).
