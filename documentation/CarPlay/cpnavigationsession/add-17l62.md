# add(_:)

**Framework**: CarPlay  
**Kind**: method

Adds one or more maneuvers, in chronological order, to the navigation session.

**Availability**:
- iOS 17.4+
- iPadOS 17.4+
- Mac Catalyst 17.4+

## Declaration

```swift
func add(_ maneuvers: [CPManeuver])
```

##### Discussion

Use this method to set all maneuvers in [`upcomingManeuvers`](cpnavigationsession/upcomingmaneuvers.md). The app needs to provide as many maneuvers as possible, as soon as they are available.
