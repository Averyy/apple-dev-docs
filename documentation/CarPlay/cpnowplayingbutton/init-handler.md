# init(handler:)

**Framework**: CarPlay  
**Kind**: init

Creates a Now Playing button that invokes a handler.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+

## Declaration

```swift
init(handler: ((CPNowPlayingButton) -> Void)? = nil)
```

## Parameters

- `handler`: A closure that the button invokes when the user taps it.
