# init(teamLogo:)

**Framework**: CarPlay  
**Kind**: init

Initialize a team logo with an image representation of this team. Provide an image no larger than 350x350; larger images will be resized down.

**Availability**:
- iOS 18.4+
- iPadOS 18.4+
- Mac Catalyst 18.4+

## Declaration

```swift
@MainActor
init(teamLogo: UIImage)
```
