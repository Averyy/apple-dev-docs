# possessionIndicator

**Framework**: CarPlay  
**Kind**: property

An optional indicator used to indicate possession by this team. Only one team should have possession at a given time.

**Availability**:
- iOS 18.4+
- iPadOS 18.4+
- Mac Catalyst 18.4+
