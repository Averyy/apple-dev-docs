# init(handler:)

**Framework**: CarPlay  
**Kind**: init

Creates a new map button.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.1+

## Declaration

```swift
init(handler: ((CPMapButton) -> Void)? = nil)
```

#### Return Value

A newly initialized map button.

## Parameters

- `handler`: The block invoked after the user taps the map button.
