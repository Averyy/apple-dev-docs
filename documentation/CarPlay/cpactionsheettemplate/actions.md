# actions

**Framework**: CarPlay  
**Kind**: property

The list of actions available on the action sheet.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.1+

## Declaration

```swift
@MainActor
var actions: [CPAlertAction] { get }
```

#### Discussion

The array must include at least one action.
