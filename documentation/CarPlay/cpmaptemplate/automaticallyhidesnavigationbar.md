# automaticallyHidesNavigationBar

**Framework**: CarPlay  
**Kind**: property

A Boolean value that indicates whether the template should automatically hide the navigation bar.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.1+

## Declaration

```swift
@MainActor
var automaticallyHidesNavigationBar: Bool { get set }
```

#### Discussion

The default is [`true`](https://developer.apple.com/documentation/swift/true).
