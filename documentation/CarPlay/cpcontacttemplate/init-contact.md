# init(contact:)

**Framework**: CarPlay  
**Kind**: init

Creates a contact template that displays the provided contact.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+

## Declaration

```swift
init(contact: CPContact)
```

#### Return Value

A new contact template that displays the provided contact.

## Parameters

- `contact`: The contact that the template displays.
