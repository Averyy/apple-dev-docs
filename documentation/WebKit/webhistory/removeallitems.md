# removeAllItems()

**Framework**: Webkit  
**Kind**: method

Removes all items from the web history.

**Availability**:
- macOS 10.3+

## Declaration

```swift
func removeAllItems()
```

#### Discussion

When successful, this method posts a notification ([`WebHistoryAllItemsRemovedNotification`](webhistoryallitemsremovednotification.md)).
