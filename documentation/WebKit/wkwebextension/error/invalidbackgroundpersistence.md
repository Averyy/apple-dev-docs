# invalidBackgroundPersistence

**Framework**: Webkit  
**Kind**: property

Indicates that the extension specified background persistence that was not compatible with the platform or features requested.

**Availability**:
- iOS 18.4+
- iPadOS 18.4+
- Mac Catalyst 18.4+
- macOS 15.4+
- visionOS 2.4+
