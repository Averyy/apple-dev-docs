**Kind**: property

**Availability**:
- iOS 18.4+
- iPadOS 18.4+
- Mac Catalyst 18.4+
- macOS 15.4+
- visionOS 2.4+

## Declaration

```swift
@MainActor
```

#### Discussion

This is the text that will appear when inspecting the pop-up web view.
