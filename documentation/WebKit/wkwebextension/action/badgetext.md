# badgeText

**Framework**: WebKit  
**Kind**: property

The badge text for the action.

**Availability**:
- iOS 18.4+
- iPadOS 18.4+
- Mac Catalyst 18.4+
- macOS 15.4+
- visionOS 2.4+

## Declaration

```swift
@MainActor
var badgeText: String { get }
```

#### Discussion

Provides the text that appears on the badge for the action. An empty string signifies that no badge should be shown.
