# defaultLocale

**Framework**: WebKit  
**Kind**: property

The default locale for the extension.

**Availability**:
- iOS 18.4+
- iPadOS 18.4+
- Mac Catalyst 18.4+
- macOS 15.4+
- visionOS 2.4+

## Declaration

```swift
@MainActor
var defaultLocale: Locale? { get }
```

#### Discussion

Returns `nil` if there was no default locale specified.
