# WKDialogResult.showDefault

**Framework**: Webkit  
**Kind**: case

A result that indicates the delegate didn’t display a message, so the web view should show the default Lockdown Mode message.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
