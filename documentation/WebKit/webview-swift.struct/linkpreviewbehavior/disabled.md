# disabled

**Framework**: WebKit  
**Kind**: property

Link previews are disabled.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst ?+
- macOS 26.0+
- visionOS 26.0+

## Declaration

```swift
static let disabled: WebView.LinkPreviewBehavior
```

#### Discussion

The web view prevents pressing a link from displaying a preview of the destination for the link.
