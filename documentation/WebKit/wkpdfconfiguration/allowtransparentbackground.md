# allowTransparentBackground

**Framework**: WebKit  
**Kind**: property

A Boolean value that indicates whether the PDF may have a transparent background.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
