# selectedString()

**Framework**: WebKit  
**Kind**: method  
**Required**: Yes

Returns the currently selected text in the web document as a string.

**Availability**:
- macOS 10.3+

## Declaration

```swift
func selectedString() -> String!
```

#### Return Value

The currently selected text in the web document.
