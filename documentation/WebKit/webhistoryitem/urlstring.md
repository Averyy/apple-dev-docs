# urlString

**Framework**: WebKit  
**Kind**: property

The string representation of the URL for the receiver’s page.

**Availability**:
- macOS 10.3+

## Declaration

```swift
var urlString: String! { get }
```

#### Discussion

This URL may differ from the original URL if the page was, for example, redirected to a new location.
