# alternateTitle

**Framework**: WebKit  
**Kind**: property

An alternate title that may be used in place of the receiver’s page title.

**Availability**:
- macOS 10.3+

## Declaration

```swift
var alternateTitle: String! { get set }
```

#### Discussion

This method returns `nil` if no alternate title exists.
