# WebPage.Configuration.MediaPlaybackBehavior.allowsInlinePlayback

**Framework**: WebKit  
**Kind**: case

Allows videos to play inline. When adding a video element to an HTML document on iPhone, you must also include the `playsinline` attribute.

**Availability**:
- iOS 18.4+
- iPadOS 18.4+
- Mac Catalyst ?+
