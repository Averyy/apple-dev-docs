# upgradeKnownHostsToHTTPS

**Framework**: WebKit  
**Kind**: property

Indicates whether the web view should automatically upgrade supported HTTP requests to HTTPS.

**Availability**:
- iOS 18.4+
- iPadOS 18.4+
- Mac Catalyst ?+
- macOS 15.4+
- visionOS 2.4+

## Declaration

```swift
@MainActor
var upgradeKnownHostsToHTTPS: Bool
```

#### Discussion

The default value of this property is `true`.
