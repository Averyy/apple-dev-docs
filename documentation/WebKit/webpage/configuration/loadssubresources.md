# loadsSubresources

**Framework**: WebKit  
**Kind**: property

Indicates whether the webpage loads all of its subresources in addition to the main resource.

**Availability**:
- iOS 18.4+
- iPadOS 18.4+
- Mac Catalyst ?+
- macOS 15.4+
- visionOS 2.4+

## Declaration

```swift
@MainActor
var loadsSubresources: Bool
```

#### Discussion

The default value of this property is `true`.
