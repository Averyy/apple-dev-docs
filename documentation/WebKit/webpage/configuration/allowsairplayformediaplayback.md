# allowsAirPlayForMediaPlayback

**Framework**: WebKit  
**Kind**: property

Indicates whether the webpage allows media playback over AirPlay.

**Availability**:
- iOS 18.4+
- iPadOS 18.4+
- Mac Catalyst ?+
- macOS 15.4+
- visionOS 2.4+

## Declaration

```swift
@MainActor
var allowsAirPlayForMediaPlayback: Bool
```

#### Discussion

The default value of this property is `true`.
