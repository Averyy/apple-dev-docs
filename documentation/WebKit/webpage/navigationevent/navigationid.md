# navigationID

**Framework**: WebKit  
**Kind**: property

The ID of the navigation that triggered this event.

**Availability**:
- iOS 18.4+
- iPadOS 18.4+
- Mac Catalyst ?+
- macOS 15.4+
- visionOS 2.4+

## Declaration

```swift
let navigationID: WebPage.NavigationID
```

#### Discussion

Multiple sequential events will have the same navigation identifier.
