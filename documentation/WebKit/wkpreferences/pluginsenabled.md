# plugInsEnabled

**Framework**: WebKit  
**Kind**: property

A Boolean value that indicates whether plug-ins are enabled.

**Availability**:
- macOS 10.10+

## Declaration

```swift
@MainActor
var plugInsEnabled: Bool { get set }
```

#### Discussion

The default value is [`false`](https://developer.apple.com/documentation/swift/false).
