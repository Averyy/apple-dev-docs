# WKPreferences.InactiveSchedulingPolicy.throttle

**Framework**: Webkit  
**Kind**: case

A policy where a web view that’s not in a window limits processing, but does not fully suspend tasks.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- visionOS 1.0+
