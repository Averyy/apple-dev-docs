# javaEnabled

**Framework**: Webkit  
**Kind**: property

A Boolean value that indicates whether Java is enabled.

**Availability**:
- macOS 10.10+

## Declaration

```swift
@MainActor
var javaEnabled: Bool { get set }
```

#### Discussion

The default value is [`false`](https://developer.apple.com/documentation/swift/false).
