# WKNavigationType.linkActivated

**Framework**: Webkit  
**Kind**: case

A link activation.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.10+
- visionOS 1.0+

## Declaration

```swift
case linkActivated
```

#### Discussion

This action occurs when the user activates a link with an `href` attribute.
