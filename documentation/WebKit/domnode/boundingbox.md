# boundingBox()

**Framework**: WebKit  
**Kind**: method

Returns a rectangle that bounds the onscreen rendering of the node.

**Availability**:
- macOS 10.5+

## Declaration

```swift
func boundingBox() -> NSRect
```

#### Return Value

The rectangle that represents the bounding box of the onscreen rendering of the node.
