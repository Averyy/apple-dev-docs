# lineBoxRects()

**Framework**: WebKit  
**Kind**: method

Returns the rectangles that bound each line of text in the node.

**Availability**:
- macOS 10.5+

## Declaration

```swift
func lineBoxRects() -> [Any]!
```

#### Return Value

An array of rectangles, in which each rectangle represents the bounding box of a line of text in the node.
