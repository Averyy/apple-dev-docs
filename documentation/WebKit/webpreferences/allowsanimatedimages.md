# allowsAnimatedImages

**Framework**: WebKit  
**Kind**: property

A Boolean that indicates whether or not the receiver allows animated images.

**Availability**:
- macOS 10.3+

## Declaration

```swift
var allowsAnimatedImages: Bool { get set }
```

#### Discussion

Set to [`true`](https://developer.apple.com/documentation/swift/true) if the web view should allow animated images, otherwise [`false`](https://developer.apple.com/documentation/swift/false).
