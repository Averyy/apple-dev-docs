# userStyleSheetEnabled

**Framework**: Webkit  
**Kind**: property

A Boolean that indicates whether or not user style sheets are enabled in the web view.

**Availability**:
- macOS 10.3+

## Declaration

```swift
var userStyleSheetEnabled: Bool { get set }
```

#### Discussion

Set to [`true`](https://developer.apple.com/documentation/swift/true) if the user style sheets should be enabled in the web view, otherwise [`false`](https://developer.apple.com/documentation/swift/false).
