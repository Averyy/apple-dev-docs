# defaultFixedFontSize

**Framework**: Webkit  
**Kind**: property

The default fixed font size of the web view.

**Availability**:
- macOS 10.3+

## Declaration

```swift
var defaultFixedFontSize: Int32 { get set }
```

#### Discussion

The default font size to use for content displayed in a fixed font if no font size is specified.
