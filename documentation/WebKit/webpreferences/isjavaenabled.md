# isJavaEnabled

**Framework**: WebKit  
**Kind**: property

A Boolean that indicates whether or not the web view allows Java.

**Availability**:
- macOS 10.3+

## Declaration

```swift
var isJavaEnabled: Bool { get set }
```

#### Discussion

Set to [`true`](https://developer.apple.com/documentation/swift/true) to indicate that the web view allows Java, otherwise [`false`](https://developer.apple.com/documentation/swift/false).
