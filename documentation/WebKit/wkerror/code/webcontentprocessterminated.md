# WKError.Code.webContentProcessTerminated

**Framework**: Webkit  
**Kind**: case

An error that indicates the web process that contains the content is no longer running.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.10+
- visionOS 1.0+
