# Menu Item Tags

**Framework**: Webkit

Tags that define the types of default menu items passed to the [`webView(_:contextMenuItemsForElement:defaultMenuItems:)`](webuidelegate/webview(_:contextmenuitemsforelement:defaultmenuitems:).md) method.

#### Overview

These tags define common actions a user might want to take with elements in a page. You can use the tags to differentiate between the different types of menu items.
