# undoManager(for:)

**Framework**: WebKit  
**Kind**: method

Returns the undo manager to be used by a web view.

**Availability**:
- macOS 10.3+

## Declaration

```swift
optional func undoManager(for webView: WebView!) -> UndoManager!
```

#### Return Value

The undo manager.

## Parameters

- `webView`: The web view that the user is editing.
