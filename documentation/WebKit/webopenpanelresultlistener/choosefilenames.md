# chooseFilenames(_:)

**Framework**: Webkit  
**Kind**: method  
**Required**: Yes

Displays a file open panel and returns the multiple selected filenames.

**Availability**:
- macOS 10.6+

## Declaration

```swift
func chooseFilenames(_ fileNames: [Any]!)
```

## Parameters

- `fileNames`: An array of selected file names.
