# Classifying Live Audio Input with a Built-in Sound Classifier

**Framework**: Sound Analysis

Detect and identify hundreds of sounds by using a trained classifier.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- Xcode 13.0+
