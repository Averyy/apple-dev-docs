# RequestFlag

**Framework**: Core NFC  
**Kind**: typealias

A set of bit mask options that, when combined, define the request flags to use when sending a command.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+

## Declaration

```swift
typealias RequestFlag = NFCISO15693RequestFlag
```
