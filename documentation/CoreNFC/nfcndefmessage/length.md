# length

**Framework**: Core NFC  
**Kind**: property

The length, in bytes, of the NDEF message when stored on an NFC tag.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+

## Declaration

```swift
var length: Int { get }
```

#### Discussion

The maximum length of an NDEF message is 128 KB.
