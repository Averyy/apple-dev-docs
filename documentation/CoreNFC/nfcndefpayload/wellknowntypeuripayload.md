# wellKnownTypeURIPayload()

**Framework**: Core NFC  
**Kind**: method

Returns the URL of a valid Well Known Type URI payload.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+

## Declaration

```swift
func wellKnownTypeURIPayload() -> URL?
```

#### Return Value

A URL when the payload contains a Well Know Type URI; otherwise, `nil`.
