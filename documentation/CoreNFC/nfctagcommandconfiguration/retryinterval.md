# retryInterval

**Framework**: Core NFC  
**Kind**: property

The time between retries, in seconds.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+

## Declaration

```swift
var retryInterval: TimeInterval { get set }
```

#### Discussion

By default, the value of this parameter is `0`.
