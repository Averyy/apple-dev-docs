# maximumRetries

**Framework**: Core NFC  
**Kind**: property

The maximum number of retries.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+

## Declaration

```swift
var maximumRetries: Int { get set }
```

#### Discussion

You can specify up to 256 retries. By default, the value of this parameter is `0`.
