# NFCWindowSceneEvent.presentation

**Framework**: Core NFC  
**Kind**: case

The user performed a gesture to present an NFC display.

**Availability**:
- iOS 17.4+
- iPadOS 17.4+
- Mac Catalyst ?+

## Declaration

case presentation

#### Discussion

This gesture can be a double-click of the side button on Face ID devices, or of the Home button on Touch ID devices.
