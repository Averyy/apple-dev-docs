# providesAppNotificationSettings

**Framework**: User Notifications  
**Kind**: property

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.1+
- macOS 10.14+
- visionOS 1.0+
- watchOS 5.0+
