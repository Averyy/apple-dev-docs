# UNShowPreviewsSetting.always

**Framework**: User Notifications  
**Kind**: case

The notification’s content is always shown, even when the device is locked.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.14+
- visionOS 1.0+
