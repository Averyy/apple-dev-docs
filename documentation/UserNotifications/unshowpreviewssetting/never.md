# UNShowPreviewsSetting.never

**Framework**: User Notifications  
**Kind**: case

The notification’s content is never shown, even when the device is unlocked

**Availability**:

- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.14+
- visionOS 1.0+
