# init(rawValue:)

**Framework**: User Notifications  
**Kind**: init

Initializes an action options object using the specified raw value.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- macOS 10.14+
- visionOS 1.0+
- watchOS 3.0+

## Declaration

```swift
init(rawValue: UInt)
```

## Parameters

- `rawValue`: The raw unsigned integer value for the action options.
