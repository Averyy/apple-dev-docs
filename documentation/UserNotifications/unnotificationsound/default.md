# default

**Framework**: User Notifications  
**Kind**: property

Returns an object representing the default sound for notifications.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- macOS 10.14+
- visionOS 1.0+
- watchOS 3.0+

## Declaration

```swift
@NSCopying
class var `default`: UNNotificationSound { get }
```

#### Return Value

A sound object that represents the default notification sound.
