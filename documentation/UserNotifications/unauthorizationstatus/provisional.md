# UNAuthorizationStatus.provisional

**Framework**: User Notifications  
**Kind**: case

The application is provisionally authorized to post noninterruptive user notifications.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.1+
- macOS 10.14+
