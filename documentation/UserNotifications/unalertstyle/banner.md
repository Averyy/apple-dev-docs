# UNAlertStyle.banner

**Framework**: User Notifications  
**Kind**: case

Banner alerts.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- macOS 10.14+
- visionOS 1.0+

## Declaration

```swift
case banner
```

#### Discussion

Alerts are displayed as a slide-down banner. Banners appear for a short time and then disappear automatically if the user does nothing.
