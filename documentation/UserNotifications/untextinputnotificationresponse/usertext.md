# userText

**Framework**: User Notifications  
**Kind**: property

The text response provided by the user.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- macOS 10.14+
- visionOS 1.0+
- watchOS 3.0+

## Declaration

```swift
var userText: String { get }
```

#### Discussion

If the user does not specify any text, this property contains an empty string.
