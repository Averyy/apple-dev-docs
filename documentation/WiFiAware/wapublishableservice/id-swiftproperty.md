# id

**Framework**: Wi-Fi Aware  
**Kind**: property

A stable ID that can be used to identify this publishable service.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+

## Declaration

```swift
var id: WAPublishableService.ID { get }
```

#### Discussion

The ID is stable for a given unique service name.
