# average

**Framework**: Wi-Fi Aware  
**Kind**: property

The average per-packet transmit latency measured for this flow’s peers.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)

## Declaration

```swift
let average: Duration?
```

#### Discussion

This property returns `nil` if the system can’t measure the transmit latency.
