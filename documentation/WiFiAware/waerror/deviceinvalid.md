# WAError.deviceInvalid(_:)

**Framework**: Wi-Fi Aware  
**Kind**: case

An error that occurs if your app specifies a paired device to which it doesn’t have access.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)

## Declaration
