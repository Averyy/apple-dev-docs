# WAError.entitlementMissing(_:)

**Framework**: Wi-Fi Aware  
**Kind**: case

An error that occurs if your app is missing the entitlement needed for the requested operation.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+

## Declaration
