# refreshRate

**Framework**: Paravirtualized Graphics  
**Kind**: property

The mode’s refresh rate.

**Availability**:
- Mac Catalyst 14.0+
- macOS 11.0+

## Declaration

```swift
var refreshRate: Double { get }
```

#### Discussion

Consider supplying only modes that have a refresh rate equal to that of the host environment’s physical display.
