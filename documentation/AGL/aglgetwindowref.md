# aglGetWindowRef

**Framework**: AGL  
**Kind**: func

Retrieves the window associated with an AGL context.

**Availability**:
- macOS 10.5+

## Declaration

```swift
extern WindowRef aglGetWindowRef(AGLContext ctx);
```

#### Return Value

The window associated with the context.

## Parameters

- `ctx`: The AGL context whose window you want to retrieve.
