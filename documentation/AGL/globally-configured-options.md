# Globally Configured Options

**Framework**: AGL

Specify options that apply globally.

#### Overview

You can pass any of these options, along with the appropriate value, to the function [`aglConfigure`](https://developer.apple.comhttps://developer.apple.com/library/archive/#id(F15029)).
