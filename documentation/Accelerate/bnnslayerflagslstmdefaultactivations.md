# BNNSLayerFlagsLSTMDefaultActivations

**Framework**: Accelerate  
**Kind**: var

A flag that ignores the specified gate activations and instructs the operation to use default activations.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+
- watchOS 7.0+
