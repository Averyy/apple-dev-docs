# BNNSRandomGeneratorMethodAES_CTR

**Framework**: Accelerate  
**Kind**: var

A constant that specifes an implementation that’s based on the Advanced Encryption Standard (AES) hash of a counter.

**Availability**:

- Mac Catalyst 15.0+
