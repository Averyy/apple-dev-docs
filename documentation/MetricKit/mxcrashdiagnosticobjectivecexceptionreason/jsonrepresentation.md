# jsonRepresentation()

**Framework**: MetricKit  
**Kind**: method

Returns the contents of the exception reason in JSON format.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- visionOS 1.0+

## Declaration

```swift
func jsonRepresentation() -> Data
```

#### Return Value

An [`NSData`](https://developer.apple.com/documentation/Foundation/NSData) object containing the JSON representation of the metrics in the payload.
