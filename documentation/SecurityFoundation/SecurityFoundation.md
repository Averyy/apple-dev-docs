# Security Foundation

**Framework**: Security Foundation  
**Kind**: module

Restrict a user’s access to particular features in your Mac app or daemon.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.3+

#### Overview

This document was previously titled  and it replaced the `SFAuthorization` chapter of the  document. The other chapters of that document have been replaced by [`Security Interface`](https://developer.apple.com/documentation/SecurityInterface).

The Security Foundation framework contains a single class, [`SFAuthorization`](sfauthorization.md), which allows you to restrict a user’s access to particular features in your Mac app or daemon.
