# PD_RS232_PARITY_SPACE

**Framework**: SerialDriverKit  
**Kind**: macro

The space state.

**Availability**:
- DriverKit 19.0+

## Declaration

```swift
#define PD_RS232_PARITY_SPACE
```

#### Discussion

This value corresponds to placing a logical `0` on the communication line.
