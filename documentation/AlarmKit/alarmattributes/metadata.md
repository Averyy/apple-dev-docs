# metadata

**Framework**: AlarmKit  
**Kind**: property

The additional data you can include in your attributes.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)

## Declaration

```swift
var metadata: Metadata?
```

#### Discussion

The additional data can be information that describes extra UI features you want to include in your countdown UI.
