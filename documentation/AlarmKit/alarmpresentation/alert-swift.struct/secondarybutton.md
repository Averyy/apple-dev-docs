# secondaryButton

**Framework**: AlarmKit  
**Kind**: property

The appearance of the secondary button.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+

## Declaration

```swift
var secondaryButton: AlarmButton?
```

#### Discussion

If you use `nil`, the secondary button is omitted.
