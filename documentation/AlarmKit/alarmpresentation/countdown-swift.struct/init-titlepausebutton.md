# init(title:pauseButton:)

**Framework**: AlarmKit  
**Kind**: init

Creates a countdown with an optional pause button.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)

## Declaration

```swift
init(title: LocalizedStringResource, pauseButton: AlarmButton? = nil)
```

## Parameters

- `title`: The title of the countdown.
- `pauseButton`: The optional pause button for a countdown timer.
