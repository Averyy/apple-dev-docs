# pauseButton

**Framework**: AlarmKit  
**Kind**: property

The pause button for a countdown timer.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)

## Declaration

```swift
var pauseButton: AlarmButton?
```

#### Discussion

If you specify, the countdown includes a pause button when presented by the templated UI.
