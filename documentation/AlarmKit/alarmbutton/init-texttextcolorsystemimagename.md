# init(text:textColor:systemImageName:)

**Framework**: AlarmKit  
**Kind**: init

Creates an alarm button.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+

## Declaration

```swift
init(text: LocalizedStringResource, textColor: Color, systemImageName: String)
```

## Parameters

- `text`: The text for a label on the button.
- `textColor`: The color for the text on the button.
- `systemImageName`: The name of the icon you use on the button.
