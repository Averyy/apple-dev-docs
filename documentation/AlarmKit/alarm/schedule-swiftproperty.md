# schedule

**Framework**: AlarmKit  
**Kind**: property

The schedule determines when the alarm alerts.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)

## Declaration

```swift
var schedule: Alarm.Schedule?
```

#### Discussion

If no schedule is supplied then the alarm will begin counting down immediately.
