# MLiMovieEventLibraryGroupTypeIdentifier

**Framework**: Media Library  
**Kind**: var

The media group that represents the event library in iMovie. The event library contains all event calendar groups.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.9+
