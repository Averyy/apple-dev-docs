# name

**Framework**: Media Library  
**Kind**: property

The name of the media group.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.9+

## Declaration

```swift
var name: String? { get }
```

#### Discussion

This string is human-readable. It is either user created (such as the name of an iTunes playlist) or already localized.
