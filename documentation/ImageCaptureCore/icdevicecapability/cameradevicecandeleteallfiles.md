# cameraDeviceCanDeleteAllFiles

**Framework**: ImageCaptureCore  
**Kind**: property

Indicates that the camera can delete all files in a single operation while it is connected.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS 10.4+
- visionOS 1.0+
