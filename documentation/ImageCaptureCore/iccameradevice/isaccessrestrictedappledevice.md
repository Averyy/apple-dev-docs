# isAccessRestrictedAppleDevice

**Framework**: ImageCaptureCore  
**Kind**: property

A Boolean value indicating whether the device is an Apple device, passcode-locked, and connected to an untrusted host.

**Availability**:

- iPadOS 13.0+
- Mac Catalyst 13.1+
