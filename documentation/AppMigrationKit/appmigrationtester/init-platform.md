# init(platform:)

**Framework**: AppMigrationKit  
**Kind**: init

Creates a tester instance for migrating to or from the specified platform.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+

## Declaration

```swift
init(platform: MigrationPlatform) async throws
```

## Parameters

- `platform`: The platform to test migration to or from.
