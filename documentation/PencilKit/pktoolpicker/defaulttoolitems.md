# defaultToolItems

**Framework**: PencilKit  
**Kind**: property

The default tool items for new tool pickers.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- visionOS 26.0+

## Declaration

```swift
class var defaultToolItems: [PKToolPickerItem] { get }
```

#### Discussion

These items are used when creating a new `PKToolPicker`.
