# PKCanvasViewDrawingPolicy.anyInput

**Framework**: PencilKit  
**Kind**: case

Allows drawing on the canvas from any input source.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- visionOS 1.0+
