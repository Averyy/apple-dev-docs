# PKContentVersion.version3

**Framework**: PencilKit  
**Kind**: case

The PencilKit version that supports barrel-roll angle data in inks.

**Availability**:
- iOS 17.5+
- iPadOS 17.5+
- Mac Catalyst 17.5+
- macOS 14.5+
- visionOS 1.2+
