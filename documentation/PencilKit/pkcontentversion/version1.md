# PKContentVersion.version1

**Framework**: PencilKit  
**Kind**: case

The PencilKit version that supports inks from iPadOS 14 and earlier, including marker, pen, and pencil.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- visionOS 1.0+
