# latest

**Framework**: PencilKit  
**Kind**: property

A property that returns latest version of PencilKit, which supports all currently available inks.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- visionOS 1.0+
