# init(_:)

**Framework**: PencilKit  
**Kind**: init

Creates an eraser tool object that removes objects wholly or partially from a canvas view.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 11.0+
- visionOS 1.0+

## Declaration

```swift
init(_ eraserType: PKEraserTool.EraserType)
```

#### Discussion

This initializer creates a new eraser tool object.

- `eraserType`: A constant that determines how the eraser affects drawn content. For a list of possible values, see  .
