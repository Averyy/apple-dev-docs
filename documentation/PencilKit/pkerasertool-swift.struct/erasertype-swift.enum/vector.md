# PKEraserTool.EraserType.vector

**Framework**: PencilKit  
**Kind**: case

An eraser that removes an entire drawn line.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 11.0+
- visionOS 1.0+

## Declaration

```swift
case vector
```

#### Discussion

When a vector eraser touches any portion of a drawn line, it removes the entire line.
