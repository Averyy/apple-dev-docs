# init(_:color:)

**Framework**: PencilKit  
**Kind**: init

Creates a new ink, specifying its type and color.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- macOS 11.0+

## Declaration

```swift
init(_ inkType: PKInk.InkType, color: NSColor = NSColor.black)
```

## Parameters

- `inkType`: The type of ink represented by one of the available   enumerations.
- `color`: The color of the ink, defaults to black.
