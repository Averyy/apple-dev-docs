# init(inkType:color:)

**Framework**: PencilKit  
**Kind**: init

Create a new ink, specifying its type, color.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- visionOS 1.0+

## Declaration

```swift
init(inkType type: __PKInkType, color: NSColor)
```

## Parameters

- `type`: The type of ink to create, from one of the available   enumerations.
- `color`: The base color for this ink.
