# opacity

**Framework**: PencilKit  
**Kind**: property

Opacity of the point.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- visionOS 1.0+

## Declaration

```swift
var opacity: CGFloat { get }
```

#### Discussion

The opacity of a [`PKStrokePointReference`](pkstrokepointreference.md) has a range of 0-2, which is a multiplier of the ink opacity.
