# creationDate

**Framework**: PencilKit  
**Kind**: property

The time at which this stroke path starts.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- visionOS 1.0+

## Declaration

```swift
var creationDate: Date { get }
```

#### Discussion

The time offset of the contained stroke points is relative to this date.
