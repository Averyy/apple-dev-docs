# fieldType

**Framework**: Authentication Services  
**Kind**: property

The type of this editable field.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
var fieldType: ASImportableEditableField.FieldType
```

#### Discussion

This value must be one of:

- `string`
- `concealed-string`
- `email`
- `number`
- `boolean`
- `date`
