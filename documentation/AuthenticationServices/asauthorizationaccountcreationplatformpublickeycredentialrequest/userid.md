# userID

**Framework**: Authentication Services  
**Kind**: property

A stable, opaque identifier for the created account. This will be saved as the userID for the created passkey.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
