# init(httpAuthorizationHeaders:)

**Framework**: Authentication Services  
**Kind**: init

Initializes an authorization with tokens stored in HTTP headers.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- visionOS 1.0+

## Declaration

