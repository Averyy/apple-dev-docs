# second

**Framework**: Authentication Services  
**Kind**: property

A second SymmetricKey that is unique to this passkey, and derived from `input2` if it was specified. If `input2` was not specified, this will be nil.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- visionOS 2.0+
