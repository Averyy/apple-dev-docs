# ASPresentationAnchor

**Framework**: Authentication Services  
**Kind**: typealias

A platform-specific type that indicates the kind of user interface element to use as a presentation anchor.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
typealias ASPresentationAnchor = NSWindow
```
