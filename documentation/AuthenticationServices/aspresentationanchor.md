# ASPresentationAnchor

**Framework**: Authentication Services  
**Kind**: typealias

**Availability**:

- iPadOS 12.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
