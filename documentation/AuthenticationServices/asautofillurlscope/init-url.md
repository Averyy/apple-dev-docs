# init(url:)

**Framework**: Authentication Services  
**Kind**: init

Initialize with the components of a URL.

**Availability**:
- iOS 26.2+
- iPadOS 26.2+
- Mac Catalyst 26.2+
- macOS 26.2+
- visionOS 26.2+

## Declaration

```swift
init?(url: URL)
```

#### Discussion

If the url string from the URL is malformed, nil is returned.

## Parameters

- `url`: The URL to expand.
