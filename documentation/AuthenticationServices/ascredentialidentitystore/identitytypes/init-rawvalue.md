# init(rawValue:)

**Framework**: Authentication Services  
**Kind**: init

**Availability**:
- iOS 17.4+
- iPadOS 17.4+
- Mac Catalyst 17.4+
- macOS 14.4+
- visionOS 1.1+

## Declaration


## Parameters
