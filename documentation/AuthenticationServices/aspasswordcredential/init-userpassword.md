# init(user:password:)

**Framework**: Authentication Services  
**Kind**: init

Initializes a password credential.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.1+
- macOS 10.15+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
init(user: String, password: String)
```

## Parameters

- `user`: The username associated with the credential.
- `password`: The password associated with the credential.
