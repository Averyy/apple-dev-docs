# ASImportableItem.ItemType.login

**Framework**: Authentication Services  
**Kind**: case

The type for an item that contains login-related credential information.

**Availability**:
- iOS 18.2+
- iPadOS 18.2+
- Mac Catalyst 18.2+
- macOS 15.2+
- visionOS 2.2+

## Declaration

```swift
case login
```

#### Discussion

This type of item contains credential information like a password or passkey, as defined in the Credential Exchange Format (CXF) specification.
