# removeObserver(_:)

**Framework**: Local Authentication  
**Kind**: method

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- visionOS 2.0+

## Declaration

```swift
func removeObserver(_ observer: any LAEnvironment.Observer)
```

#### Discussion

Removes the previously registered observer.

If the observer is deallocated, it will be removed automatically.
