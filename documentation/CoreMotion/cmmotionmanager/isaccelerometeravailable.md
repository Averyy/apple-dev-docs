# isAccelerometerAvailable

**Framework**: Core Motion  
**Kind**: property

A Boolean value that indicates whether an accelerometer is available on the device.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.1+
- visionOS 1.0+
- watchOS 2.0+
