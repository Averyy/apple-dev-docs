# deviceMotion

**Framework**: Core Motion  
**Kind**: property

The latest device-motion data.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 14.0+
- watchOS 7.0+

## Declaration

```swift
var deviceMotion: CMDeviceMotion? { get }
```

#### Discussion

This property is `nil` when there is no device-motion data.
