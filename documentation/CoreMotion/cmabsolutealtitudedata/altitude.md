# altitude

**Framework**: Core Motion  
**Kind**: property

The absolute altitude of the device relative to sea level, measured in meters.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
var altitude: Double { get }
```

#### Discussion

This value can be positive or negative.
