# endDate

**Framework**: Core Motion  
**Kind**: property

The result’s end time and date.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.1+
- watchOS 5.0+

## Declaration

```swift
var endDate: Date { get }
```

#### Discussion

The result’s `endDate` is always one minute or less after the [`startDate`](cmdyskineticsymptomresult/startdate.md).
