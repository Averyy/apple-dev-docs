# MACaptionAppearanceBehavior.useContentIfAvailable

**Framework**: Media Accessibility  
**Kind**: case

The preference setting should be used unless the content media being played has its own custom value for this setting.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- macOS 10.9+
- tvOS 17.0+
- visionOS 1.0+

