# kMADimFlashingLightsChangedNotification

**Framework**: Media Accessibility  
**Kind**: var

A notification that posts when a person changes the flashing lights setting on the device.

**Availability**:
- iOS 16.4+
- iPadOS 16.4+
- Mac Catalyst 16.4+
- macOS 13.3+
- tvOS 16.4+
- visionOS 1.0+
