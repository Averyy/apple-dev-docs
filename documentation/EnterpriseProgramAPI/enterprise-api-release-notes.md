# Enterprise Program API Release Notes

**Framework**: Enterprise Program API

Learn about new features and updates in the Enterprise Program API.

#### Overview

The Enterprise Program API is a REST API that enables the automation of actions you take in developer.apple.com. Download the latest [`OpenAPI specification`](https://developer.apple.comhttp://developer.apple.com/sample-code/enterprise/enterprise-program-openapi-oas.zip).
