# OSLogBoolFormat.answer

**Framework**: os  
**Kind**: case

An option to format Boolean values as yes or no strings.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst ?+
- macOS 11.0+
- tvOS 14.0+
- visionOS ?+
- watchOS 7.0+

## Declaration

```swift
case answer
```

#### Discussion

This option formats a Boolean true value as `YES` and a Boolean false value as `NO`.
