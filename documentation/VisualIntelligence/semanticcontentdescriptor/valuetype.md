# SemanticContentDescriptor.ValueType

**Framework**: Visual Intelligence  
**Kind**: typealias

A type that represents the value of a semantic content descriptor.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+

## Declaration

```swift
typealias ValueType = SemanticContentDescriptor
```

#### Discussion

This type defines the value type that the system uses when working with a [`SemanticContentDescriptor`](semanticcontentdescriptor.md) instance. It’s an essential part of App Intents’ type system for processing visual intelligence data.
