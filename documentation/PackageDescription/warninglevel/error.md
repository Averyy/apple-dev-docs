# WarningLevel.error

**Framework**: PackageDescription  
**Kind**: case

Treat as an error.

**Availability**:
- SwiftPM 6.2+

## Declaration

```swift
case error
```

#### Discussion

Warnings will be elevated to errors, causing the build to fail if any such warnings occur.
