# LanguageTag.StringLiteralType

**Framework**: PackageDescription  
**Kind**: typealias

A type that represents a string literal.

## Declaration

```swift
typealias StringLiteralType = String
```

#### Discussion

Valid types for `StringLiteralType` are `String` and `StaticString`.
