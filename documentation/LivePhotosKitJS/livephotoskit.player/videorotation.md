# videoRotation

**Framework**: LivePhotosKit JS  
**Kind**: property

The angle that the video is displayed.

**Availability**:
- LivePhotosKit JS 1.3+

## Declaration

```swift
attribute Number videoRotation;
```

#### Discussion

This property takes on a value of: 0, 90, 180, or 270, and is automatically determined as part of the video loading process.

A video might be provided in a different format, and that video is rotated in a way that the Player or the browser canâ€™t compensate for. In this situation, you can manually assign a different value to correct the issue without having to rotate the source video.
