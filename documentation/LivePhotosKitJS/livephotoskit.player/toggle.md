# toggle

**Framework**: LivePhotosKit JS  
**Kind**: method

Starts playing the video component of Live Photo if the video is paused, or pauses the video if it is playing.

**Availability**:
- LivePhotosKit JS 1.0+

## Declaration

```swift
void toggle();
```
