# ContentIdentifier

**Framework**: SiriKit Cloud Media  
**Kind**: typealias

An identifier for a song, podcast, ad, or other media content. The identifier must be stable and unique within a queue.

**Availability**:
- SiriKit Cloud Media 1.0.2+

## Declaration

```swift
string ContentIdentifier
```
