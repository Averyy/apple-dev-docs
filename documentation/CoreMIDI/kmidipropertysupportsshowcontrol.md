# kMIDIPropertySupportsShowControl

**Framework**: Core MIDI  
**Kind**: var

A Boolean value that indicates whether the device implements the MIDI Show Control specification.

**Availability**:
- iOS 4.2+
- iPadOS 4.2+
- Mac Catalyst 13.1+
- macOS 10.4+
