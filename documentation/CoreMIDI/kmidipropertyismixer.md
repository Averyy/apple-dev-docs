# kMIDIPropertyIsMixer

**Framework**: Core MIDI  
**Kind**: var

A Boolean value that indicates whether the device or entity mixes external audio signals.

**Availability**:
- iOS 4.2+
- iPadOS 4.2+
- Mac Catalyst 13.1+
- macOS 10.2+
- visionOS 1.0+
