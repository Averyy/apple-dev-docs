# init(host:)

**Framework**: Core MIDI  
**Kind**: init

Creates a connection to the specified host.

**Availability**:
- iOS 4.2+
- iPadOS 4.2+
- Mac Catalyst 13.1+
- macOS 10.15+
- visionOS 1.0+

## Declaration

```swift
convenience init(host: MIDINetworkHost)
```

## Parameters

- `host`: The host with which to establish a connection.
