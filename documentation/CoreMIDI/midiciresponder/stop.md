# stop()

**Framework**: Core MIDI  
**Kind**: method

Stops receiving initiator requests and disconnects all connected initiators.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- visionOS 1.0+
