# start()

**Framework**: Core MIDI  
**Kind**: method

Starts receiving initiator requests.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- visionOS 1.0+

## Declaration

```swift
func start() -> Bool
```

#### Return Value

A Boolean value that indicates whether the the responder started.
