# LockedCameraCaptureExtensionScene

**Framework**: LockedCameraCapture  
**Kind**: protocol

**Availability**:

- iPadOS 18.0+

#### Overview

Implement this protocol with the [`LockedCameraCaptureUIScene`](lockedcameracaptureuiscene.md) providing the UI.
