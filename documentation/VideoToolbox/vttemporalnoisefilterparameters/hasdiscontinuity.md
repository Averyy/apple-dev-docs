# hasDiscontinuity

**Framework**: Video Toolbox  
**Kind**: property

A Boolean that indicates sequence discontinuity, forcing the processor to reset prior to processing the source frame.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
