# init(width:height:)

**Framework**: Videotoolbox  
**Kind**: init

Creates a size with a width and height.

**Availability**:
- iOS 6.0+
- iPadOS 6.0+
- Mac Catalyst 13.0+
- macOS 10.8+
- tvOS 10.2+
- visionOS 1.0+

## Declaration

```swift
init(width: Int32, height: Int32)
```

## Parameters

- `width`: The width of the size.
- `height`: The height of the size.
