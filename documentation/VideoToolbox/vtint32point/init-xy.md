# init(x:y:)

**Framework**: Video Toolbox  
**Kind**: init

Creates a point with coordinates specified as integer values.

**Availability**:
- iOS 6.0+
- iPadOS 6.0+
- Mac Catalyst 13.0+
- macOS 10.8+
- tvOS 10.2+
- visionOS 1.0+

## Declaration

```swift
init(x: Int32, y: Int32)
```

## Parameters

- `x`: The x-coordinate of the point.
- `y`: The y-coordinate of the point.
