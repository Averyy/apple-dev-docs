# extensionName

**Framework**: Video Toolbox  
**Kind**: property

A dictionary key for the localized extension name.

**Availability**:
- macOS 15.0+

## Declaration

```swift
static let extensionName: VTExtensionPropertiesKey
```

#### Discussion

This key points to a `CFStringRef` value with the localized extension name.
