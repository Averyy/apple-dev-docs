# extensionURL

**Framework**: Video Toolbox  
**Kind**: property

A dictionary key for the URL of the extension.

**Availability**:
- macOS 15.0+

## Declaration

```swift
static let extensionURL: VTExtensionPropertiesKey
```

#### Discussion

This key points to a `CFURLRef` value with the URL for the extension.
