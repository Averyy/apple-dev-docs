# init(excluding:)

**Framework**: MapKit  
**Kind**: init

Creates an address filter with options for excluding results in a search.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+

## Declaration

```swift
init(excluding options: MKAddressFilter.Options)
```

## Parameters

- `options`: A set of filter options that excludes results.
