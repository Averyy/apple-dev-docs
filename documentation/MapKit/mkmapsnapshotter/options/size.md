# size

**Framework**: MapKit  
**Kind**: property

The size of the image that you want to create.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- macOS 10.9+
- tvOS 9.2+
- visionOS 1.0+
- watchOS 1.0+

## Declaration

```swift
var size: NSSize { get set }
```

#### Discussion

The default value of this property is 256 by 256 points.
