# appearance

**Framework**: MapKit  
**Kind**: property

The visual style that MapKit uses when rendering the snapshot.

**Availability**:
- macOS 10.14+

## Declaration

```swift
var appearance: NSAppearance { get }
```

#### Discussion

Use this property to determine whether MapKit applied a light or dark appearance to your map.
