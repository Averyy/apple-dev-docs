# init(scope:)

**Framework**: MapKit  
**Kind**: init

Creates a new map compass with the scope you specify.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst ?+
- macOS 14.0+
- tvOS 17.0+
- visionOS ?+
- watchOS 10.0+

## Declaration

```swift
@MainActor
@preconcurrency init(scope: Namespace.ID? = nil)
```

## Parameters

- `scope`: A   value that identifies a namespace and that you use to associate this control with a map instance.
