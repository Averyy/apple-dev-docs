# init(request:)

**Framework**: MapKit  
**Kind**: init

Creates and returns a search object for fetching points of interest.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+
- watchOS 7.0+

## Declaration

```swift
init(request: MKLocalPointsOfInterestRequest)
```

## Parameters

- `request`: The search parameters to fetch points of interest.
