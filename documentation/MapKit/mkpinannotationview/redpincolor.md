# redPinColor()

**Framework**: MapKit  
**Kind**: method

Returns the standard color for red pins.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
class func redPinColor() -> UIColor
```

#### Return Value

The red pin color.

#### Discussion

The system uses red pins to indicate destination points on the map.
