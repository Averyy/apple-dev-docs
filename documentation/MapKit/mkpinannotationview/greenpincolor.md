# greenPinColor()

**Framework**: MapKit  
**Kind**: method

Returns the standard color for green pins.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
class func greenPinColor() -> NSColor
```

#### Return Value

The green pin color.

#### Discussion

The system uses green pins to indicate starting points on the map.
