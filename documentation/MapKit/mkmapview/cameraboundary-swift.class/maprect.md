# mapRect

**Framework**: MapKit  
**Kind**: property

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+

## Declaration


#### Discussion

Both the [`mapRect`](mkmapview/cameraboundary-swift.class/maprect.md) and the [`region`](mkmapview/cameraboundary-swift.class/region.md) represent the same camera boundary.
