# init(coordinateRegion:)

**Framework**: MapKit  
**Kind**: init

Creates a camera boundary using the provided coordinate region.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
init?(coordinateRegion region: MKCoordinateRegion)
```

## Parameters

- `region`: The   region that describes the camera boundary.
