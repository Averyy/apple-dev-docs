# init(mapItem:)

**Framework**: MapKit  
**Kind**: init

Creates a LookAround scene with the location described by the specified map item.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- visionOS 1.0+

## Declaration

