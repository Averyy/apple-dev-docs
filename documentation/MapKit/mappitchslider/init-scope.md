# init(scope:)

**Framework**: MapKit  
**Kind**: init

Creates a new map pitch slider with the scope you specify.

**Availability**:
- Mac Catalyst 14.0+
- macOS 14.0+

## Declaration

```swift
@MainActor
@preconcurrency init(scope: Namespace.ID? = nil)
```

## Parameters

- `scope`: A   value that identifies this namespace and that you use to associate this control with a map instance.
