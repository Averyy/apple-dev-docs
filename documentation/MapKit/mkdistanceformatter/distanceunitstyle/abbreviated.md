# MKDistanceFormatter.DistanceUnitStyle.abbreviated

**Framework**: MapKit  
**Kind**: case

Abbreviates units.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- macOS 10.9+
- tvOS 9.2+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
case abbreviated
```

#### Discussion

For example, if the units are metric the map displays “km” instead of “kilometer.”
