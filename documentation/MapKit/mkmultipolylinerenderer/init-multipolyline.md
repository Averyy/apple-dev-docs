# init(multiPolyline:)

**Framework**: MapKit  
**Kind**: init

Creates an object that renders a visual representation of multiple polyline objects.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
init(multiPolyline: MKMultiPolyline)
```

## Parameters

- `multiPolyline`: The collection of polyline shapes to render.
