# init(mapRect:)

**Framework**: MapKit  
**Kind**: init

Creates the largest possible circle centered within the given map rectangle.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst ?+
- macOS 14.0+
- tvOS 17.0+
- visionOS ?+
- watchOS 10.0+

## Declaration

```swift
init(mapRect: MKMapRect)
```

## Parameters

- `mapRect`: The rectangle to center the circle within.
