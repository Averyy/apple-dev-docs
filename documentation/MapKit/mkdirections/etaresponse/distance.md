# distance

**Framework**: MapKit  
**Kind**: property

**Availability**:

- iPadOS 9.0+
- Mac Catalyst 13.1+

- tvOS 9.0+

## Declaration


#### Discussion

This property contains the overall distance traversed by the route.
