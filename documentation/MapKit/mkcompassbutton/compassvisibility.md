# compassVisibility

**Framework**: MapKit  
**Kind**: property

The visibility of the compass button.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.0+
- macOS 11.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var compassVisibility: MKFeatureVisibility { get set }
```

#### Discussion

You can configure a compass button to be visible all the time or only when the compass heading changes.
