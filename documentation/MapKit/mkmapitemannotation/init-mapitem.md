# init(mapItem:)

**Framework**: MapKit  
**Kind**: init

Creates a map item annotation

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+

## Declaration

```swift
init?(mapItem: MKMapItem)
```

#### Discussion

If the map item does not have valid coordinate data, the result will be nil.

## Parameters

- `mapItem`: The map item this annotation will represent
