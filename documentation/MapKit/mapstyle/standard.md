# standard

**Framework**: MapKit  
**Kind**: property

A map style that represents the default map presentation, which is a street map that shows the position of all roads and some road names, depending upon the zoom level of the map.

**Availability**:

- iPadOS 17.0+
- Mac Catalyst ?+
