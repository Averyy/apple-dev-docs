# init(mapItem:)

**Framework**: MapKit  
**Kind**: init

Create a map item detail view controller.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- visionOS 2.0+

## Declaration

```swift
@MainActor
init(mapItem: MKMapItem?)
```

#### Discussion

Displays an inline map with the place data.

## Parameters

- `mapItem`: The map item to display, or   to indicate the item is loading.
