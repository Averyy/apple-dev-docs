# title

**Framework**: MapKit  
**Kind**: property

The title of the shape annotation.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.1+
- macOS 10.9+
- tvOS 9.2+
- visionOS 1.0+

## Declaration

```swift
var title: String? { get set }
```

#### Discussion

The shape displays this string in the callout for the associated annotation view. The default value of this property is `nil`.
