# init(mapView:)

**Framework**: MapKit  
**Kind**: init

Creates a pitch control and associates it with the specified map view.

**Availability**:
- Mac Catalyst 14.0+
- macOS 11.0+

## Declaration

```swift
@MainActor
convenience init(mapView: MKMapView?)
```

## Parameters

- `mapView`: The map to associate with the zoom control.
