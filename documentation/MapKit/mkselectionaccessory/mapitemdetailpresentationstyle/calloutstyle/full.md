# MKSelectionAccessory.MapItemDetailPresentationStyle.CalloutStyle.full

**Framework**: MapKit  
**Kind**: case

A rich, detailed callout style that is suitable for large map views.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- visionOS 2.0+

## Declaration

```swift
case full
```

#### Discussion

Use this style for map views on iPadOS and macOS. Use the sheet presentation to display full detail place information on iOS.
