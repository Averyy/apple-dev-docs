# animatesWhenAdded

**Framework**: MapKit  
**Kind**: property

A Boolean that indicates whether the marker animates into position onscreen.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 11.0+
- tvOS 11.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var animatesWhenAdded: Bool { get set }
```

#### Discussion

The default value of this property is `false`.
