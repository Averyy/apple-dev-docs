# full

**Framework**: MapKit  
**Kind**: property

A rich, detailed callout style that is suitable for large map views.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst ?+
- macOS 15.0+
- visionOS 2.0+

## Declaration

```swift
static var full: MapItemDetailSelectionAccessoryStyle.CalloutStyle { get }
```

#### Discussion

Use this style for map views on iPadOS and macOS. Use the sheet presentation to display full detail place information on iOS.
