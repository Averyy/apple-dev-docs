# init(mapView:)

**Framework**: MapKit  
**Kind**: init

Initializes a newly created bar button item with the specified map view.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- Mac Catalyst 13.1+
- visionOS 1.0+

## Declaration

```swift
@MainActor
init(mapView: MKMapView?)
```

#### Return Value

The initialized bar button item.

## Parameters

- `mapView`: The map view used by this bar button item.
