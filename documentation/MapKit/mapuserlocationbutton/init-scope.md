# init(scope:)

**Framework**: MapKit  
**Kind**: init

Creates a new user location button with the scope you specify.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst ?+
- macOS 14.0+
- tvOS 17.0+
- visionOS ?+
- watchOS 10.0+

## Declaration

```swift
@MainActor
@preconcurrency init(scope: Namespace.ID? = nil)
```

## Parameters

- `scope`: A   value that identifies this namespace and that you use to associate the user location indicator with a map instance.
