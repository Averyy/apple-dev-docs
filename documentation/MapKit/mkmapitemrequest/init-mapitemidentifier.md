# init(mapItemIdentifier:)

**Framework**: MapKit  
**Kind**: init

Create a request with a map item identifier.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+

## Declaration

```swift
init(mapItemIdentifier identifier: MKMapItem.Identifier)
```

## Parameters

- `identifier`: The identifier to use for the request.
