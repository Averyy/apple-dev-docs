# init(rawValue:)

**Framework**: MapKit  
**Kind**: init

Creates a new feature option structure with the specified value.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- visionOS 1.0+

## Declaration

```swift
init(rawValue: Int)
```

## Parameters

- `rawValue`: An integer value that represents one of the available map features.
