# init(_:)

**Framework**: MapKit  
**Kind**: init

Creates a multipolyline object using the provided polylines.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
init(_ polylines: [MKPolyline])
```

## Parameters

- `polylines`: An array of   objects to initialize the object with.
