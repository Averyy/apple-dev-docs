# init(_:)

**Framework**: MapKit  
**Kind**: init

Creates a multipolygon object using the provided polygons.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
init(_ polygons: [MKPolygon])
```

## Parameters

- `polygons`: An array of polygons to initialize the array with.
