# preferredLocale

**Framework**: MapKit  
**Kind**: property

A value that indicates the preferred locale for the addresses the request returns, or `nil` if the framework should use the device locale.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
- tvOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)
- watchOS 26.0+ (Beta)

