# supportsDevice(_:)

**Framework**: MetalFX  
**Kind**: method

Returns a Boolean value that indicates whether the spatial scaler works with a GPU.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+

## Declaration

```swift
class func supportsDevice(_ device: any MTLDevice) -> Bool
```

## Parameters

- `device`: An   instance that represents a GPU.
