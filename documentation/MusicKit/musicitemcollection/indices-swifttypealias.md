# MusicItemCollection.Indices

**Framework**: MusicKit  
**Kind**: typealias

A type that represents the indices that are valid for subscripting the collection, in ascending order.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
typealias Indices = Array<MusicItemType>.Indices
```
