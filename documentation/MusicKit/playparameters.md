# PlayParameters

**Framework**: MusicKit  
**Kind**: struct

An opaque object that represents parameters to initiate playback of a playable music item using a music player.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
