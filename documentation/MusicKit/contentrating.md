# ContentRating

**Framework**: MusicKit  
**Kind**: enum

The rating of the content that potentially plays while playing a resource.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
enum ContentRating
```

#### Overview

A nil value means no rating is available for this resource.
