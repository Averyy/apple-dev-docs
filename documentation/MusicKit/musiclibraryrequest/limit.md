# limit

**Framework**: MusicKit  
**Kind**: property

A limit for the number of items to return in the library response.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 16.0+
- visionOS 1.0+
- watchOS 9.0+

## Declaration

```swift
var limit: Int
```

#### Discussion

A limit of 0 length means no limit.
