# init(playlist:startingAt:)

**Framework**: MusicKit  
**Kind**: init

Creates a playback queue with a playlist and a specific playlist entry for the player to start playback.

**Availability**:
- iOS 16.4+
- iPadOS 16.4+
- Mac Catalyst 16.4+
- macOS 14.0+
- tvOS 16.4+
