# ==(_:_:)

**Framework**: MusicKit  
**Kind**: op

Returns a Boolean value indicating whether two values are equal.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 14.0+
- tvOS 15.0+
- visionOS 1.0+

## Declaration

```swift
static func == (left: MusicPlayer.Queue, right: MusicPlayer.Queue) -> Bool
```

#### Discussion

Equality is the inverse of inequality. For any values `a` and `b`, `a == b` implies that `a != b` is `false`.

## Parameters

- `lhs`: A value to compare.
- `rhs`: Another value to compare.
