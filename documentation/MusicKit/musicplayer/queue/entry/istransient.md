# isTransient

**Framework**: MusicKit  
**Kind**: property

A Boolean value that indicates whether this entry of the playback queue has a transient music item.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 14.0+
