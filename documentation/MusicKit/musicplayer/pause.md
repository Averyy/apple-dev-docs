# pause()

**Framework**: MusicKit  
**Kind**: method

Pauses playback of the current entry.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 14.0+
- tvOS 15.0+
- visionOS 1.0+
