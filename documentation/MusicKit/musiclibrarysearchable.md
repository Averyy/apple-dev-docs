# MusicLibrarySearchable

**Framework**: MusicKit  
**Kind**: protocol

A protocol for music items that your app can fetch by using a library search request.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 16.0+
- visionOS 1.0+
- watchOS 9.0+
