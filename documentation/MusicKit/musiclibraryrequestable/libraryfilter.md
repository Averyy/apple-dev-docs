# LibraryFilter

**Framework**: MusicKit  
**Kind**: associatedtype  
**Required**: Yes

The associated type that contains the set of music item properties your app uses as a filter for a library request.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 16.0+
- visionOS 1.0+
- watchOS 9.0+
