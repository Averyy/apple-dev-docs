# ApplicationMusicPlayer.Queue.Entries.Index

**Framework**: MusicKit  
**Kind**: typealias

A type that represents a position in the collection.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 14.0+
- tvOS 15.0+
- visionOS 1.0+

## Declaration

```swift
typealias Index = Array<MusicPlayer.Queue.Entry>.Index
```

#### Discussion

Valid indices consist of the position of every element and a “past the end” position that’s not valid for use as a subscript argument.
