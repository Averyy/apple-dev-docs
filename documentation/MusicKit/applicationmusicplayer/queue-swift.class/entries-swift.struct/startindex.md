# startIndex

**Framework**: MusicKit  
**Kind**: property

The position of the first element in a nonempty collection.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 14.0+
- tvOS 15.0+
- visionOS 1.0+

## Declaration

```swift
var startIndex: ApplicationMusicPlayer.Queue.Entries.Index { get }
```

#### Discussion

If the collection is empty, `startIndex` is equal to `endIndex`.
