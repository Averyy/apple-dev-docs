# messageIdentifier

**Framework**: MusicKit  
**Kind**: property

An identifier for selecting the main message that the subscription offer sheet presents to the user.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+

## Declaration

var messageIdentifier: MusicSubscriptionOffer.MessageIdentifier
