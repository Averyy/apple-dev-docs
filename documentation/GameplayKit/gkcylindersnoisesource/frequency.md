# frequency

**Framework**: GameplayKit  
**Kind**: property

A value that determines the size and spacing of concentric cylinders.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- macOS 10.12+
- tvOS 10.0+
- visionOS 1.0+

## Declaration

```swift
var frequency: Double { get set }
```

#### Discussion

Higher frequencies increase the number of and decrease the spacing between concentric cylinders. The default value is `1.0`.
