# init(gridPosition:)

**Framework**: GameplayKit  
**Kind**: init

Initializes a graph node with the specified position on a grid.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
init(gridPosition: vector_int2)
```

#### Return Value

A new graph node representing the specified grid position.

#### Discussion

For more information, see [`GameplayKit Programming Guide`](https://developer.apple.comhttps://developer.apple.com/library/archive/documentation/General/Conceptual/GameplayKit_Guide/index.html#//apple_ref/doc/uid/TP40015172).

## Parameters

- `gridPosition`: The position for the node on a discrete integer grid.
