# squareSize

**Framework**: GameplayKit  
**Kind**: property

The size (both width and height) of squares in the generated checkerboard pattern.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- macOS 10.12+
- tvOS 10.0+
- visionOS 1.0+

## Declaration

```swift
var squareSize: Double { get set }
```

#### Discussion

The default value is `1.0`.
