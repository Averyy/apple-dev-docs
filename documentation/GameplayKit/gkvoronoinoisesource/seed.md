# seed

**Framework**: GameplayKit  
**Kind**: property

The value that determines the specific configuration of noise produced by the noise source.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- macOS 10.12+
- tvOS 10.0+
- visionOS 1.0+

## Declaration

```swift
var seed: Int32 { get set }
```

#### Discussion

Like the [`seed`](gkarc4randomsource/seed.md) value of a random source, a noise sourceâ€™s seed value determines the pseudorandom aspect of noise generated by the noise source. Changing the seed value produces different overall structure to the generated noise, while changing other parameters produces variation upon that structure.
