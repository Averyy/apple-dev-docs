# d20()

**Framework**: GameplayKit  
**Kind**: method

Creates a random distribution equivalent to a twenty-sided die.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
class func d20() -> Self
```

#### Return Value

A uniform random distribution that produces values in the range `[1, 20]`.

#### Discussion

The newly created distribution uses its own [`GKRandomSource`](gkrandomsource.md) instance, so the random behavior of the distribution is independent from that of other randomizers.
