# init()

**Framework**: GameplayKit  
**Kind**: init

Initializes a random source from a nondeterministic seed.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
convenience init()
```

#### Return Value

A new, independent random source.

#### Discussion

Calling this method seeds the random source with a sequence of bytes sampled from a high-entropy system source. To later replicate the sequence of numbers generated by this source, read the [`seed`](gkarc4randomsource/seed.md) property and use the resulting data to initialize another random source with the [`init(seed:)`](gkarc4randomsource/init(seed:).md) initializer.

For more information, see [`GameplayKit Programming Guide`](https://developer.apple.comhttps://developer.apple.com/library/archive/documentation/General/Conceptual/GameplayKit_Guide/index.html#//apple_ref/doc/uid/TP40015172).
