# addEntity(_:)

**Framework**: GameplayKit  
**Kind**: method

Adds a GameplayKit entity to the list of entities managed by the scene.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- macOS 10.12+
- tvOS 10.0+
- visionOS 1.0+

## Declaration

```swift
func addEntity(_ entity: GKEntity)
```

## Parameters

- `entity`: The entity to be added to the scene.
