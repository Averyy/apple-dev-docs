# init(identifier:)

**Framework**: Background Tasks  
**Kind**: init

Return a new refresh task request for the specified identifier.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
init(identifier: String)
```

## Parameters

- `identifier`: The string identifier of the refresh task associated with the request.
