# ALAssetPropertyOrientation

**Framework**: Assets Library  
**Kind**: var

The key to retrieve the orientation of the asset.

## Declaration

```swift
let ALAssetPropertyOrientation: String
```

#### Discussion

The corresponding value is an [`NSNumber`](https://developer.apple.com/documentation/Foundation/NSNumber) object containing an asset’s orientation as described by the TIFF format.
