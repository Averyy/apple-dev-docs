# kCGImagePropertyDNGPrivateData

**Framework**: Image I/O  
**Kind**: var

Private data that manufacturers may store with an image and use in their own converters.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- macOS 10.12+
- tvOS 10.0+
- visionOS 1.0+
