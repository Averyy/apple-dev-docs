# kCGImagePropertyPNGDictionary

**Framework**: Image I/O  
**Kind**: var

A dictionary of key-value pairs for an image that uses Portable Network Graphics (PNG) format.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.1+
- macOS 10.4+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
let kCGImagePropertyPNGDictionary: CFString
```
