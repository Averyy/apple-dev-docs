# TIFF Image Properties

**Framework**: Image I/O

Metadata keys for the Tagged Image File Format (TIFF).

## Topics

### Dictionary
- [let kCGImagePropertyTIFFDictionary: CFString](kcgimagepropertytiffdictionary.md)
  A dictionary of key-value pairs for an image that uses Tagged Image File Format (TIFF). See [`TIFF Image Properties`](tiff-image-properties.md).
### Image Quality
- [let kCGImagePropertyTIFFCompression: CFString](kcgimagepropertytiffcompression.md)
  The compression scheme used on the image data.
- [let kCGImagePropertyTIFFPhotometricInterpretation: CFString](kcgimagepropertytiffphotometricinterpretation.md)
  The color space of the image data.
- [let kCGImagePropertyTIFFTransferFunction: CFString](kcgimagepropertytifftransferfunction.md)
  The transfer function, in tabular format, used to map pixel components from a nonlinear form into a linear form.
### Canvas Details
- [let kCGImagePropertyTIFFOrientation: CFString](kcgimagepropertytifforientation.md)
  The image orientation.
- [let kCGImagePropertyTIFFXResolution: CFString](kcgimagepropertytiffxresolution.md)
  The number of pixels per resolution unit in the image width direction.
- [let kCGImagePropertyTIFFYResolution: CFString](kcgimagepropertytiffyresolution.md)
  The number of pixels per resolution unit in the image height direction.
- [let kCGImagePropertyTIFFResolutionUnit: CFString](kcgimagepropertytiffresolutionunit.md)
  The units of resolution.
- [let kCGImagePropertyTIFFWhitePoint: CFString](kcgimagepropertytiffwhitepoint.md)
  The white point of the image.
- [let kCGImagePropertyTIFFPrimaryChromaticities: CFString](kcgimagepropertytiffprimarychromaticities.md)
  The chromaticities of the primaries of the image.
- [let kCGImagePropertyTIFFTileLength: CFString](kcgimagepropertytifftilelength.md)
- [let kCGImagePropertyTIFFTileWidth: CFString](kcgimagepropertytifftilewidth.md)
### Descriptive Information
- [let kCGImagePropertyTIFFDocumentName: CFString](kcgimagepropertytiffdocumentname.md)
  The document name.
- [let kCGImagePropertyTIFFImageDescription: CFString](kcgimagepropertytiffimagedescription.md)
  The image description.
- [let kCGImagePropertyTIFFArtist: CFString](kcgimagepropertytiffartist.md)
  The artist who created the image.
- [let kCGImagePropertyTIFFCopyright: CFString](kcgimagepropertytiffcopyright.md)
  Copyright information.
- [let kCGImagePropertyTIFFDateTime: CFString](kcgimagepropertytiffdatetime.md)
  The date and time that the image was created.
- [let kCGImagePropertyTIFFMake: CFString](kcgimagepropertytiffmake.md)
  The name of the manufacturer of the camera or input device.
- [let kCGImagePropertyTIFFModel: CFString](kcgimagepropertytiffmodel.md)
  The camera or input device model.
- [let kCGImagePropertyTIFFSoftware: CFString](kcgimagepropertytiffsoftware.md)
  The name and version of the software used for image creation.
- [let kCGImagePropertyTIFFHostComputer: CFString](kcgimagepropertytiffhostcomputer.md)
  The computer or operating system used when the image was created.
