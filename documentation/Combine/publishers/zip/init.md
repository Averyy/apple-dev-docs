# init(_:_:)

**Framework**: Combine  
**Kind**: init

Creates a publisher that applies the zip function to two upstream publishers.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
init(_ a: A, _ b: B)
```

## Parameters

- `a`: A publisher to zip.
- `b`: Another publisher to zip.
