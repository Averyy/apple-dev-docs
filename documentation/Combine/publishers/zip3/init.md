# init(_:_:_:)

**Framework**: Combine  
**Kind**: init

Creates a publisher that applies the zip function to three upstream publishers.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
init(_ a: A, _ b: B, _ c: C)
```

## Parameters

- `a`: A publisher to zip.
- `b`: A second publisher to zip.
- `c`: A third publisher to zip.
