# init(_:_:)

**Framework**: Combine  
**Kind**: init

Creates a publisher that receives and combines the latest elements from two publishers.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
init(_ a: A, _ b: B)
```

## Parameters

- `a`: The first upstream publisher.
- `b`: The second upstream publisher.
