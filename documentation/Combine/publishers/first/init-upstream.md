# init(upstream:)

**Framework**: Combine  
**Kind**: init

Creates a publisher that publishes the first element of a stream, then finishes.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
init(upstream: Upstream)
```

## Parameters

- `upstream`: The publisher from which this publisher receives elements.
