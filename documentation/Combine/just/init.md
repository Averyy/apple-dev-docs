# init(_:)

**Framework**: Combine  
**Kind**: init

Initializes a publisher that emits the specified output just once.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
init(_ output: Output)
```

## Parameters

- `output`: The one element that the publisher emits.
