# -=(_:_:)

**Framework**: Combine  
**Kind**: op

Subtracts the second value from the first and stores the difference in the left-hand-side variable.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
static func -= (lhs: inout Self, rhs: Self)
```

## Parameters

- `lhs`: A numeric value.
- `rhs`: The value to subtract from  .
