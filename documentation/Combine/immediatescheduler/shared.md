# shared

**Framework**: Combine  
**Kind**: property

The shared instance of the immediate scheduler.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
static let shared: ImmediateScheduler
```

#### Discussion

You cannot create instances of the immediate scheduler yourself. Use only the shared instance.
