# init(webPage:)

**Framework**: BrowserEngineKit  
**Kind**: init

**Availability**:
- iOS 17.4+
- iPadOS 17.4+

## Declaration

```swift
init(webPage url: URL)
```

## Parameters

- `url`: A URL that identifies this media environment.
