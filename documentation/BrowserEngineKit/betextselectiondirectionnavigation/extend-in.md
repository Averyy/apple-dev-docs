# extend(in:)

**Framework**: BrowserEngineKit  
**Kind**: method  
**Required**: Yes

Extends text selection in the specified directions, such as in response to an arrow key press while shift is held.

**Availability**:
- iOS 17.4+
- iPadOS 17.4+
