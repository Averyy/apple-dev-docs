# addTextRect(_:forCharacterRange:)

**Framework**: BrowserEngineKit  
**Kind**: method

Adds a text `rect` for the given character `range` The CGRects representing each character range are specified in -textInputView coordinates.

**Availability**:
- iOS 17.4+
- iPadOS 17.4+
- tvOS 17.4+
- visionOS 1.1+

## Declaration

