# elapsedTime

**Framework**: Webkitjs  
**Kind**: instp

The duration of the transition, in seconds, since this event was sent. This value is not affected by the value of the CSS `transition-delay` property.

**Availability**:
- Safari Desktop 10.0+
- Safari Mobile 10.0+

## Declaration

```swift
readonly attribute double elapsedTime;
```
