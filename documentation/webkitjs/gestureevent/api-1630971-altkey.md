# altKey

**Framework**: Webkitjs  
**Kind**: instp

A Boolean value indicating whether the alt key is pressed.

**Availability**:
- Safari Desktop 10.1+
- Safari Mobile 2.0+

## Declaration

```swift
readonly attribute boolean altKey;
```

#### Discussion

If `true`, the alt key is pressed; otherwise, it is not. If there is no keyboard, this value is `false`.
