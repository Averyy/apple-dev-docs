# ctrlKey

**Framework**: Webkitjs  
**Kind**: instp

A Boolean value indicating whether the control key is pressed.

**Availability**:
- Safari Desktop 10.1+
- Safari Mobile 2.0+

## Declaration

```swift
readonly attribute boolean ctrlKey;
```

#### Discussion

If `true`, the control key is pressed; otherwise, it is not. If there is no keyboard, this value is `false`.
