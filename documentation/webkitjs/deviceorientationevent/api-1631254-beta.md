# beta

**Framework**: WebKit JS  
**Kind**: instp

The rotation, in degrees, of the device frame around its x-axis.

**Availability**:
- Safari Mobile 4.2+

## Declaration

```swift
readonly attribute unrestricted double beta;
```

#### Discussion

Values are between -90 and 90. This property is `null` if the device cannot provide this angle.
