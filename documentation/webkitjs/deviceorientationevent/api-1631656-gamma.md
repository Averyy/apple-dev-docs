# gamma

**Framework**: WebKit JS  
**Kind**: instp

The rotation, in degrees, of the device frame around its y-axis.

**Availability**:
- Safari Mobile 4.2+

## Declaration

```swift
readonly attribute unrestricted double gamma;
```

#### Discussion

Values are between -180 and 180.  This property is `null` if the device cannot provide this angle.
