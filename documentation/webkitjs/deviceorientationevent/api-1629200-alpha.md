# alpha

**Framework**: WebKit JS  
**Kind**: instp

The rotation, in degrees, of the device frame around its z-axis.

**Availability**:
- Safari Mobile 4.2+

## Declaration

```swift
readonly attribute unrestricted double alpha;
```

#### Discussion

Values are between 0 and 360. This property is `null` if the device cannot provide this angle.
