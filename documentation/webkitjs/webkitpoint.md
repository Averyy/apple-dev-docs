# WebKitPoint

**Framework**: Webkitjs  
**Kind**: cl

`WebKitPoint` objects represent a point in two-dimensional space used by CSS transitions specified using the -webkit-transition property.

**Availability**:
- Safari Desktop 10.0+
- Safari Mobile 2.0+
