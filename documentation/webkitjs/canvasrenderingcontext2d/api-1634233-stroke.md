# stroke

**Framework**: WebKit JS  
**Kind**: instm

Draws the outline of the current path using the current stroke style and line width.

**Availability**:
- Safari Desktop 10.0+
- Safari Mobile 10.0+

## Declaration

```swift
void stroke(
    DOMPath path
);
```

#### Discussion

The stroke style can be a color, gradient, or pattern.
