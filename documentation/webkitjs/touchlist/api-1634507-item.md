# item

**Framework**: WebKit JS  
**Kind**: instm

Returns the [`Touch`](touch.md) object at the given index.

**Availability**:
- Safari Desktop 10.1+
- Safari Mobile 2.0+

## Declaration

```swift
getter Touch? item(
    unsigned long index
);
```

## Parameters

- `index`: The index of the item to return.
