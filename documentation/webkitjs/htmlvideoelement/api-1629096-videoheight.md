# videoHeight

**Framework**: WebKit JS  
**Kind**: instp

The native height of the video in CSS pixels. (read-only)

**Availability**:
- Safari Desktop 10.0+
- Safari Mobile 10.0+

## Declaration

```swift
readonly attribute unsigned long videoHeight;
```

#### Discussion

If no video data is available, this property is `0`.
