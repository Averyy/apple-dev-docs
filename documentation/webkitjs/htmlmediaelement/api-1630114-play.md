# play

**Framework**: Webkitjs  
**Kind**: instm

Begins playing the media resource.

**Availability**:
- Safari Desktop 3.1+
- Safari Mobile 3.0+

## Declaration

```swift
Promise <void> play();
```

#### Discussion

If the media resource is not available for playback, this method loads it.
