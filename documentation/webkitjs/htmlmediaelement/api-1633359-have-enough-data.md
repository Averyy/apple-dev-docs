# HAVE_ENOUGH_DATA

**Framework**: Webkitjs  
**Kind**: clconst

Enough data is available to play at the default playback rate to the end of the media resource without having to pause to rebuffer.

**Availability**:
- Safari Desktop 10.0+
- Safari Mobile 10.0+

## Declaration

```swift
const unsigned short HAVE_ENOUGH_DATA;
```
