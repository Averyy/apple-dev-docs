# HAVE_METADATA

**Framework**: WebKit JS  
**Kind**: clconst

Enough of the media resource has been loaded to know the duration, and in the case of a `video` element, the dimensions.

**Availability**:
- Safari Desktop 4.0+
- Safari Mobile 3.0+

## Declaration

```swift
const unsigned short HAVE_METADATA;
```
