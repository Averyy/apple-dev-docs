# networkState

**Framework**: Webkitjs  
**Kind**: instp

The state of downloading the media resource. (read-only)

**Availability**:
- Safari Desktop 3.1+
- Safari Mobile 3.0+

## Declaration

```swift
readonly attribute unsigned short networkState;
```

#### Discussion

Possible values are described in [`Network States`](htmlmediaelement/network_states.md).
