# muted

**Framework**: WebKit JS  
**Kind**: instp

A Boolean value that determines whether the audio content should be muted.

**Availability**:
- Safari Desktop 3.1+
- Safari Mobile 3.0+

## Declaration

```swift
attribute boolean muted;
```

#### Discussion

If `true`, the audio track is muted; otherwise, it is not.
