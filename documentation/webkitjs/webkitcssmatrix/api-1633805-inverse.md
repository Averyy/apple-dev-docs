# inverse

**Framework**: WebKit JS  
**Kind**: instm

Returns the inverse of this matrix.

**Availability**:
- Safari Desktop 4.0+
- Safari Mobile 2.0+

## Declaration

```swift
WebKitCSSMatrix inverse();
```

#### Return_value

A new matrix that is the inverse of this matrix.

#### Discussion

This matrix is not modified by this method.
