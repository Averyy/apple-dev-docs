# CSSRule Additions Reference

**Framework**: WebKit JS

These extensions to the `CSSRule` class are used to access animation keyframe and keyframes rules.

## Topics

### Constants
- [WEBKIT_KEYFRAMES_RULE](cssrule/1632070-webkit_keyframes_rule.md)
  Keyframes rule type. Read [`WebKitCSSKeyframesRule`](webkitcsskeyframesrule.md) for details.
- [WEBKIT_KEYFRAME_RULE](cssrule/1633440-webkit_keyframe_rule.md)
  Keyframe rule type. Read [`WebKitCSSKeyframesRule`](webkitcsskeyframesrule.md) for details.
