# WEBKIT_KEYFRAMES_RULE

**Framework**: WebKit JS  
**Kind**: clconst

Keyframes rule type. Read [`WebKitCSSKeyframesRule`](webkitcsskeyframesrule.md) for details.

**Availability**:
- Safari Desktop 4.0+
- Safari Mobile 2.0+

## Declaration

```swift
const unsigned short WEBKIT_KEYFRAMES_RULE;
```

#### Discussion

```javascript
const unsigned short WEBKIT_KEYFRAMES_RULE = 7;
```

> **Note**: In Safari 4.0 and later, the value is `8`.
