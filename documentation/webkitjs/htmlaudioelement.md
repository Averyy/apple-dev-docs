# HTMLAudioElement

**Framework**: Webkitjs  
**Kind**: cl

A class representing the HTML `audio` element that plays a sound or audio stream in a webpage. All of the methods and properties in this class are inherited from [`HTMLMediaElement`](htmlmediaelement.md).

**Availability**:
- Safari Desktop 10.0+
- Safari Mobile 3.0+
