# force

**Framework**: WebKit JS  
**Kind**: instp

The force of the touch, using a positive linear scale where a value of `0.0` indicates the absence of force.

**Availability**:
- Safari Desktop 10.1+
- Safari Mobile 9.0+

## Declaration

```swift
readonly attribute double force;
```
