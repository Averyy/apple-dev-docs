# init(intensity:)

**Framework**: Create ML Components  
**Kind**: init

**Availability**:

- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+

## Declaration

```swift

```

## Parameters

- `intensity`: The intensity of the random noise to add to the image.
