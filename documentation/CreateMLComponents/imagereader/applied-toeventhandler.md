# applied(to:eventHandler:)

**Framework**: Create ML Components  
**Kind**: method

Reads an image URL as a `CIImage`.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+

## Declaration

```swift
func applied(to url: URL, eventHandler: EventHandler? = nil) throws -> CIImage
```

#### Return Value

An image.

## Parameters

- `url`: A image URL.
- `eventHandler`: An event handler.
