# init(radius:)

**Framework**: Create ML Components  
**Kind**: init

Creates a transformer that blurs an image.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+

## Declaration

```swift
init(radius: Double)
```

## Parameters

- `radius`: The radius determines how many pixels are used to create the blur. The larger the radius,   the blurrier the result.
