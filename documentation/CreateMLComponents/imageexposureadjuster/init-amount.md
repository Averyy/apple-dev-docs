# init(amount:)

**Framework**: Create ML Components  
**Kind**: init

Creates an image exposure adjusting transformer.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+

## Declaration

```swift
init(amount: Double)
```

## Parameters

- `amount`: The amount to adjust the exposure of the image. The larger the value, the brighter the exposure.
