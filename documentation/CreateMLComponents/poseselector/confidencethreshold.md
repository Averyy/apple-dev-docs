# confidenceThreshold

**Framework**: Create ML Components  
**Kind**: property

A threshold confidence between 0 to 1 for the joints to be considered valid in pose selection. The default value is 0.2.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+
