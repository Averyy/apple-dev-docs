# GLKMathRadiansToDegrees(_:)

**Framework**: GLKit  
**Kind**: func

Converts an angle measured in radians to degrees.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- macOS 10.8+
- tvOS 9.0+

## Declaration

```swift
func GLKMathRadiansToDegrees(_ radians: Float) -> Float
```

#### Return Value

The converted angle in degrees.

## Parameters

- `radians`: An angle in radians.
