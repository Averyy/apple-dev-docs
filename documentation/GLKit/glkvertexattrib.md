# GLKVertexAttrib

**Framework**: GLKit  
**Kind**: enum

Values used as indices in OpenGL code to associate vertex data with an attribute in a named shader effect.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- macOS 10.8+
- tvOS 9.0+

## Declaration

enum GLKVertexAttrib
