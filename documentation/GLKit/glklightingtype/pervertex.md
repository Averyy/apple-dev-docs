# GLKLightingType.perVertex

**Framework**: GLKit  
**Kind**: case

Indicates that the lighting calculations are performed at each vertex in a triangle and then interpolated across the triangle.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- macOS 10.8+
- tvOS 9.0+

```swift
case perVertex
```
