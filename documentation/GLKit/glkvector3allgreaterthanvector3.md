# GLKVector3AllGreaterThanVector3(_:_:)

**Framework**: GLKit  
**Kind**: func

Returns a Boolean value that indicates whether each component of the first vector is greater than the corresponding component of a second vector.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- macOS 10.8+
- tvOS 9.0+

## Declaration

```swift
func GLKVector3AllGreaterThanVector3(_ vectorLeft: GLKVector3, _ vectorRight: GLKVector3) -> Bool
```

#### Return Value

[`true`](https://developer.apple.com/documentation/swift/true) if each component in the first vector is greater than the corresponding component of the second vector, [`false`](https://developer.apple.com/documentation/swift/false) otherwise.

## Parameters

- `vectorLeft`: The first vector.
- `vectorRight`: The second vector.
