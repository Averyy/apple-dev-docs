# NSStringFromGLKQuaternion(_:)

**Framework**: GLKit  
**Kind**: func

Returns a string that represents the contents of a quaternion.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- macOS 10.8+
- tvOS 9.0+

## Declaration

```swift
func NSStringFromGLKQuaternion(_ quaternion: GLKQuaternion) -> String
```

## Parameters

- `quaternion`: A quaternion.
