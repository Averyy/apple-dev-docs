# spotDirection

**Framework**: GLKit  
**Kind**: property

A vector indicating the direction the spotlight is projecting.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- macOS 10.8+
- tvOS 9.0+

## Declaration

```swift
var spotDirection: GLKVector3 { get set }
```

#### Discussion

The default direction is `[0.0, 0.0, -1.0]`.
