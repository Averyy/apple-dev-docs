# center

**Framework**: GLKit  
**Kind**: property

The center of the skybox.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- macOS 10.8+
- tvOS 9.0+

## Declaration

```swift
var center: GLKVector3 { get set }
```

#### Discussion

The property holds the `[x,y,z]` coordinates of the center of the skybox.
