# NSStringFromGLKVector2(_:)

**Framework**: GLKit  
**Kind**: func

Returns a string that represents the contents of a vector.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- macOS 10.8+
- tvOS 9.0+

## Declaration

```swift
func NSStringFromGLKVector2(_ vector: GLKVector2) -> String
```

## Parameters

- `vector`: A 2-component vector.
