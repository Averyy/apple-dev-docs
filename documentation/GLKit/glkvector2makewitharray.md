# GLKVector2MakeWithArray(_:)

**Framework**: GLKit  
**Kind**: func

Returns a new two-component vector created from an array of components.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- macOS 10.8+
- tvOS 9.0+

## Declaration

```swift
func GLKVector2MakeWithArray(_ values: UnsafeMutablePointer<Float>!) -> GLKVector2
```

#### Return Value

The array

#### Discussion

An initialized vector.
