# GLKVector3Distance(_:_:)

**Framework**: GLKit  
**Kind**: func

Returns the distance between two points.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- macOS 10.8+
- tvOS 9.0+

## Declaration

```swift
func GLKVector3Distance(_ vectorStart: GLKVector3, _ vectorEnd: GLKVector3) -> Float
```

#### Return Value

The distance between the two points.

## Parameters

- `vectorStart`: A vector providing the coordinates of the starting point.
- `vectorEnd`: A vector providing the coordinates of the ending point.
