# GLKVector2Make(_:_:)

**Framework**: GLKit  
**Kind**: func

Returns a new two-component vector created from individual component values.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- macOS 10.8+
- tvOS 9.0+

## Declaration

```swift
func GLKVector2Make(_ x: Float, _ y: Float) -> GLKVector2
```

#### Return Value

An initialized vector.

## Parameters

- `x`: The first component.
- `y`: The second component.
