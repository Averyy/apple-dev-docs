# GLKTextureInfoAlphaState.none

**Framework**: GLKit  
**Kind**: case

Indicates that the texture has no alpha information.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- macOS 10.8+
- tvOS 9.0+
