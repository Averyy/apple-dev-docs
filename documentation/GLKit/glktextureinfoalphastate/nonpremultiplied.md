# GLKTextureInfoAlphaState.nonPremultiplied

**Framework**: GLKit  
**Kind**: case

Indicates that the color values in the texture were not premultiplied by the alpha value.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- macOS 10.8+
- tvOS 9.0+
