# GLKMathDegreesToRadians(_:)

**Framework**: GLKit  
**Kind**: func

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- macOS 10.8+
- tvOS 9.0+

## Declaration

```swift
func GLKMathDegreesToRadians(_ degrees: Float) -> Float
```

#### Return Value
