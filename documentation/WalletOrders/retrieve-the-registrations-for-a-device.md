# Retrieve the registrations for a device

**Framework**: Walletorders  
**Kind**: httpRequest

Retrieves the identifiers of the orders that the device registered for.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- macOS 13.0+
