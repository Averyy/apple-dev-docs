# init(url:)

**Framework**: SpriteKit  
**Kind**: init

Initializes a video node using a URL.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 1.0+

## Declaration

```swift
init(url: URL)
```

#### Return Value

An initialized video node.

## Parameters

- `url`: The URL for the video to play.
