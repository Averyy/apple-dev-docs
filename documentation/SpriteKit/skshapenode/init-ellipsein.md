# init(ellipseIn:)

**Framework**: SpriteKit  
**Kind**: init

Creates a shape node with an elliptical path that fills the specified rectangle.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 1.0+

## Declaration

```swift
convenience init(ellipseIn rect: CGRect)
```

#### Return Value

A new shape node.

## Parameters

- `rect`: A rectangle, relative to the node’s origin.
