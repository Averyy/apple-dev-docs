# init(rectOf:)

**Framework**: SpriteKit  
**Kind**: init

Creates a shape node with a rectangular path centered on the node’s origin.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 1.0+

## Declaration

```swift
convenience init(rectOf size: CGSize)
```

#### Return Value

A new shape node.

## Parameters

- `size`: The size of the rectangle.
