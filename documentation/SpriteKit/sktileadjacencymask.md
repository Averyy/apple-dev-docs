# SKTileAdjacencyMask

**Framework**: SpriteKit  
**Kind**: struct

An enumeration defining how neighboring tiles are automatically placed next to each other.

- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- macOS 10.12+
