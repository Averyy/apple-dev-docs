# SKNodeFocusBehavior.focusable

**Framework**: SpriteKit  
**Kind**: case

Node is focusable and prevents nodes that it visually obscures from becoming focusable.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
