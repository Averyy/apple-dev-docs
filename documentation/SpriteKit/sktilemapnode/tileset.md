# tileSet

**Framework**: SpriteKit  
**Kind**: property

The tile set being used by this tile map. The tile map object can only display tiles that exist in this set.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- macOS 10.12+
- tvOS 10.0+
- visionOS 1.0+
- watchOS 3.0+
