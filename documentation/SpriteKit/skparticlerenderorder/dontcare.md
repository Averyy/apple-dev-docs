# SKParticleRenderOrder.dontCare

**Framework**: SpriteKit  
**Kind**: case

The particles can be rendered in any order. SpriteKit may choose to reorder the particles to improve rendering performance.

- iPadOS 9.0+
- Mac Catalyst 13.1+
