# init(texture:normalTexture:size:)

**Framework**: SpriteKit  
**Kind**: init

Initializes a new tile definition with a single texture and separate normal texture for simulating 3D lighting.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- macOS 10.12+
- tvOS 10.0+
- visionOS 1.0+
