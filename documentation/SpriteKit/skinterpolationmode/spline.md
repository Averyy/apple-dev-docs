#### Discussion

Given a sequence with the key values of `[5, 1, 10, 0]` and times of `[0, 0.25, 0.5, 1]`, spline interpolation produces a set of values that follow a smooth curve.

![Playground graph generated by SKKeyframeSequence using spline interpolation](https://docs-assets.developer.apple.com/published/274a44f6e27d19b5886852aa6988bec1/media-2852038%402x.png)
