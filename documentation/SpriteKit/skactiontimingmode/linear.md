# SKActionTimingMode.linear

**Framework**: SpriteKit  
**Kind**: case

Specifies linear pacing. Linear pacing causes an animation to occur evenly over its duration.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- macOS 10.9+
- tvOS ?+
- visionOS 1.0+
- watchOS 2.0+
