# floatValue

**Framework**: SpriteKit  
**Kind**: property

The receiver’s floating point value.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
var floatValue: Float { get set }
```

#### Discussion

If the receiver’s original value is a vector, [`floatValue`](skattributevalue/floatvalue.md) is the first element of the vector.
