# init(vectorFloat2:)

**Framework**: SpriteKit  
**Kind**: init

Creates and initializes a new attribute value object that holds a vector of two floating point numbers.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
convenience init(vectorFloat2 value: vector_float2)
```

#### Return Value

A new attribute value object.

## Parameters

- `value`: The vector of two point values for the new attribute value.
