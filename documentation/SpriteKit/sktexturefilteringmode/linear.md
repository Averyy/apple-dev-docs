# SKTextureFilteringMode.linear

**Framework**: SpriteKit  
**Kind**: case

Each pixel is drawn by using a linear filter of multiple texels in the texture. This mode produces higher quality results but may be slower.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- macOS 10.9+
- tvOS ?+
- visionOS 1.0+
- watchOS 2.0+
