# SKTextureFilteringMode.nearest

**Framework**: SpriteKit  
**Kind**: case

Each pixel is drawn using the nearest point in the texture. This mode is faster, but the results are often pixelated.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- macOS 10.9+
- tvOS ?+
