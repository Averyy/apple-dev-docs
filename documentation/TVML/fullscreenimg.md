# fullscreenImg

**Framework**: TVML

Displays an image over an entire element.

#### Overview

The `fullscreenImg` element contains a link to an image that is to be displayed over an entire element. Typically this is used to display an image that fills the entire screen. Hereâ€™s an example that shows how to place a full screen image.

```xml
<fullscreenImg src="path to server/image" />
```
