# tumblerItem

**Framework**: TVML

Contains title information for a tumbler header.

#### Overview

The user swipes left and right on the remote to display the previous and next `tumblerItem` elements inside of the tumbler header. Hereâ€™s an example of a tumbler bar containing two tumbler items.

```xml
<tumblerBar>
    <tumblerItem>
        <title>Title 1</title>
        <subtitle>5 Items</subtitle>
    </tumblerItem>
    <tumblerItem>
        <title>Title 2</title>
        <subtitle>12 Items</subtitle>
    </tumblerItem>
</tumblerBar>
```
