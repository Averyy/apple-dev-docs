# decorationImage

**Framework**: TVML

Displays images inside of other elements.

#### Overview

The `decorationImage` element contains a link to an image that is displayed. Hereâ€™s an example that displays a chevron image.

```xml
<decorationImage src="resource://list-chevron" />
```
