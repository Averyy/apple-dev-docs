# tv-placeholder

**Framework**: TVML

Sets a default image for an `img` or `monogram` element.

#### Overview

Here’s an example that displays the default movie image.

```xml
<lockup>
   <img style="tv-placeholder:movie" width="250" height="376" />
   <title>Movie 1</title>
</lockup>
```
