# footer

**Framework**: TVML

Contains information displayed at the bottom of an element.

#### Overview

The `footer` element displays additional information that describes its containing element. Hereâ€™s an example that displays a button at the bottom of a `formTemplate` element.

```xml
<formTemplate>
   <footer>
      <button>
         <text>Submit</text>
      </button>
   </footer>
</formTemplate>
```

##### Subelements of Footer

- [`text`](text.md)

##### Elements That Use Footer
