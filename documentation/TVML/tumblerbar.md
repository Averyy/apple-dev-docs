# tumblerBar

**Framework**: TVML

Displays a list of `tumblerItem` elements.

#### Overview

The `tumblerBar` element contains a list of `tumblerItem` elements that are displayed horizontally. Use the remote to move between the contained `tumblerItem` elements. Hereâ€™s an example of a tumbler bar containing two tumbler items.

```xml
<tumblerBar>
    <tumblerItem>
        <title>Title 1</title>
        <subtitle>5 Items</subtitle>
    </tumblerItem>
    <tumblerItem>
        <title>Title 2</title>
        <subtitle>12 Items</subtitle>
    </tumblerItem>
</tumblerBar>
```
