# isPrimary

**Framework**: HomeKit  
**Kind**: property

A Boolean value that indicates whether this is the primary home for its home manager.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 8.0+
- tvOS 10.0+
