# HMCharacteristicTypeLabelNamespace

**Framework**: HomeKit  
**Kind**: var

The naming schema used to label the services on an accessory with multiple services of the same type.

**Availability**:
- iOS 10.3+
- iPadOS 10.3+
- Mac Catalyst 10.3+
- tvOS 10.2+
- visionOS 1.0+
- watchOS 3.2+

## Declaration

```swift
let HMCharacteristicTypeLabelNamespace: String
```

#### Discussion

The corresponding value is one of the constants in the [`HMCharacteristicValueLabelNamespace`](hmcharacteristicvaluelabelnamespace.md) enumeration.
