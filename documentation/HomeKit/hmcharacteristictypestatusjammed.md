# HMCharacteristicTypeStatusJammed

**Framework**: HomeKit  
**Kind**: var

An indicator of whether an accessory is jammed.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 9.0+
- tvOS 10.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

