# HMCharacteristicTypeStatusTampered

**Framework**: HomeKit  
**Kind**: var

An indicator of whether an accessory has been tampered with.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 9.0+
- tvOS 10.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
let HMCharacteristicTypeStatusTampered: String
```

#### Discussion

The corresponding value is one of the constants in the [`HMCharacteristicValueTamperedStatus`](hmcharacteristicvaluetamperedstatus.md) enumeration.
