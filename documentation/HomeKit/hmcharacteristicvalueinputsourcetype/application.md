# HMCharacteristicValueInputSourceType.application

**Framework**: HomeKit  
**Kind**: case

A virtual input source type for any preinstalled or downloaded applications on the television.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+
