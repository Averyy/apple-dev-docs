# HMCharacteristicTypeOutputState

**Framework**: HomeKit  
**Kind**: var

The output state of a programmable switch.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 9.0+
- tvOS 10.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
let HMCharacteristicTypeOutputState: String
```

#### Discussion

The value represents the state of a programmable switch. For a binary switch, a value of `1` indicates that the switch is in the `ON` position.
