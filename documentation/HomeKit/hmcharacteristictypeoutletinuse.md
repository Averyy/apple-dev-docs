# HMCharacteristicTypeOutletInUse

**Framework**: HomeKit  
**Kind**: var

The state of an outlet.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 8.0+
- tvOS 10.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
let HMCharacteristicTypeOutletInUse: String
```

#### Discussion

The corresponding value is a Boolean which is set to `true` when the outlet is in use, and `false` otherwise.
