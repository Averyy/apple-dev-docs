# HMCharacteristicTypeBrightness

**Framework**: HomeKit  
**Kind**: var

The brightness of a light.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 8.0+
- tvOS 10.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
let HMCharacteristicTypeBrightness: String
```

#### Discussion

The corresponding value is an integer representing a percentage of the maximum brightness.
