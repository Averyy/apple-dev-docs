# init(source:)

**Framework**: HomeKit  
**Kind**: init

Creates a new camera view using the given source.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- tvOS 14.0+
- visionOS 1.0+
- watchOS 7.0+

## Declaration

```swift
nonisolated
init(source: HMCameraSource)
```

## Parameters

- `source`: The source of a video stream or an image snapshot that a camera view renders.
