# HMCharacteristicTypeHeatingThreshold

**Framework**: HomeKit  
**Kind**: var

The temperature below which heating will be active.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 8.0+
- tvOS 10.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
let HMCharacteristicTypeHeatingThreshold: String
```

#### Discussion

The corresponding value is a floating point number in degrees Celsius.
