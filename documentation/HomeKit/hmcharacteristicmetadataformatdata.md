# HMCharacteristicMetadataFormatData

**Framework**: HomeKit  
**Kind**: var

Indicates that the characteristic has data blob values.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 8.0+
- tvOS 10.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
let HMCharacteristicMetadataFormatData: String
```

#### Discussion

The value is an [`NSData`](https://developer.apple.com/documentation/Foundation/NSData) object containing bytes of data.
