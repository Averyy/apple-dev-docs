# HMCharacteristicTypeLogs

**Framework**: HomeKit  
**Kind**: var

Log data for the accessory.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 8.0+
- tvOS 10.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
let HMCharacteristicTypeLogs: String
```

#### Discussion

The corresponding value is TLV8-encoded data.
