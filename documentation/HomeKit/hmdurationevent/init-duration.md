# init(duration:)

**Framework**: HomeKit  
**Kind**: init

Creates a duration event with the specified time duration.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 11.0+
- tvOS 11.0+
- visionOS 1.0+
- watchOS 4.0+

## Declaration

```swift
init(duration: TimeInterval)
```

#### Return Value

An initialized duration event.

## Parameters

- `duration`: The duration before the event ends, specified in seconds.
