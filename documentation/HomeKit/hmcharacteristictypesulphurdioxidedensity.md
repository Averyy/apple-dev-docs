# HMCharacteristicTypeSulphurDioxideDensity

**Framework**: HomeKit  
**Kind**: var

The measured density of sulphur dioxide.

**Availability**:
- iOS 10.2+
- iPadOS 10.2+
- Mac Catalyst 10.2+
- tvOS 10.1+
- visionOS 1.0+
- watchOS 3.1.1+

## Declaration

```swift
let HMCharacteristicTypeSulphurDioxideDensity: String
```

#### Discussion

The corresponding value is a number in units of micrograms per cubic meter.
