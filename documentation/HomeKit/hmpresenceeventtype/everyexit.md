# HMPresenceEventType.everyExit

**Framework**: HomeKit  
**Kind**: case

Triggers the event every time a user leaves the home.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 11.0+
- tvOS 11.0+
- visionOS 1.0+
- watchOS 4.0+

## Declaration

```swift
case everyExit
```

#### Discussion

This presence type cannot be used as a predicate.
