# presenceEventType

**Framework**: HomeKit  
**Kind**: property

The event type that triggers the presence event.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 11.0+
- tvOS 11.0+
- visionOS 1.0+
- watchOS 4.0+

## Declaration

```swift
var presenceEventType: HMPresenceEventType { get set }
```

#### Discussion

Use this property to change the presence type that triggers this event.
