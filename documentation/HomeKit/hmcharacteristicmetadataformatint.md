# HMCharacteristicMetadataFormatInt

**Framework**: HomeKit  
**Kind**: var

Indicates that the characteristic has `int` values.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 8.0+
- tvOS 10.0+
- visionOS 1.0+
- watchOS 2.0+
