# HMError.Code.nilParameter

**Framework**: HomeKit  
**Kind**: case

An error indicating that `nil` was passed for an operation that does not accept `nil`.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 8.0+
- tvOS 10.0+
- visionOS 1.0+
- watchOS 2.0+
