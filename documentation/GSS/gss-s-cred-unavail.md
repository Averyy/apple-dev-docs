# GSS_S_CRED_UNAVAIL

**Framework**: GSS  
**Kind**: var

Unavailable credential.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- Mac Catalyst 13.0+
- macOS 10.14+
- visionOS 1.0+

## Declaration

```swift
var GSS_S_CRED_UNAVAIL: UInt { get }
```

#### Discussion

This value is currently an alias of the generic [`GSS_S_FAILURE`](gss_s_failure.md).
