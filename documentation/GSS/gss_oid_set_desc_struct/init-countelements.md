# init(count:elements:)

**Framework**: GSS  
**Kind**: init

Initialize a new object identifier set with the given identifiers.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- Mac Catalyst 13.0+
- macOS 10.14+
- visionOS 1.0+

## Declaration

```swift
init(count: Int, elements: gss_OID!)
```

## Parameters

- `count`: The number of object identifiers in the   array.
- `elements`: An array containing the identifiers with which to initialize the new set.
