# kAEUseHTTPProxyAttr

**Framework**: Core Services  
**Kind**: data

A value of type `typeBoolean`. Specifies whether to manually specify the proxy host and port. Defaults to `true`.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.2+
