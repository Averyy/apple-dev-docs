# kUCCollateTypeSourceMask

**Framework**: Core Services  
**Kind**: data

You can use this mask, in conjunction with the `kUCCollateTypeShiftBits` constant, to obtain a value identifying a fixed ordering scheme.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+
