# kSKSearchOptionSpaceMeansOR

**Framework**: Core Services  
**Kind**: data

This option alters query behavior so that spaces are interpreted as Boolean `OR` operators.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.4+
