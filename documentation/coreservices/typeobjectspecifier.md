# typeObjectSpecifier

**Framework**: Core Services  
**Kind**: data

Specifies a descriptor used with the `keyAEContainer` keyword in a keyword-specified descriptor. The key data for the descriptor is an object specifier.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.12+
