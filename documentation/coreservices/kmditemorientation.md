# kMDItemOrientation

**Framework**: Core Services  
**Kind**: data

The orientation of the document contents. Possible values are 0 (landscape) and 1 (portrait). A CFNumber.

**Availability**:
- macOS 10.4+

## Declaration

```swift
let kMDItemOrientation: CFString!
```
