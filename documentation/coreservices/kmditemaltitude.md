# kMDItemAltitude

**Framework**: Core Services  
**Kind**: data

 The altitude of the item in meters above sea level, expressed using the WGS84 datum.  Negative values lie below sea level. A CFString.

**Availability**:
- macOS 10.5+

let kMDItemAltitude: CFString!
