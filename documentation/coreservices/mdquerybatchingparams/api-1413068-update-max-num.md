# update_max_num

**Framework**: Core Services  
**Kind**: structp

The maximum number of results that can accumulatebefore an update notification is sent. This value is used only duringthe live-update phase of a query.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.4+
