# typeNull

**Framework**: Core Services  
**Kind**: data

A null data storage pointer. When resolving an object specifier, an object with a null storage pointer specifies the default container at the top of the container hierarchy.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+
