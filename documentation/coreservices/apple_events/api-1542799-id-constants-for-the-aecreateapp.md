# ID Constants for the AECreateAppleEvent Function

**Framework**: Core Services

Specify values for the ID parameters of the `AECreateAppleEvent` function.

#### Overview

You use these constants with the [`AECreateAppleEvent(_:_:_:_:_:_:)`](1448525-aecreateappleevent.md) function. 

## Topics

### Constants
- [var kAutoGenerateReturnID: Int](kautogeneratereturnid.md)
  If you pass this value for the `returnID` parameter of the [`AECreateAppleEvent(_:_:_:_:_:_:)`](1448525-aecreateappleevent.md) function, the Apple Event Manager assigns to the created Apple event a return ID that is unique to the current session.
- [var kAnyTransactionID: Int](kanytransactionid.md)
