# kLSUnknownType

**Framework**: Core Services  
**Kind**: data

The value to supply as the file type (for example,to the `LSGetApplicationForInfo` function)if no file type information is available.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+
