# OSLCompareUPP

**Framework**: Core Services  
**Kind**: tdef

Defines a data type for the universal procedure pointer for the `OSLCompareProcPtr` callback function pointer.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+

## Declaration

