# kCoreEventClass

**Framework**: Core Services  
**Kind**: data

An Apple event sent by the Mac OS; applications that present a graphical interface to the user should be able to any events sent by the Mac OS that apply to the application.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+
