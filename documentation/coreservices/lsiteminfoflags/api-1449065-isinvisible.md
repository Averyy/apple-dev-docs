# isInvisible

**Framework**: Core Services  
**Kind**: structdata

Item is invisible, because either its name begins with a period or its `isInvisible` Finder flag is set.

**Availability**:
- Mac Catalyst 13.0+ - Deprecated in 13.1
- macOS 10.0+ - Deprecated in 10.11
