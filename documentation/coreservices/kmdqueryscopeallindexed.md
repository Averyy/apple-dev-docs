# kMDQueryScopeAllIndexed

**Framework**: Core Services  
**Kind**: data

Specifies that the search should be restricted to indexed, locally mounted volumes and indexed user mounted remote volumes, plus the user's home directory.

**Availability**:
- macOS 10.6+
