# defaults

**Framework**: Core Services  
**Kind**: structdata

Requests launching in the default manner (as if the only flags set were `kLSLaunchNoParams`, `kLSLaunchAsync`, and `kLSLaunchStartClassic`).

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+
