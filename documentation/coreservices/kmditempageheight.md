# kMDItemPageHeight

**Framework**: Core Services  
**Kind**: data

Height of the document page, in points (72 points per inch). For PDF files this indicates the height of the first page only. A CFNumber.

**Availability**:
- macOS 10.4+

## Declaration

```swift
let kMDItemPageHeight: CFString!
```
