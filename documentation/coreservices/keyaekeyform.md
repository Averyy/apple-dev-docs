# keyAEKeyForm

**Framework**: Core Services  
**Kind**: data

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+

## Declaration

```swift
var keyAEKeyForm: Int { get }
```

#### Discussion

A four-character code that identifies the key form for the specified object or objects.

The constants for specifying the key form are described in [`Key Form and Descriptor Type Object Specifier Constants`](apple_events/1572731-key_form_and_descriptor_type_obj.md).
