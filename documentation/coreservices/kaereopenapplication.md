# kAEReopenApplication

**Framework**: Core Services  
**Kind**: data

Event that reopens an application. Sent, for example, when your application is running and a user clicks your application icon in the Dock.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+
