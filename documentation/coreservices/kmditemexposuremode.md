# kMDItemExposureMode

**Framework**: Core Services  
**Kind**: data

The exposure mode used to acquire the document contents. A CFNumber.

**Availability**:
- macOS 10.4+

## Declaration

```swift
let kMDItemExposureMode: CFString!
```

#### Discussion

Possible values are 0 (auto exposure), 1 (manual exposure) and 2 (auto bracket).
