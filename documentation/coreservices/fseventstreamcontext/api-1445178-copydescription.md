# copyDescription

**Framework**: Core Services  
**Kind**: structp

The callback used to create a descriptive string representation of the info pointer (or the data pointed to by the info pointer) for debugging purposes. This can be NULL.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.5+
