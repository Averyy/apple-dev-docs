# kMDItemCity

**Framework**: Core Services  
**Kind**: data

Identifies city of origin according to guidelines established by the provider. A CFString.

**Availability**:
- macOS 10.4+

## Declaration

```swift
let kMDItemCity: CFString!
```

#### Discussion

For example, "New York", "Cupertino", or "Toronto".
