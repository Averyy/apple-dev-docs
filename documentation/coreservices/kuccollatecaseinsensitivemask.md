# kUCCollateCaseInsensitiveMask

**Framework**: Core Services  
**Kind**: data

If the corresponding bit is set, then uppercase and titlecase characters are treated as equivalent to the corresponding lowercase characters.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+
