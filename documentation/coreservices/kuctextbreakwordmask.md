# kUCTextBreakWordMask

**Framework**: Core Services  
**Kind**: data

If the bit specified by this mask is set, boundaries of words may be located. This can be used to determine what to highlight as the result of a double-click.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+
