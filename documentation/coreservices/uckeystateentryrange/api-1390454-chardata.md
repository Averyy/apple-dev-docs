# charData

**Framework**: Core Services  
**Kind**: structp

A value of type `UCKeyCharSeq`. This base character value is used to determine the actual Unicode character(s) produced when a given dead-key state terminates. 

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+
