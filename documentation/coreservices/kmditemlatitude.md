# kMDItemLatitude

**Framework**: Core Services  
**Kind**: data

 The latitude of the item in degrees north of the equator, expressed using the WGS84 datum.  Negative values lie south of the equator. A CFString.

**Availability**:
- macOS 10.5+

```swift
let kMDItemLatitude: CFString!
```
