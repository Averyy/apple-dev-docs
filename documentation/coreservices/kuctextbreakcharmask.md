# kUCTextBreakCharMask

**Framework**: Core Services  
**Kind**: data

If the bit specified by this mask is set, boundaries of characters may be located (with surrogate pairs treated as a single character).

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+
