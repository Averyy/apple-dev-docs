# keyLayoutDataVersion

**Framework**: Core Services  
**Kind**: structp

An unsigned 16-bit integer identifying the version of the data in the resource, in binary code decimal format. For example, 0x0100 would equal version 1.0.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+
