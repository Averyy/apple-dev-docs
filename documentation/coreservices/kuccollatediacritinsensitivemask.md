# kUCCollateDiacritInsensitiveMask

**Framework**: Core Services  
**Kind**: data

If the corresponding bit is set, then characters with diacritics are treated as equivalent to the corresponding characters without diacritics.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+
