# AEArrayType

**Framework**: Core Services  
**Kind**: tdef

Stores a value that specifies an array type.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+

## Declaration

```swift
typealias AEArrayType = Int8
```

#### Discussion

You use this data type with the [`AEGetArray(_:_:_:_:_:_:_:)`](1445720-aegetarray.md) function and the [`AEPutArray(_:_:_:_:_:_:)`](1442535-aeputarray.md) function to specify an array type, using one of the constants from [`Data Array Constants`](apple_events/1542848-data_array_constants.md).
