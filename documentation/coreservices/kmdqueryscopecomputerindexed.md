# kMDQueryScopeComputerIndexed

**Framework**: Core Services  
**Kind**: data

Specifies that the search should be restricted to indexed, locally mounted volumes, plus the user's home directory (which may be on a remote volume).

**Availability**:
- macOS 10.6+
