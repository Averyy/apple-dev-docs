# CLSProgressReportingCapability.Kind.score

**Framework**: ClassKit  
**Kind**: case

A score.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- visionOS 1.0+

## Declaration

```swift
case score
```

#### Discussion

Add this capability to indicate that a context reports a score at the end of a task using a [`CLSScoreItem`](clsscoreitem.md) instance.
