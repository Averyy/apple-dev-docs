# default()

**Framework**: Safari Services  
**Kind**: method

Returns the Safari Reading List singleton object.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 14.0+
- visionOS 1.0+

## Declaration

```swift
class func `default`() -> SSReadingList?
```

#### Return Value

The shared instance of the Reading List, or `nil` if access to the Reading List is not permitted.
