# webpageURL

**Framework**: Safari Services  
**Kind**: property

The URL specified when initializing the receiver.

**Availability**:
- macOS 10.15+

## Declaration

```swift
var webpageURL: URL { get }
```

#### Discussion

This property returns the URL passed when initializing the receiver. This property is read-only.
