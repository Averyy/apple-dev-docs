# init(manifest:)

**Framework**: Safari Services  
**Kind**: init

Initializes a Home Screen information object with the supplied web app manifest.

**Availability**:
- iOS 18.2+
- iPadOS 18.2+
- visionOS 2.2+

## Declaration

```swift
init(manifest: BEWebAppManifest)
```

## Parameters

- `manifest`: The web app manifest for the bookmark.
