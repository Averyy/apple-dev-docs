# SFExtensionProfileKey

**Framework**: Safari Services  
**Kind**: var

A string the system uses as a key in a user info dictionary to identify a profile identifier.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.1+
- macOS 14.0+
- visionOS 1.0+

## Declaration

```swift
let SFExtensionProfileKey: String
```
