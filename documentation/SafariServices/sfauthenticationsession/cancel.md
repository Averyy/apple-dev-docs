# cancel()

**Framework**: Safari Services  
**Kind**: method

Cancels the currently running session when the consent dialog is up or when the webpage is up. This is how applications stop the login process.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
