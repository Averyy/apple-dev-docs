# deviceModel

**Framework**: Core Media I/O  
**Kind**: property

A property key for the device model.

**Availability**:
- Mac Catalyst 15.4+
- macOS 12.3+

## Declaration

```swift
static let deviceModel: CMIOExtensionProperty
```

#### Discussion

The property state for this property is a string with a read-only attribute.
