# kCTErrorDomainMach

**Framework**: Core Telephony  
**Kind**: var

The domain value for errors reported by Mach.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 14.0+
- macOS 10.10+

## Declaration

```swift
var kCTErrorDomainMach: Int { get }
```

#### Discussion

Mach error codes are listed in `mach/error.h`.
