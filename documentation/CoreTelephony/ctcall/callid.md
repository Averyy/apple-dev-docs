# callID

**Framework**: Core Telephony  
**Kind**: property

A unique identifier for the cellular call.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.1+

## Declaration

```swift
var callID: String { get }
```

#### Discussion

Use this value to differentiate multiple active cellular calls.
