# kCTErrorDomainPOSIX

**Framework**: Core Telephony  
**Kind**: var

The domain value for errors from POSIX.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 14.0+
- macOS 10.10+

## Declaration

```swift
var kCTErrorDomainPOSIX: Int { get }
```

#### Discussion

POSIX error codes are listed in `sys/errno.h`.
