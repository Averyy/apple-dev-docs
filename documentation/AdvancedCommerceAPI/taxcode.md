# taxCode

**Framework**: Advanced Commerce API  
**Kind**: typealias

A tax code that applies to a SKU.

**Availability**:
- Advanced Commerce API 1.0+

## Declaration

```swift
string taxCode
```

#### Discussion

You select a tax code for every SKU. For more information, see [`Choosing tax codes for your SKUs`](taxcodes.md).
