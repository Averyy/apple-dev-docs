# init(scale:)

**Framework**: ML Compute  
**Kind**: init

Creates a gram matrix layer with the scaling factor you specify.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+

## Declaration

```swift
convenience init(scale: Float)
```

## Parameters

- `scale`: The scaling factor you want to use for the output.
