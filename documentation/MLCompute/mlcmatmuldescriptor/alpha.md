# alpha

**Framework**: ML Compute  
**Kind**: property

A scalar value you specify to scale the result in C = alpha x A x B.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+

## Declaration

```swift
var alpha: Float { get }
```

#### Discussion

The default value is `1.0`.
