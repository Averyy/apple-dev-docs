# transposesX

**Framework**: ML Compute  
**Kind**: property

A Boolean that specifies whether you choose to transpose the last two dimensions of x.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+

## Declaration

```swift
var transposesX: Bool { get }
```

#### Discussion

The default value is `false`.
