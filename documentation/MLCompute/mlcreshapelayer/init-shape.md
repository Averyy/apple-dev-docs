# init(shape:)

**Framework**: ML Compute  
**Kind**: init

Creates a reshape layer with the shape you specify.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+

## Declaration

```swift
convenience init?(shape: [Int])
```

## Parameters

- `shape`: An array that contains the size of each dimension.
