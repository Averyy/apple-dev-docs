# init(dimension:)

**Framework**: ML Compute  
**Kind**: init

Creates a concatenation layer with the dimension you specify.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+

## Declaration

```swift
convenience init(dimension: Int)
```

## Parameters

- `dimension`: The dimension, or axis, along which you concatenate tensors.

## See Also
