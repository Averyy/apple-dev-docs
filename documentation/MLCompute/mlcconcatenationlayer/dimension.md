# dimension

**Framework**: ML Compute  
**Kind**: property

The dimension, or axis, along which you concatenate tensors.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+

## Declaration

```swift
var dimension: Int { get }
```

#### Discussion

The default value is `1`, which typically represents features channels.
