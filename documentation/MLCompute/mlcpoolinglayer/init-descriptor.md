# init(descriptor:)

**Framework**: ML Compute  
**Kind**: init

Creates a pooling layer with the descriptor you specify.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+

## Declaration

```swift
convenience init(descriptor: MLCPoolingDescriptor)
```

## Parameters

- `descriptor`: An object you use to configure the pooling layer.
