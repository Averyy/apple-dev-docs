# init(descriptor:)

**Framework**: ML Compute  
**Kind**: init

Creates an activation layer with the descriptor you specify.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+

## Declaration

```swift
convenience init(descriptor: MLCActivationDescriptor)
```

## Parameters

- `descriptor`: An object you use to configure the activation layer.
