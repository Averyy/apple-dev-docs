# weight

**Framework**: ML Compute  
**Kind**: property

The scale factor you apply to each element of a result.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+

## Declaration

```swift
var weight: Float { get }
```

#### Discussion

The default value is `1.0`.
