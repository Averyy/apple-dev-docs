# init(start:end:stride:)

**Framework**: ML Compute  
**Kind**: init

Creates a slice layer with the start, end, and stride you specify.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+

## Declaration

```swift
convenience init?(start: [Int], end: [Int], stride: [Int]?)
```

## Parameters

- `start`: The start vector.
- `end`: The end vector.
- `stride`: The stride vector.
