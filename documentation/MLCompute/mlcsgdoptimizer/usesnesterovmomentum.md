# usesNesterovMomentum

**Framework**: ML Compute  
**Kind**: property

A Boolean that indicates whether you enable Nesterov momentum.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+

## Declaration

```swift
var usesNesterovMomentum: Bool { get }
```

#### Discussion

The default value is `false`.
