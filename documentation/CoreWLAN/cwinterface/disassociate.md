# disassociate()

**Framework**: Core WLAN  
**Kind**: method

Disassociates from the current network.

**Availability**:
- macOS 10.6+

## Declaration

```swift
func disassociate()
```

#### Discussion

This method has no effect if the interface is not associated to a network. This operation may require an administrator password.
