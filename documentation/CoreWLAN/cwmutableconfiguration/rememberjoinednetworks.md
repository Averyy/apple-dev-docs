# rememberJoinedNetworks

**Framework**: Core WLAN  
**Kind**: property

A Boolean value that determines whether to remember all joined Wi-Fi networks unless the user specifies otherwise when joining a particular Wi-Fi network.

**Availability**:
- macOS 10.7+
