# ssidData

**Framework**: Core WLAN  
**Kind**: property

The service set identifier (SSID) for the network profile, returned as data.

**Availability**:
- macOS 10.7+

## Declaration

```swift
var ssidData: Data? { get }
```

#### Discussion

The SSID is 1-32 octets.
