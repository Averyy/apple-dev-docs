# init()

**Framework**: Core WLAN  
**Kind**: init

Initializes a Wi-Fi client object.

**Availability**:
- macOS 10.10+

## Declaration

```swift
init()
```

#### Discussion

Instead of initializing a Wi-Fi client directly, consider using the shared instance available from the [`shared()`](cwwificlient/shared().md) class method.
