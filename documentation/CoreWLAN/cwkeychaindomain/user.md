# CWKeychainDomain.user

**Framework**: Core WLAN  
**Kind**: case

The user keychain domain.

**Availability**:
- macOS 10.10+

## Declaration

```swift
case user
```

#### Discussion

If iCloud Keychain is enabled, the iCloud keychain is the user keychain.
