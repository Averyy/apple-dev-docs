# text

**Framework**: ClockKit  
**Kind**: property

The long version of text that you want to display.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
var text: String { get set }
```

#### Discussion

The long version of the text is the preferred version and is used whenever possible.
