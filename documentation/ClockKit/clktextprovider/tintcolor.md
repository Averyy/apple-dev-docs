# tintColor

**Framework**: ClockKit  
**Kind**: property

The tint color to use for text.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
var tintColor: UIColor { get set }
```

#### Discussion

On clock faces that support custom colors, this color is applied to the text in your text provider.
