# imageProvider

**Framework**: ClockKit  
**Kind**: property

The image to display.

**Availability**:
- watchOS 5.0+

## Declaration

```swift
@NSCopying
var imageProvider: CLKFullColorImageProvider { get set }
```

#### Discussion

This image provider produces a full-color image, masked to a circle.
