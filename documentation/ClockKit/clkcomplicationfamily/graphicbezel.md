# CLKComplicationFamily.graphicBezel

**Framework**: ClockKit  
**Kind**: case

A circular area with optional curved text that ClockKit displays along the bezel of the Infograph watch face.

**Availability**:
- watchOS 5.0+

## Declaration

```swift
case graphicBezel
```
