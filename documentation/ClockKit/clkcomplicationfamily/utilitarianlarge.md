# CLKComplicationFamily.utilitarianLarge

**Framework**: ClockKit  
**Kind**: case

A large rectangular area that spans the width of the screen in the Utility and Mickey watch faces.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
case utilitarianLarge
```
