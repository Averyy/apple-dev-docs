# CLKComplicationFamily.graphicCircular

**Framework**: ClockKit  
**Kind**: case

A circular area that ClockKit displays on the Infograph and Infograph Modular watch faces.

**Availability**:
- watchOS 5.0+

## Declaration

```swift
case graphicCircular
```
