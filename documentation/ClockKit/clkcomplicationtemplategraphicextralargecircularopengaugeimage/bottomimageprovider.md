# bottomImageProvider

**Framework**: ClockKit  
**Kind**: property

The image to display at the bottom of the gauge.

**Availability**:
- watchOS 7.0+

## Declaration

```swift
@NSCopying
var bottomImageProvider: CLKFullColorImageProvider { get set }
```

#### Discussion

This image provider produces a full-color image, masked to a circle.
