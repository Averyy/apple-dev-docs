# CLKGaugeProviderStyle.fill

**Framework**: ClockKit  
**Kind**: case

A gauge that fills in as the value increases.

**Availability**:
- watchOS 5.0+

## Declaration

```swift
case fill
```

#### Discussion

The system draws a gauge with the color bar filled in up to the selected value. The rest of the gauge is shown as empty.
