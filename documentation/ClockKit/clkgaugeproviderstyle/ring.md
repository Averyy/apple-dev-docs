# CLKGaugeProviderStyle.ring

**Framework**: ClockKit  
**Kind**: case

A gauge that indicates a value with a sliding ring.

**Availability**:
- watchOS 5.0+

## Declaration

```swift
case ring
```

#### Discussion

The system draws the entire gauge and positions a ring along the gauge to indicate the fill value.
