# accessibilityLabel

**Framework**: ClockKit  
**Kind**: property

A succinct label that succinctly identifies the purpose of the image.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
var accessibilityLabel: String? { get set }
```

#### Discussion

An accessibility label should be a very short, localized string that identifies the purpose of the image presented by the image provider.
