# fillFraction

**Framework**: ClockKit  
**Kind**: property

The value displayed by the gauge.

**Availability**:
- watchOS 5.0+

## Declaration

```swift
var fillFraction: Float { get }
```

#### Discussion

Contains a value between `0.0` and `1.0`, or `CLKSimpleGaugeProviderFillFractionEmpty` for an empty gauge.
