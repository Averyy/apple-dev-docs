# CLKRelativeDateStyle.naturalAbbreviated

**Framework**: ClockKit  
**Kind**: case

An abbreviated, natural date style.

**Availability**:
- watchOS 5.0+

## Declaration

```swift
case naturalAbbreviated
```

#### Discussion

The smallest possible abbreviation of the natural date style. For example, ten hours and nine minutes always formats as “10H 9M.” The [`CLKRelativeDateStyle.natural`](clkrelativedatestyle/natural.md) style may provide a longer string like “10hrs 9min” if there’s enough space.
