# column2Alignment

**Framework**: ClockKit  
**Kind**: property

The alignment of the text in the second column.

**Availability**:
- watchOS 3.0+

## Declaration

```swift
var column2Alignment: CLKComplicationColumnAlignment { get set }
```

#### Discussion

This alignment value applies to the text in column two of all rows.
