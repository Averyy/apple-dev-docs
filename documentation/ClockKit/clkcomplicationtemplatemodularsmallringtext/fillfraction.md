# fillFraction

**Framework**: ClockKit  
**Kind**: property

The fraction of the ring to fill.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
var fillFraction: Float { get set }
```

#### Discussion

Specify a value between `0.0` and `1.0`, where `0.0` represents no ring and `1.0` represents a complete ring.
