# endDate

**Framework**: ClockKit  
**Kind**: property

The ending time and date for the gauge’s time interval.

**Availability**:
- watchOS 5.0+

## Declaration

```swift
var endDate: Date { get }
```

#### Discussion

The end date is equal to or later than the start date.
