# buttonsInBottomBar

**Framework**: StoreKit  
**Kind**: property

A hybrid placement that positions subscription controls within the main scroll view, and places auxiliary buttons in the bottom bar.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- macOS 15.0+
- visionOS 2.0+
