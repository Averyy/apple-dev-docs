# priceLocale

**Framework**: StoreKit  
**Kind**: property

**Availability**:

- iPadOS 11.2+
- Mac Catalyst 13.1+
- macOS 10.13.2+
- tvOS 11.2+
- visionOS 1.0+
- watchOS 6.2+
