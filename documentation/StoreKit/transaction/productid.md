# productID

**Framework**: StoreKit  
**Kind**: property

The product identifier of the in-app purchase.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
let productID: String
```

#### Discussion

You create product identifiers for in-app purchases in App Store Connect.
