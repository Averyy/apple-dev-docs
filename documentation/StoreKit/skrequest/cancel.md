# cancel()

**Framework**: StoreKit  
**Kind**: method

Cancels a previously started request.

**Availability**:
- iOS 3.0+
- iPadOS 3.0+
- Mac Catalyst 13.1+
- macOS 10.7+
- tvOS ?+
- visionOS 1.0+
- watchOS 6.2+

## Declaration

```swift
func cancel()
```

#### Discussion

When you cancel a request, the delegate is not called with an error.
