# id

**Framework**: StoreKit  
**Kind**: property

The in-app binding identifier.

**Availability**:
- iOS 16.4+
- iPadOS 16.4+
- visionOS 1.0+

## Declaration

```swift
let id: String
```

#### Discussion

This identifier is the in-app binding identifier that your server receives from the Apple server when your server initiates payment method binding.
