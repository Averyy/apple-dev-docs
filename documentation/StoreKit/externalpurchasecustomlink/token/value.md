# value

**Framework**: StoreKit  
**Kind**: property

A Base64URL-encoded JSON string that represents the external purchase token.

**Availability**:
- iOS 18.1+
- iPadOS 18.1+
- macOS 15.1+
- tvOS 18.1+
- visionOS 2.1+
- watchOS 11.1+
