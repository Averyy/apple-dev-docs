# purchase()

**Framework**: StoreKit  
**Kind**: method

Initiates a purchase action for the product.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+
- watchOS 10.0+

## Declaration

```swift
func purchase()
```

#### Discussion

Use this method instead of calling [`purchase(options:)`](product/purchase(options:).md)directly on the product.
