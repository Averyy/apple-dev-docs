# !=(_:_:)

**Framework**: FamilyControls  
**Kind**: op

Returns a Boolean value that indicates whether two error values aren’t equal.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS ?+

## Declaration

```swift
static func != (lhs: Self, rhs: Self) -> Bool
```

#### Return Value

A Boolean value that’s set to `true` if the two error values aren’t equal.

## Parameters

- `lhs`: The first error value to compare.
- `rhs`: The second error value to compare.
