# init(includeEntireCategory:)

**Framework**: FamilyControls  
**Kind**: init

Creates a new activity selection instance.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+

## Declaration

```swift
init(includeEntireCategory: Bool)
```

## Parameters

- `includeEntireCategory`: A Boolean value   that indicates whether the selection should include   applications and web domains from the selected categories.
