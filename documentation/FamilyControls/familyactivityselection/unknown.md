# !=(_:_:)

**Framework**: FamilyControls  
**Kind**: op

Returns a Boolean value that indicates whether two activity selections aren’t equal.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+

## Declaration

```swift
static func != (lhs: Self, rhs: Self) -> Bool
```

#### Return Value

A Boolean value that’s set to `true` if the two activity selection aren’t equal.

## Parameters

- `lhs`: The first activity selection to compare.
- `rhs`: The second activity selection to compare.
