# kMEVideoDecoderCodecNameKey

**Framework**: MediaExtension  
**Kind**: var

A user-readable string describing the name of the codec format.

**Availability**:
- Mac Catalyst 18.0+
- macOS 15.0+

## Declaration

```swift
var kMEVideoDecoderCodecNameKey: String { get }
```

#### Discussion

This string might be displayed as format information for the video track in a player application.
