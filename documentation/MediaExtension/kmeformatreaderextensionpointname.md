# kMEFormatReaderExtensionPointName

**Framework**: MediaExtension  
**Kind**: var

A key to the extension point name for format readers.

**Availability**:
- Mac Catalyst 18.0+
- macOS 15.0+

## Declaration

```swift
var kMEFormatReaderExtensionPointName: String { get }
```

#### Discussion

The value for this string is `com.apple.mediaextension.formatreader`.
