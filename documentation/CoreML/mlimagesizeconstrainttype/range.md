# MLImageSizeConstraintType.range

**Framework**: Core ML  
**Kind**: case

The image feature accepts image sizes defined by a range of widths and a range of heights.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.1+
- macOS 10.14+
- tvOS 12.0+
- visionOS 1.0+
- watchOS 5.0+
