# isOptional

**Framework**: Core ML  
**Kind**: property

A Boolean value that indicates whether this feature is optional.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.13+
- tvOS 11.0+
- visionOS 1.0+
- watchOS 4.0+

## Declaration

```swift
var isOptional: Bool { get }
```

#### Discussion

Optional values can be `nil`, for models that have inputs related to features being present or not.
