# init(undefined:)

**Framework**: Core ML  
**Kind**: init

Creates a feature value with a type that represents an undefined or missing value.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.13+
- tvOS 11.0+
- visionOS 1.0+
- watchOS 4.0+

## Declaration

```swift
convenience init(undefined type: MLFeatureType)
```

## Parameters

- `type`: The type of the undefined or missing value.
