# trackingButtonEnabled

**Framework**: DockKit  
**Kind**: property

The state of the tracking button on the dock accessory.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+

## Declaration

```swift
let trackingButtonEnabled: Bool
```

#### Discussion

Every dock accessory has a tracking button that a person presses to turn on or off subject tracking.
