# rect

**Framework**: DockKit  
**Kind**: property

The coordinates of the subject in the frame.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+

## Declaration

```swift
let rect: CGRect
```

#### Discussion

This is a unit rectangle with coordinate values between `0` and `1`.
