# DockKitError.frameRateTooHigh

**Framework**: DockKit  
**Kind**: case

The call rate for the method is too frequent.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+

## Declaration

```swift
case frameRateTooHigh
```

#### Discussion

Reduce the number of times per second you call the method.
