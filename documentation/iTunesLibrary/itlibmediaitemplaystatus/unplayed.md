# ITLibMediaItemPlayStatus.unplayed

**Framework**: iTunes Library  
**Kind**: case

The user hasn’t played the media item.

**Availability**:
- Mac Catalyst 14.0+
- macOS 10.13+

## Declaration

```swift
case unplayed
```

#### Discussion

The user can also set this state manually.
