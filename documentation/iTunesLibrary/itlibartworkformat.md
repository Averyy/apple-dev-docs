# ITLibArtworkFormat

**Framework**: iTunes Library  
**Kind**: enum

These constants specify the possible formats of the data that [`imageDataFormat`](itlibartwork/imagedataformat.md) returns.

**Availability**:
- Mac Catalyst 14.0+
- macOS 10.13+

## Declaration

```swift
enum ITLibArtworkFormat
```
