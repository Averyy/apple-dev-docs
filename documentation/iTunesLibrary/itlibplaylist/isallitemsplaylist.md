# isAllItemsPlaylist

**Framework**: iTunes Library  
**Kind**: property

Indicates whether the API exposes every item in the playlist.

**Availability**:
- Mac Catalyst 14.0+
- macOS 10.13+

## Declaration

```swift
var isAllItemsPlaylist: Bool { get }
```

#### Discussion

This property always returns [`true`](https://developer.apple.com/documentation/Swift/true).
