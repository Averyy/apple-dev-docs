# EnableLostModeResponse

**Framework**: Device Management  
**Kind**: dictionary

A response from the device after it processes the command to enable Lost Mode on a device, which provides a message and phone number on the Lock Screen.

**Availability**:
- iOS 9.3+
- iPadOS 9.3+
