# StatusReason

**Framework**: Device Management  
**Kind**: dictionary

Provides details about an error for an item in a status report.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.1+
- watchOS 10.0+

## Declaration

```swift
object StatusReason
```

#### Discussion

Each status item defines its own set of `code`, `description`, and `details` values.
