# AppManagedInstallBehaviorObject

**Framework**: Device Management  
**Kind**: dictionary

A dictionary that describes how and when to install an app.

**Availability**:
- iOS 17.2+
- iPadOS 17.2+
- macOS 26.0+ (Beta)
- visionOS 2.4+
