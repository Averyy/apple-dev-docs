# StatusDeviceModelFamily

**Framework**: Device Management  
**Kind**: dictionary

A status report of the device’s hardware family.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.1+
- watchOS 10.0+
