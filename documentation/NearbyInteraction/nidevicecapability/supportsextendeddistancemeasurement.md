# supportsExtendedDistanceMeasurement

**Framework**: Nearby Interaction  
**Kind**: property  
**Required**: Yes

A Boolean value that indicates whether this device supports extended distance measurement.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- watchOS 10.0+
