# NINearbyObject.RemovalReason.peerEnded

**Framework**: Nearby Interaction  
**Kind**: case

The peer ended the session.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- watchOS 7.3+

## Declaration

```swift
case peerEnded
```

#### Discussion

The framework provides this removal reason when the peer app calls [`invalidate()`](nisession/invalidate().md).
