# init(name:image:)

**Framework**: Push to Talk  
**Kind**: init

Creates a participant with the name and image you specify.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+

## Declaration

```swift
init(name: String, image: UIImage?)
```

## Parameters

- `name`: The name of the participant.
- `image`: The image you associate with the participant.
