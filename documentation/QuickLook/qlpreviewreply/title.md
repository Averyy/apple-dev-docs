# title

**Framework**: Quick Look  
**Kind**: property

Custom display title for the preview. If left as the empty string, QuickLook will use the file name.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- visionOS 1.0+
