# allowsContentScaling

**Framework**: Quick Look  
**Kind**: property

Whether or not AR Quick Look allows content scaling in AR mode. Defaults to `YES` which allows scaling content in AR mode.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- visionOS 1.0+

## Declaration

```swift
var allowsContentScaling: Bool { get set }
```
