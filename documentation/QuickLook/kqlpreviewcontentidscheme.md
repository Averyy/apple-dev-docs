# kQLPreviewContentIDScheme

**Framework**: Quick Look  
**Kind**: var

The content ID URL scheme.

**Availability**:
- macOS 10.0+

## Declaration

```swift
let kQLPreviewContentIDScheme: CFString!
```

#### Discussion

The `cid` URL scheme permits the HTML of a mail message to refer to the images or other data included in the message. For more information, read [`RFC 2111`](https://developer.apple.comhttps://www.rfc-archive.org/getrfc.php?rfc=2111).
