# id

**Framework**: Visionkit  
**Kind**: property

A unique identifier for the recognized item.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- visionOS 1.0+

## Declaration

```swift
var id: UUID { get }
```

#### Discussion

If the same item resides in more than one video frame, the `id` remains the same.
