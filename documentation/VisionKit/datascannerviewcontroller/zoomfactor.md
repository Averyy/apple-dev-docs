# zoomFactor

**Framework**: Visionkit  
**Kind**: property

The zoom factor for the live video in the camera.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var zoomFactor: Double { get set }
```

#### Discussion

Set this property to a value between the [`minZoomFactor`](datascannerviewcontroller/minzoomfactor.md) and [`maxZoomFactor`](datascannerviewcontroller/maxzoomfactor.md) properties.
