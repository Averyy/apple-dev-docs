# viewWillAppear(_:)

**Framework**: Visionkit  
**Kind**: method

Performs some action before the view appears.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
override dynamic func viewWillAppear(_ animated: Bool)
```

## Parameters

- `animated`: A Boolean value that indicates whether the view will appear   using an animation.
