# viewDidDisappear(_:)

**Framework**: VisionKit  
**Kind**: method

Performs some action after the view disappears.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
override dynamic func viewDidDisappear(_ animated: Bool)
```

## Parameters

- `animated`: A Boolean value that indicates whether the view disappears   using an animation.
