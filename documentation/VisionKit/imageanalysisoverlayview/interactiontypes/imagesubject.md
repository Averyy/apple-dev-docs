# imageSubject

**Framework**: VisionKit  
**Kind**: property

An option that enables people to click or tap a subject in an image to separate it from the background.

**Availability**:
- macOS 13.0+

## Declaration

```swift
static let imageSubject: ImageAnalysisOverlayView.InteractionTypes
```

#### Discussion

For more information about image subjects, see [`ImageAnalysisOverlayView.Subject`](imageanalysisoverlayview/subject.md).
