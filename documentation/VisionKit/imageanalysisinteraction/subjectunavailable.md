# ImageAnalysisInteraction.SubjectUnavailable

**Framework**: Visionkit  
**Kind**: enum

Error conditions that can occur during subject analysis.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- visionOS 1.0+

## Declaration

```swift
enum SubjectUnavailable
```

#### Overview

This enumeration contains a failure [`ImageAnalysisInteraction.SubjectUnavailable.imageUnavailable`](imageanalysisinteraction/subjectunavailable/imageunavailable.md) that can occur when the [`ImageAnalysisInteraction.Subject`](imageanalysisinteraction/subject.md) property [`image`](imageanalysisinteraction/subject/image.md) fails to produce a result.
