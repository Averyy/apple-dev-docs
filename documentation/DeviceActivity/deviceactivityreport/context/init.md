# init(_:)

**Framework**: DeviceActivity  
**Kind**: init

Creates a new instance with the specified raw value.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+

## Declaration

```swift
init(_ rawValue: String)
```

## Parameters

- `rawValue`: The raw value to use for the new instance.
