# users

**Framework**: DeviceActivity  
**Kind**: property

The users to include in a report.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+

## Declaration

```swift
let users: DeviceActivityFilter.Users?
```

#### Discussion

If `nil`, then the filter only includes data for the current user.
