# devices

**Framework**: DeviceActivity  
**Kind**: property

The devices to include in a report.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+

## Declaration

```swift
let devices: DeviceActivityFilter.Devices?
```

#### Discussion

If `nil`, then the filter only includes data for the current device.
