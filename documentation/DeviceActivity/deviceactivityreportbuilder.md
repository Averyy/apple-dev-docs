# DeviceActivityReportBuilder

**Framework**: DeviceActivity  
**Kind**: struct

A result builder that combines one or more `DeviceActivityReportScene`s into a single scene.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+

## Declaration

```swift
@resultBuilder
struct DeviceActivityReportBuilder
```
