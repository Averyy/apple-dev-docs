# dateInterval

**Framework**: DeviceActivity  
**Kind**: property

The date interval of the activity segment.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+

## Declaration

```swift
var dateInterval: DateInterval
```

#### Discussion

This is determined by the `DeviceActivityFilter.SegmentInterval` that your app requests.
