# transactionDate

**Framework**: FinanceKit  
**Kind**: property

The time the transaction took place, if available.

**Availability**:
- iOS 17.4+
- iPadOS 17.4+

## Declaration

```swift
let transactionDate: Date
```

#### Discussion

This may be different from the time the transaction posted to the account.
