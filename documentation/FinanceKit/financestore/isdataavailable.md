# isDataAvailable(_:)

**Framework**: FinanceKit  
**Kind**: method

Returns a Boolean value that indicates if data that represents the provided type is available in the finance store.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
