# disableAllBackgroundDelivery()

**Framework**: FinanceKit  
**Kind**: method

Disables all background delivery.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)

## Declaration

```swift
func disableAllBackgroundDelivery()
```

#### Discussion

Your background delivery extension will no longer receive updates when data changes in the finance store.
