# IOUSBDeviceQualifierDescriptor

**Framework**: USBDriverKit  
**Kind**: struct

The structure for storing a USB device qualifier descriptor.

**Availability**:
- DriverKit 19.0+

## Declaration

```swift
struct IOUSBDeviceQualifierDescriptor;
```

#### Overview

For information about this descriptor, see section 9.6.2 of the USB 2.0 specification at [`http://www.usb.org`](https://developer.apple.comhttp://www.usb.org).
