# IOUSBInterfaceAssociationDescriptor

**Framework**: USBDriverKit  
**Kind**: struct

The USB Interface Association Descriptor.

**Availability**:
- DriverKit 19.0+

## Declaration

```swift
struct IOUSBInterfaceAssociationDescriptor;
```

#### Overview

This type defines the ECN to the USB 2.0 Spec. See the USB Specification at [`http://www.usb.org`](https://developer.apple.comhttp://www.usb.org). USB 3.0 9.6.4: Interface Association.
