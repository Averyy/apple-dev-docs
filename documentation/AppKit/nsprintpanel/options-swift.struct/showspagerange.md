# showsPageRange

**Framework**: AppKit  
**Kind**: property

The Print panel includes a set of fields for manipulating the range of pages being printed.

**Availability**:
- macOS 10.5+

## Declaration

```swift
static var showsPageRange: NSPrintPanel.Options { get }
```

#### Discussion

These fields are separate from any accessory views.
