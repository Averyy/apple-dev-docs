# behavior

**Framework**: AppKit  
**Kind**: property

The set of allowed behaviors for the status item.

**Availability**:
- macOS 10.12+

## Declaration

```swift
var behavior: NSStatusItem.Behavior { get set }
```

#### Discussion

By default, this property includes no behavior options. See [`NSStatusItem.Behavior`](nsstatusitem/behavior-swift.struct.md) for a list of available behavior options and their effects.
