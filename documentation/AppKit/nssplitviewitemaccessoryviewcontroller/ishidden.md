# isHidden

**Framework**: AppKit  
**Kind**: property

When set, this property will collapse the accessory view to 0 height (animatable) but not remove it from the window. Set through the animator object to animate it.

**Availability**:
- macOS 26.0+ (Beta)

## Declaration

```swift
@MainActor
var isHidden: Bool { get set }
```
