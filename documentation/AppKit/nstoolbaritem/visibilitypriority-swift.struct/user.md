# user

**Framework**: AppKit  
**Kind**: property

The highest priority for items in the toolbar.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS ?+

## Declaration

```swift
static let user: NSToolbarItem.VisibilityPriority
```

#### Discussion

The toolbar pushes these items to the overflow menu last.
