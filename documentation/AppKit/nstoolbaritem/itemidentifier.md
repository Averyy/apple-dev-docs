# itemIdentifier

**Framework**: AppKit  
**Kind**: property

The value you use to identify the toolbar item.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS ?+

## Declaration

```swift
@MainActor
var itemIdentifier: NSToolbarItem.Identifier { get }
```

#### Discussion

Use this property to distinguish one toolbar item from another on the same toolbar.
