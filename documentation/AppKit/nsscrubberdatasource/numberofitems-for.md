# numberOfItems(for:)

**Framework**: AppKit  
**Kind**: method  
**Required**: Yes

Asks the data source for the number of items in the scrubber.

**Availability**:
- macOS 10.12.2+

## Declaration

```swift
@MainActor
func numberOfItems(for scrubber: NSScrubber) -> Int
```

#### Return Value

The number of items in the scrubber.

## Parameters

- `scrubber`: The scrubber whose item count is being requested.
