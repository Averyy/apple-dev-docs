# NSButton.BezelStyle.disclosure

**Framework**: AppKit

**Availability**:
- macOS ?+

#### Discussion

Use this style of button when you want to reveal more information. When you use this bezel style along with the [`NSButton.ButtonType.pushOnPushOff`](nsbutton/buttontype/pushonpushoff.md) button type, the button points itâ€™s disclosure triangle to the right representing a closed state. When someone clicks the button the triangle animates down representing an open state.
