# intAttribute(_:forGlyphAt:)

**Framework**: AppKit  
**Kind**: method

Returns the value of the attribute identified by the given attribute tag for the glyph at the given index.

**Availability**:
- macOS 10.0+

## Declaration

```swift
func intAttribute(_ attributeTag: Int, forGlyphAt glyphIndex: Int) -> Int
```

#### Return Value

The value of the attribute identified by `attributeTag` and `glyphIndex`.

#### Discussion

Subclasses that define their own custom attributes must override this method to access their own storage for the attribute values. Nonnegative tags are reserved by Apple; you can define your own attributes with negative tags and set values using [`setIntAttribute(_:value:forGlyphAt:)`](nslayoutmanager/setintattribute(_:value:forglyphat:).md).

If noncontiguous layout is not enabled, this method causes generation of all glyphs up to and including `glyphIndex`. This method is primarily for the use of the glyph generator and typesetter.

## Parameters

- `attributeTag`: The attribute whose value is returned.
- `glyphIndex`: Index of the glyph whose attribute value is returned.
