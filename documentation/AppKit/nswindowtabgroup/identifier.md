# identifier

**Framework**: AppKit  
**Kind**: property

The unique identifier for a tabbed window group.

**Availability**:
- macOS 10.13+

## Declaration

```swift
var identifier: NSWindow.TabbingIdentifier { get }
```

#### Discussion

This identifier is shared by all windows that participate in this tab group.
