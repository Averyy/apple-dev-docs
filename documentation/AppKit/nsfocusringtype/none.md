# NSFocusRingType.none

**Framework**: AppKit  
**Kind**: case

No focus ring.

**Availability**:
- macOS ?+

## Declaration

```swift
case none
```

#### Discussion

If you set the focus ring type to this value, [`NSView`](nsview.md) and [`NSCell`](nscell.md) do not draw any focus ring.
