# itemCount

**Framework**: AppKit  
**Kind**: property

The number of items the badge displays.

**Availability**:
- macOS 14.0+

## Declaration

```swift
var itemCount: Int { get }
```

#### Discussion

If you create a badge with a custom string, this value is `0`.
