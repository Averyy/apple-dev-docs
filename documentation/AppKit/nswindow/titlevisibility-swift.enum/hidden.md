# NSWindow.TitleVisibility.hidden

**Framework**: AppKit  
**Kind**: case

**Availability**:
- macOS 10.10+

## Declaration

```swift
case hidden
```

#### Discussion

The window hides the title and moves the toolbar up into the area previously occupied by the title.
