# NSWindow.ToolbarStyle.unifiedCompact

**Framework**: AppKit  
**Kind**: case

A style indicating that the toolbar appears next to the window title and with reduced margins to allow more focus on the window’s contents.

**Availability**:
- macOS 11.0+
