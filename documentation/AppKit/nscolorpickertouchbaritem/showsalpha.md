# showsAlpha

**Framework**: AppKit  
**Kind**: property

A Boolean value that controls whether the color picker allows picking of colors with alpha values other than `1.0`.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS 10.12.2+
