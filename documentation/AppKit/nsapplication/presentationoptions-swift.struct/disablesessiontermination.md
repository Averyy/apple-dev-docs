# disableSessionTermination

**Framework**: AppKit  
**Kind**: property

The panel that shows the Restart, Shut Down, and Log Out options that are displayed as a result of pushing the power key is disabled.

**Availability**:
- macOS 10.6+

## Declaration

```swift
static var disableSessionTermination: NSApplication.PresentationOptions { get }
```
