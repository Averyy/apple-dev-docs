# context

**Framework**: AppKit  
**Kind**: property

The graphics context associated with the app.

**Availability**:
- macOS 10.0+

## Declaration

```swift
@MainActor
var context: NSGraphicsContext? { get }
```

#### Discussion

This property contains the graphics context most recently used by your app.
