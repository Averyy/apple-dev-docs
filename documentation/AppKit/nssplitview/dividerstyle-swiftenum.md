# NSSplitView.DividerStyle

**Framework**: AppKit  
**Kind**: enum

Constants that specify the style of the split view’s dividers.

**Availability**:
- macOS 10.5+

## Declaration

```swift
enum DividerStyle
```

#### Overview

These constants specify the possible divider styles that [`NSSplitView`](nssplitview.md) uses.
