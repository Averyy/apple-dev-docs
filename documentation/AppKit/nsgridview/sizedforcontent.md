# sizedForContent

**Framework**: AppKit  
**Kind**: property

The default value for row and column sizes.

**Availability**:
- macOS 10.12+

## Declaration

```swift
class let sizedForContent: CGFloat
```

#### Discussion

This constant indicates that a row or column should automatically fit the content views.
