# minDate

**Framework**: AppKit  
**Kind**: property

The date picker’s minimum date value.

**Availability**:
- macOS ?+

## Declaration

```swift
@MainActor
var minDate: Date? { get set }
```

#### Discussion

This property represents the minimum value that the date picker allows as input. `nil` indicates no minimum date.
