# maxDate

**Framework**: AppKit  
**Kind**: property

The date picker’s maximum date value.

**Availability**:
- macOS ?+

## Declaration

```swift
@MainActor
var maxDate: Date? { get set }
```

#### Discussion

This property represents the maximum value that the date picker allows as input. `nil` indicates no maximum date.
