# setInterlineSpacing(_:)

**Framework**: AppKit  
**Kind**: method

Sets the spacing between entries

**Availability**:
- macOS 10.0+

## Declaration

```swift
@MainActor
func setInterlineSpacing(_ spacing: CGFloat)
```

## Parameters

- `spacing`: The spacing between entries, measured in points in the user coordinate space.
