# selectText(at:)

**Framework**: AppKit  
**Kind**: method

Selects the entry at the specified index.

**Availability**:
- macOS 10.0+

## Declaration

```swift
@MainActor
func selectText(at index: Int)
```

## Parameters

- `index`: The index of the entry to select. If the specified index is invalid, this method does nothing.
