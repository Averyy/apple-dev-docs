# indexOfSelectedItem()

**Framework**: AppKit  
**Kind**: method

Returns the index of the selected entry.

**Availability**:
- macOS 10.0+

## Declaration

```swift
@MainActor
func indexOfSelectedItem() -> Int
```

#### Return Value

The index of the selected entry, or `-1` if no entry is selected.
