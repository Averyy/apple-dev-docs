# localizedName

**Framework**: AppKit  
**Kind**: property

The localized name of the color space.

**Availability**:
- macOS ?+

## Declaration

```swift
var localizedName: String? { get }
```

#### Discussion

This property holds the name of the color space as a localized string or `nil` if no localized name exists.
