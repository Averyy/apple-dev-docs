# adobeRGB1998

**Framework**: AppKit  
**Kind**: property

A color space object that represents an Adobe RGB (1998) color space.

**Availability**:
- macOS 10.5+

## Declaration

```swift
class var adobeRGB1998: NSColorSpace { get }
```

#### Return Value

The `NSColorSpace` object. This color-additive color space has red, green, blue, and alpha components.

#### Discussion

The Adobe RGB (1998) color space was designed to encompass most of the colors achievable on CMYK color printers, but by using RGB primary colors on a device such as the computer display. For more information on this color space, go to [`http://www.adobe.com/digitalimag/adobergb.html`](https://developer.apple.comhttp://www.adobe.com/digitalimag/adobergb.html).
