# deviceSize

**Framework**: AppKit  
**Kind**: property

The range of the touch device in points, such as 72 ppi.

**Availability**:
- macOS 10.6+

## Declaration

```swift
var deviceSize: NSSize { get }
```

#### Discussion

The lower-left corner of the surface is considered (0,0).
