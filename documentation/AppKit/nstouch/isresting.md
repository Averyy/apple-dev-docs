# isResting

**Framework**: AppKit  
**Kind**: property

The indicator for a resting touch.

**Availability**:
- macOS 10.6+

## Declaration

```swift
var isResting: Bool { get }
```

#### Discussion

Resting touches occur when a user simply rests their thumb on the trackpad device.
