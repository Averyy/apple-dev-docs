# isAnyApplicationSpeaking

**Framework**: AppKit  
**Kind**: property

A Boolean value indicating whether any application is currently speaking through the sound output device.

**Availability**:
- macOS 10.3+

## Declaration

```swift
class var isAnyApplicationSpeaking: Bool { get }
```

#### Discussion

This property is [`true`](https://developer.apple.com/documentation/swift/true) when another application is producing speech through the sound output device. You usually invoke this method to prevent your application from speaking over speech being generated by another application or system component.
