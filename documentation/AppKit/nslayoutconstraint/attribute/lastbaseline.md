# NSLayoutConstraint.Attribute.lastBaseline

**Framework**: AppKit  
**Kind**: case

The object’s baseline.

**Availability**:
- macOS ?+

## Declaration

```swift
case lastBaseline
```

#### Discussion

For objects with more than one line of text, this is the baseline for the bottommost line of text.
