# NSNotApplicableMarker

**Framework**: AppKit  
**Kind**: var

This marker indicates that an object is not key-value coding compliant for the requested key.

**Availability**:
- macOS 10.0+

## Declaration

```swift
var NSNotApplicableMarker: AnyObject
```
