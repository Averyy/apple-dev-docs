# removeAccessoryController(_:)

**Framework**: AppKit  
**Kind**: method

Removes the specified controller of an accessory view.

**Availability**:
- macOS 10.5+

## Declaration

```swift
@MainActor
func removeAccessoryController(_ accessoryController: NSViewController)
```

## Parameters

- `accessoryController`: The controller to remove.
