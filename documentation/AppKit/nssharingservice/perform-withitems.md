# perform(withItems:)

**Framework**: AppKit  
**Kind**: method

Manually performs the service on the provided items.

**Availability**:
- macOS 10.8+

## Declaration

```swift
func perform(withItems items: [Any])
```

#### Discussion

In most cases this will display a sharing window.

## Parameters

- `items`: The items to share.
