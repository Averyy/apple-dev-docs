# init(rawValue:)

**Framework**: AppKit  
**Kind**: init

Creates a set of CloudKit options using the specified raw value.

**Availability**:
- macOS 10.12+

## Declaration

```swift
init(rawValue: UInt)
```

## Parameters

- `rawValue`: An integer that represents the preferred CloudKit options.
