# minimumSize

**Framework**: AppKit  
**Kind**: property

The minimum size necessary for the tab view to display tabs in a useful way.

**Availability**:
- macOS ?+

## Declaration

```swift
@MainActor
var minimumSize: NSSize { get }
```

#### Discussion

You can use the value of this property to limit how much a user can resize a tab view.
