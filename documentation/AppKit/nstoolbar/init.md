# init()

**Framework**: AppKit  
**Kind**: init

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS 10.13+

## Declaration

```swift
@MainActor
convenience init()
```

#### Return Value
