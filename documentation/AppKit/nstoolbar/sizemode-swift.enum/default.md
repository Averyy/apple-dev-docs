# NSToolbar.SizeMode.default

**Framework**: AppKit  
**Kind**: case

The toolbar uses the system-defined default size, which is `NSToolbarSizeModeRegular`.

**Availability**:
- Mac Catalyst 13.1+
- macOS 10.0+
