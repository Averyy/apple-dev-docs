# init()

**Framework**: AppKit  
**Kind**: init

Initializes and returns an empty text block object.

**Availability**:
- macOS ?+

## Declaration

```swift
init()
```

#### Return Value

An initialized text block object.

#### Discussion

This is the designated initializer for `NSTextBlock`.
