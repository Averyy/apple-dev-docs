# interpolation

**Framework**: AppKit  
**Kind**: property

**Availability**:

## Declaration


The value for this key is an [`NSNumber`](https://developer.apple.com/documentation/Foundation/NSNumber) with an [`NSImageInterpolation`](nsimageinterpolation.md) value.
