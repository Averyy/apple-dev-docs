# itemKey

**Framework**: AppKit  
**Kind**: property

A key that specifies the toolbar item associated with the notification.

**Availability**:
- Mac Catalyst 16.1+
- macOS 13.0+

## Declaration

```swift
static let itemKey: NSToolbarUserInfoKey
```

#### Discussion

The value of this key is an [`NSToolbarItem`](nstoolbaritem.md) object.
