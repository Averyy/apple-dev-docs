# markerLocation

**Framework**: AppKit  
**Kind**: property

The location of the receiver in the coordinate system of the ruler view’s client view.

**Availability**:
- macOS ?+

## Declaration

```swift
var markerLocation: CGFloat { get set }
```

#### Discussion

This is an x position for a horizontal ruler, a y position for a vertical ruler.
