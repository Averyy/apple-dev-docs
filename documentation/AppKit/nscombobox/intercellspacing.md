# intercellSpacing

**Framework**: AppKit  
**Kind**: property

The horizontal and vertical spacing between cells in the pop-up list.

**Availability**:
- macOS ?+

## Declaration

```swift
@MainActor
var intercellSpacing: NSSize { get set }
```

#### Discussion

Spacing values are measured in points. The default spacing is (`3.0`, `2.0`).
