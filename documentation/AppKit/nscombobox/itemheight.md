# itemHeight

**Framework**: AppKit  
**Kind**: property

The height of each item in the pop-up list.

**Availability**:
- macOS ?+

## Declaration

```swift
@MainActor
var itemHeight: CGFloat { get set }
```

#### Discussion

The height of items is measured in points. The default item height is `16.0` points.
