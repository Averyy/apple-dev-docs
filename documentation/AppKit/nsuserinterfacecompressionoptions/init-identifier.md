# init(identifier:)

**Framework**: AppKit  
**Kind**: init

Creates an option object with the given identifier string.

**Availability**:
- macOS 10.13+

## Declaration

```swift
init(identifier: String)
```

#### Discussion

Use this initializer to create custom compression options.
