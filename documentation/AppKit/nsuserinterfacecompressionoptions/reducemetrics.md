# reduceMetrics

**Framework**: AppKit  
**Kind**: property

An option specifying that views should reduce their internal metrics.

**Availability**:
- macOS 10.13+

## Declaration

```swift
@NSCopying
class var reduceMetrics: NSUserInterfaceCompressionOptions { get }
```

#### Discussion

Use this compression option to reduce the padding in system controls.
