# indentationForDropOperation

**Framework**: AppKit  
**Kind**: property

Defines the amount the drag target for a row should be indented.

**Availability**:
- macOS 10.7+

## Declaration

```swift
@MainActor
var indentationForDropOperation: CGFloat { get set }
```

#### Discussion

The default is `0`.
