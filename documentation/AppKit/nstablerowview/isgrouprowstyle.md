# isGroupRowStyle

**Framework**: AppKit  
**Kind**: property

Specifies whether this row view is a group row.

**Availability**:
- macOS 10.7+

## Declaration

```swift
@MainActor
var isGroupRowStyle: Bool { get set }
```

#### Discussion

When [`true`](https://developer.apple.com/documentation/swift/true) this row is a group row and will draw appropriately.
