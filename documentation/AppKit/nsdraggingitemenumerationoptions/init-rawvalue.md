# init(rawValue:)

**Framework**: AppKit  
**Kind**: init

Creates a dragging enumeration option with the specified raw value.

**Availability**:
- macOS 10.7+

## Declaration

```swift
init(rawValue: UInt)
```

## Parameters

- `rawValue`: An unsigned integer value that corresponds to a dragging enumeration option.
