# set()

**Framework**: AppKit  
**Kind**: method

Sets the shadow of subsequent drawing operations to the current shadow.

**Availability**:
- macOS 10.0+

## Declaration

```swift
func set()
```

#### Discussion

The shadow attributes of the receiver are used until another shadow is set or until the graphics state is restored.
