# centerXAnchor

**Framework**: AppKit  
**Kind**: property

A layout anchor representing the horizontal center of the layout guide’s frame.

**Availability**:
- macOS 10.11+

## Declaration

```swift
var centerXAnchor: NSLayoutXAxisAnchor { get }
```

#### Discussion

Use this anchor to create constraints with the layout guide’s horizontal center. You can combine this anchor only with other [`NSLayoutXAxisAnchor`](https://developer.apple.com/documentation/UIKit/NSLayoutXAxisAnchor) anchors. For more information, see [`NSLayoutAnchor`](https://developer.apple.com/documentation/UIKit/NSLayoutAnchor).
