# init(content:)

**Framework**: AppKit  
**Kind**: init

Initializes and returns an `NSObjectController` object with the given content.

**Availability**:
- macOS ?+

## Declaration

```swift
init(content: Any?)
```

#### Return Value

The initialized object controller, with its content object set to `content`.

## Parameters

- `content`: The content for the receiver.
