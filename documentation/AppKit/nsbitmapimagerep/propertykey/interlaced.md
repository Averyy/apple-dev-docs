# interlaced

**Framework**: AppKit  
**Kind**: property

Identifies an `NSNumber` object containing a Boolean value that indicates whether the image is interlaced.

**Availability**:
- macOS ?+

## Declaration

```swift
static let interlaced: NSBitmapImageRep.PropertyKey
```

#### Discussion

Used only when writing out PNG files.
