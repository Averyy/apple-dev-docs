# alphaNonpremultiplied

**Framework**: AppKit  
**Kind**: property

A format where alpha values are not premultiplied.

**Availability**:
- macOS ?+

## Declaration

```swift
static var alphaNonpremultiplied: NSBitmapImageRep.Format { get }
```

#### Discussion

If this option is not specified, alpha values are premultiplied.
