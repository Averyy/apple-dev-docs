# NSLineBreakMode.byWordWrapping

**Framework**: AppKit  
**Kind**: case

The value that indicates wrapping occurs at word boundaries, unless the word doesn’t fit on a single line.

**Availability**:
- macOS 10.0+

## Declaration

```swift
case byWordWrapping
```
