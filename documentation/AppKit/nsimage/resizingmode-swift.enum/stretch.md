# NSImage.ResizingMode.stretch

**Framework**: AppKit  
**Kind**: case

The image stretches when it resizes.

**Availability**:
- macOS 10.10+

## Declaration

```swift
case stretch
```

#### Discussion

The interior region of the original image scales to fill in the interior region of the newly resized imaged.
