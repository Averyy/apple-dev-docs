# NSImage.ResizingMode.tile

**Framework**: AppKit  
**Kind**: case

The image tiles when it resizes.

**Availability**:
- macOS 10.10+

## Declaration

```swift
case tile
```

#### Discussion

The interior region of the original image repeats to fill in the interior region of the newly resized image.
