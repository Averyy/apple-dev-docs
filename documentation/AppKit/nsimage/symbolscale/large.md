# NSImage.SymbolScale.large

**Framework**: AppKit  
**Kind**: case

The symbol uses the large scale variant.

**Availability**:
- macOS 11.0+

## Declaration

```swift
case large
```

#### Discussion

This scale is approximately 30% larger than the [`NSImage.SymbolScale.medium`](nsimage/symbolscale/medium.md) scale.
