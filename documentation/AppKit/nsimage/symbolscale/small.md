# NSImage.SymbolScale.small

**Framework**: AppKit  
**Kind**: case

The symbol uses the small scale variant.

**Availability**:
- macOS 11.0+

## Declaration

```swift
case small
```

#### Discussion

This scale is approximately 20% smaller than the [`NSImage.SymbolScale.medium`](nsimage/symbolscale/medium.md) scale.
