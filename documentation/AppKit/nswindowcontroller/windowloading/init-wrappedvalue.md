# init(wrappedValue:)

**Framework**: AppKit  
**Kind**: init

Creates a property wrapper that loads the window controller’s window before accessing the property.

**Availability**:
- macOS 13.3+
- Swift 5.1+

## Declaration

```swift
init(wrappedValue: Value)
```

## Parameters

- `wrappedValue`: The underlying value tied to the loading of the window.
