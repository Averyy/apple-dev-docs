# options

**Framework**: AppKit  
**Kind**: property

The options specified for the receiver.

**Availability**:
- macOS 10.5+

## Declaration

```swift
var options: NSTrackingArea.Options { get }
```

#### Discussion

The options for an `NSTrackingArea` object are specified when the object is created.
