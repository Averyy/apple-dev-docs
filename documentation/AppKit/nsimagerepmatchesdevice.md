# NSImageRepMatchesDevice

**Framework**: AppKit  
**Kind**: var

A constant indicating that the value of certain attributes, such as the number of colors or bits per sample, will change to match the display device.

**Availability**:
- macOS ?+

## Declaration

```swift
var NSImageRepMatchesDevice: Int { get }
```

#### Discussion

This value can be passed in (or received back) as the value of [`bitsPerSample`](nsimagerep/bitspersample.md), [`pixelsWide`](nsimagerep/pixelswide.md), and [`pixelsHigh`](nsimagerep/pixelshigh.md).
