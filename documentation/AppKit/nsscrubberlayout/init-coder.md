# init(coder:)

**Framework**: AppKit  
**Kind**: init

Initializes and returns a newly allocated scrubber layout object from a storyboard or nib file.

**Availability**:
- macOS 10.12.2+

## Declaration

```swift
@MainActor
init(coder: NSCoder)
```

#### Return Value

A properly initialized scrubber layout object.
