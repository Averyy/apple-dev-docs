# beep()

**Framework**: AppKit  
**Kind**: method

Plays the system beep.

**Availability**:
- macOS 10.9+
- Swift 4.0+

## Declaration

```swift
static func beep()
```

#### Discussion

Users can select a sound to be played as the system beep in the Sound pane of System Preferences.
