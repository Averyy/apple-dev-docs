# push()

**Framework**: AppKit  
**Kind**: method

Puts the receiver on top of the cursor stack and makes it the current cursor.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS 10.0+
