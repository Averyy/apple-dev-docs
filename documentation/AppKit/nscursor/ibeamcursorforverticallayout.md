# iBeamCursorForVerticalLayout

**Framework**: AppKit  
**Kind**: property

Returns the cursor for editing vertical layout text.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS 10.7+

## Declaration

```swift
class var iBeamCursorForVerticalLayout: NSCursor { get }
```

#### Return Value

The vertical layout text cursor. This cursor is used when editing vertical layout text.
