# NSCursor.FrameResizeDirection

**Framework**: AppKit  
**Kind**: enum

The direction in which a rectangular frame can be resized.

**Availability**:
- Mac Catalyst 18.0+
- macOS 15.0+

## Declaration

```swift
enum FrameResizeDirection
```

## Topics

### Structures

### Enumeration Cases
- [NSCursor.FrameResizeDirection.inward](nscursor/frameresizedirection/inward.md)
  Indicates that the rectangular frame can be resized inwards to be smaller.
- [NSCursor.FrameResizeDirection.outward](nscursor/frameresizedirection/outward.md)
  Indicates that the rectangular frame can be resized outwards to be larger.
