# NSView.BackgroundStyle.raised

**Framework**: AppKit  
**Kind**: case

A style that makes the background appear higher than the content drawn on it.

**Availability**:
- macOS 10.5+

## Declaration

```swift
case raised
```

#### Discussion

You might need to inset the cell’s content to achieve this effect.
