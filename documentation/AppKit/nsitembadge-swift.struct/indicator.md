# indicator

**Framework**: AppKit  
**Kind**: property

Creates a badge styled as an indicator. In this context, an indicator is simply a badge without any text.

**Availability**:
- Mac Catalyst ?+
- macOS 26.0+ (Beta)

## Declaration

```swift
static var indicator: NSItemBadge
```

#### Return Value

A new `NSItemBadge` instance styled as an indicator.
