# slider

**Framework**: AppKit  
**Kind**: property

The slider displayed by the bar item.

**Availability**:
- macOS 10.12.2+

## Declaration

```swift
@MainActor
var slider: NSSlider { get set }
```

#### Discussion

The slider is automatically created by the item, but you can provide a custom subclass to this property to customize behavior.
