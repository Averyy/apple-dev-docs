# style

**Framework**: AppKit  
**Kind**: property

The style applied to the action button.

**Availability**:
- macOS 10.11+

## Declaration

```swift
var style: NSTableViewRowAction.Style { get }
```

#### Discussion

The value of this property is set at creation time and cannot be changed later.
