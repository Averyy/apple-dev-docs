# init(identifier:)

**Framework**: AppKit  
**Kind**: init

Performs default initialization for the receiver.

**Availability**:
- macOS ?+

## Declaration

```swift
init(identifier: Any?)
```

#### Discussion

Sets the receiver’s identifier object to `identifier`, if it is not `nil`. Use this method when creating tab view items programmatically.
