# view

**Framework**: AppKit  
**Kind**: property

Sets the view associated with the receiver to `view`.

**Availability**:
- macOS ?+

## Declaration

```swift
var view: NSView? { get set }
```

#### Discussion

This is the view displayed when a user clicks the tab. When you set a new view, the old view is released.
