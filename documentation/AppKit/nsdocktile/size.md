# size

**Framework**: AppKit  
**Kind**: property

The size of the tile.

**Availability**:
- macOS 10.5+

## Declaration

```swift
var size: NSSize { get }
```

#### Discussion

This corresponds to the size of the backing store in the dock, which may be bigger than the actual tile displayed on the screen.
