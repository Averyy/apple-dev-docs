# target

**Framework**: AppKit  
**Kind**: property

The target of the animation.

**Availability**:
- macOS ?+

## Declaration

```swift
static let target: NSViewAnimation.Key
```

#### Discussion

The target can be either an [`NSView`](nsview.md) object or an [`NSWindow`](nswindow.md) object. This property is required.
