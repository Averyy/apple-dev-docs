**Framework**: AppKit  
**Kind**: case

**Availability**:
- macOS ?+

## Declaration

```swift
case standard
```

#### Discussion

The standard display style and behavior. All path component cells are displayed with an icon image and component name. If the path can not fully be displayed, the middle parts are truncated as required.
