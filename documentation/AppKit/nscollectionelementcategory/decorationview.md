# NSCollectionElementCategory.decorationView

**Framework**: AppKit  
**Kind**: case

The element is a decoration view. Decoration views represent visual adornments that do not contain any data of their own.

**Availability**:
- Mac Catalyst 13.1+
- macOS 10.11+

## Declaration

```swift
case decorationView
```
