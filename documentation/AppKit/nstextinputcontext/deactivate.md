# deactivate()

**Framework**: AppKit  
**Kind**: method

Deactivates the receiver.

**Availability**:
- macOS 10.6+

## Declaration

```swift
func deactivate()
```

#### Discussion

You should not call this method directly; it is invoked by the system. It is provided as an override point for subclasses.
