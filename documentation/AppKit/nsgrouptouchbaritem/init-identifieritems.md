# init(identifier:items:)

**Framework**: AppKit  
**Kind**: init

Initializes and returns a group item whose bar is constructed from the supplied items.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS 10.12.2+
