# NSPasteboard.AccessBehavior.alwaysAllow

**Framework**: AppKit  
**Kind**: case

The system will automatically allow all pasteboard access, without notifying the user.  The app is listed in the corresponding System Settings pane.

**Availability**:
- macOS 15.4+
