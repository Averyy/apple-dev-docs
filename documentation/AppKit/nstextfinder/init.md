# init()

**Framework**: AppKit  
**Kind**: init

Initializes and returns a new `NSTextFinder` instance.

**Availability**:
- macOS 10.7+

## Declaration

```swift
init()
```

#### Return Value

An initialized NSTextFinder instance.

#### Discussion

This is the designated initiator for the `NSTextFinder` class.
