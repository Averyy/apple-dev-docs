# NSCollectionLayoutSectionOrthogonalScrollingBehavior.continuous

**Framework**: AppKit  
**Kind**: case

The section allows users to scroll its content orthogonally with continuous scrolling.

**Availability**:
- macOS 10.15+

## Declaration

```swift
case continuous
```
