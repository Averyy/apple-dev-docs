# isSeparatorItem

**Framework**: AppKit  
**Kind**: property

A Boolean value indicating whether the menu item is a separator item.

**Availability**:
- macOS ?+

## Declaration

```swift
var isSeparatorItem: Bool { get }
```

#### Discussion

This menu item is disabled. The default separator item is blank space.
