# interiorBackgroundStyle

**Framework**: AppKit  
**Kind**: property

The view’s interior background style.

**Availability**:
- macOS 10.10+

## Declaration

```swift
@MainActor
var interiorBackgroundStyle: NSView.BackgroundStyle { get }
```

#### Discussion

The background style may be light or dark, depending on the selected material.
