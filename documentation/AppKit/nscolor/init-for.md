# init(for:)

**Framework**: AppKit  
**Kind**: init

Returns the color object specified by the given control tint.

**Availability**:
- macOS 10.0+

## Declaration

```swift
init(for controlTint: NSControlTint)
```

#### Return Value

The `NSColor` object.

## Parameters

- `controlTint`: The control tint for which to return a color object. This is one of the tint settings.
