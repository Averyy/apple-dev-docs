# init()

**Framework**: AppKit  
**Kind**: init

Initializes the color object.

**Availability**:
- macOS ?+

## Declaration

```swift
init()
```

#### Discussion

Do not call this method directly. Instead, retrieve one of the predefined color objects or create a color using color component values.
