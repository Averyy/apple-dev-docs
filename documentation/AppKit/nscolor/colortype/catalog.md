# NSColor.ColorType.catalog

**Framework**: AppKit  
**Kind**: case

Colors that are retrieved from an asset catalog.

**Availability**:
- macOS ?+

## Declaration

```swift
case catalog
```

#### Discussion

Colors of this type have values for the [`catalogNameComponent`](nscolor/catalognamecomponent.md) and [`colorNameComponent`](nscolor/colornamecomponent.md) properties. A color of this type may also have device- and appearance-specific representations, allowing you to convert it to other types.
