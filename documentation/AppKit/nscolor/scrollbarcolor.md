# scrollBarColor

**Framework**: AppKit  
**Kind**: property

The system color used for scroll “bars”—that is, for the groove in which a scroller’s knob moves

**Availability**:
- macOS 10.0+

## Declaration

```swift
class var scrollBarColor: NSColor { get }
```

#### Return Value

The system color used for scroll bars. For general information about system colors, see [`Accessing System Colors`](https://developer.apple.comhttps://developer.apple.com/library/archive/documentation/Cocoa/Conceptual/DrawColor/Tasks/SystemColors.html#//apple_ref/doc/uid/20000790).
