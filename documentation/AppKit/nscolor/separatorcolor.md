# separatorColor

**Framework**: AppKit  
**Kind**: property

The color to use for separators between different sections of content.

**Availability**:
- macOS 10.14+

## Declaration

```swift
class var separatorColor: NSColor { get }
```

#### Discussion

Do not use this color for split view dividers or window chrome dividers.
