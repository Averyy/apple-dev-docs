# textColor

**Framework**: AppKit  
**Kind**: property

The color to use for text.

**Availability**:
- macOS ?+

## Declaration

```swift
class var textColor: NSColor { get }
```

#### Discussion

When text is selected, its color changes to the return value of [`selectedTextColor`](nscolor/selectedtextcolor.md).
