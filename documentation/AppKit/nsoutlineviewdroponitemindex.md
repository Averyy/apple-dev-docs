# NSOutlineViewDropOnItemIndex

**Framework**: AppKit  
**Kind**: var

**Availability**:
- macOS ?+

## Declaration

```swift
var NSOutlineViewDropOnItemIndex: Int { get }
```

#### Discussion

In this case, the drop will happen directly on the target item.
