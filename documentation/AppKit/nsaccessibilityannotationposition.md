# NSAccessibilityAnnotationPosition

**Framework**: AppKit  
**Kind**: enum

Constants that specify the position where the annotation applies.

**Availability**:
- macOS 10.13+

## Declaration

```swift
enum NSAccessibilityAnnotationPosition
```

#### Overview

Use these constants as values for the [`location`](nsaccessibility-swift.struct/annotationattributekey/location.md) key.
