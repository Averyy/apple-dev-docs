# removeItems(at:)

**Framework**: AppKit  
**Kind**: method

Removes the items at the specified indexes from the scrubber.

**Availability**:
- macOS 10.12.2+

## Declaration

```swift
@MainActor
func removeItems(at indexes: IndexSet)
```

## Parameters
