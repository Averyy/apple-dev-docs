# highlightedIndex

**Framework**: AppKit  
**Kind**: property

The index of the highlighted item in the scrubber.

**Availability**:
- macOS 10.12.2+

## Declaration

```swift
@MainActor
var highlightedIndex: Int { get }
```

#### Discussion

If no item is highlighted, the value of this property is `-1`.
