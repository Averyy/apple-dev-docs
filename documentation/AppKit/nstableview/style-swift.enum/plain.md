# NSTableView.Style.plain

**Framework**: AppKit  
**Kind**: case

The table view style resolves to a plain style.

**Availability**:
- macOS 11.0+

## Declaration

```swift
case plain
```

#### Discussion

A plain-style table applies no additional insets or padding around its content, except for the spacing that the [`intercellSpacing`](nstableview/intercellspacing.md) property specifies.
