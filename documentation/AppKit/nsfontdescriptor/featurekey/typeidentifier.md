# typeIdentifier

**Framework**: AppKit  
**Kind**: property

A key that indicates the type of the font feature.

**Availability**:
- macOS 10.5+

## Declaration

```swift
static let typeIdentifier: NSFontDescriptor.FeatureKey
```

#### Discussion

The value of this key is an [`NSNumber`](https://developer.apple.com/documentation/Foundation/NSNumber) object specifying a font feature type such as ligature, character shape, and so on.
