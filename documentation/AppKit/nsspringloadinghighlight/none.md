# NSSpringLoadingHighlight.none

**Framework**: AppKit  
**Kind**: case

A constant that indicates no highlighting.

**Availability**:
- macOS 10.11+

## Declaration

```swift
case none
```

#### Discussion

Don’t apply any highlighting style to the destination’s user interface. The destination is probably not spring-loadable.
