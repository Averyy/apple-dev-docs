# NSSegmentedControl.Style.texturedSquare

**Framework**: AppKit  
**Kind**: case

The control is displayed using the textured square style.

**Availability**:
- macOS 10.5+

## Declaration

```swift
case texturedSquare
```

- [NSSegmentedControl.Style.smallSquare](nssegmentedcontrol/style/smallsquare.md)
