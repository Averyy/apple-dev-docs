# NSEvent.EventType.pressure

**Framework**: AppKit  
**Kind**: case

An event that reports a change in pressure on a pressure-sensitive device.

**Availability**:
- macOS 10.10.3+

## Declaration

```swift
case pressure
```

#### Discussion

This type of event requires a 64-bit processor.
