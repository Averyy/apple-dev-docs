# NSEvent.EventType.tabletPoint

**Framework**: AppKit  
**Kind**: case

The user touched a point on a tablet.

**Availability**:
- macOS ?+

## Declaration

```swift
case tabletPoint
```

#### Discussion

Tablets generate tablet-point events between a mouse-down event and the first mouse drag event.
