# insertLine

**Framework**: AppKit  
**Kind**: property

The insert line key.

**Availability**:
- macOS 10.9+

## Declaration

```swift
static let insertLine: NSEvent.SpecialKey
```

#### Discussion

This key is unavailable on most Macintosh keyboards.
