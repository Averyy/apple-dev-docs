# init(name:)

**Framework**: TV Services  
**Kind**: init

Creates a new app profile descriptor object with the specified name.

**Availability**:
- tvOS 13.0+

## Declaration

```swift
init(name: String)
```

#### Return Value

An initialized app profile descriptor object.

## Parameters

- `name`: The user-visible string to display for the profile.
