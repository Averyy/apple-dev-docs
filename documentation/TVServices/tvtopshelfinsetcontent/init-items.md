# init(items:)

**Framework**: TV Services  
**Kind**: init

Creates an inset content object and populates it with the specified set of items.

**Availability**:
- tvOS 13.0+

## Declaration

```swift
init(items: [TVTopShelfItem])
```

#### Return Value

A new inset content object containing the specified set of items.

## Parameters

- `items`: An array of   objects representing the content that you want to display in the top shelf. This method stores a copy of the array.
