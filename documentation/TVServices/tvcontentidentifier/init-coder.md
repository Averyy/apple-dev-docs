# init(coder:)

**Framework**: TV Services  
**Kind**: init

Returns an object initialized from data in a given unarchiver.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
init?(coder: NSCoder)
```

#### Return Value

A newly initialized content identifier object.

## Parameters

- `coder`: An unarchiver object.
