# kSecCSCheckNestedCode

**Framework**: Security  
**Kind**: var

For code in bundle form, locate and recursively check embedded code.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+

## Declaration

```swift
var kSecCSCheckNestedCode: UInt32 { get }
```

#### Discussion

Only code in standard locations is considered.
