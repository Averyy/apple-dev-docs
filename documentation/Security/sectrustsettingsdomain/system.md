# SecTrustSettingsDomain.system

**Framework**: Security  
**Kind**: case

System trust settings.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+

## Declaration

```swift
case system
```

#### Discussion

These trust settings are immutable and comprise the set of trusted root certificates supplied in macOS. These settings are read-only, even by root.
