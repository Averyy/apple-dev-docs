# kSecCFErrorResourceAltered

**Framework**: Security  
**Kind**: var

A key whose value is a URL pointing to the resource on disk that has been altered.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+

## Declaration

```swift
let kSecCFErrorResourceAltered: CFString
```

#### Discussion

This key is present when a modified resource is found.
