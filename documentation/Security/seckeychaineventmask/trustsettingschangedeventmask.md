# trustSettingsChangedEventMask

**Framework**: Security  
**Kind**: property

If the bit specified by this mask is set, your callback function is invoked when there is a change in certificate trust settings.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+
