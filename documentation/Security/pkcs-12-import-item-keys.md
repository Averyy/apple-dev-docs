# PKCS #12 Import Item Keys

**Framework**: Security

Recognized the dictionary keys returned by an import operation.

#### Overview

These keys appear in the dictionary returned by the [`SecPKCS12Import(_:_:_:)`](secpkcs12import(_:_:_:).md) function.
