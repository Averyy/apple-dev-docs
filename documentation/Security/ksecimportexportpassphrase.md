# kSecImportExportPassphrase

**Framework**: Security  
**Kind**: var

A passphrase (represented by a `CFStringRef` object) to be used when exporting to or importing from PKCS#12 format.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.6+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
let kSecImportExportPassphrase: CFString
```
