# kSecCSDoNotValidateExecutable

**Framework**: Security  
**Kind**: var

Do not validate the contents of the main executable.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+

## Declaration

```swift
var kSecCSDoNotValidateExecutable: UInt32 { get }
```

#### Discussion

The contents of the main executable are normally validated.
