# debugged

**Framework**: Security  
**Kind**: property

The code has been debugged by another process that was allowed to do so.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+

## Declaration

```swift
static var debugged: SecCodeStatus { get }
```

#### Discussion

The debugger sets this flag when it attaches.
