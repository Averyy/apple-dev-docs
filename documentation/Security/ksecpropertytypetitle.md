# kSecPropertyTypeTitle

**Framework**: Security  
**Kind**: var

Specifies a key whose value is a string containing the title (display name) of the certificate.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- macOS 10.7+
- tvOS 9.0+
- visionOS 1.0+
