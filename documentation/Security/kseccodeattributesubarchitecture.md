# kSecCodeAttributeSubarchitecture

**Framework**: Security  
**Kind**: var

A key whose value is a string indicating a specific processor type, such as `i686` or `core2`.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+
