# SSL Cipher Suite Values

**Framework**: Security

Recognize the set of valid SSL cipher suite values.

#### Overview

These cipher suite constants are part of the TLS specification. For more information, see [`RFC 5246`](https://developer.apple.comhttps://tools.ietf.org/html/rfc5246).
