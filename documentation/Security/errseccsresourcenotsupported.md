# errSecCSResourceNotSupported

**Framework**: Security  
**Kind**: var

Found an unsupported resource.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+

## Declaration

```swift
var errSecCSResourceNotSupported: OSStatus { get }
```

#### Discussion

Resources other than directories, plain files, and symbolic links are not supported.
