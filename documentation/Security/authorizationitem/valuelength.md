# valueLength

**Framework**: Security  
**Kind**: property

The number of bytes in the value field.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+

## Declaration

```swift
var valueLength: Int
```

#### Discussion

Set this field to `0` if you set the `value` field to `NULL`.
