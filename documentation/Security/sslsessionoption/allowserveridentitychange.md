# SSLSessionOption.allowServerIdentityChange

**Framework**: Security  
**Kind**: case

Allow server identity change on renegotiation.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.2+

## Declaration

```swift
case allowServerIdentityChange
```

#### Discussion

Disallow by default to avoid the Triple Handshake attack.
