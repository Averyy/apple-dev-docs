# kSecPropertyTypeDate

**Framework**: Security  
**Kind**: var

Specifies a key whose value is a string containing a date (or a string listing the bytes of an invalid date).

**Availability**:
- macOS 10.7+

## Declaration

```swift
let kSecPropertyTypeDate: CFString
```
