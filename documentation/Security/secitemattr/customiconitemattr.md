# SecItemAttr.customIconItemAttr

**Framework**: Security  
**Kind**: case

Identifies the custom icon attribute.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+

## Declaration

```swift
case customIconItemAttr
```

#### Discussion

Use of this attribute is deprecated. Custom icons for keychains are not supported in macOS.
