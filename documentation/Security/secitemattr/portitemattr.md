# SecItemAttr.portItemAttr

**Framework**: Security  
**Kind**: case

Identifies the port attribute.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+

## Declaration

```swift
case portItemAttr
```

#### Discussion

You use this tag to set or get a value of type `UInt32` that represents the Internet port number. This is unique to Internet password attributes.
