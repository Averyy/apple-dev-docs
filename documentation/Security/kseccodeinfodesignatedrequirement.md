# kSecCodeInfoDesignatedRequirement

**Framework**: Security  
**Kind**: var

A keys whose value is the designated requirement of the code.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+

## Declaration

```swift
let kSecCodeInfoDesignatedRequirement: CFString
```

#### Discussion

The value is a [`SecRequirement`](secrequirement.md) object. If this object is identical to the one returned by the [`kSecCodeInfoImplicitDesignatedRequirement`](kseccodeinfoimplicitdesignatedrequirement.md) key, then there is no explicit designated requirement and the designated requirement in use was generated by the system.

Specify the [`kSecCSRequirementInformation`](kseccsrequirementinformation.md) flag when calling the [`SecCodeCopySigningInformation(_:_:_:)`](seccodecopysigninginformation(_:_:_:).md) function to get this information.
