# kSecMatchPolicy

**Framework**: Security  
**Kind**: var

A key whose value indicates a policy with which a matching certificate or identity must verify.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.6+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
let kSecMatchPolicy: CFString
```

#### Discussion

The corresponding value is of type [`SecPolicy`](secpolicy.md).
