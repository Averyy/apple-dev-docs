# interactionAllowed

**Framework**: Security  
**Kind**: property

A flag that permits user interaction as needed.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+

## Declaration

```swift
static var interactionAllowed: AuthorizationFlags { get }
```

#### Discussion

If this flag is set, the Security Server is permitted to interact with the user as needed.
