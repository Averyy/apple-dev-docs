# System Identity Domains

**Framework**: Security

Set or obtain a system identity for domains.

#### Overview

These constants can be used with the [`SecIdentitySetSystemIdentity(_:_:)`](secidentitysetsystemidentity(_:_:).md) and [`SecIdentityCopySystemIdentity(_:_:_:)`](secidentitycopysystemidentity(_:_:_:).md) functions.

## Topics

### Constants
- [let kSecIdentityDomainDefault: CFString](ksecidentitydomaindefault.md)
  The system-wide default identity.
- [let kSecIdentityDomainKerberosKDC: CFString](ksecidentitydomainkerberoskdc.md)
  Kerberos Key Distribution Center (KDC) identity.
