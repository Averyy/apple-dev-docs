# kSecUseAuthenticationUIAllow

**Framework**: Security  
**Kind**: var

A value that indicates user authentication is allowed.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
let kSecUseAuthenticationUIAllow: CFString
```

#### Discussion

The user may be prompted for authentication. This is the default value.
