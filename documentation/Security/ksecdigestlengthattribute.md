# kSecDigestLengthAttribute

**Framework**: Security  
**Kind**: var

The digest length.

**Availability**:
- macOS 10.7+

## Declaration

```swift
let kSecDigestLengthAttribute: CFString
```

#### Discussion

The value is a [`CFNumber`](https://developer.apple.com/documentation/CoreFoundation/CFNumber) that contains the digest length.
