# kSecTransformErrorInvalidInputDictionary

**Framework**: Security  
**Kind**: var

A dictionary used to import a transform has invalid data.

**Availability**:
- macOS 10.7+

## Declaration

```swift
var kSecTransformErrorInvalidInputDictionary: CFIndex { get }
```

#### Discussion

This error may occur when trying to import a transform from a data representation using the [`SecTransformCreateFromExternalRepresentation(_:_:)`](sectransformcreatefromexternalrepresentation(_:_:).md) function.
