# kSecCodeAttributeUniversalFileOffset

**Framework**: Security  
**Kind**: var

A key whose value indicates the offset of a Mach-O specific slice of a universal Mach-O file.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+
