# errAuthorizationBadAddress

**Framework**: Security  
**Kind**: var

The requested socket address is invalid.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+

## Declaration

```swift
var errAuthorizationBadAddress: OSStatus { get }
```

#### Discussion

The socket address must be in the range (0, 1023) inclusive.
