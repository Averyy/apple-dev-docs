# kSecEncryptionMode

**Framework**: Security  
**Kind**: var

The encryption mode.

**Availability**:
- macOS 10.7+

## Declaration

```swift
let kSecEncryptionMode: CFString
```

#### Discussion

If you do not supply this key, an appropriate value will be supplied for you. See [`Encryption Modes`](transform-attributes#Encryption-Modes.md) for a list of possible values.
