# kSecPropertyKeyLabel

**Framework**: Security  
**Kind**: var

A key whose value is the label for a certificate property.

**Availability**:
- macOS 10.7+

## Declaration

```swift
let kSecPropertyKeyLabel: CFString
```

#### Discussion

This label may contain a descriptive (localized) string or an OID string.
