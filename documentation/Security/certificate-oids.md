# Certificate OIDs

**Framework**: Security

Use OIDs as keys in the dictionary representing certificate values.

#### Overview

These Object Identifiers (OIDs) are the keys that may appear in the dictionary returned by a call to the [`SecCertificateCopyValues(_:_:_:)`](seccertificatecopyvalues(_:_:_:).md) function. The values associated with these keys are themselves dictionaries that represent the given property of a certificate, and that may have the keys listed in [`Certificate Property Keys`](certificate-property-keys.md).
