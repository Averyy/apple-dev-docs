# kSecCodeAttributeArchitecture

**Framework**: Security  
**Kind**: var

A key whose value is a string that indicates an architecture, such as `i386` or `x86_64`.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+

## Declaration

```swift
let kSecCodeAttributeArchitecture: CFString
```
