# Code Signing Architecture Flags

**Framework**: Security

Use these supplemental flags to get static code.

#### Overview

These flags supplement the flags described in [`SecCSFlags`](seccsflags.md). Use these additional constants with the `flags` parameter of the [`SecCodeCopyStaticCode(_:_:_:)`](seccodecopystaticcode(_:_:_:).md) function.

## Topics

### Constants
- [var kSecCSUseAllArchitectures: UInt32](kseccsuseallarchitectures.md)
  Flag for requesting all architectures.
