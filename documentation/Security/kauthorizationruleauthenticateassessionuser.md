# kAuthorizationRuleAuthenticateAsSessionUser

**Framework**: Security  
**Kind**: var

Indicates a delegate rule definition constant specifying that the user must authenticate as the session owner (logged-in user).

**Availability**:
- Mac Catalyst 13.0+
