# init(intValue:)

**Framework**: SwiftData  
**Kind**: init

Creates a new instance from the specified integer.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+
- Swift 5.9+

## Declaration

```swift
init?(intValue: Int)
```

#### Discussion

If the value passed as `intValue` does not correspond to any instance of this type, the result is `nil`.

## Parameters

- `intValue`: The integer value of the desired key.
