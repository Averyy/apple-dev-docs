# startIndex

**Framework**: SwiftData  
**Kind**: property

The position of the first element in a nonempty collection.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+
- watchOS 10.0+
- Swift 5.9+

## Declaration

```swift
var startIndex: Int { get }
```

#### Discussion

If the collection is empty, `startIndex` is equal to `endIndex`.
