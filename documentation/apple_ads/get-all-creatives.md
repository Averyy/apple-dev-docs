# Get All Creatives

**Framework**: Apple Ads  
**Kind**: httpRequest

Fetches all creatives within an organization.

**Availability**:
- Search Ads 5.0+

#### Discussion

Use this endpoint to fetch details of all assigned [`Creative`](creative.md) objects for your organization.

##### Payload Example Get All Creatives
