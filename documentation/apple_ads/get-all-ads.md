# Get All Ads

**Framework**: Apple Ads  
**Kind**: httpRequest

Fetches all ads assigned to an ad group.

**Availability**:
- Search Ads 5.0+

#### Discussion

Use this endpoint to fetch all `ads` assigned to an ad group.

##### Payload Example Get All Ads
