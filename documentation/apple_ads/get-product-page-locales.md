# Get Product Page Locales

**Framework**: Apple Ads  
**Kind**: httpRequest

Fetches product page locales by identifier.

**Availability**:
- Search Ads 5.0+

#### Discussion

Use this endpoint to return localized product page metadata using your `adamId` and `productPageId` in the resource path.
