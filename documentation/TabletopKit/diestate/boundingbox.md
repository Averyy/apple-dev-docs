# boundingBox

**Framework**: TabletopKit  
**Kind**: property

A 3D bounding box that encloses the die.

**Availability**:
- visionOS 2.0+

## Declaration

```swift
var boundingBox: Rect3D
```

#### Discussion

This bounding box is object oriented, so remains the same regardless of the equipment orientation.
