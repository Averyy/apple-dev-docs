# DebugDrawOptions

**Framework**: TabletopKit  
**Kind**: struct

Types of items in a rendering that you want to debug.

**Availability**:
- visionOS 2.0+

## Declaration

```swift
struct DebugDrawOptions
```

#### Overview

Use the [`debugDraw(options:)`](tabletopgame/debugdraw(options:).md) method to draw a debug representation of the rendered items.
