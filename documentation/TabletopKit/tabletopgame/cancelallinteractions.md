# cancelAllInteractions()

**Framework**: TabletopKit  
**Kind**: method

Cancels all local and remote interactions. This releases control of all the equipment and rolls back all the actions added to the canceled interaction.

**Availability**:
- visionOS 26.0+ (Beta)
