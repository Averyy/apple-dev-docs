# SLComposeViewControllerResult.cancelled

**Framework**: Social  
**Kind**: case

The view controller is dismissed without sending the post.

**Availability**:
- iOS 6.0+
- iPadOS 6.0+
- Mac Catalyst 13.0+

## Declaration

```swift
case cancelled
```

#### Discussion

This result occurs when the user selects Cancel or the account is unavailable.
