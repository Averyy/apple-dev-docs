# ILMessageFilterAction.transaction

**Framework**: SMS and Call Reporting  
**Kind**: case

Prevents the system from showing the message normally, filtered as a Transactional message.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- visionOS 1.0+
