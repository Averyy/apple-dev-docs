# ILMessageFilterAction.junk

**Framework**: SMS and Call Reporting  
**Kind**: case

Prevents the system from showing the message normally, filtered as a Junk message.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.0+
- visionOS 1.0+
