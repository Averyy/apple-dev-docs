# Localization-friendly layouts in macOS

**Framework**: Xcode

This project demonstrates localization-friendly auto layout constraints.

**Availability**:
- macOS 10.14+
- Xcode 12.0+

#### Overview

The sample uses `NSGridView` as a container view to achieve localized layouts.

#### Overview

> **Note**: This sample code project is associated with WWDC 2020 session [`10219: Build localization-friendly layouts using Xcode`](https://developer.apple.comhttps://developer.apple.com/videos/play/wwdc2020/10219/).
