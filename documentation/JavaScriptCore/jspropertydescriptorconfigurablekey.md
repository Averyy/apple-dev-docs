# JSPropertyDescriptorConfigurableKey

**Framework**: JavaScriptCore  
**Kind**: var

The Boolean value for this key determines whether the property deleted from its JavaScript object or its descriptor changed.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 13.0+
- macOS 10.5+
- tvOS 9.0+
- visionOS 1.0+
