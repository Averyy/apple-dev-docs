# JSObjectGetPrivate(_:)

**Framework**: JavaScriptCore  
**Kind**: func

Gets an object’s private data.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 13.0+
- macOS 10.5+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
func JSObjectGetPrivate(_ object: JSObjectRef!) -> UnsafeMutableRawPointer!
```

#### Return Value

A pointer to the object’s private data, if the object has private data; otherwise, `NULL`.

## Parameters

- `object`: A   with the private data you want to get.
