# init(_:id:)

**Framework**: WorkoutKit  
**Kind**: init

Creates a new workout plan from the provided workout and ID.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- watchOS 10.0+

## Declaration

```swift
init(_ workout: WorkoutPlan.Workout, id: UUID = UUID())
```

## Parameters

- `workout`: The workout represented by this plan.
- `id`: A unique ID for the plan.
