# power(_:unit:)

**Framework**: WorkoutKit  
**Kind**: method

Returns a new power threshold alert for the target value.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- watchOS 10.0+

## Declaration

```swift
static func power(_ value: Double, unit: UnitPower) -> Self
```

## Parameters

- `value`: The target value of the power threshold.
- `unit`: The power units used by the target value.
