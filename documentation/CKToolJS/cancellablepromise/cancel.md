# cancel

**Framework**: CKTool JS  
**Kind**: method

Stops any work the promise is doing.

**Availability**:
- CKTool JS 1.2.15+

## Declaration

```swift
void cancel();
```

#### Discussion

When called, it throws a `CancelledError`.
