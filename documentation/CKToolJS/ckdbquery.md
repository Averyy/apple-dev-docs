# CKDBQuery

**Framework**: CKTool JS  
**Kind**: struct

An object that represents a query for searching records.

**Availability**:
- CKTool JS 1.2.15+

## Declaration

```swift
dictionary CKDBQuery {
	string recordType;
	CKDBQueryFilter[]? filters;
	CKDBQuerySort[]? sorts;
};
```

#### Overview

In JavaScript, this is a plain object with properties as described.

In TypeScript, this type is imported in the following way:

```javascript
import type { CKDBQuery } from "@apple/cktool.database";
```

## Topics

### Instance Properties
- [filters](ckdbquery/filters.md)
  The array of filters used to determine whether a record matches the query.
- [recordType](ckdbquery/recordtype.md)
  The record type to perform this query on.
- [sorts](ckdbquery/sorts.md)
  The array of sort descriptors that specify how to order the fetched records.
