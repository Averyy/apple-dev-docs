# zoneName

**Framework**: CKTool JS  
**Kind**: property

The name that identifies the record zone.

**Availability**:
- CKTool JS 1.2.15+

## Declaration

```swift
attribute string zoneName;
```

#### Discussion

The default value is `_defaultZone`, which indicates the default zone of the current database.
