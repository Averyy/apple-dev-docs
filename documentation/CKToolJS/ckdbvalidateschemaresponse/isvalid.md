# isValid

**Framework**: CKTool JS  
**Kind**: property

The status of the schema string validation.

**Availability**:
- CKTool JS 1.2.15+

## Declaration

```swift
attribute boolean isValid;
```

#### Discussion

`true` if the schema can be applied to the container environment. Otherwise `false`.
