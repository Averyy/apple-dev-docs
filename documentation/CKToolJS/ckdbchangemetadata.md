# CKDBChangeMetadata

**Framework**: CKTool JS  
**Kind**: struct

An object that represents metadata about a record change via creation or modification.

**Availability**:
- CKTool JS 1.2.15+

## Declaration

```swift
dictionary CKDBChangeMetadata {
	string? deviceId;
	date timestamp;
	string userRecordName;
};
```

#### Overview

In JavaScript, this is a plain object with properties as described.

In TypeScript, this type is imported in the following way:

```javascript
import type { CKDBChangeMetadata } from "@apple/cktool.database";
```

## Topics

### Instance Properties
- [deviceId](ckdbchangemetadata/deviceid.md)
  The device where the change occurred.
- [timestamp](ckdbchangemetadata/timestamp.md)
  The time of the change.
- [userRecordName](ckdbchangemetadata/userrecordname.md)
  The record name that represents the user who made the change.
