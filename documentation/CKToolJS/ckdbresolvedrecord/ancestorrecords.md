# ancestorRecords

**Framework**: CKTool JS  
**Kind**: property

The array of record objects that represent the ancestor records.

**Availability**:
- CKTool JS 1.2.15+

## Declaration

```swift
attribute CKDBRecord[]? ancestorRecords;
```

#### Discussion

Ancestor records are previous versions of the record. Each value in this array is a `CKDBRecord` object.
