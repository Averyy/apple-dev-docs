# WatchKitError.Code.invalidArgument

**Framework**: Watchkit  
**Kind**: case

An invalid argument error.

**Availability**:
- watchOS ?+

## Declaration

```swift
case invalidArgument
```

## Overview

WatchKit reports this error when you specify invalid settings for one of the system supplied interfaces.
