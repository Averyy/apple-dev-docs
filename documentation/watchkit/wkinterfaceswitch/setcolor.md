# setColor(_:)

**Framework**: WatchKit  
**Kind**: method

Changes the tint color of the switch when it is on.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
func setColor(_ color: UIColor?)
```

## Parameters

- `color`: The tint color to use for the switch. Specify   to use the default switch color.
