# WKHapticType.directionUp

**Framework**: Watchkit  
**Kind**: case

Indicates an increase in a specific value or when a value has gone above a certain threshold.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
case directionUp
```

## Overview

For example, you could use this feedback when adjusting the value of a thermostat.
