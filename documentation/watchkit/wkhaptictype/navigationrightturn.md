# WKHapticType.navigationRightTurn

**Framework**: Watchkit  
**Kind**: case

Indicates that the user should turn right.

**Availability**:
- watchOS 7.0+

## Declaration

```swift
case navigationRightTurn
```

## Overview

You can only use this haptic type when your app is running a continuous background location session.
