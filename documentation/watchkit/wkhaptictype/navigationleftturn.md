# WKHapticType.navigationLeftTurn

**Framework**: WatchKit  
**Kind**: case

Indicates that the user should turn left.

**Availability**:
- watchOS 7.0+

## Declaration

```swift
case navigationLeftTurn
```

#### Discussion

You can only use this haptic type when your app is running a continuous background location session.
