# WKHapticType.directionDown

**Framework**: Watchkit  
**Kind**: case

Indicates a decrease in a specific value or when a value has gone below a certain threshold.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
case directionDown
```

## Overview

For example, you could use this feedback when adjusting the value of a thermostat.
