# WKHapticType.stop

**Framework**: Watchkit  
**Kind**: case

Indicates the end of an action.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
case stop
```

## Overview

For example, a stopwatch app uses this haptic when the user stops the stopwatch.
