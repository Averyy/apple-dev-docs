# play()

**Framework**: WatchKit  
**Kind**: method

Begins playback of the current item.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
func play()
```

#### Discussion

This method sets the value of the [`rate`](https://developer.apple.com/documentation/watchkit/wkaudiofileplayer/rate) property to `1.0`.
