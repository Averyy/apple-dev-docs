# pause()

**Framework**: WatchKit  
**Kind**: method

Pauses playback of the associated item.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
func pause()
```

#### Discussion

This method sets the value of the [`rate`](https://developer.apple.com/documentation/watchkit/wkaudiofileplayer/rate) property to `0.0`.
