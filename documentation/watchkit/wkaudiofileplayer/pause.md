# pause()

**Framework**: Watchkit  
**Kind**: method

Pauses playback of the associated item.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
func pause()
```

#### Discussion

This method sets the value of the [`rate`](wkaudiofileplayer/rate.md) property to `0.0`.
