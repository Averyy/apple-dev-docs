# screenScale

**Framework**: WatchKit  
**Kind**: property

The number of pixels per point for the current screen.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
var screenScale: CGFloat { get }
```

#### Discussion

The screen scale value applies to both the vertical and horizontal dimensions. For Apple Watch, this value is `2.0`.
