# systemVersion

**Framework**: Watchkit  
**Kind**: property

The version of the operating system.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
var systemVersion: String { get }
```

#### Discussion

This property contains the numerical version number of the system. For example, the value is `2.0` for the initial release of watchOS 2.
