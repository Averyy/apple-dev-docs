# localizedModel

**Framework**: WatchKit  
**Kind**: property

The localized version of the model information.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
var localizedModel: String { get }
```

#### Discussion

This property contains the localized string for the value in the [`model`](https://developer.apple.com/documentation/watchkit/wkinterfacedevice/model) property.
