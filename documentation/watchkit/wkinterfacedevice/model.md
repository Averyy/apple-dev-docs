# model

**Framework**: Watchkit  
**Kind**: property

The model information for the device.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
var model: String { get }
```

#### Discussion

For Apple Watch, the value of this string is `Apple Watch`.
