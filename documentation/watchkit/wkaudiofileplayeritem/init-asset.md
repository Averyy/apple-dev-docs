# init(asset:)

**Framework**: WatchKit  
**Kind**: init

Creates and returns a player item for the specified audio file asset.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
init(asset: WKAudioFileAsset)
```

#### Return Value

An initialized player item.

## Parameters

- `asset`: The audio file asset to be played.
