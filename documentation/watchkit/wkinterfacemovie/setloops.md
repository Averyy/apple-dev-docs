# setLoops(_:)

**Framework**: WatchKit  
**Kind**: method

Sets a Boolean value indicating whether the movie plays in a continuous loop.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
func setLoops(_ loops: Bool)
```

## Parameters

- `loops`: A Boolean value indicating the looping behavior. Specify   to play the movie in a continuous loop or   to play the movie once and then stop playback.
