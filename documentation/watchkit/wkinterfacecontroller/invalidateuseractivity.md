# invalidateUserActivity()

**Framework**: Watchkit  
**Kind**: method

Invalidates the most recent user activity.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
@MainActor
func invalidateUserActivity()
```

#### Discussion

Use this method to invalidate an activity that should no longer be performed.
