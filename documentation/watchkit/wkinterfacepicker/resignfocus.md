# resignFocus()

**Framework**: Watchkit  
**Kind**: method

Removes focus from the picker, causing it to stop receiving input from the Digital Crown.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
func resignFocus()
```

#### Discussion

When no picker has focus, input from the Digital Crown is once again used to scroll the contents of the screen.
