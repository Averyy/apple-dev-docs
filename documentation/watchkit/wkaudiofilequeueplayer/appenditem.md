# appendItem(_:)

**Framework**: WatchKit  
**Kind**: method

**Availability**:
- watchOS 2.0+

## Declaration

```swift
func appendItem(_ item: WKAudioFilePlayerItem)
```

## Parameters

- `item`: The player item to add to the queue.
