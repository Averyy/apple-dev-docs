# removeAllItems()

**Framework**: Watchkit  
**Kind**: method

Removes all items from the queue.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
func removeAllItems()
```

## Overview

This method has the side effect of stopping playback.
