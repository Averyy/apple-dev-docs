# advanceToNextItem()

**Framework**: Watchkit  
**Kind**: method

Ends playback of the current item and begins playing the next item in the queue.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
func advanceToNextItem()
```

## Overview

The method removes the current item from the queue before beginning playback of the next item.
