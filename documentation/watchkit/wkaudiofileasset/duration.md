# duration

**Framework**: WatchKit  
**Kind**: property

The duration (in seconds) of the audio file.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
var duration: TimeInterval { get }
```

#### Discussion

The duration of the audio file is obtained from the audio file itself.
