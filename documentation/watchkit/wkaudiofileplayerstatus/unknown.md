# WKAudioFilePlayerStatus.unknown

**Framework**: WatchKit  
**Kind**: case

The status of the item is unknown because the player has not yet loaded the audio file for playback.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
case unknown
```
