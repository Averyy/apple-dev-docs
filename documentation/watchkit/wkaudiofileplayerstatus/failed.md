# WKAudioFilePlayerStatus.failed

**Framework**: Watchkit  
**Kind**: case

The player can no longer play the audio because of an error. Use the [`error`](wkaudiofileplayer/error.md) property to get information about the error that occurred.

**Availability**:
- watchOS 2.0+

case failed
