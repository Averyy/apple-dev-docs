# presentScene(_:)

**Framework**: WatchKit  
**Kind**: method

Presents a scene.

**Availability**:
- watchOS 3.0+

## Declaration

```swift
func presentScene(_ scene: SKScene?)
```

#### Discussion

The new scene immediately replaces the current scene, if one exists.

## Parameters

- `scene`: The scene to present.
