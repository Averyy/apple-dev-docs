# stopAnimating()

**Framework**: Watchkit  
**Kind**: method  
**Required**: Yes

Stops any in-progress animations.

**Availability**:
- watchOS ?+

## Declaration

```swift
func stopAnimating()
```

## Overview

If no animation is in progress, this method does nothing.
