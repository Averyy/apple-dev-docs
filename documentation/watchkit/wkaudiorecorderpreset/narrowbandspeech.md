# WKAudioRecorderPreset.narrowBandSpeech

**Framework**: Watchkit  
**Kind**: case

Audio quality suitable for basic speech recording. This preset records audio with an 8 kHz sampling rate using either the LPCM 128 kbps or AAC 24 kbps format.

**Availability**:
- watchOS 2.0+
