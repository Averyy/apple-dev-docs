# WKInterfaceObjectVerticalAlignment.bottom

**Framework**: WatchKit  
**Kind**: case

Bottom alignment. The object’s content aligns with the bottom edge of the container and extends upward.

**Availability**:
- watchOS 2.0+

case bottom
