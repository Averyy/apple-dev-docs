# startIndex

**Framework**: hvf  
**Kind**: property

The position of the first element in a nonempty collection.

**Availability**:
- iOS 18.4+
- iPadOS 18.4+
- Mac Catalyst ?+
- macOS 15.4+
- tvOS 18.4+
- visionOS 2.4+
- watchOS 11.4+

## Declaration

```swift
final let startIndex: Int
```

#### Discussion

If the collection is empty, `startIndex` is equal to `endIndex`.
