# disconnect()

**Framework**: GameKit  
**Kind**: method

Disconnects the local player from the match.

**Availability**:
- iOS 4.1+
- iPadOS 4.1+
- Mac Catalyst 13.1+
- macOS 10.8+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
func disconnect()
```

#### Discussion

Call this method before ending a match.
