# local

**Framework**: GameKit  
**Kind**: property

The shared instance of the local player.

**Availability**:
- iOS 4.1+
- iPadOS 4.1+
- Mac Catalyst 13.1+
- macOS 10.8+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 3.0+

## Declaration

```swift
class var local: GKLocalPlayer { get }
```

#### Return Value

The local player object.

#### Discussion

You never directly create a local player object. Instead, use this method to get the shared instance.
