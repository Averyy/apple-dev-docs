# showHighlights

**Framework**: GameKit  
**Kind**: property

A Boolean value that indicates whether to display highlights for achievements and current ranks for leaderboards.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+
