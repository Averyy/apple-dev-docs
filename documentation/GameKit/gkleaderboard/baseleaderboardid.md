# baseLeaderboardID

**Framework**: GameKit  
**Kind**: property

The ID that Game Center uses to identify this leaderboard.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+
- watchOS 7.0+

## Declaration

```swift
var baseLeaderboardID: String { get }
```

#### Discussion

This is the same ID that App Store Connect uses to identify this leaderboard.
