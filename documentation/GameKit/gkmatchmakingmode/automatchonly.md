# GKMatchmakingMode.automatchOnly

**Framework**: GameKit  
**Kind**: case

A mode that matches the local player only with players who are also actively looking for a match.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
