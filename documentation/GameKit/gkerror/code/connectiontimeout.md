# GKError.Code.connectionTimeout

**Framework**: GameKit  
**Kind**: case

The system can’t complete the requested operation because the connection timed out.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+
