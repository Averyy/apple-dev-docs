# GKError.Code.iCloudUnavailable

**Framework**: GameKit  
**Kind**: case

The system can’t complete the requested operation because it can’t access the player’s iCloud account.

**Availability**:
- iOS 17.2+
- iPadOS 17.2+
- Mac Catalyst 17.2+
- macOS 14.2+
- tvOS 17.2+
- visionOS 1.1+
- watchOS 10.2+

## Declaration

```swift
case iCloudUnavailable
```
