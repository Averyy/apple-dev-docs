# GKError.Code.appUnlisted

**Framework**: GameKit  
**Kind**: case

The system can’t complete the requested operation because the game isn’t available on the App Store.

**Availability**:
- iOS 17.4+
- iPadOS 17.4+
- Mac Catalyst 17.4+
- macOS 14.4+
- tvOS 17.4+
- visionOS 1.1+
- watchOS 10.4+
