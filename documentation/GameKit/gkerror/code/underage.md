# GKError.Code.underage

**Framework**: GameKit  
**Kind**: case

The system can’t complete the requested operation because this feature isn’t available to underage players.

**Availability**:
- iOS 4.1+
- iPadOS 4.1+
- Mac Catalyst 13.1+
- macOS 10.8+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 3.0+
