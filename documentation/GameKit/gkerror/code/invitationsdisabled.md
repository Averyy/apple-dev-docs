# GKError.Code.invitationsDisabled

**Framework**: GameKit  
**Kind**: case

The system can’t complete the requested operation because the receiving player has disabled invitations.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- macOS 10.10+
