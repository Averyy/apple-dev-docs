# isFriend

**Framework**: GameKit  
**Kind**: property

A Boolean value that indicates whether the player is a friend of the local player.

**Availability**:
- iOS 4.1+
- iPadOS 4.1+
- Mac Catalyst 13.1+
- macOS 10.8+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
var isFriend: Bool { get }
```

#### Discussion

Players use the Game Center app to declare other players as friends.
