# fallbackURL

**Framework**: GameKit  
**Kind**: property

A fallback URL that can be used to construct a game-specific URL for players to share or join, if the joining device does not support the default URL.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+

## Declaration

```swift
var fallbackURL: URL? { get }
```
