# show()

**Framework**: GameKit  
**Kind**: method

Displays the peer picker dialog to the user.

**Availability**:
- visionOS 1.0+

## Declaration

```swift
func show()
```

#### Discussion

On iOS 3.1 or later, the peer picker is retained when it is shown, and autoreleased when it is dismissed.
