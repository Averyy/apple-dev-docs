# decline()

**Framework**: GameKit  
**Kind**: method

Declines a challenge that another player issues to the local player.

**Availability**:
- iOS 6.0+
- iPadOS 6.0+
- Mac Catalyst 13.1+
- macOS 10.8+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
func decline()
```

#### Discussion

If you implement an interface to present challenges, it should call this method when the local player declines a challenge.
