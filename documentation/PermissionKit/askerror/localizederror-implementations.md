# LocalizedError Implementations

**Framework**: PermissionKit

## Topics

### Instance Properties
- [var errorDescription: String?](askerror/errordescription.md)
  A localized message describing what error occurred.
- [var failureReason: String?](askerror/failurereason.md)
  A localized message describing the reason for the failure.
- [var helpAnchor: String?](askerror/helpanchor.md)
  A localized message providing “help” text if the user requests help.
- [var recoverySuggestion: String?](askerror/recoverysuggestion.md)
  A localized message describing how one might recover from the failure.
