# expirationDate

**Framework**: PermissionKit  
**Kind**: property

The date that this question expires, if any.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- visionOS 26.0+

## Declaration

```swift
final var expirationDate: Date?
```

#### Discussion

Once the date passes, the person that receives the question can no longer respond.
