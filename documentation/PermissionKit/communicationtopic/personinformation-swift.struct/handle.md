# handle

**Framework**: PermissionKit  
**Kind**: property

A handle that can be used to communicate with the person.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
var handle: CommunicationHandle
```

#### Discussion

A handle can be an email address, phone number, username, or any other person identifier.
