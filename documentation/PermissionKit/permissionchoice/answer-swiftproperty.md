# answer

**Framework**: PermissionKit  
**Kind**: property

The type of answer this choice represents.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- visionOS 26.0+

## Declaration

```swift
var answer: PermissionChoice.Answer { get set }
```

#### Discussion

The system uses this value to style how the choice appears to the person making the decision.
