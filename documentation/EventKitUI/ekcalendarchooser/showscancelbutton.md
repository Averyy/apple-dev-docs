# showsCancelButton

**Framework**: EventKit UI  
**Kind**: property

A Boolean that determines whether to display a Cancel button.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var showsCancelButton: Bool { get set }
```

#### Discussion

Use [`true`](https://developer.apple.com/documentation/Swift/true) to show a Cancel button; otherwise, [`false`](https://developer.apple.com/documentation/Swift/false).
