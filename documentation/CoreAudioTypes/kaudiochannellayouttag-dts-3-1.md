# kAudioChannelLayoutTag_DTS_3_1

**Framework**: Core Audio Types  
**Kind**: var

A Blu-ray Disc audio layout, defined by DTS (Digital Theater Systems Ltd.).

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- macOS 10.7+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 3.0+

## Declaration

```swift
var kAudioChannelLayoutTag_DTS_3_1: AudioChannelLayoutTag { get }
```

#### Discussion

Your channel layout must be in the following order:

- Center
- Left
- Right
- Low-frequency effects
