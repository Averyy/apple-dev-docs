# init(rawValue:)

**Framework**: Core Audio Types  
**Kind**: init

Creates a audio channel flags structure with a unsigned integer value.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
init(rawValue: UInt32)
```

## Parameters

- `rawValue`: An unsigned integer value that represents an audio channel flag.
