# kAudioChannelLayoutTag_AudioUnit_5_0

**Framework**: Core Audio Types  
**Kind**: var

A 5-channel surround-based layout, recommended for use by audio units.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- macOS 10.2+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 3.0+

## Declaration

```swift
var kAudioChannelLayoutTag_AudioUnit_5_0: AudioChannelLayoutTag { get }
```

#### Discussion

Your channels must be laid out in the following order:

- Left
- Right
- Left surround
- Right surround
- Center
