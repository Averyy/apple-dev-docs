# kAudioChannelLayoutTag_EAC3_6_1_C

**Framework**: Core Audio Types  
**Kind**: var

A Blu-ray Disc audio layout for Enhanced AC-3, also known as Dolby Digital Plus.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- macOS 10.7+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 3.0+

## Declaration

```swift
var kAudioChannelLayoutTag_EAC3_6_1_C: AudioChannelLayoutTag { get }
```

#### Discussion

Your channel layout must be in the following order:

- Left
- Center
- Right
- Left surround
- Right surround
- Low-frequency effects
- Vertical height center
