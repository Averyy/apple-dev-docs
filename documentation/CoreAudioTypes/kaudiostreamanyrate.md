# kAudioStreamAnyRate

**Framework**: Core Audio Types  
**Kind**: var

A value that indicates that an audio stream can use any sample rate.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 3.0+
