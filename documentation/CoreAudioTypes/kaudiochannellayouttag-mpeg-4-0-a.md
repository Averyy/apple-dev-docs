# kAudioChannelLayoutTag_MPEG_4_0_A

**Framework**: Core Audio Types  
**Kind**: var

An MPEG 4-channel, configuration A, audio layout.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- macOS 10.2+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 3.0+

## Declaration

```swift
var kAudioChannelLayoutTag_MPEG_4_0_A: AudioChannelLayoutTag { get }
```

#### Discussion

Your channels must be laid out in the following order:

- Left
- Right
- Center
- Center surround
