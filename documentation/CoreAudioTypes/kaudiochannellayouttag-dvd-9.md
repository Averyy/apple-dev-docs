# kAudioChannelLayoutTag_DVD_9

**Framework**: Core Audio Types  
**Kind**: var

A DVD 5-channel audio layout.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- macOS 10.2+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 3.0+

## Declaration

```swift
var kAudioChannelLayoutTag_DVD_9: AudioChannelLayoutTag { get }
```

#### Discussion

Your channels must be laid out in the following order:

- Left
- Right
- Center
- Left surround
- Right surround
