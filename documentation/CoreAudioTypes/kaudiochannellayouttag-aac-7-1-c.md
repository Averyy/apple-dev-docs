# kAudioChannelLayoutTag_AAC_7_1_C

**Framework**: Core Audio Types  
**Kind**: var

An AAC 7.1-channel, configuration C, surround-based layout.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- macOS 10.9+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 3.0+

## Declaration

```swift
var kAudioChannelLayoutTag_AAC_7_1_C: AudioChannelLayoutTag { get }
```

#### Discussion

Your channels must be laid out in the following order:

- Center
- Left
- Right
- Left surround
- Rear surround
- Low-frequency effects
- Vertical height left
- Vertical height right
