# MPEG4ObjectID.AAC_SBR

**Framework**: Core Audio Types  
**Kind**: case

A constant that specifies spectral band replication, which reconstructs high-frequency content from lower frequencies and side information.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- macOS 10.3+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 3.0+
