# hostHasController

**Framework**: CoreAudioKit  
**Kind**: property

A Boolean value that indicates whether the host shows its own control surface in this view configuration.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.13+
- visionOS 1.0+
