# upperBound

**Framework**: Declared Age Range  
**Kind**: property

The upper limit of the person’s age range.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)

## Declaration

```swift
var upperBound: Int?
```

#### Discussion

If nil, then there’s no upper bound of the person’s age range.
