# hasNotes

**Framework**: EventKit  
**Kind**: property

A Boolean value that indicates whether the calendar item has notes.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- Mac Catalyst 13.1+
- macOS 10.8+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
var hasNotes: Bool { get }
```

#### Discussion

If [`true`](https://developer.apple.com/documentation/Swift/true), the calendar item has notes; otherwise it does not.
