# init(absoluteDate:)

**Framework**: EventKit  
**Kind**: init

Creates and returns an alarm with an absolute date.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.1+
- macOS 10.8+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
init(absoluteDate date: Date)
```

#### Return Value

The created alarm.

## Parameters

- `date`: The date for the alarm.
