# init(mapItem:)

**Framework**: EventKit  
**Kind**: init

Creates a new structured location with the specified map item.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
convenience init(mapItem: MKMapItem)
```

## Parameters

- `mapItem`: The   object that represents the location.
