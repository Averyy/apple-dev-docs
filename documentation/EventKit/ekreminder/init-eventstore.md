# init(eventStore:)

**Framework**: EventKit  
**Kind**: init

Creates and returns a new reminder in the given event store.

**Availability**:
- iOS 6.0+
- iPadOS 6.0+
- Mac Catalyst 13.1+
- macOS 10.8+
- visionOS 1.0+
- watchOS 2.0+

```swift

```
