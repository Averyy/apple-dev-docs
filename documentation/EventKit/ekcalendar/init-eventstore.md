# init(eventStore:)

**Framework**: EventKit  
**Kind**: init

Creates and returns a calendar belonging to a specified event store.

**Availability**:
- watchOS 2.0+

## Declaration

```swift
init(eventStore: EKEventStore)
```

#### Return Value

The created calendar.

## Parameters

- `eventStore`: The event store to which the event belongs.
