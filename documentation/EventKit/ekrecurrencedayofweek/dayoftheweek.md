# dayOfTheWeek

**Framework**: EventKit  
**Kind**: property

The day of the week.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.1+
- macOS 10.8+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
var dayOfTheWeek: EKWeekday { get }
```

#### Discussion

Values are from `1` to `7`, with Sunday being `1`.
