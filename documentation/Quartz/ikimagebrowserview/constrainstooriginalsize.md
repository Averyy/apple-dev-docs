# constrainsToOriginalSize()

**Framework**: Quartz  
**Kind**: method

Returns whether the receiver constrains the cell’s image to its original size.

**Availability**:
- macOS 10.4+

## Declaration

```swift
@MainActor
func constrainsToOriginalSize() -> Bool
```

#### Return Value

[`false`](https://developer.apple.com/documentation/Swift/false) if the image is not constrained; otherwise [`true`](https://developer.apple.com/documentation/Swift/true).
