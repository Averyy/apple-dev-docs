# reloadItem(at:)

**Framework**: Quartz  
**Kind**: method

Reloads the data for a slideshow, starting at the specified index.

**Availability**:
- macOS 10.4+

## Declaration

```swift
func reloadItem(at index: Int)
```

## Parameters

- `index`: The index that species where to reload the slideshow data.
