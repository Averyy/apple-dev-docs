# hasParameters()

**Framework**: Quartz  
**Kind**: method

Checks whether the composition that is currently edited by the composition parameter view has any input parameters.

**Availability**:
- macOS 10.4+

## Declaration

```swift
@MainActor
func hasParameters() -> Bool
```

#### Return Value

[`true`](https://developer.apple.com/documentation/swift/true) if the composition has any input parameters.
