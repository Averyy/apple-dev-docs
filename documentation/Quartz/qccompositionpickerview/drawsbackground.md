# drawsBackground()

**Framework**: Quartz  
**Kind**: method

Returns whether the composition picker view draws its background.

**Availability**:
- macOS 10.4+

## Declaration

```swift
@MainActor
func drawsBackground() -> Bool
```

#### Return Value

[`true`](https://developer.apple.com/documentation/Swift/true) if the composition picker view draws its background; otherwise [`false`](https://developer.apple.com/documentation/Swift/false).
