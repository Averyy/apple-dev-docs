# showsCompositionNames()

**Framework**: Quartz  
**Kind**: method

Retrieves whether composition names can be shown in the composition picker view.

**Availability**:
- macOS 10.4+

## Declaration

```swift
@MainActor
func showsCompositionNames() -> Bool
```

#### Return Value

[`true`](https://developer.apple.com/documentation/Swift/true) if the display of names is enabled; otherwise [`false`](https://developer.apple.com/documentation/Swift/false).
