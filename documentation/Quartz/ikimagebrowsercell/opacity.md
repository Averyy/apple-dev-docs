# opacity()

**Framework**: Quartz  
**Kind**: method

Returns the opacity of the receiver.

**Availability**:
- macOS 10.4+

## Declaration

```swift
func opacity() -> CGFloat
```

#### Return Value

The cell’s opacity.

#### Discussion

Possible values are between 0.0 (transparent) and 1.0 (opaque).

Subclasses can override this method to customize the opacity of the cell.
