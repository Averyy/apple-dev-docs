# scroll(to:)

**Framework**: Quartz  
**Kind**: method

Scrolls the view so that it includes the provided rectangular area.

**Availability**:
- macOS 10.5+

## Declaration

```swift
@MainActor
func scroll(to rect: NSRect)
```

## Parameters

- `rect`: The rectangular area to include in the view.
