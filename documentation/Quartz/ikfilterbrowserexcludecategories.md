# IKFilterBrowserExcludeCategories

**Framework**: Quartz  
**Kind**: var

The key for excluding filter categories.

**Availability**:
- macOS 10.4+

## Declaration

```swift
let IKFilterBrowserExcludeCategories: String
```

#### Discussion

The associated value is an [`NSArray`](https://developer.apple.com/documentation/Foundation/NSArray) object that lists the categories that you do  want to display in the filter browser.
