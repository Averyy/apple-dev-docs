# maxRenderingFrameRate()

**Framework**: Quartz  
**Kind**: method

Returns the maximum frame rate for rendering.

**Availability**:
- macOS 10.4+

## Declaration

```swift
@MainActor
func maxRenderingFrameRate() -> Float
```

#### Return Value

The maximum frame rate for rendering. A value of `0.0` specifies that there is no limit.
