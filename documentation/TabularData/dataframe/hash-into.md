# hash(into:)

**Framework**: TabularData  
**Kind**: method

Hashes the essential components of the data frame by feeding them into a hasher.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
func hash(into hasher: inout Hasher)
```

## Parameters

- `hasher`: A hasher the method uses to combine the components of the data frame.
