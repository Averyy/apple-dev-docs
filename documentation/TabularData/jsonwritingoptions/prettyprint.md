# prettyPrint

**Framework**: TabularData  
**Kind**: property

A Boolean value that indicates whether to split lines and indent the generated JSON.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+
- watchOS 9.0+

## Declaration

```swift
var prettyPrint: Bool
```

#### Discussion

Defaults to `false`.
