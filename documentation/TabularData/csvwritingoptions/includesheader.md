# includesHeader

**Framework**: TabularData  
**Kind**: property

A Boolean value that indicates whether to write a header with the column names.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
var includesHeader: Bool
```

#### Discussion

Defaults to `true`.
