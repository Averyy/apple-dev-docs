# delimiter

**Framework**: TabularData  
**Kind**: property

The character the CSV file generator uses to separate data fields in a CSV file.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
var delimiter: Character
```

#### Discussion

Defaults to comma (`,`).
