# falseEncoding

**Framework**: TabularData  
**Kind**: property

The string the CSV file generator uses to represent false Boolean values.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
var falseEncoding: String
```

#### Discussion

Defaults to `false`.
