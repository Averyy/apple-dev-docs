# dateFormat

**Framework**: TabularData  
**Kind**: property

The format the CSV file generator uses to create date strings.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+

## Declaration

```swift
var dateFormat: String?
```

#### Discussion

Defaults to `nil`, which uses ISO 8601 encoding.
