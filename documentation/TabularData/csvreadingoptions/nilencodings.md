# nilEncodings

**Framework**: TabularData  
**Kind**: property

The set of strings that stores acceptable spellings for empty values.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
var nilEncodings: Set<String>
```

#### Discussion

Defaults to `["", "#N/A", "#N/A N/A", "#NA", "N/A", "NA", "NULL", "n/a", "null"]`.
