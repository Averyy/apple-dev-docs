# falseEncodings

**Framework**: TabularData  
**Kind**: property

The set of strings that stores acceptable spellings for false Boolean values.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
var falseEncodings: Set<String>
```

#### Discussion

Defaults to `["0", "False", "FALSE", "false"]`.
