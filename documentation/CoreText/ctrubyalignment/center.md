# CTRubyAlignment.center

**Framework**: Core Text  
**Kind**: case

Centers the ruby text within the width of the base text.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
case center
```

#### Discussion

If the ruby text is wider than the base text, Core Text centers the base text within the width of the ruby text.
