# kCTTabColumnTerminatorsAttributeName

**Framework**: Core Text

Specifies the terminating character for a tab column.

#### Overview

The value associated with this attribute is a CFCharacterSet object. The character set is used to determine the terminating character for a tab column. The tab and newline characters are implied even if they donâ€™t exist in the character set. This attribute can be used to implement decimal tabs, for instance. This attribute is optional.
