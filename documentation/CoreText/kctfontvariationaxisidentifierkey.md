# kCTFontVariationAxisIdentifierKey

**Framework**: Core Text  
**Kind**: var

Key to get the variation axis identifier.

**Availability**:
- iOS 3.2+
- iPadOS 3.2+
- Mac Catalyst 13.1+
- macOS 10.5+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
let kCTFontVariationAxisIdentifierKey: CFString
```

#### Discussion

Key to get the variation axis identifier value as a reference to a [`CFNumber`](https://developer.apple.com/documentation/CoreFoundation/CFNumber) object.
