# kCTFontFeatureSelectorNameKey

**Framework**: Core Text  
**Kind**: var

**Availability**:
- iOS 3.2+
- iPadOS 3.2+
- Mac Catalyst 13.1+
- macOS 10.5+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
let kCTFontFeatureSelectorNameKey: CFString
```

#### Discussion

Key to be used with a selector dictionary to get the localized name string for the selector as a `CFStringRef` object.
