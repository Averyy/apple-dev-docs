# CTFontDescriptorCreateWithNameAndSize(_:_:)

**Framework**: Core Text  
**Kind**: func

Creates a new font descriptor with the provided PostScript name and size.

**Availability**:
- iOS 3.2+
- iPadOS 3.2+
- Mac Catalyst 13.1+
- macOS 10.5+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
func CTFontDescriptorCreateWithNameAndSize(_ name: CFString, _ size: CGFloat) -> CTFontDescriptor
```

#### Return Value

A new font descriptor reference with the given PostScript name and point size.

## Parameters

- `name`: The PostScript name to be used for the font descriptor as a   object.
- `size`: The point size. If  , the font size attribute ( ) is omitted from the returned font descriptor.
