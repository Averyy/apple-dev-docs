# kCTFontMiniSystemFontType

**Framework**: Core Text  
**Kind**: property

The standard miniature system font used for mini controls and utility window labels and text.

**Availability**:
- iOS 3.2+
- iPadOS 3.2+
- Mac Catalyst 13.1+
- macOS 10.5+
- visionOS 1.0+
