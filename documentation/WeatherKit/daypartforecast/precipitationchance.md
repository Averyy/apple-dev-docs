# precipitationChance

**Framework**: Weatherkit  
**Kind**: property

The probability of precipitation for the day part.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+

## Declaration

```swift
var precipitationChance: Double
```

#### Discussion

The value is from `0` (no chance of precipitation) to `1` (100% chance of precipitation).
