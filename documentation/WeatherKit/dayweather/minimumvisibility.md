# minimumVisibility

**Framework**: Weatherkit  
**Kind**: property

The minimum distance, in meters, at which terrain is visible for the day.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+

## Declaration

```swift
var minimumVisibility: Double
```

#### Discussion

The amount of light, and weather conditions like fog, mist, and smog affect visibility.
