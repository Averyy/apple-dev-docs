# precipitationChance

**Framework**: WeatherKit  
**Kind**: property

The probability of precipitation in this minute.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+
- watchOS 9.0+

## Declaration

```swift
var precipitationChance: Double
```

#### Discussion

The value is from `0` (no chance of precipitation) to `1` (100% chance of precipitation).
