# eac3

**Framework**: AVFoundation  
**Kind**: property

The UTI for the enhanced AC3 audio file format.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
static let eac3: AVFileType
```

#### Discussion

The value of this UTI is `public.enhanced-ac3-audio`. Files of this type have an `.eac3` extension.
