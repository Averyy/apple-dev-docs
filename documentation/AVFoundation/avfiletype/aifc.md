# aifc

**Framework**: AVFoundation  
**Kind**: property

The UTI for the AIFC audio file format.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.1+
- macOS 10.7+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 1.0+

## Declaration

```swift
static let aifc: AVFileType
```

#### Discussion

The value of this UTI is `public.aifc-audio`. Files of this type have an `.aifc` or `.cdda` extension.
