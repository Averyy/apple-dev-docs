# removeFontStyle(in:)

**Framework**: AVFoundation  
**Kind**: method

Removes a font style from a range of text.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 15.0+
- macOS 12.0+

## Declaration

```swift
@nonobjc
func removeFontStyle(in range: NSRange)
```

## Parameters

- `range`: The range from which to remove the style.
