# setTextColor(_:in:)

**Framework**: AVFoundation  
**Kind**: method

Sets the text color for a range of text.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 15.0+
- macOS 12.0+

## Declaration

```swift
@nonobjc
func setTextColor(_ textColor: CGColor, in range: NSRange)
```

## Parameters

- `textColor`: The text color.
- `range`: The range to which this text color applies.
