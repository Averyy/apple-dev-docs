# AVAuthorizationStatus.denied

**Framework**: AVFoundation  
**Kind**: case

A status that indicates the user has explicitly denied an app permission to capture media.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 14.0+
- macOS 10.14+
- tvOS 17.0+
- visionOS 1.0+
