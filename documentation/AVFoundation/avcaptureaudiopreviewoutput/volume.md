# volume

**Framework**: AVFoundation  
**Kind**: property

The output volume of the audio preview.

**Availability**:
- macOS 10.7+

## Declaration

```swift
var volume: Float { get set }
```

#### Discussion

A value of `1.0` indicates maximum volume, and a value of `0.0` mutes the audio preview.
