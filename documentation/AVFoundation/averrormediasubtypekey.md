# AVErrorMediaSubTypeKey

**Framework**: AVFoundation  
**Kind**: var

The user information key to retrieve the media subtype.

**Availability**:
- iOS 4.3+
- iPadOS 4.3+
- Mac Catalyst 13.1+
- macOS 10.7+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 1.0+

## Declaration

```swift
let AVErrorMediaSubTypeKey: String
```

#### Discussion

The associated value is an array of integers that indicate the media subtype.
