# minSimulatedAperture

**Framework**: AVFoundation  
**Kind**: property

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
- tvOS 26.0+ (Beta)

## Declaration

```swift
var minSimulatedAperture: Float { get }
```

#### Discussion

Minimum supported shallow depth of field simulated aperture.

On devices that do not support changing the simulated aperture value, this will return a value of 0.
