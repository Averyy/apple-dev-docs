# videoFieldOfView

**Framework**: AVFoundation  
**Kind**: property

Indicates the format’s horizontal field of view in degrees.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 14.0+
- tvOS 17.0+

## Declaration

```swift
var videoFieldOfView: Float { get }
```

#### Discussion

Returns zero if the format’s field of view is unknown.
