# externalUnknown

**Framework**: AVFoundation  
**Kind**: property

An unknown external device type.

**Availability**:
- macOS 10.15+

## Declaration

```swift
static let externalUnknown: AVCaptureDevice.DeviceType
```

#### Discussion

In macOS, use this type to specify external devices, such as an iPhone camera.
