# mindingInterval

**Framework**: AVFoundation  
**Kind**: property

The amount of time between checks for additional movie fragments.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS 10.10+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
var mindingInterval: TimeInterval { get set }
```

#### Discussion

The default interval is `10.0`.
