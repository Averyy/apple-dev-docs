# availableLanguages

**Framework**: AVFoundation  
**Kind**: property

Provides available language choices.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
- tvOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)
- watchOS 26.0+ (Beta)

## Declaration

```swift
var availableLanguages: [String] { get }
```

#### Discussion

Each string in the array is intended to be interpreted as a BCP 47 language tag.
