# url()

**Framework**: AVFoundation  
**Kind**: method

Returns the URL used to initialize the receiver.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
func url() -> URL?
```

#### Return Value

The [`NSURL`](https://developer.apple.com/documentation/Foundation/NSURL) object used to initialize the receiver. This value can be `nil`.
