# init(playerItem:)

**Framework**: AVFoundation  
**Kind**: init

Creates a new player to play the specified player item.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.1+
- macOS 10.7+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 1.0+

## Declaration

```swift
nonisolated
init(playerItem item: AVPlayerItem?)
```

#### Return Value

A new player initialized to play `item`.

## Parameters

- `item`: The player item to play.
