# url

**Framework**: AVFoundation  
**Kind**: property

A URL to a QuickTime or ISO base media file.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS 10.10+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
var url: URL? { get }
```

#### Discussion

The value is `nil` if you didn’t initialize the movie with a URL.
