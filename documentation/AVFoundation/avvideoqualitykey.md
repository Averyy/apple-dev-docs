# AVVideoQualityKey

**Framework**: AVFoundation  
**Kind**: var

A key to set the JPEG compression quality of the video.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- Mac Catalyst 13.1+
- macOS 10.7+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
let AVVideoQualityKey: String
```

#### Discussion

The corresponding value is an instance of `NSNumber` 0.0-1.0.
