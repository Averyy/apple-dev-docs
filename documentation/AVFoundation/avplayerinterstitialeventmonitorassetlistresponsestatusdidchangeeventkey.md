# AVPlayerInterstitialEventMonitorAssetListResponseStatusDidChangeEventKey

**Framework**: AVFoundation  
**Kind**: var

A key to retrieve the interstitial event that has an asset list response status change.

**Availability**:
- iOS 16.4+
- iPadOS 16.4+
- Mac Catalyst 16.4+
- macOS 13.3+
- tvOS 16.4+
- visionOS 1.0+
- watchOS 9.4+

## Declaration

```swift
let AVPlayerInterstitialEventMonitorAssetListResponseStatusDidChangeEventKey: String
```

#### Discussion

Use this key to retrieve the [`AVPlayerInterstitialEvent`](avplayerinterstitialevent.md) object that has an updates asset list response.
