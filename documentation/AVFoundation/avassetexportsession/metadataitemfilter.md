# metadataItemFilter

**Framework**: AVFoundation  
**Kind**: property

An object the export session uses to filter the metadata items it transfers to the output asset.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- macOS 10.9+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

var metadataItemFilter: AVMetadataItemFilter? { get set }
