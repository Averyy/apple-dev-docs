# init(asset:presetName:)

**Framework**: AVFoundation  
**Kind**: init

Creates an export session with a preset configuration.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.1+
- macOS 10.7+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
init?(asset: AVAsset, presetName: String)
```

## Parameters

- `asset`: The asset to export.
- `presetName`: A string constant that specifies the preset template for the export. See   for available values.
