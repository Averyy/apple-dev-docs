# init(asset:)

**Framework**: AVFoundation  
**Kind**: init

Creates an object that generates images for times within a video asset.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.1+
- macOS 10.7+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
init(asset: AVAsset)
```

## Parameters

- `asset`: A video asset from which to generate images.
