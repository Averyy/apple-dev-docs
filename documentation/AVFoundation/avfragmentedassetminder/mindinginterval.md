# mindingInterval

**Framework**: AVFoundation  
**Kind**: property

An interval that specifies when to perform a check for additional fragments.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 12.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
var mindingInterval: TimeInterval { get set }
```

#### Discussion

The default interval in `10.0` seconds.
