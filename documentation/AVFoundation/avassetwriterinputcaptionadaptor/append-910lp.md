# append(_:)

**Framework**: AVFoundation  
**Kind**: method

**Availability**:

- iPadOS 18.0+
- Mac Catalyst 15.0+
- macOS 12.0+

## Declaration

#### Return Value

A Boolean value that indicates whether the operation succeeded.
