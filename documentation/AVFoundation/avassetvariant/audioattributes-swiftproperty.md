# audioAttributes

**Framework**: AVFoundation  
**Kind**: property

The audio rendition attributes for the variant.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
var audioAttributes: AVAssetVariant.AudioAttributes? { get }
```

#### Discussion

This property value is `nil` if the variant defines no audio attributes.
