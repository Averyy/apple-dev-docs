# preferredVolume

**Framework**: AVFoundation  
**Kind**: property

The asset’s volume preference for playing its audible media.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
var preferredVolume: Float { get set }
```

#### Discussion

This value is typically, but not always, 1.0.
