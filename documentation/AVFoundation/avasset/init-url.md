# init(url:)

**Framework**: AVFoundation  
**Kind**: init

Creates an asset that models the media at the specified URL.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.1+
- macOS 10.7+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 1.0+

## Declaration

```swift
convenience init(url URL: URL)
```

## Parameters

- `URL`: A URL to a local, remote, or HTTP Live Streaming media resource.
