# AVVideoAppleProRAWBitDepthKey

**Framework**: AVFoundation  
**Kind**: var

A key to access the Apple ProRAW bit depth.

**Availability**:
- iOS 14.3+
- iPadOS 14.3+
- Mac Catalyst 14.3+
- macOS 11.1+
- tvOS 14.3+
- visionOS 1.0+
- watchOS 7.2+

## Declaration

```swift
let AVVideoAppleProRAWBitDepthKey: String
```

#### Discussion

The value is an integer value from 8 to16 bits.
