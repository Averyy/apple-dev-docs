# trackGroups

**Framework**: AVFoundation  
**Kind**: property

The track groups an asset contains.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- macOS 10.9+
- tvOS 9.0+

## Declaration

```swift
var trackGroups: [AVAssetTrackGroup] { get }
```

#### Discussion

This value is an empty array if the composition has no track groups.
