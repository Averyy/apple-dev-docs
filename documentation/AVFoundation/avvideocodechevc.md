# AVVideoCodecHEVC

**Framework**: AVFoundation  
**Kind**: var

A key to access the name of the HEVC codec used to encode the video.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.13+
- tvOS 11.0+
