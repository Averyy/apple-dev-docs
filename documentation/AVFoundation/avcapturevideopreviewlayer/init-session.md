# init(session:)

**Framework**: AVFoundation  
**Kind**: init

Creates a layer to preview the visual output of a capture session.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 14.0+
- macOS 10.7+
- tvOS 17.0+

## Declaration

```swift
init(session: AVCaptureSession)
```

## Parameters

- `session`: A capture session to preview.
