# AVCaptureColorSpace.appleLog

**Framework**: AVFoundation  
**Kind**: case

The Apple Log Color space, which uses BT2020 as the color primaries, and an Apple-defined Log curve as a transfer function.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- tvOS 17.0+
