# captions

**Framework**: AVFoundation  
**Kind**: property

The captions to render.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 15.0+
- macOS 12.0+

## Declaration

```swift
var captions: [AVCaption] { get set }
```

#### Discussion

This property value is an empty array if there are no captions for the system to render.
