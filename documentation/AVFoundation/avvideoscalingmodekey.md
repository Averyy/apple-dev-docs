# AVVideoScalingModeKey

**Framework**: AVFoundation  
**Kind**: var

A key to retrieve the video scaling mode from a dictionary.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- Mac Catalyst 13.1+
- macOS 10.7+
- tvOS 9.0+
- visionOS 1.0+
