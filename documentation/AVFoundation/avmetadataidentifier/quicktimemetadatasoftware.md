# quickTimeMetadataSoftware

**Framework**: AVFoundation  
**Kind**: property

An identifier that represents the name of software used to create the movie file content.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 1.0+
