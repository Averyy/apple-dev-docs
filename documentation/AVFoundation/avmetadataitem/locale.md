# locale

**Framework**: AVFoundation  
**Kind**: property

The locale of the metadata item.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.1+
- macOS 10.7+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 1.0+

## Declaration

```swift
var locale: Locale? { get }
```

#### Discussion

The locale may be `nil` if no locale information is available for the metadata item.
