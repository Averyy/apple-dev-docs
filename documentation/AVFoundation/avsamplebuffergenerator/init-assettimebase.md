# init(asset:timebase:)

**Framework**: AVFoundation  
**Kind**: init

Creates a new sample buffer generator.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 10.10+
- tvOS 16.0+
- visionOS 1.0+
- watchOS 9.0+

## Declaration

```swift
init(asset: AVAsset, timebase: CMTimebase?)
```

#### Return Value

An initialized `AVSampleBufferGenerator` instance.

## Parameters

- `asset`: The asset.
- `timebase`: If  , requests will be handled synchronously.
