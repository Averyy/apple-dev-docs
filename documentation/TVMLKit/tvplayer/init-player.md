# init(player:)

**Framework**: TVMLKit  
**Kind**: init

Creates a new customizable player from an existing player.

**Availability**:
- tvOS 12.0+

## Declaration

```swift
init(player: AVPlayer)
```

## Parameters

- `player`: The player used to create the new player.
