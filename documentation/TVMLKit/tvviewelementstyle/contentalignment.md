# contentAlignment

**Framework**: TVMLKit  
**Kind**: property

A value indicating how items inside of an element are aligned.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
var contentAlignment: TVElementContentAlignment { get }
```

#### Discussion

This property aligns items inside of an element on the vertical axis. For possible values, see [`TVElementContentAlignment`](tvelementcontentalignment.md).
