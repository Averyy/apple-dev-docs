# value(propertyName:)

**Framework**: TVMLKit  
**Kind**: method

Returns the value associated with a given style name.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
func value(propertyName name: String) -> Any?
```

#### Return Value

An object containing the value associated with the style name.

## Parameters

- `name`: A user defined string representing the name of the style.
