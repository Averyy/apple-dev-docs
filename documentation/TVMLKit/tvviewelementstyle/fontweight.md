# fontWeight

**Framework**: TVMLKit  
**Kind**: property

A string indicating how thick or thin the font is.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
var fontWeight: String? { get }
```

#### Discussion

Valid values for this property are `bold`, `light`, `medium`, and `regular`.
