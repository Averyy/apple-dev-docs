# textMinimumScaleFactor

**Framework**: TVMLKit  
**Kind**: property

The minimum size text can be if the original text size does not fit in an element.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
var textMinimumScaleFactor: CGFloat { get }
```

#### Discussion

The minimum size is defined as a percentage of the original text size.
