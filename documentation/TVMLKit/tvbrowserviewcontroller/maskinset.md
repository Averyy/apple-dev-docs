# maskInset

**Framework**: TVMLKit  
**Kind**: property

The amount by which the content of the cell is inset.

**Availability**:
- tvOS 13.0+

## Declaration

```swift
@MainActor
var maskInset: UIEdgeInsets { get set }
```

#### Discussion

The default is `UIEdgeInsetsMake(32.0, 120.0, 0.0, 120.0)`.
