# cornerRadius

**Framework**: TVMLKit  
**Kind**: property

The corner radius, in points, of each full-screen browser item.

**Availability**:
- tvOS 13.0+

## Declaration

```swift
@MainActor
var cornerRadius: CGFloat { get set }
```

#### Discussion

The default value is 36.0 points.
