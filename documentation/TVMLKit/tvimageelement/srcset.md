# srcset

**Framework**: TVMLKit  
**Kind**: property

A dictionary specifying versions of the same image for different resolutions.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
var srcset: [String : URL]? { get }
```

#### Discussion

Each dictionary entry contains a string and a URL for a specific image.
