# wrapMode

**Framework**: USD

An option that determines the flow of the text.

#### Overview

The default value is `flowing`.

##### Wrap Modes

##### Declaration

```other
token wrapMode = "flowing" (
    allowedTokens = ["singleLine", "hardBreaks", "flowing"]
)
```
