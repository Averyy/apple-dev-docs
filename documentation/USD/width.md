# width

**Framework**: USD

The width of the text’s bounding box.

#### Overview

Supply a positive value for this property in stage linear units. For more information, see [`Encoding Stage Linear Units`](https://developer.apple.comhttps://openusd.org/docs/api/group___usd_geom_linear_units__group.html).

The runtime ignores this value if [`wrapMode`](wrapmode.md) is `singleLine`.

##### Declaration

```other
float width
```
