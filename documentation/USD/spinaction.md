# SpinAction

**Framework**: USD

An action that spins a prim.

#### Overview

The default configuration of this action spins an object horizontally on its axis, like a top.

##### Declaration

```other
class Preliminary_Action "SpinAction"
```

### Properties

  A vector that describes the axis of rotation.
