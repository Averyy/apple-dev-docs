# revocationDate

**Framework**: App Store Server API  
**Kind**: typealias

The UNIX time, in milliseconds, that the App Store refunded the transaction or revoked it from Family Sharing.

**Availability**:
- App Store Server API 1.0+

## Declaration

```swift
timestamp revocationDate
```
