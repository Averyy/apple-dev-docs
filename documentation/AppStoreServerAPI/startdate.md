# startDate

**Framework**: App Store Server API  
**Kind**: typealias

The start date of a timespan, expressed in UNIX time, in milliseconds.

**Availability**:
- App Store Server API 1.5+

## Declaration

```swift
timestamp startDate
```

#### Discussion

The start date must be earlier than the [`endDate`](enddate.md).
