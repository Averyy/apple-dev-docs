# endDate

**Framework**: App Store Server API  
**Kind**: typealias

The end date of a timespan, expressed in UNIX time, in milliseconds.

**Availability**:
- App Store Server API 1.5+

## Declaration

```swift
timestamp endDate
```

#### Discussion

The end date must be later than the [`startDate`](startdate.md).
