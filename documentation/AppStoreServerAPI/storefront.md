# storefront

**Framework**: App Store Server API  
**Kind**: typealias

The three-letter code that represents the country or region associated with the App Store storefront of the purchase.

**Availability**:
- App Store Server API 1.8+

## Declaration

```swift
string storefront
```

## Mentions

- [App Store Server API changelog](app-store-server-api-changelog.md)

#### Discussion

This property uses the ISO 3166-1 alpha-3 country code representation. This property is the same as the [`countryCode`](https://developer.apple.com/documentation/StoreKit/Storefront/countryCode) in StoreKit.
