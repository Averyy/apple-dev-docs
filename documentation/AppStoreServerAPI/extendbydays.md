# extendByDays

**Framework**: App Store Server API  
**Kind**: typealias

The number of days to extend the subscription renewal date.

**Availability**:
- App Store Server API 1.1+

## Declaration

```swift
int32 extendByDays
```

#### Discussion

The number of days is a number from 1 to 90.
