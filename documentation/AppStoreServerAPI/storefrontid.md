# storefrontId

**Framework**: App Store Server API  
**Kind**: typealias

An Apple-defined value that uniquely identifies an App Store storefront.

**Availability**:
- App Store Server API 1.8+

## Declaration

```swift
string storefrontId
```

## Mentions

- [App Store Server API changelog](app-store-server-api-changelog.md)

#### Discussion

This value is the same as the [`id`](https://developer.apple.com/documentation/storekit/storefront/id-n42) value in StoreKit.
