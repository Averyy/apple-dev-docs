# invalidate()

**Framework**: ContactProvider  
**Kind**: method  
**Required**: Yes

Invalidates the enumerator.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+

## Declaration

```swift
func invalidate() async
```

#### Discussion

The system calls this method before terminating the extension. The extension may terminate before this method returns.
