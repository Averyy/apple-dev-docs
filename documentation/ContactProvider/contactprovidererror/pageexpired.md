# ContactProviderError.pageExpired

**Framework**: ContactProvider  
**Kind**: case

The page expired and is no longer valid.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+

## Declaration

```swift
case pageExpired
```

#### Discussion

When this error occurs, the system discards locally cached data and restarts a content enumeration.
