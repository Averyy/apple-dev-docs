# functionID

**Framework**: CarKey  
**Kind**: property

The vehicle-specific code that identifies which feature you want to control.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.3+
- watchOS 9.0+

## Declaration

```swift
let functionID: FunctionIdentifier
```

#### Discussion

You define the function identifiers for your vehicles and what features they represent.
