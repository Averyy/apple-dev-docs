# init(application:)

**Framework**: XCTest  
**Kind**: init

Creates a CPU metric that records data for the requested app.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 7.0+

## Declaration

```swift
init(application: XCUIApplication)
```

## Parameters

- `application`: The application to be tested.
