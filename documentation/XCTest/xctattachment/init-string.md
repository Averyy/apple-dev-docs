# init(string:)

**Framework**: XCTest  
**Kind**: init

Creates an attachment containing the provided string.

## Declaration

```swift
convenience init(string: String)
```

#### Discussion

Creates an attachment with a [`uniformTypeIdentifier`](xctattachment/uniformtypeidentifier.md) of `"``public.plain-text"`.

## Parameters

- `string`: The string to wrap as an attachment.
