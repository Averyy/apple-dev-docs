# name

**Framework**: XCTest  
**Kind**: property

The attachment’s name, or `nil` if the attachment is unnamed.

## Declaration

```swift
var name: String? { get set }
```

#### Discussion

Set an attachment’s [`name`](xctattachment/name.md) property to a custom string to provide a descriptive name for the attachment within Xcode’s test reports.
