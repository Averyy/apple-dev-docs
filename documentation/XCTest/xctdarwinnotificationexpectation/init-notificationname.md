# init(notificationName:)

**Framework**: Xctest  
**Kind**: init

Creates an expectation that waits for a Darwin notification with the specified name to be posted.

## Declaration

```swift
init(notificationName: String)
```

## Parameters

- `notificationName`: The notification name to watch for.
