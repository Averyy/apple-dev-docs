# init(forBundlePath:)

**Framework**: Xctest  
**Kind**: init

Creates a test suite with the bundle at the specified path.

## Declaration

```swift
convenience init(forBundlePath bundlePath: String)
```

## Parameters

- `bundlePath`: A string that represents a file path to a bundle.
