# kCGPDFContextTitle

**Framework**: Core Graphics  
**Kind**: var

The corresponding value is a string that represents the title of the document. This key is optional.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.4+
- tvOS ?+
- visionOS 1.0+
- watchOS 2.0+
