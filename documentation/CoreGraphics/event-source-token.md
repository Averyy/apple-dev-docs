# Event Source Token

**Framework**: Core Graphics

Specifies any input event type.

#### Overview

This constant is typically used with the function [`secondsSinceLastEventType(_:eventType:)`](cgeventsource/secondssincelasteventtype(_:eventtype:).md) to specify that you want the elapsed time since the last input event of any type.
