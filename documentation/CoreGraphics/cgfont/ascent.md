# ascent

**Framework**: Core Graphics  
**Kind**: property

Returns the ascent of a font.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.5+
- tvOS ?+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
var ascent: Int32 { get }
```

#### Discussion

The ascent is the maximum distance above the baseline of glyphs in a font. The value is specified in glyph space  units.
