# setFillColor(_:)

**Framework**: Core Graphics  
**Kind**: method

Sets the current fill color in a graphics context, using a CGColor.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.3+
- tvOS ?+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
func setFillColor(_ color: CGColor)
```

## Parameters

- `color`: The new fill color.
