# init(domain:)

**Framework**: File Provider  
**Kind**: init  
**Required**: Yes

Creates an instance of the file provider for the specified domain.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- macOS 11.0+
- visionOS 1.0+

## Declaration

```swift
init(domain: NSFileProviderDomain)
```

## Parameters

- `domain`: The domain for the file provider.
