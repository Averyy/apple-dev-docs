# init(for:)

**Framework**: File Provider  
**Kind**: init

Returns a newly created file provider manager for the specified domain.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- macOS 11.0+
- visionOS 1.0+

## Declaration

```swift
convenience init?(for domain: NSFileProviderDomain)
```
