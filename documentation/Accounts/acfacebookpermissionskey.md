# ACFacebookPermissionsKey

**Framework**: Accounts  
**Kind**: var

The array of strings of permissions to request (for example, `email`, `publish_stream`, and so on).

**Availability**:
- iOS 6.0+
- iPadOS 6.0+
- Mac Catalyst 13.1+
- macOS 10.8+
