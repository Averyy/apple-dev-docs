# init(accountType:)

**Framework**: Accounts  
**Kind**: init

Initializes a new account of the specified type.

**Availability**:
- iOS 6.0+
- iPadOS 6.0+
- Mac Catalyst 13.1+
- macOS 10.8+

## Declaration

```swift
init!(accountType type: ACAccountType!)
```

#### Return Value

The newly initialized account.

## Parameters

- `type`: The type of this account.
