# GroupActivityActivationResult.cancelled

**Framework**: Group Activities  
**Kind**: case

A result that indicates the user canceled the activation request.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+

## Declaration

```swift
case cancelled
```

#### Discussion

When you receive this result, ignore the activity.
