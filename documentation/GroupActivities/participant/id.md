# id

**Framework**: Group Activities  
**Kind**: property

A globally unique identifier for the session participant.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+

## Declaration

```swift
let id: UUID
```

#### Discussion

Use this property to differentiate among the devices active in a session.
