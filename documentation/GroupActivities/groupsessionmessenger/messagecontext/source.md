# source

**Framework**: Group Activities  
**Kind**: property

The participant device that sent the message.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+

## Declaration

```swift
var source: Participant
```

#### Discussion

Each device has a unique participant. Use this property to determine the origin of the message.
