# kCMTextFormatDescriptionExtension_TextJustification

**Framework**: Core Media  
**Kind**: var

The extension flag that represents the text justification.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.1+
- macOS 10.7+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
let kCMTextFormatDescriptionExtension_TextJustification: CFString
```

#### Discussion

The value needs to be `SInt8` and containing a [`CMTextJustificationValue`](cmtextjustificationvalue.md).
