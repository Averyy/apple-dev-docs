# init(start:end:)

**Framework**: Core Media  
**Kind**: init

Creates a valid time range from a start and end time.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst ?+
- macOS 10.7+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
init(start: CMTime, end: CMTime)
```

## Parameters

- `start`: The start time of the range.
- `end`: The end time of the range.
