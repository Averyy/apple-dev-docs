# CMPixelFormatType

**Framework**: Core Media  
**Kind**: typealias

A pixel format type.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.1+
- macOS 10.7+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
typealias CMPixelFormatType = FourCharCode
```

#### Discussion

Only some codec types are pixel formats.
