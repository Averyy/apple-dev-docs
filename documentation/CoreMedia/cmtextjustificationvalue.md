# CMTextJustificationValue

**Framework**: Core Media  
**Kind**: typealias

An integer value that describes the justification modes for text media.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.1+
- macOS 10.7+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
typealias CMTextJustificationValue = Int8
```
