# CMAudioFormatDescriptionMask

**Framework**: Core Media  
**Kind**: typealias

A type for mask bits that represent parts of an audio format description.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.1+
- macOS 10.7+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 6.0+
