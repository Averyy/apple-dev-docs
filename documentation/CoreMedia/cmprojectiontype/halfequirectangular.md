# CMProjectionType.halfEquirectangular

**Framework**: Core Media  
**Kind**: case

Video content displays as a 180 degree equirectangular projection.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+
- watchOS 10.0+

## Declaration

```swift
case halfEquirectangular
```

#### Discussion

The video frames map onto the interior of a 3D half-sphere projection.
