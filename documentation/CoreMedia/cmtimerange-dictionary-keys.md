# Dictionary Keys

**Framework**: Core Media

Keys to use when working with dictionary representations of a time range.

#### Overview

To convert a `CMTimeRange` to and from a `CFDictionary`, see [`CMTimeRangeCopyAsDictionary(_:allocator:)`](cmtimerangecopyasdictionary(_:allocator:).md) and [`CMTimeRangeMakeFromDictionary(_:)`](cmtimerangemakefromdictionary(_:).md).

## Topics

### Constants
- [let kCMTimeRangeStartKey: CFString](kcmtimerangestartkey.md)
  The key for the start field of a time range.
- [let kCMTimeRangeDurationKey: CFString](kcmtimerangedurationkey.md)
  The key for the timescale of a time range.
