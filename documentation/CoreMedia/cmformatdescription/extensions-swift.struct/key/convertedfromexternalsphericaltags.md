# convertedFromExternalSphericalTags

**Framework**: Core Media  
**Kind**: property

Indicates that the image description contains projected media data converted from an external format.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst ?+
- macOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
static let convertedFromExternalSphericalTags: CMFormatDescription.Extensions.Key
```

#### Discussion

This format description extension is included if boxes containing external stereoscopic or spherical projection information were parsed.
