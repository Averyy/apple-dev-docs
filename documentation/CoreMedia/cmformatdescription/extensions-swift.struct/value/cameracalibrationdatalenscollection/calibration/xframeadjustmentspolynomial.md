# xFrameAdjustmentsPolynomial

**Framework**: Core Media  
**Kind**: property

Three element polynomial for mapping x axis UV parameters with an adjustment using the equation `x' = polynomialX[0] + polynomialX[1]*x + polynomialX[2]*x^3`.
