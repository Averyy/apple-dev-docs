# yFrameAdjustmentsPolynomial

**Framework**: Core Media  
**Kind**: property

Three element polynomial for mapping y axis UV parameters with an adjustment using the equation `y' = polynomialY[0] + polynomialY[1]*y + polynomialY[2]*y^3`.
