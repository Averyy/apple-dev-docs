# additionalViews

**Framework**: Core Media  
**Kind**: property

A flag indicating that the video content contains additional views beyond the left or right eye.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+
- watchOS 10.0+
