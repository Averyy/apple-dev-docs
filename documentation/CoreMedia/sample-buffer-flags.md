# Sample Buffer Flags

**Framework**: Core Media

### Constants
- [var kCMSampleBufferFlag_AudioBufferList_Assure16ByteAlignment: UInt32](kcmsamplebufferflag_audiobufferlist_assure16bytealignment.md)
  Indicates that memory involved in audio buffer lists is 16-byte aligned.
