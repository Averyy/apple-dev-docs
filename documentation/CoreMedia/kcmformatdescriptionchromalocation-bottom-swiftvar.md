# kCMFormatDescriptionChromaLocation_Bottom

**Framework**: Core Media  
**Kind**: var

The chroma sample is horizontally centered, but co-sited with the bottom row of the luma samples.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
