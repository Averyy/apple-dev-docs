# Metadata Registry Error Codes

**Framework**: Core Media

Error codes that indicate metadata registry errors.

## Topics

### Constants
- [var kCMMetadataDataTypeRegistryError_AllocationFailed: OSStatus](kcmmetadatadatatyperegistryerror_allocationfailed.md)
  An error that indicates the allocation fails.
- [var kCMMetadataDataTypeRegistryError_BadDataTypeIdentifier: OSStatus](kcmmetadatadatatyperegistryerror_baddatatypeidentifier.md)
  An error that indicates a bad datatype identifier.
- [var kCMMetadataDataTypeRegistryError_DataTypeAlreadyRegistered: OSStatus](kcmmetadatadatatyperegistryerror_datatypealreadyregistered.md)
  An error that indicates the datatype is in a registered state.
- [var kCMMetadataDataTypeRegistryError_MultipleConformingBaseTypes: OSStatus](kcmmetadatadatatyperegistryerror_multipleconformingbasetypes.md)
  An error that indicates that type you provide has more than one base data type.
- [var kCMMetadataDataTypeRegistryError_RequiredParameterMissing: OSStatus](kcmmetadatadatatyperegistryerror_requiredparametermissing.md)
  An error that indicates a parameter the function requires is empty.
- [var kCMMetadataDataTypeRegistryError_RequiresConformingBaseType: OSStatus](kcmmetadatadatatyperegistryerror_requiresconformingbasetype.md)
  An error that indicates the data type you specify requires a conforming data type that resolves to a base data type.
