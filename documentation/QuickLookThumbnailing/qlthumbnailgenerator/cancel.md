# cancel(_:)

**Framework**: Quick Look Thumbnailing  
**Kind**: method

Cancels the generation of a thumbnail for a given request.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS 10.15+
- visionOS 1.0+
