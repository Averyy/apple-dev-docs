# extensionBadge

**Framework**: Quick Look Thumbnailing  
**Kind**: property

A short string that identifies the file type that the system uses as a badge when producing an icon thumbnail.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- visionOS 1.0+
