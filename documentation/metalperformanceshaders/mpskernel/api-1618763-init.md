# init(device:)

**Framework**: Metal Performance Shaders  
**Kind**: instctr

Initializes a new kernel object.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.0+
- macOS 10.13+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
init(device: any MTLDevice)
```

#### Return_value

An initialized kernel object.

#### Discussion

This method fails if the device is not supported. Query the [`MPSSupportsMTLDevice(_:)`](1618849-mpssupportsmtldevice.md) function to determine whether the device is supported.

## Parameters

- `device`: The Metal device on which the kernel will be used.
