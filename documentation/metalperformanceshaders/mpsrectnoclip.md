# MPSRectNoClip

**Framework**: Metal Performance Shaders  
**Kind**: data

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.0+
- macOS 10.13+
- tvOS 11.0+
- visionOS 1.0+

## Declaration

```swift
let MPSRectNoClip: MTLRegion
```

#### Discussion

If this constant is used in an image filter operation, no clipping is done and the entire image is used.
