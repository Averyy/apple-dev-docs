# MPSAlphaType.nonPremultiplied

**Framework**: Metal Performance Shaders  
**Kind**: case

The image is not premultiplied by alpha.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.0+
- macOS 10.13+
- tvOS 10.0+
- visionOS 1.0+

## Declaration

```swift
case nonPremultiplied
```

#### Discussion

The image uses non-premultiplied alpha. Alpha is not guaranteed to be 1.
