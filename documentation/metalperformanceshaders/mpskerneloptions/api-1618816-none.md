# none

**Framework**: Metal Performance Shaders  
**Kind**: structdata

The default option for the kernel. Kernels created with this option will not skip any API validation and will not use reduced precision.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.0+
- macOS 10.13+
- tvOS 9.0+
- visionOS 1.0+
