# init(device:)

**Framework**: Metal Performance Shaders  
**Kind**: init

**Availability**:

- iPadOS 11.3+
- Mac Catalyst 13.0+
- macOS 10.13.4+
- tvOS 11.3+
- visionOS 1.0+

## Declaration


## Parameters

- `device`: The device that the filter runs on.
