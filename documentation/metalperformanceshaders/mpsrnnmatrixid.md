# MPSRNNMatrixId

**Framework**: Metal Performance Shaders  
**Kind**: enum

Options that define which matrix is copied in or out of a trainable RNN layer.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.0+
- macOS 10.14+
- tvOS 12.0+
- visionOS 1.0+
