# init(device:imageDescriptor:)

**Framework**: Metal Performance Shaders  
**Kind**: instctr

Initializes an empty image.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.0+
- macOS 10.13+
- tvOS 10.0+
- visionOS 1.0+

## Declaration

```swift
convenience init(device: any MTLDevice, imageDescriptor: MPSImageDescriptor)
```

#### Return_value

A valid [`MPSImage`](mpsimage.md) object or `nil`, if failure.

#### Discussion

Storage for the image data is allocated lazily on the first use of the [`MPSImage`](mpsimage.md) object, or when the [`texture`](mpsimage/1648903-texture.md) property is first read.

## Parameters

- `device`: The device on which the image will be used.
- `imageDescriptor`: The image descriptor.
