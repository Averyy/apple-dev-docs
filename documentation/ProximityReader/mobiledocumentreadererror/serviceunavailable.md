# MobileDocumentReaderError.serviceUnavailable

**Framework**: ProximityReader  
**Kind**: case

An error that occurs when mobile document reader service is unavailable.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- visionOS 1.0+
