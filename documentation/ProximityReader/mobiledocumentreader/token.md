# MobileDocumentReader.Token

**Framework**: ProximityReader  
**Kind**: struct

A reader token generated by your server.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- visionOS 1.0+

## Declaration

```swift
struct Token
```

#### Overview

Use the `Token` to create a [`MobileDocumentReaderSession`](mobiledocumentreadersession.md). You can cache the token and reuse it to create new sessions for up to 48 hours.
