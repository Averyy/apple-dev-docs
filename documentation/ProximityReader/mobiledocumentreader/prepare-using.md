# prepare(using:)

**Framework**: ProximityReader  
**Kind**: method

Configures the device to read mobile documents.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+

## Declaration

```swift
final func prepare(using token: MobileDocumentReader.Token? = nil) async throws -> MobileDocumentReaderSession
```

#### Return Value

[`MobileDocumentReaderSession`](mobiledocumentreadersession.md) when successful.

#### Discussion

This method prepares the device for mobile document reading.

This method returns a [`MobileDocumentReaderSession`](mobiledocumentreadersession.md) object, which you use to read mobile documents.

After calling this method, there are three possible scenarios:

- The method returns an existing [`MobileDocumentReaderSession`](mobiledocumentreadersession.md) immediately if that session is still valid.
- The method retrieves and returns a new [`MobileDocumentReaderSession`](mobiledocumentreadersession.md) from the server.
- This method throws a [`MobileDocumentReaderError`](mobiledocumentreadererror.md) if a problem occurs.

> **Note**: A [`MobileDocumentReaderError`](mobiledocumentreadererror.md) if the method fails to configure the device.
