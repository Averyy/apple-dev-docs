# init(vasMerchants:)

**Framework**: ProximityReader  
**Kind**: init

Creates a new options structure with the specified list of merchants.

**Availability**:
- iOS 15.4+
- iPadOS 15.4+
- Mac Catalyst 17.0+

## Declaration

```swift
init(vasMerchants: [VASRequest.Merchant] = [])
```

## Parameters

- `vasMerchants`: An array of merchants with supported loyalty cards.
