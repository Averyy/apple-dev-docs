# Emoji Rangers: Supporting Live Activities, interactivity, and animations

**Framework**: Widgetkit

Offer Live Activities, controls, animate data updates, and add interactivity to widgets.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- watchOS 11.0+
- Xcode 16.2+
