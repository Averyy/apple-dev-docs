# vibrant

**Framework**: Widgetkit  
**Kind**: property

The system desaturates the widget, making a monochrome version that it uses to create an adaptive, vibrant effect.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- watchOS 9.0+

## Declaration

```swift
static let vibrant: WidgetRenderingMode
```
