# init(context:)

**Framework**: Widgetkit  
**Kind**: init

Creates a new widget relevance that is relevant in a specific context.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- watchOS 11.0+

## Declaration

```swift
init(context: RelevantContext)
```

## Parameters

- `context`: The relevant context where this widget is relevant.
