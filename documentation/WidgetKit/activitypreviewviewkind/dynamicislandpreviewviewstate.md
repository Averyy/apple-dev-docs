# ActivityPreviewViewKind.DynamicIslandPreviewViewState

**Framework**: WidgetKit  
**Kind**: enum

Values that represent the different presentations of a Live Activity in the Dynamic Island for use in Xcode previews.

**Availability**:
- iOS 16.2+
- iPadOS 16.2+

## Declaration

```swift
@preconcurrency

```
