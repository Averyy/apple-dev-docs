# accented

**Framework**: Widgetkit  
**Kind**: property

Specifies that the `Image` should be included as part of the accented widget group.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- watchOS 11.0+

## Declaration

```swift
static let accented: WidgetAccentedRenderingMode
```

#### Discussion

It is equivalent to applying the `.widgetAccentable()` modifier to the `Image`.
