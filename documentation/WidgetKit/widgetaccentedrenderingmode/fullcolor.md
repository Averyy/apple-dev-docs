# fullColor

**Framework**: WidgetKit  
**Kind**: property

Specifies that the `Image` should be rendered at full color with no other color modifications. Only applies to iOS.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst ?+
- macOS 15.0+
- visionOS 26.0+
- watchOS 11.0+

## Declaration

```swift
static let fullColor: WidgetAccentedRenderingMode
```
