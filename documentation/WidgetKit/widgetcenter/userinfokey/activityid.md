# activityID

**Framework**: Widgetkit  
**Kind**: property

A key you use to access the activity ID if the widget represents a Live Activity.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- watchOS 9.0+
