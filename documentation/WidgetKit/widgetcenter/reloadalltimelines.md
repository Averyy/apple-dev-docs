# reloadAllTimelines()

**Framework**: Widgetkit  
**Kind**: method

Reloads the timelines for all configured widgets belonging to the containing app.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- watchOS 9.0+
