# simplified

**Framework**: WidgetKit  
**Kind**: property

The level of detail should be simplified.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst ?+
- visionOS 26.0+ (Beta)

## Declaration

```swift

```

## Mentions

#### Discussion

This suggests the app should lay out its content and show a simplified version of its content. On visionOS specifically, this signals that the user is at a further distance.
