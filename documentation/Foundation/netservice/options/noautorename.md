# noAutoRename

**Framework**: Foundation  
**Kind**: property

Specifies that the network service should not rename itself in the event of a name collision.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.2+
- tvOS 9.0+
- visionOS 1.0+
