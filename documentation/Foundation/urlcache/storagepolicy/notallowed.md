# URLCache.StoragePolicy.notAllowed

**Framework**: Foundation  
**Kind**: case

Storage in [`URLCache`](urlcache.md) is not allowed in any fashion, either in memory or on disk.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.0+
- macOS 10.0+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+
