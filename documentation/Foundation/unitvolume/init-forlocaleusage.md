# init(forLocale:usage:)

**Framework**: Foundation  
**Kind**: init

Creates a `UnitVolume` which the specified `locale` prefers for the specific `usage`.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+
