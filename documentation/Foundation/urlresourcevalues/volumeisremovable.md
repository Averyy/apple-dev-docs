# volumeIsRemovable

**Framework**: Foundation  
**Kind**: property

A Boolean value that indicates whether the volume’s media is removable from the drive mechanism.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 8.0+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+
