# open()

**Framework**: Foundation  
**Kind**: method

Opens the receiving stream.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.0+
- macOS 10.0+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
func open()
```

#### Discussion

A stream must be created before it can be opened. Once opened, a stream cannot be closed and reopened.
