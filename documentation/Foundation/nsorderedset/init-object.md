# init(object:)

**Framework**: Foundation  
**Kind**: init

Initializes a new ordered set with the object.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.0+
- macOS 10.0+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
convenience init(object: Any)
```

#### Return Value

A new ordered set that contains a single member, `object`.

## Parameters

- `object`: The object to add to the new ordered set
