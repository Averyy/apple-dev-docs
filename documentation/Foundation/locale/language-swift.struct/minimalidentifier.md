# minimalIdentifier

**Framework**: Foundation  
**Kind**: property

Returns a BCP-47 identifier in a minimalist form. Script and region may be omitted. For example, “zh-TW”, “en”

**Availability**:

- iPadOS 16.0+
- Mac Catalyst 16.0+
