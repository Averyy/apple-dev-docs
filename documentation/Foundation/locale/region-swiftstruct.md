# Locale.Region

**Framework**: Foundation  
**Kind**: struct

A type that represents a geographic region, for use in specifying a locale or language.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+
- watchOS 9.0+

## Declaration

```swift
struct Region
```
