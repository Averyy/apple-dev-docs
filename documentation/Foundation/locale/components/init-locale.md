# init(locale:)

**Framework**: Foundation  
**Kind**: init

Creates a language components instance from an existing locale.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+
- watchOS 9.0+

## Declaration

```swift
init(locale: Locale)
```

## Parameters

- `locale`: A locale from which to copy component values.
