# init(_:)

**Framework**: Foundation  
**Kind**: init

Creates a region from a BCP 47 identifier.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+
- watchOS 9.0+

## Declaration

```swift
init(_ identifier: String)
```

## Parameters

- `identifier`: A BCP 47 identifier, such as   for the United States. This parameter is case-insensitive.
