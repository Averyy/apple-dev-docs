# JSONDecoder.KeyDecodingStrategy.useDefaultKeys

**Framework**: Foundation  
**Kind**: case

A key decoding strategy that doesn’t change key names during decoding.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 8.0+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
case useDefaultKeys
```

#### Discussion

The [`JSONDecoder.KeyDecodingStrategy.useDefaultKeys`](jsondecoder/keydecodingstrategy-swift.enum/usedefaultkeys.md) strategy is the strategy used if you don’t specify one.
