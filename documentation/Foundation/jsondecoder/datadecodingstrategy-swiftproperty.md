# dataDecodingStrategy

**Framework**: Foundation  
**Kind**: property

The strategy that a decoder uses to decode raw data.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 8.0+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
var dataDecodingStrategy: JSONDecoder.DataDecodingStrategy { get set }
```

#### Discussion

The default strategy is the [`JSONDecoder.DataDecodingStrategy.base64`](jsondecoder/datadecodingstrategy-swift.enum/base64.md) strategy.
