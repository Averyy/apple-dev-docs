# init(symbol:)

**Framework**: Foundation  
**Kind**: init

Initializes a new unit with the specified symbol.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- macOS 10.12+
- tvOS 10.0+
- visionOS 1.0+
- watchOS 3.0+

## Declaration

```swift
init(symbol: String)
```

#### Return Value

A new unit with the specified symbol.

## Parameters

- `symbol`: The symbol used to represent the unit.
