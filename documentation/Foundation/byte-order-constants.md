# Byte Order

**Framework**: Foundation

Recognize constants that specify a byte-order format.

#### Overview

These constants are returned by [`NSHostByteOrder()`](nshostbyteorder().md).

## Topics

### Constants
- [var NS_BigEndian: Int](ns_bigendian.md)
  The byte order is big endian.
- [var NS_LittleEndian: Int](ns_littleendian.md)
  The byte order is little endian.
- [var NS_UnknownByteOrder: Int](ns_unknownbyteorder.md)
  The byte order is unknown.
