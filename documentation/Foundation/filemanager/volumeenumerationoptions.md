# FileManager.VolumeEnumerationOptions

**Framework**: Foundation  
**Kind**: struct

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.1+
- macOS 10.6+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+
