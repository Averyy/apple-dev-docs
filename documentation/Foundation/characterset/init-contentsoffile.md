# init(contentsOfFile:)

**Framework**: Foundation  
**Kind**: init

Initialize with the contents of a file.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 8.0+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
init?(contentsOfFile file: String)
```

#### Discussion

Returns `nil` if there was an error reading the file.

## Parameters

- `file`: The file to read.
