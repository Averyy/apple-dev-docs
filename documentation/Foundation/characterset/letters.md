# letters

**Framework**: Foundation  
**Kind**: property

Returns a character set containing the characters in Unicode General Category L* & M*.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 8.0+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

static var letters: CharacterSet { get }
