# commandDescription

**Framework**: Foundation  
**Kind**: property

Returns the command description for the command.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+

## Declaration

```swift
var commandDescription: NSScriptCommandDescription { get }
```

#### Discussion

Once a command is created, its command description is immutable.
