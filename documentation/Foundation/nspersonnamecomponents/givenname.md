# givenName

**Framework**: Foundation  
**Kind**: property

Name bestowed upon an individual to differentiate them from other members of a group that share a family name .

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
var givenName: String? { get set }
```

#### Discussion

In some locales, this is also known as a “first name” or “forename.”
