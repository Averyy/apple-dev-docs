# HTTPCookie.AcceptPolicy.always

**Framework**: Foundation  
**Kind**: case

Accept all cookies.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.0+
- macOS 10.0+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
case always
```

#### Discussion

This is the default cookie accept policy.
