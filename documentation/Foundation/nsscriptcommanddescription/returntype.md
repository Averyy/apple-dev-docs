# returnType

**Framework**: Foundation  
**Kind**: property

Returns the return type of the command.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+

## Declaration

```swift
var returnType: String? { get }
```

#### Return Value

The receiver’s command return type; for example, `"NSNumber"` or `"NSDictionary"`).
