# init(rawValue:)

**Framework**: Foundation  
**Kind**: init

Creates a set of JSON formatting options from an integer that represents those options.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- Mac Catalyst 13.1+
- macOS 10.7+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
init(rawValue: UInt)
```

## Parameters

- `rawValue`: An option set of reading options defined by this type.
