# fragmentsAllowed

**Framework**: Foundation  
**Kind**: property

Specifies that the parser allows top-level objects that aren’t arrays or dictionaries.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- Mac Catalyst 13.1+
- macOS 10.7+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
static var fragmentsAllowed: JSONSerialization.ReadingOptions { get }
```
