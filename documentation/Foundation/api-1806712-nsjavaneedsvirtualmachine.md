# NSJavaNeedsVirtualMachine

**Framework**: Foundation

Returns a Boolean value that indicates whether a Java virtual machine is required.

#### Overview

Returns [`true`](https://developer.apple.com/documentation/Swift/true) if `plist` contains a key saying that it requires Java.
