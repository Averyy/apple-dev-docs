# setUserInfoObject(_:forKey:)

**Framework**: Foundation  
**Kind**: method

Sets a value in the user info dictionary.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- macOS 10.9+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
func setUserInfoObject(_ objectOrNil: Any?, forKey key: ProgressUserInfoKey)
```

#### Discussion

Use this method to set a value in the [`userInfo`](progress/userinfo.md) dictionary, with appropriate KVO notification for properties with values that can depend on values in the user info dictionary, like [`localizedDescription`](progress/localizeddescription.md).

Supply a value of `nil` to remove an existing dictionary entry for a specified key.

## Parameters

- `objectOrNil`: The object to set for the specified key, or   to remove an existing entry in the dictionary.
- `key`: The key for storing the specified object.
