# estimatedTimeRemaining

**Framework**: Foundation  
**Kind**: property

A value that indicates the estimated amount of time remaining to complete the progress.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 11.0+
- macOS 10.13+
- tvOS 11.0+
- visionOS 1.0+
- watchOS 4.0+

## Declaration

```swift
var estimatedTimeRemaining: TimeInterval? { get set }
```

#### Discussion

If present, [`Progress`](progress.md) presents additional information in its localized description by setting a value in the `userInfo` dictionary.
