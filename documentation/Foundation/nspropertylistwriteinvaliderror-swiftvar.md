# NSPropertyListWriteInvalidError

**Framework**: Foundation  
**Kind**: var

Writing failed because of an invalid property list object, or an invalid property list type was specified.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
