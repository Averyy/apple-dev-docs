# NSJavaObjectNamedInPath

**Framework**: Foundation

Creates an instance of the named class using the class loader previously specified at the given path.

#### Overview

Returns a new instance of the class `name`. The class loader must be already be set up for the specified `path` (you can do this using a function such as doc:object_runtime/java_support/1806697-nsjavaclassesfrompath).
