# subjectForm

**Framework**: Foundation  
**Kind**: property

The subject pronoun form to apply when using this custom pronoun behavior.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
var subjectForm: String? { get set }
```

#### Discussion

In the English phrase “she reads,” the pronoun “she” exhibits the subject form.
