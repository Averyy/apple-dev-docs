# string(for:)

**Framework**: Foundation  
**Kind**: method

Returns a string formatted to contain the data from an offset structure.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.0+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
class func string(for offset: UIOffset) -> String
```

#### Return Value

A string that corresponds to `offset`.

## Parameters

- `offset`: A UIKit offset data structure.
