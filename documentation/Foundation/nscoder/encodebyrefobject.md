# encodeByrefObject(_:)

**Framework**: Foundation  
**Kind**: method

An encoding method for subclasses to override such that it creates a proxy, rather than a copy, when decoded.

- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.0+
- macOS 10.0+
- tvOS 9.0+
