# soundName

**Framework**: Foundation  
**Kind**: property

Specifies the name of the sound to play when the notification is delivered.

**Availability**:
- macOS 10.8+

## Declaration

```swift
var soundName: String? { get set }
```

#### Discussion

Passing the [`NSUserNotificationDefaultSoundName`](nsusernotificationdefaultsoundname.md) constant causes the default notification center sound to be played.

A value of `nil` means no sound is played.
