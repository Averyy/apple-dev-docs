# isRemote

**Framework**: Foundation  
**Kind**: property

Specifies whether the remote was generated by a push notification.

**Availability**:
- macOS 10.8+

## Declaration

```swift
var isRemote: Bool { get }
```

#### Discussion

If this property is [`true`](https://developer.apple.com/documentation/swift/true) then the user notification was generated by a push notification (that is, remotely); if [`false`](https://developer.apple.com/documentation/swift/false) it was generated locally.
