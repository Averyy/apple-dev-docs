# NSSpecifierTest.TestComparisonOperation.greaterThan

**Framework**: Foundation  
**Kind**: case

Binary comparison operator that results in [`true`](https://developer.apple.com/documentation/swift/true) if the value of the test object is greater than the value of the other object.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+
