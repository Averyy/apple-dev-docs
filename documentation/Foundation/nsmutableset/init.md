# init()

**Framework**: Foundation  
**Kind**: init

Initializes a newly allocated set.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.0+
- macOS 10.0+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
init()
```

#### Return Value

A set.

#### Discussion

This method is a designated initializer of `NSMutableSet`.
