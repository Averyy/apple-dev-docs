# NSURLAuthenticationMethodNegotiate

**Framework**: Foundation  
**Kind**: var

Negotiate whether to use Kerberos or NTLM authentication for this protection space.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.5+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+
