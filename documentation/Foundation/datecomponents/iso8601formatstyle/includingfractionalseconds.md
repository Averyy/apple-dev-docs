# includingFractionalSeconds

**Framework**: Foundation  
**Kind**: property

If set, fractional seconds will be present in formatted output. Fractional seconds may be present in parsing regardless of the setting of this property.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+
- watchOS 26.0+
