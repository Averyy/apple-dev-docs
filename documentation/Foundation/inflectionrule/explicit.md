# InflectionRule.explicit(_:)

**Framework**: Foundation  
**Kind**: case

An inflection rule that uses a morphology instance to determine how to inflect attribued strings.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
case explicit(Morphology)
```

## Parameters

- `Morphology`: The   instance to use when applying this rule.
