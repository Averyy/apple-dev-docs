# timeZone

**Framework**: Foundation  
**Kind**: property

The time zone used to create and parse date representations.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
var timeZone: TimeZone { get set }
```

#### Discussion

The default time zone is Greenwich Mean Time (GMT).
