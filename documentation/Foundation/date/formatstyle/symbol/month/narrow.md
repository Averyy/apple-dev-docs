# narrow

**Framework**: Foundation  
**Kind**: property

The shortest representation of a month.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
static var narrow: Date.FormatStyle.Symbol.Month { get }
```

#### Discussion

This custom format style conveys the shortest representation of a month. For example, it may represent September as `S`.
