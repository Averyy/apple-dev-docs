# threeDigits

**Framework**: Foundation  
**Kind**: property

Custom format style portraying the three-digit numeric day of the year, zero-padded if necessary.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
static var threeDigits: Date.FormatStyle.Symbol.DayOfYear { get }
```

#### Discussion

For example, `001`, `018`, `317`.
