# wide

**Framework**: Foundation  
**Kind**: property

The full representation of a standalone month.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
static var wide: Date.FormatStyle.Symbol.StandaloneMonth { get }
```

#### Discussion

This custom format style conveys the complete representation of a month, like `October` for October.
