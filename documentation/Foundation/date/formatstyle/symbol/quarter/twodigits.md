# twoDigits

**Framework**: Foundation  
**Kind**: property

The two-digit representation of a quarter.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
static var twoDigits: Date.FormatStyle.Symbol.Quarter { get }
```

#### Discussion

This custom format style conveys a two-digit representation of a quarter. For example, `02`, for the second quarter.
