# abbreviated

**Framework**: Foundation  
**Kind**: property

The abbreviated representation of a quarter.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
static var abbreviated: Date.FormatStyle.Symbol.Quarter { get }
```

#### Discussion

This custom format style conveys an abbreviated representation of a quarter. For example, `Q2`.
