# init(timeIntervalSince1970:)

**Framework**: Foundation  
**Kind**: init

Creates a date value initialized relative to 00:00:00 UTC on 1 January 1970 by a given number of seconds.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 8.0+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
init(timeIntervalSince1970: TimeInterval)
```
