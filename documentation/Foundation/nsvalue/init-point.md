# init(point:)

**Framework**: Foundation  
**Kind**: init

Creates a new value object containing the specified Foundation point structure.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+

## Declaration

```swift
init(point: NSPoint)
```

#### Return Value

A new value object that contains the point information.

## Parameters

- `point`: The value for the new object.
