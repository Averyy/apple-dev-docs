# init(range:)

**Framework**: Foundation  
**Kind**: init

Creates a new value object containing the specified Foundation range structure.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.0+
- macOS 10.0+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
init(range: NSRange)
```

#### Return Value

A new value object that contains the range information.

## Parameters

- `range`: The value for the new object.
