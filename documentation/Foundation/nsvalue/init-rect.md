# init(rect:)

**Framework**: Foundation  
**Kind**: init

Creates a new value object containing the specified Foundation rectangle structure.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+

## Declaration

```swift
init(rect: NSRect)
```

#### Return Value

A new value object that contains the data in the `rect` structure.

## Parameters

- `rect`: The value for the new object.
