# append(_:)

**Framework**: Foundation  
**Kind**: method

Appends the content of another data object to the receiver.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.0+
- macOS 10.0+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
func append(_ other: Data)
```

## Parameters

- `other`: The data object whose content is to be appended to the contents of the receiver.
