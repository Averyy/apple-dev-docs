# appending(queryItems:)

**Framework**: Foundation  
**Kind**: method

Returns a new URL formed by appending a list of query items to the URL.

**Availability**:

- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+


## Parameters

- `queryItems`: An array of   instances to append to the URL.
