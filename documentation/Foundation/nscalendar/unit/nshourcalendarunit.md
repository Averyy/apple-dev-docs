# NSHourCalendarUnit

**Framework**: Foundation  
**Kind**: property

Specifies the hour unit.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.4+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
static var NSHourCalendarUnit: NSCalendar.Unit { get }
```

#### Discussion

The corresponding value is an `NSInteger`. Equal to `kCFCalendarUnitHour`.
