# hour

**Framework**: Foundation  
**Kind**: property

Identifier for the hour unit.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.0+
- macOS 10.0+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
static var hour: NSCalendar.Unit { get }
```

#### Discussion

The corresponding value is an `NSInteger`. Equal to `kCFCalendarUnitHour`.
