# iso8601

**Framework**: Foundation  
**Kind**: property

A style for formatting a date in accordance with the ISO-8601 standard.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
static var iso8601: Date.ISO8601FormatStyle { get }
```

#### Discussion

Use the dot-notation form of this type property when the call point allows the use of [`Date.ISO8601FormatStyle`](date/iso8601formatstyle.md); in other words, when the value type is `Date`. Typically, you use this with the [`formatted(_:)`](date/formatted(_:).md) method of `Date`.
