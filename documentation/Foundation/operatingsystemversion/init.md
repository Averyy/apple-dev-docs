# init()

**Framework**: Foundation  
**Kind**: init

Creates an empty operating system version.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.0+
- macOS 10.0+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
init()
```

#### Discussion

After initialization, the [`majorVersion`](operatingsystemversion/majorversion.md), [`minorVersion`](operatingsystemversion/minorversion.md), and [`patchVersion`](operatingsystemversion/patchversion.md) are all `0`.
