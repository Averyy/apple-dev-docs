# NSLocaleSensitivePredicateOption

**Framework**: Foundation

Indicates that strings to be compared using `<`, `<=`, `=`, `=>`, `>` should be handled in a locale-aware fashion.

#### Overview

You represent this option in a predicate format string using a `[l]` following one of the `<`, `<=`, `=`, `=>`, `>` operators (for example, `"straÃŸe" >[l] "strasse"`).
