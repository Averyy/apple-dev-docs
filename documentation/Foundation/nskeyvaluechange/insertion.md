# NSKeyValueChange.insertion

**Framework**: Foundation  
**Kind**: case

Indicates that an object has been inserted into the to-many relationship that is being observed.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.0+
- macOS 10.0+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
case insertion
```
