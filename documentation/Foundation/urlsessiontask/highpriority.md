# highPriority

**Framework**: Foundation  
**Kind**: property

A high URL session task priority, with a floating point value above the default value and below the maximum of `1.0`.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+
