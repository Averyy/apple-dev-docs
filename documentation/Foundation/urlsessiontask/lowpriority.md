# lowPriority

**Framework**: Foundation  
**Kind**: property

A low URL session task priority, with a floating point value above the minimum of `0` and below the default value.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+
