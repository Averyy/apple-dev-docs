# init()

**Framework**: Foundation  
**Kind**: init

Initializes a newly allocated mutable ordered set.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- Mac Catalyst 13.1+
- macOS 10.7+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
init()
```

#### Return Value

A mutable ordered set.

#### Discussion

This method is a designated initializer of `NSMutableOrderedSet`.
