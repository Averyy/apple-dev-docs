# DateFormatter.Style.long

**Framework**: Foundation  
**Kind**: case

**Availability**:

- iPadOS 2.0+
- Mac Catalyst 13.0+
- macOS 10.0+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
case long
```

#### Discussion

Specifies a long style, typically with full text, such as “November 23, 1937” or “3:30:32 PM PST”.

Equal to `kCFDateFormatterLongStyle`.
