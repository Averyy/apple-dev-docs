# portKey

**Framework**: Foundation  
**Kind**: property

Value is an `NSNumber` object containing an integer that represents the port on which the proxy listens.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.3+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
static let portKey: StreamSOCKSProxyConfiguration
```
