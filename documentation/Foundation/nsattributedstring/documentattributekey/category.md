# category

**Framework**: Foundation  
**Kind**: property

The document’s category.

**Availability**:
- macOS 10.6+

## Declaration

```swift
static let category: NSAttributedString.DocumentAttributeKey
```

#### Discussion

The value of this attribute is an [`NSString`](nsstring.md) object containing the document’s category.
