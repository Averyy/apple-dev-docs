# imageURL

**Framework**: Foundation  
**Kind**: property

The URL for an image in Markdown text.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
static let imageURL: NSAttributedString.Key
```

#### Discussion

The value of this key is an [`NSURL`](nsurl.md) with the link to the image.
