# suggestionHighlight

**Framework**: Foundation  
**Kind**: property

A highlight associated with a Spotlight suggestion.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- visionOS 1.0+

## Declaration

```swift
static let suggestionHighlight: NSAttributedString.Key
```

#### Discussion

The value of this attribute is an [`NSString`](nsstring.md) object with the text that Spotlight suggested.
