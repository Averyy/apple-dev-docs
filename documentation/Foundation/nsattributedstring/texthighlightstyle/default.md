# default

**Framework**: Foundation  
**Kind**: property

The default highlight style to apply to text.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+

## Declaration

```swift
static let `default`: NSAttributedString.TextHighlightStyle
```

#### Discussion

Use this constant as the value for the [`textHighlightStyle`](nsattributedstring/key/texthighlightstyle.md) attribute. The system applies the default highlight color to your text. To specify a different highlight color, add the [`textHighlightColorScheme`](nsattributedstring/key/texthighlightcolorscheme.md) attribute to your text and set its value to the color you want.
