# letterpressStyle

**Framework**: Foundation  
**Kind**: property

A graphical text effect that gives glyphs the appearance of letterpress printing, which involves pressing the type into the paper.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- macOS 10.10+
- tvOS 9.0+
