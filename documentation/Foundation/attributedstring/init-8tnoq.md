# init(_:)

**Framework**: Foundation  
**Kind**: init

Creates an attributed string from an attributed substring.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
init(_ substring: AttributedSubstring)
```

## Parameters

- `substring`: An attributed substring to create the new attributed string from.
