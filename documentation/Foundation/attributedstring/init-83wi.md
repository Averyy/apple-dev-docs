# init(_:)

**Framework**: Foundation  
**Kind**: init

- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+

## Declaration

```swift
init(_ substring: DiscontiguousAttributedSubstring)
```

## Parameters

- `substring`: A discontiguous attributed substring to create the new attributed string from.
