# inflected()

**Framework**: Foundation  
**Kind**: method

Applies automatic grammar agreement inflection rules to the attributed string and returns the result.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
func inflected() -> AttributedString
```

#### Return Value

An attributed string that represents the effect of applying inflection rules to the source attributed string.
