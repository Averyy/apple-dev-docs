# init()

**Framework**: Foundation  
**Kind**: init

Initializes an interval with start and end dates set to the current date and the duration set to `0`.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 10.0+
- macOS 10.12+
- tvOS 10.0+
- visionOS 1.0+
- watchOS 3.0+
