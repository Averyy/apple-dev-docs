# NSMakeSize(_:_:)

**Framework**: Foundation  
**Kind**: func

Returns a new `NSSize` from the specified values.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+

## Declaration

```swift
func NSMakeSize(_ w: Double, _ h: Double) -> NSSize
```

#### Return Value

An `NSSize` having the specified `width` and `height`.
