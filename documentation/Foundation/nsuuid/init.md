# init()

**Framework**: Foundation  
**Kind**: init

Initializes a new UUID with RFC 4122 version 4 random bytes.

**Availability**:
- iOS 6.0+
- iPadOS 6.0+
- Mac Catalyst 13.1+
- macOS 10.8+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
init()
```

#### Return Value

A new UUID object.
