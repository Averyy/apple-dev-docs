# NSURLProtectionSpace protocol types

**Framework**: Foundation

These constants describe the supported protocols for a protection space, as returned by [`protocol`](urlprotectionspace/protocol.md).

## Topics

### Protocol types
- [let NSURLProtectionSpaceHTTP: String](nsurlprotectionspacehttp.md)
  The protocol type for HTTP.
- [let NSURLProtectionSpaceHTTPS: String](nsurlprotectionspacehttps.md)
  The protocol type for HTTPS.
- [let NSURLProtectionSpaceFTP: String](nsurlprotectionspaceftp.md)
  The protocol type for FTP.
