# init(options:)

**Framework**: Foundation  
**Kind**: init

Returns an `NSPointerFunctions` object initialized with the given options.

**Availability**:
- iOS 6.0+
- iPadOS 6.0+
- Mac Catalyst 13.1+
- macOS 10.5+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
init(options: NSPointerFunctions.Options = [])
```

#### Return Value

The `NSPointerFunctions` object, initialized with the given options.

## Parameters

- `options`: The options for the new   object.
