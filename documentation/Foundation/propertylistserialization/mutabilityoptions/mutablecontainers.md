# mutableContainers

**Framework**: Foundation  
**Kind**: property

Causes the returned property list to have mutable containers but immutable leaves.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.0+
- macOS 10.0+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+
