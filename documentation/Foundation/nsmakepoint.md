# NSMakePoint(_:_:)

**Framework**: Foundation  
**Kind**: func

Creates a new `NSPoint` from the specified values.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.0+

## Declaration

```swift
func NSMakePoint(_ x: Double, _ y: Double) -> NSPoint
```

#### Return Value

An `NSPoint` having the coordinates `x` and `y`.
