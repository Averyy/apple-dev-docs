# ListFormatStyle.Width.narrow

**Framework**: Foundation  
**Kind**: case

Specifies a narrow list style, the shortest list style.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
case narrow
```

#### Discussion

This width creates a list like `One, Two, Three` in U.S. English.

The rawValue of `narrow` is 2.
