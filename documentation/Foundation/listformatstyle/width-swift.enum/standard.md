# ListFormatStyle.Width.standard

**Framework**: Foundation  
**Kind**: case

Specifies a standard list style.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
case standard
```

#### Discussion

This width creates a list like `One, Two, and Three` in U.S. English.

The rawValue of `standard` is 0.
