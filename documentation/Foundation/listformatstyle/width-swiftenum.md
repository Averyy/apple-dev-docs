# ListFormatStyle.Width

**Framework**: Foundation  
**Kind**: enum

The type representing the width of a list.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+

## Declaration

```swift
enum Width
```

#### Overview

The possible values of a [`width`](listformatstyle/width-swift.property.md) are `standard`, `short`, and `narrow`.
