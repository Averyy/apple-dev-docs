# memoryFailure

**Framework**: Foundation  
**Kind**: property

During a page fault, the target address refers to a memory object that has been destroyed.  This failure is permanent.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 8.0+
- macOS 10.10+
