# init(url:)

**Framework**: Foundation  
**Kind**: init

Initializes a metadata item with a given URL.

**Availability**:
- macOS 10.9+

## Declaration

```swift
init?(url: URL)
```

#### Return Value

A metadata item for the file identified by `url`.

## Parameters

- `url`: The URL for the metadata item.
