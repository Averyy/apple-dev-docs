# quotationBeginDelimiterKey

**Framework**: Foundation  
**Kind**: property

The begin quotation symbol associated with the locale.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.1+
- macOS 10.6+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
static let quotationBeginDelimiterKey: NSLocale.Key
```

#### Discussion

The corresponding value is an `NSString` object; for example, `"“"`, `"„"`, `"«"`, or `"「"`.
