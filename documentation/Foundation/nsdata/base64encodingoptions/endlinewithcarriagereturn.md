# endLineWithCarriageReturn

**Framework**: Foundation  
**Kind**: property

When a maximum line length is set, specify that the line ending to insert should include a carriage return.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- macOS 10.9+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+
