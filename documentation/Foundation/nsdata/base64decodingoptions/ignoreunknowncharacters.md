# ignoreUnknownCharacters

**Framework**: Foundation  
**Kind**: property

Modify the decoding algorithm so that it ignores unknown non-Base-64 bytes, including line ending characters.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- macOS 10.9+
- tvOS 9.0+
- visionOS 1.0+
