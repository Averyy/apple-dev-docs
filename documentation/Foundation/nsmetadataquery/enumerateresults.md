# enumerateResults(_:)

**Framework**: Foundation  
**Kind**: method

Enumerates the current set of results using the given block.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- macOS 10.9+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
func enumerateResults(_ block: (Any, Int, UnsafeMutablePointer<ObjCBool>) -> Void)
```

#### Discussion

This method disables the query at the start of the iteration and reenables it upon completion. Use [`enumerateResults(options:using:)`](nsmetadataquery/enumerateresults(options:using:).md) if you want to use concurrent or reverse iteration.

## Parameters

- `block`: The block to execute for each current result.
