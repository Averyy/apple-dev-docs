# UTI Subtypes for Data Detector Types

**Framework**: Foundation

These constants represent sub-Uniform Type Identifier of `com.apple.structured-text`

#### Overview

These constants represent sub-[`Uniform Type Identifier`](https://developer.apple.comhttps://developer.apple.com/library/archive/documentation/General/Conceptual/DevPedia-CocoaCore/UniformTypeIdentifier.html#//apple_ref/doc/uid/TP40008195-CH60) of `com.apple.structured-text`, which conforms to `public.plain-text`. Use these constants to specify the type of data your extension works with.
