# familyName

**Framework**: Foundation  
**Kind**: property

Name bestowed upon an individual to denote membership in a group or family.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 9.0+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
var familyName: String? { get set }
```

#### Discussion

An example of a family name would be Ruiz.
