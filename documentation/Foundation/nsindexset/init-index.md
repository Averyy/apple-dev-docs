# init(index:)

**Framework**: Foundation  
**Kind**: init

Initializes an allocated [`NSIndexSet`](nsindexset.md) object with an index.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.0+
- macOS 10.0+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
convenience init(index value: Int)
```

#### Return Value

Initialized [`NSIndexSet`](nsindexset.md) object with `index`.

## Parameters

- `value`: An index. Must be in the range  .
