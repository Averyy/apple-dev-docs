# NumberFormatter.Style.scientific

**Framework**: Foundation  
**Kind**: case

A scientific style format.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.0+
- macOS 10.0+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
case scientific
```

#### Discussion

For example, in the en_US locale, the number 1234.5678 is represented as 1.2345678E3.
