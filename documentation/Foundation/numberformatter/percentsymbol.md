# percentSymbol

**Framework**: Foundation  
**Kind**: property

The string used to represent a percent symbol.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.0+
- macOS 10.0+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
var percentSymbol: String! { get set }
```

#### Discussion

By default, this property is set to the percent sign (%).
