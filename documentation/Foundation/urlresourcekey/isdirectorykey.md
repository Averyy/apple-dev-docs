# isDirectoryKey

**Framework**: Foundation  
**Kind**: property

A key for determining whether the resource is a directory.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.1+
- macOS 10.6+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
static let isDirectoryKey: URLResourceKey
```

#### Discussion

The corresponding value is a read-only Boolean `NSNumber` object.
