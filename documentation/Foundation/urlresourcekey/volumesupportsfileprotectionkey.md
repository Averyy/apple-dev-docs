# volumeSupportsFileProtectionKey

**Framework**: Foundation  
**Kind**: property

A Boolean value that indicates the volume supports data protection for files.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+
- watchOS 7.0+
