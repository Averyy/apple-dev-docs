# volumeIsEncryptedKey

**Framework**: Foundation  
**Kind**: property

A key for determining whether the volume is encrypted.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- macOS 10.12+
- tvOS 10.0+
- visionOS 1.0+
- watchOS 3.0+

## Declaration

```swift
static let volumeIsEncryptedKey: URLResourceKey
```

#### Discussion

The corresponding value is a Boolean `NSNumber` object.
