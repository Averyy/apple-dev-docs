# systemVersion

**Framework**: Foundation  
**Kind**: property

The system version number in effect when the archive was created.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.0+

## Declaration

```swift
var systemVersion: UInt32 { get }
```

#### Discussion

This information is available as soon as the receiver has been initialized.
