# DateIntervalFormatter.Style.none

**Framework**: Foundation  
**Kind**: case

No information for the date or time. Use this style when you do not want to include date or time information in the resulting string.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+
