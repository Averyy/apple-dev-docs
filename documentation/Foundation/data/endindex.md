# endIndex

**Framework**: Foundation  
**Kind**: property

The end index into the data.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 8.0+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
var endIndex: Data.Index { get }
```

#### Discussion

This is the “one-past-the-end” position, and will always be equal to the `count`.
