# popFirst()

**Framework**: Foundation  
**Kind**: method

Removes and returns the first element of the collection.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.0+
- macOS 10.0+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
mutating func popFirst() -> Self.Element?
```

#### Return Value

The first element of the collection if the collection is not empty; otherwise, `nil`.

#### Discussion

> **Note**: O(1)
