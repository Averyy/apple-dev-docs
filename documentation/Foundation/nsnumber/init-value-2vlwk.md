# init(value:)

**Framework**: Foundation  
**Kind**: init

Returns an `NSNumber` object initialized to contain a given value, treated as a `float`.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.0+
- macOS 10.0+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
init(value: Float)
```

#### Return Value

An `NSNumber` object containing `value`, treating it as a `float`.

## Parameters

- `value`: The value for the new number.
