# estimatedTimeRemainingKey

**Framework**: Foundation  
**Kind**: property

A key with a corresponding value that represents the time remaining, in seconds.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- macOS 10.9+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
static let estimatedTimeRemainingKey: ProgressUserInfoKey
```

#### Discussion

If present, the autogenerated [`localizedAdditionalDescription`](progress/localizedadditionaldescription.md) uses the value to provide a string that presents more information.
