# Classes Bridged to Swift Standard Library Value Types

- [class NSMutableString](nsmutablestring.md)

### Arrays
- [class NSArray](nsarray.md)
  A static ordered collection of objects.
- [class NSMutableArray](nsmutablearray.md)

### Dictionaries
- [class NSDictionary](nsdictionary.md)
  A static collection of objects associated with unique keys.
- [class NSMutableDictionary](nsmutabledictionary.md)

### Sets
- [class NSSet](nsset.md)
  A static, unordered collection of unique objects.
- [class NSMutableSet](nsmutableset.md)
  A dynamic unordered collection of unique objects.
