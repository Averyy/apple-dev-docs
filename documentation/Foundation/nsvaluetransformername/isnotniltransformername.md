# isNotNilTransformerName

**Framework**: Foundation  
**Kind**: property

This value transformer returns true if the value is non-nil.

**Availability**:
- iOS 3.0+
- iPadOS 3.0+
- Mac Catalyst 13.1+
- macOS 10.3+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
static let isNotNilTransformerName: NSValueTransformerName
```

#### Discussion

This transformer is not reversible.
