# init(signOf:magnitudeOf:)

**Framework**: Foundation  
**Kind**: init

Creates and initializes a decimal with the sign and magnitude of the given decimals.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 8.0+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
init(signOf: Decimal, magnitudeOf magnitude: Decimal)
```

## Parameters

- `signOf`: A   to use for the sign of the newly-created  .
- `magnitude`: A   to use for the magnitude of the newly-created  .
