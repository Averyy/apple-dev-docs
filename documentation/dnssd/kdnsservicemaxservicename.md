# kDNSServiceMaxServiceName

**Framework**: dnssd  
**Kind**: var

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.0+
- macOS 10.12+
- tvOS 10.0+
- visionOS 1.0+
- watchOS 3.0+

## Declaration

```swift
var kDNSServiceMaxServiceName: Int32 { get }
```

#### Discussion

Maximum length, in bytes, of a service name represented as a literal C string, including the terminating NULL at the end.
