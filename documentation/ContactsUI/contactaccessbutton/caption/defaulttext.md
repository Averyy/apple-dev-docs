# ContactAccessButton.Caption.defaultText

**Framework**: Contacts UI  
**Kind**: case

A caption option that displays a default value.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+

## Declaration

```swift
case defaultText
```

#### Discussion

The default option omits the caption from the contact access button, showing only the contact name.
