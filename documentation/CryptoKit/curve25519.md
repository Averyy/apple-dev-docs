# Curve25519

**Framework**: Apple CryptoKit  
**Kind**: enum

An elliptic curve that enables X25519 key agreement and Ed25519 signatures.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+

## Topics

### Performing operations
- [Curve25519.KeyAgreement](curve25519/keyagreement.md)

- [Curve25519.Signing](curve25519/signing.md)
