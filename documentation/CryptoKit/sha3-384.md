#### Overview

The [`SHA3_384`](sha3_384.md) hash implements the [`HashFunction`](hashfunction.md) protocol for the specific case of SHA-3 hashing with a 384-bit digest ([`SHA3_384Digest`](sha3_384digest.md)). Larger digests take more space but are more secure.

Applications can compute the digest by calling the static `hash(data:)` method once. Alternatively, if the data to hash is too large to fit in memory, applications can compute the digest iteratively by creating a new hash instance, calling the `update(data:)` method repeatedly with blocks of data, and then calling the [`finalize()`](sha3_384/finalize().md) method to get the result.
