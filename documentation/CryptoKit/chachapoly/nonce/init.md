# init()

**Framework**: Apple CryptoKit  
**Kind**: init

Creates a new random nonce.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
init()
```

#### Discussion

The default nonce is a 12-byte random nonce.
