# ==(_:_:)

**Framework**: Apple CryptoKit  
**Kind**: op

Determines whether two digests are equal.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
static func == (lhs: Self, rhs: Self) -> Bool
```

#### Return Value

A Boolean value set to `true` if the two digests are equal.

## Parameters

- `lhs`: The first digest to compare.
- `rhs`: The second digest to compare.
