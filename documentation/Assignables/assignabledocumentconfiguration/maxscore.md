# maxScore

**Framework**: Assignables  
**Kind**: property  
**Required**: Yes

An optional maximum score for this assessment. If `nil`, this value will be synthesized from the question data and associated annotations.

**Availability**:
- iOS 17.4+
- iPadOS 17.4+
- Mac Catalyst 17.4+
- visionOS ?+
