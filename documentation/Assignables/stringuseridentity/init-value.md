# init(value:)

**Framework**: Assignables  
**Kind**: init

Initializes a `StringUserIdentity` with the given string value.

**Availability**:
- iOS 17.4+
- iPadOS 17.4+
- Mac Catalyst 17.4+
- visionOS ?+

## Declaration

```swift
init(value: String)
```

## Parameters

- `value`: The value of the string to contain in this user identity.
