# symbolVariableValueMode(_:)

**Framework**: Assignables  
**Kind**: method

Sets the variable value mode mode for symbol images within this view.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
- tvOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)
- watchOS 26.0+ (Beta)

## Declaration

```swift
nonisolated
func symbolVariableValueMode(_ mode: SymbolVariableValueMode?) -> some View
```

#### Return Value

A view that uses the symbol variable value mode you supply.

## Parameters

- `mode`: The symbol variable value mode to use.
