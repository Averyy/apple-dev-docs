# ContextsRequest

**Framework**: ClassKit Catalog API  
**Kind**: dictionary

A request that you make when modifying context information.

**Availability**:
- ClassKit 1.0+

## Declaration

```swift
object ContextsRequest
```

#### Discussion

Use this request object to define one or more contexts that your app provides.
