# ocean

**Framework**: Core Location  
**Kind**: property

The name of the ocean associated with the placemark.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- Mac Catalyst 13.1+
- macOS 10.8+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
var ocean: String? { get }
```

#### Discussion

For coordinates that lie over an ocean, this property contains the name of the ocean.
