# init(center:radius:)

**Framework**: Core Location  
**Kind**: init

Creates a circular geographic condition with a center point and radius you specify.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+

## Declaration

```swift
init(center: CLLocationCoordinate2D, radius: CLLocationDistance)
```

## Parameters

- `center`: The center point of the circular geographic condition.
- `radius`: The radius of the circular geographic condition, in meters.
