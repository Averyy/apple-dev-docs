# init(_:)

**Framework**: Core Location  
**Kind**: init

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+

## Declaration

```swift
init(_ name: String) async
```

## Parameters

- `name`: The name of the location monitor.
