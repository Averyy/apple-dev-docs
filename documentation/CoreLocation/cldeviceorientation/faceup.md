# CLDeviceOrientation.faceUp

**Framework**: Core Location  
**Kind**: case

The device is held parallel to the ground with the screen facing upwards.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.0+
- macOS 10.6+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+
