# latitude

**Framework**: Core Location  
**Kind**: property

The latitude in degrees.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.0+
- macOS 10.6+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
var latitude: CLLocationDegrees
```

#### Discussion

Positive values indicate latitudes north of the equator. Negative values indicate latitudes south of the equator.
