# CLLocationSpeed

**Framework**: Core Location  
**Kind**: typealias

The velocity (measured in meters per second) at which the device is moving.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.0+
- macOS 10.6+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+
