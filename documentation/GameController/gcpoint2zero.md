# GCPoint2Zero

**Framework**: Game Controller  
**Kind**: var

The origin for a two dimensional point.

**Availability**:
- iOS 17.4+
- iPadOS 17.4+
- Mac Catalyst 17.4+
- macOS 14.3+
- tvOS 17.4+
- visionOS 1.1+

## Declaration

```swift
let GCPoint2Zero: GCPoint2
```

#### Discussion

This constant represents the `(0, 0)` origin.
