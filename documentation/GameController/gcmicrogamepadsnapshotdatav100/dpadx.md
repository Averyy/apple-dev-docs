# dpadX

**Framework**: Game Controller  
**Kind**: property

The value of the horizontal axis of the dpad.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
var dpadX: Float
```

#### Discussion

The range of permissible values is `-1.0` to `1.0`, inclusive.
