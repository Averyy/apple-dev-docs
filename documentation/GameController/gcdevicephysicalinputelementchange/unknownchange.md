# GCDevicePhysicalInputElementChange.unknownChange

**Framework**: Game Controller  
**Kind**: case

It’s unknown whether there’s a change to the input value.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+

## Declaration

```swift
case unknownChange
```

## Mentions

#### Discussion

This occurs if Game Controller removes older input states from the queue.
