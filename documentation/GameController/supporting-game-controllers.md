# Supporting Game Controllers

**Framework**: Gamecontroller

Support a physical controller or add a virtual controller to enhance how people interact with your game through haptics, lighting, and motion sensing.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- macOS 11.0+
- tvOS 14.0+
- Xcode 12.5+
