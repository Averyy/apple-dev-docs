# Extended gamepad input names

**Framework**: Game Controller

Constants for names of extended gamepad elements.

#### Overview

Use these constants as the keys when accessing [`GCExtendedGamepad`](gcextendedgamepad.md) elements by name using the [`allElements`](gcphysicalinputprofile/allelements.md) property, `subscript(_:)` notation, or similar [`GCPhysicalInputProfile`](gcphysicalinputprofile.md) APIs.
