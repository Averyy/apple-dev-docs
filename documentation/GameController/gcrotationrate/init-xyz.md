# init(x:y:z:)

**Framework**: Game Controller  
**Kind**: init

Creates a rotation rate structure with the specified values.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
init(x: Double, y: Double, z: Double)
```

## Parameters

- `x`: The value for the x-axis.
- `y`: The value for the y-axis.
- `z`: The value for the z-axis.
