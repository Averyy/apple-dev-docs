# microGamepad

**Framework**: Game Controller  
**Kind**: property

The micro gamepad profile.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- macOS 10.12+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
var microGamepad: GCMicroGamepad? { get }
```

#### Discussion

If the controller supports the micro gamepad profile, this property is a [`GCMicroGamepad`](gcmicrogamepad.md) object that you use to access the input elements of the controller. If the controller doesnâ€™t support the micro gamepad profile, this property is `nil`.
