# DualShock controller input names

**Framework**: Game Controller

Constants for names of DualShock 4 elements.

#### Overview

Use these constants as the keys when accessing [`GCDualShockGamepad`](gcdualshockgamepad.md) elements by name using the [`allElements`](gcphysicalinputprofile/allelements.md) property, `subscript(_:)` notation, or similar [`GCPhysicalInputProfile`](gcphysicalinputprofile.md) APIs.

## Topics

### DualShock controller button names
- [var GCInputDualShockTouchpadButton: String](gcinputdualshocktouchpadbutton-74k1l.md)
  The name of the button functionality of the DualShock 4 touchpad.
- [var GCInputDualShockTouchpadOne: String](gcinputdualshocktouchpadone-55wgz.md)
  The name of the first finger element to touch down on the DualShock 4 touchpad.
- [var GCInputDualShockTouchpadTwo: String](gcinputdualshocktouchpadtwo-5f8r9.md)
  The name of the second finger element to touch down on the DualShock 4 touchpad.
