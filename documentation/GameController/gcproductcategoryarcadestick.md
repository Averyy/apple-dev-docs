# GCProductCategoryArcadeStick

**Framework**: Game Controller  
**Kind**: var

The arcade stick product category.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+

## Declaration

```swift
let GCProductCategoryArcadeStick: String
```

#### Discussion

Use this product category to determine whether a [`GCController`](gccontroller.md) object is an arcade stick.
