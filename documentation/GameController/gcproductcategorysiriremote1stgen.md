# GCProductCategorySiriRemote1stGen

**Framework**: Game Controller  
**Kind**: var

The first-generation Siri Remote or first-generation Apple TV Remote product category.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+

## Declaration

```swift
let GCProductCategorySiriRemote1stGen: String
```

#### Discussion

Remotes in this category feature motion and a touch surface for navigation.
