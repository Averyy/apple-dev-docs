# GCPoint2Equal(_:_:)

**Framework**: Game Controller  
**Kind**: func

Returns whether two points are equal.

**Availability**:
- iOS 17.4+
- iPadOS 17.4+
- Mac Catalyst 17.4+
- macOS 14.3+
- tvOS 17.4+
- visionOS 1.1+

## Declaration

```swift
func GCPoint2Equal(_ point1: GCPoint2, _ point2: GCPoint2) -> Bool
```

#### Return Value

[`true`](https://developer.apple.com/documentation/swift/true) if the points are the same; otherwise, [`false`](https://developer.apple.com/documentation/swift/false).

## Parameters

- `point1`: The first point to compare.
- `point2`: The second point to compare.
