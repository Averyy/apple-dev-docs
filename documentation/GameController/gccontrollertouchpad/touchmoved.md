# touchMoved

**Framework**: Game Controller  
**Kind**: property

The block that the element calls when the user continues touching the touchpad, not when the user begins or ends touching the touchpad.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
