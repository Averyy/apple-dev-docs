# init(x:y:)

**Framework**: Game Controller  
**Kind**: init

Creates a two dimensional point with the given coordinates.

**Availability**:
- iOS 17.4+
- iPadOS 17.4+
- Mac Catalyst 17.4+
- macOS 14.3+
- tvOS 17.4+
- visionOS 1.1+

## Declaration

```swift
init(x: Float, y: Float)
```

## Parameters

- `x`: The x-coordinate for the point.
- `y`: The y-coordinate for the point.
