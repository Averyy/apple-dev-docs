# hasRotationRate

**Framework**: Game Controller  
**Kind**: property

A Boolean value that indicates whether the controller provides rotation data.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+

## Declaration

```swift
var hasRotationRate: Bool { get }
```

#### Discussion

[`true`](https://developer.apple.com/documentation/swift/true) if the controller provides rotation data; otherwise, [`false`](https://developer.apple.com/documentation/swift/false).
