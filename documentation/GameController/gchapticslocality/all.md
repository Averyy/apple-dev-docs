# all

**Framework**: Game Controller  
**Kind**: property

All locations of haptics actuators on a game controller.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+

## Declaration

```swift
static let all: GCHapticsLocality
```

#### Discussion

Controllers must support this location, and it can be the same as [`default`](gchapticslocality/default.md).
