# GCProductCategoryUniversalElectronicsRemote

**Framework**: Game Controller  
**Kind**: var

The product category for a Universal Electronics remote that works with Apple TV.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
