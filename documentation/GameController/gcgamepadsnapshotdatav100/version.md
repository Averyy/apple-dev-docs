# version

**Framework**: Game Controller  
**Kind**: property

A value that indicates the version number of the data structure.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- macOS 10.9+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
var version: UInt16
```

#### Discussion

For the structure defined here, the Game Controller framework uses a value of `0x0100`.
