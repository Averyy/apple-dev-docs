# VSErrorInfoKeySAMLResponse

**Framework**: Videosubscriberaccount  
**Kind**: var

The subscription provider’s SAML error response.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- macOS ?+
- tvOS 10.0+
- visionOS 1.0+

## Declaration

```swift
let VSErrorInfoKeySAMLResponse: String
```

#### Discussion

This key is a string in the [`userInfo`](https://developer.apple.com/documentation/foundation/nserror/1411580-userinfo) dictionary that the system returns with the error.
