# kSCCompAnyRegex

**Framework**: System Configuration  
**Kind**: var

A regular expression pattern that matches any component.

**Availability**:
- macOS 10.1+

## Declaration

```swift
let kSCCompAnyRegex: CFString
```

#### Discussion

This constant represents the regular expression pattern `[^/]+`.
