# init(coder:)

**Framework**: Playground Bluetooth  
**Kind**: instctr

Creates a connection view initialized from data you supply.

**Availability**:
- Xcode 10.2+
- Swift Playgrounds 2.0+

## Declaration

```swift
required init?(coder aDecoder: NSCoder)
```

## Parameters

- `coder`: An archive containing data that can be used to create a connection view.
