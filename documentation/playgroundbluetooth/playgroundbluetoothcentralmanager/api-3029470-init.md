# init(services:queue:)

**Framework**: Playground Bluetooth  
**Kind**: instctr

Creates a central manager that supports communicating with Bluetooth peripherals.

**Availability**:
- Xcode 10.2+
- Swift Playgrounds 2.0+

## Declaration

```swift
init(services: [CBUUID]?, queue: DispatchQueue = .main)
```
