# init()

**Framework**: Create ML  
**Kind**: init

Constructs an invalid Column.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 10.14+
- tvOS 16.0+
- visionOS 1.0+

## Declaration

```swift
init()
```

#### Discussion

Assigning an invalid Column to a column name in a DataTable will remove any Column previously stored under that name.
