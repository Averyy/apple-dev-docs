# min()

**Framework**: Create ML  
**Kind**: method

Returns the element with the lowest value in a column of doubles.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 10.14+
- tvOS 16.0+
- visionOS 1.0+

## Declaration

```swift
func min() -> Double?
```

#### Return Value

A [`Double`](https://developer.apple.com/documentation/Swift/Double); otherwise `nil` if the column is empty or invalid.
