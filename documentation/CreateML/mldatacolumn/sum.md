# sum()

**Framework**: Create ML  
**Kind**: method

**Availability**:

- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 10.14+
- tvOS 16.0+
- visionOS 1.0+

## Declaration

```swift
func sum() -> Double?
```

#### Return Value

A [`Double`](https://developer.apple.com/documentation/Swift/Double); otherwise `nil` if the column is invalid.
