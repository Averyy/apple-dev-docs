# dropMissing()

**Framework**: Create ML  
**Kind**: method

Creates a subset of the column by removing all elements without a value.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 10.14+
- tvOS 16.0+
- visionOS 1.0+
