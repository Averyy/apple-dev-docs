# ==(_:_:)

**Framework**: Create ML  
**Kind**: op

Returns a Boolean value indicating whether two values are equal.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 10.14+
- tvOS 16.0+
- visionOS 1.0+

## Declaration

```swift
static func == (lhs: MLDataTable.ColumnNames, rhs: MLDataTable.ColumnNames) -> Bool
```

#### Discussion

Equality is the inverse of inequality. For any values `a` and `b`, `a == b` implies that `a != b` is `false`.

## Parameters

- `lhs`: A value to compare.
- `rhs`: Another value to compare.
