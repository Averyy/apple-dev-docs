#### Overview

Use one or more [`MLDataTable.Aggregator`](mldatatable/aggregator.md) instances when you use the [`group(columnsNamed:aggregators:)`](mldatatable/group(columnsnamed:aggregators:).md) method. Each aggregator performs a sequence of one or more [`MLDataTable.Aggregator.Operations`](mldatatable/aggregator/operations-swift.enum.md) for a single column. Each operation makes a new column in the new data table generated by the method.
