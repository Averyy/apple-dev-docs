# renameColumn(named:to:)

**Framework**: Create ML  
**Kind**: method

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 10.14+
- tvOS 16.0+
- visionOS 1.0+

## Declaration

```swift

```

## Parameters

- `named`: The name of the column to rename.
- `to`: The new name for the column.
