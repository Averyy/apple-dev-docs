# sort(columnNamed:byIncreasingOrder:)

**Framework**: Create ML  
**Kind**: method

Creates a new data table by sorting the table by the given column.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 10.14+
- tvOS 16.0+
- visionOS 1.0+

## Declaration

```swift
func sort(columnNamed: String, byIncreasingOrder: Bool = true) -> MLDataTable
```

#### Return Value

A new data table.

## Parameters

- `columnNamed`: The name of the column to sort the rows of data table.
- `byIncreasingOrder`: Set this value to   to sort the   table in ascending order.
