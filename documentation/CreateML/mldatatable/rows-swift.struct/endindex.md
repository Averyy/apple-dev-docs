# endIndex

**Framework**: Create ML  
**Kind**: property

The DataTable’s “past the end” position—that is, the position one greater than the last valid subscript argument.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 10.14+
- tvOS 16.0+
- visionOS 1.0+
