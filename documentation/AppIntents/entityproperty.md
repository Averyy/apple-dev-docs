# EntityProperty

**Framework**: App Intents  
**Kind**: class

A property wrapper that exposes the associated property to the system.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+
- watchOS 9.0+

## Declaration

```swift
@propertyWrapper
final class EntityProperty<Value> where Value : _IntentValue, Value : Sendable
```
