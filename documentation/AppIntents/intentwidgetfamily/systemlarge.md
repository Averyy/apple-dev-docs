# IntentWidgetFamily.systemLarge

**Framework**: App Intents  
**Kind**: case

A large widget.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- visionOS 1.0+
- watchOS 10.0+

## Declaration

```swift
case systemLarge
```

#### Discussion

The large system widget can appear on the Home Screen or in the Today View in iOS or iPadOS, or in the Notification Center on macOS.
