# fileExporterFilenameLabel(_:)

**Framework**: App Intents  
**Kind**: method

On macOS, configures the `fileExporter` with a text to use as a label for the file name field.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst ?+

- visionOS ?+
