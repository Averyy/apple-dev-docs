# OpenURLIntent.SummaryContent

**Framework**: App Intents  
**Kind**: typealias

The type of parameter summary representing this intent.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst ?+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+

## Declaration

```swift
typealias SummaryContent = some ParameterSummary
```

#### Discussion

When you create an intent, Swift infers this type from your implementation of the [`parameterSummary`](openurlintent/parametersummary.md) property.
