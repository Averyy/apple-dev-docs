# appContext

**Framework**: App Intents  
**Kind**: property  
**Required**: Yes

An app context that is associated with the focus configuration. The system will retrieve this app context and adapt the system behavior based on the context provided.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst ?+
- macOS 13.0+
- tvOS 16.0+
- visionOS ?+
- watchOS 9.0+
