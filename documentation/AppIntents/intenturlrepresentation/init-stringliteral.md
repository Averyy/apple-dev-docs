# init(stringLiteral:)

**Framework**: App Intents  
**Kind**: init

- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+

## Declaration


## Parameters
