# IntentParameter

**Framework**: App Intents  
**Kind**: class

A property wrapper that indicates the associated property is an input argument of the app intent.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst ?+
- macOS 13.0+
- tvOS 16.0+
- visionOS ?+
- watchOS 9.0+

## Declaration

```swift
@propertyWrapper
final class IntentParameter<Value> where Value : _IntentValue, Value : Sendable
```

## Mentions

- [Adding parameters to an app intent](adding-parameters-to-an-app-intent.md)

#### Overview

When you implement an [`AppIntent`](appintent.md) type, declare its parameters using the `@Parameter` property wrapper.
