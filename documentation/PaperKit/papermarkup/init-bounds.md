# init(bounds:)

**Framework**: PaperKit  
**Kind**: init

Initializes and returns a new paper model with the specified bounds.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- macOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
init(bounds: CGRect)
```

## Parameters

- `bounds`: The bounds of the markup container.
