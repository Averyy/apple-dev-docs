# init(traitCollection:)

**Framework**: PaperKit  
**Kind**: init

Creates the most suitable options for rendering on a device with the specified traits.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- macOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
init(traitCollection: UITraitCollection)
```

## Parameters

- `traitCollection`: The trait collection to use.
