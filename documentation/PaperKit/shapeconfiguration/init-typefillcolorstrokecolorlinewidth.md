# init(type:fillColor:strokeColor:lineWidth:)

**Framework**: PaperKit  
**Kind**: init

Create a new shape configuration.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- macOS 26.0+
- visionOS 26.0+

## Declaration

```swift
init(type: ShapeConfiguration.Shape, fillColor: CGColor? = CGColor(srgbRed: 0, green: 0, blue: 0, alpha: 1), strokeColor: CGColor? = nil, lineWidth: CGFloat = 0)
```

## Parameters

- `type`: The type of shape.
- `fillColor`: The fill color of the shape.
- `strokeColor`: The stroke color of the shape.
- `lineWidth`: The line width of the shape.
