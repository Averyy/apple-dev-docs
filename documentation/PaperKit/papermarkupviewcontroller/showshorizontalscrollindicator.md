# showsHorizontalScrollIndicator

**Framework**: PaperKit  
**Kind**: property

A Boolean value that controls whether the horizontal scroll indicator is visible.

**Availability**:
- iOS 26.1+
- iPadOS 26.1+
- macOS 26.1+
- visionOS 26.1+

## Declaration

```swift
@MainActor
@preconcurrency var showsHorizontalScrollIndicator: Bool { get set }
```

#### Discussion

Default is `true`.
