# shapes

**Framework**: PaperKit  
**Kind**: property

The supported shape types.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- macOS 26.0+
- visionOS 26.0+

## Declaration

```swift
var shapes: Set<ShapeConfiguration.Shape>
```

#### Discussion

Default is all shapes. Set to an empty set to disable all shapes.
