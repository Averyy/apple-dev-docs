# FeatureSet.Feature.stickers

**Framework**: PaperKit  
**Kind**: case

Supports inserting stickers.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- macOS 26.0+
- visionOS 26.0+

## Declaration

```swift
case stickers
```

#### Discussion

Once inserted stickers are images, a data model can not contain stickers. If images are not supported, then stickers can not be inserted.
