# insert(_:)

**Framework**: PaperKit  
**Kind**: method

Inserts the given feature in the set if it is not already present.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- macOS 26.0+
- visionOS 26.0+

## Declaration

```swift
mutating func insert(_ newFeature: FeatureSet.Feature)
```

## Parameters

- `newFeature`: A feature to insert into the set.
