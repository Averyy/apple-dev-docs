# cancel()

**Framework**: Translation  
**Kind**: method

Attempt to stop all ongoing work for this session. Future requests will throw an error that the session is cancelled already.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
