# CFNetServiceGetTargetHost(_:)

**Framework**: CFNetwork  
**Kind**: func

Queries a CFNetService for its target hosts.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.4+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
func CFNetServiceGetTargetHost(_ theService: CFNetService) -> Unmanaged<CFString>?
```

#### Return Value

The target host name of the machine providing the service or `NULL` is of the serviceâ€™s target host is not known. (The target host will not be known if it has not been resolved.)

#### Discussion

This function is thread safe, but the target host name is not safe if the service is altered from another thread.

## Parameters

- `theService`: Network service to be queried.
