# CFNetServiceMonitor

**Framework**: CFNetwork  
**Kind**: class

An opaque reference for a service monitor.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.0+
- macOS 10.8+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
class CFNetServiceMonitor
```

#### Discussion

Service monitor references are used to monitor record changes on a CFNetServiceRef.
