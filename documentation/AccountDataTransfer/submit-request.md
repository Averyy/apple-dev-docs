# Submit request

**Framework**: Account Data Transfer  
**Kind**: httpRequest

Starts preparing someone’s data for download.

**Availability**:
- Account Data Transfer 1.0+

#### Overview

Request the `app-store` data type to get App Store information and app-install activity.
