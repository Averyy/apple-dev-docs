# updatesNowPlayingInfoCenter

**Framework**: AVKit  
**Kind**: property

A Boolean value that indicates whether the view controller updates Now Playing information.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var updatesNowPlayingInfoCenter: Bool { get set }
```

#### Discussion

The default value is `true`.
