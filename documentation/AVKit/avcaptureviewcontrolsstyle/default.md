# default

**Framework**: AVKit  
**Kind**: property

The view’s default controls style.

**Availability**:
- macOS 10.10+

## Declaration

```swift
static var `default`: AVCaptureViewControlsStyle { get }
```

#### Discussion

The default controls style is [`AVCaptureViewControlsStyle.inline`](avcaptureviewcontrolsstyle/inline.md).
