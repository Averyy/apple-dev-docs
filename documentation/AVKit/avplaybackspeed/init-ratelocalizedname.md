# init(rate:localizedName:)

**Framework**: AVKit  
**Kind**: init

Creates a playback speed with a rate and localized name.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+

## Declaration

```swift
init(rate: Float, localizedName: String)
```

## Parameters

- `rate`: The rate to use when you select this playback speed.
- `localizedName`: A localized name to display in a user interface for this speed.
