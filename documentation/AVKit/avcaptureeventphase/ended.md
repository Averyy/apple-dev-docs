# AVCaptureEventPhase.ended

**Framework**: AVKit  
**Kind**: case

A phase that indicates the end of a capture event.

**Availability**:
- iOS 17.2+
- iPadOS 17.2+
- Mac Catalyst 17.2+

## Declaration

```swift
case ended
```

#### Discussion

This phase corresponds to a user pressing up on a hardware button.
