# AVCaptureEventPhase.began

**Framework**: AVKit  
**Kind**: case

A phase that indicates the beginning of a capture event.

**Availability**:
- iOS 17.2+
- iPadOS 17.2+
- Mac Catalyst 17.2+

## Declaration

```swift
case began
```

#### Discussion

This phase corresponds to a user pressing down on a hardware button.
