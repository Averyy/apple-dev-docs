# AVDisplayDynamicRange.standard

**Framework**: AVKit  
**Kind**: case

Defines a standard dynamic range. Restricts the video content dynamic range to the standard range regardless of the actual range of the video content.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
