# AVDisplayDynamicRange.high

**Framework**: AVKit  
**Kind**: case

Defines a high dynamic range. Allows video content to use extended dynamic range if it has dynamic range content.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
