# isPictureInPictureActive

**Framework**: AVKit  
**Kind**: property

A Boolean value that indicates whether the Picture in Picture window is onscreen.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.15+
- tvOS 14.0+
- visionOS 1.0+

```swift
var isPictureInPictureActive: Bool { get }
```
