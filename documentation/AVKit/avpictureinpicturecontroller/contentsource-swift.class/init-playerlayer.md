# init(playerLayer:)

**Framework**: AVKit  
**Kind**: init

Creates a content source with a player layer.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+

## Declaration

```swift
init(playerLayer: AVPlayerLayer)
```

## Parameters

- `playerLayer`: The player layer to show in Picture in Picture.
