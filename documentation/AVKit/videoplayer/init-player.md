# init(player:)

**Framework**: AVKit  
**Kind**: init

Creates a video-player user interface for the player object.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+
- watchOS 7.0+

## Declaration

```swift
@MainActor
@preconcurrency init(player: AVPlayer?)
```

## Parameters

- `player`: The player that plays the audiovisual content.
