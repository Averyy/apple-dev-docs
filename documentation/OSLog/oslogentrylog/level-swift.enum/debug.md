# OSLogEntryLog.Level.debug

**Framework**: OSLog  
**Kind**: case

A log level that captures diagnostic information.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 10.15+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
case debug
```

#### Discussion

This level captures information that may be useful during development or while troubleshooting a specific problem.
