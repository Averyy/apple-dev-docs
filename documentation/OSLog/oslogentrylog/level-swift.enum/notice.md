# OSLogEntryLog.Level.notice

**Framework**: OSLog  
**Kind**: case

The log level that captures notifications.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 10.15+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
case notice
```

#### Discussion

This log level captures normal `os_log` entries used to generate notifications on system errors.
