# OSLogEntryLog.Level.error

**Framework**: OSLog  
**Kind**: case

The log level that captures errors.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 10.15+
- tvOS 15.0+
- visionOS 1.0+

## Declaration

```swift
case error
```

#### Discussion

This log level captures process-level information to report errors in the process.
