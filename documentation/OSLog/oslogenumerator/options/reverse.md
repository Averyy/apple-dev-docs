# reverse

**Framework**: OSLog  
**Kind**: property

Tells the framework to iterate backwards.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 10.15+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
static var reverse: OSLogEnumerator.Options { get }
```

#### Discussion

If there is no starting position specified, start at the latest entry.
