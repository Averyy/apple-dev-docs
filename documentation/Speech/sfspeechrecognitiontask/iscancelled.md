# isCancelled

**Framework**: Speech  
**Kind**: property

A Boolean value that indicates whether the speech recognition task was canceled.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- macOS 10.15+
- visionOS 1.0+

## Declaration

```swift
var isCancelled: Bool { get }
```

#### Discussion

By default, the value of this property is `false`.
