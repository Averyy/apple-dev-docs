# isFinishing

**Framework**: Speech  
**Kind**: property

A Boolean value that indicates whether audio input has stopped.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 10.0+
- macOS 10.15+
- visionOS 1.0+

## Declaration

```swift
var isFinishing: Bool { get }
```

#### Discussion

By default, the value of this property is `false`.
