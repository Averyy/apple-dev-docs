# endRetention()

**Framework**: Speech  
**Kind**: method

Releases all models held by cached recognizer instances. The method does not return until all models are released.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)
