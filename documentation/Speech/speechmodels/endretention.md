# endRetention()

**Framework**: Speech  
**Kind**: method

Releases all models held by cached recognizer instances. The method does not return until all models are released.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- visionOS 26.0+
