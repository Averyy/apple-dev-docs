# Notification Types

**Framework**: CloudKit JS

Constants indicating the type of event that generated the push notification.

## Topics

### Constants
- [NOTIFICATION_TYPE_QUERY](cloudkit.notification/notification_type_query.md)
  A notification generated based on the conditions set forth in a subscription object.
- [NOTIFICATION_TYPE_RECORD_ZONE](cloudkit.notification/notification_type_record_zone.md)
  A notification generated when the contents of a record zone changed.
