# moreComing

**Framework**: CloudKit JS  
**Kind**: property

A Boolean value that indicates there are more database changes to fetch.

**Availability**:
- CloudKit JS 1.0+

## Declaration

```swift
readonly attribute Boolean moreComing;
```

#### Discussion

`true` if there are more database changes to fetch; otherwise, `false`.
