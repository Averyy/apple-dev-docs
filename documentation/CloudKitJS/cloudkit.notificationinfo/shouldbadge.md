# shouldBadge

**Framework**: CloudKit JS  
**Kind**: property

A Boolean value indicating whether a badge should be displayed. If `true`, a badge is displayed; otherwise, it is not. The default value is `false`.

**Availability**:
- CloudKit JS 1.0+

```swift
attribute Boolean shouldBadge;
```
