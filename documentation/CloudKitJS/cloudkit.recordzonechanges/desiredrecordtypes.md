# desiredRecordTypes

**Framework**: CloudKit JS  
**Kind**: property

An array of record types that limit the amount of records returned. Only the record types specified in the array are returned.

**Availability**:
- CloudKit JS 1.0+

## Declaration

```swift
attribute String[] desiredRecordTypes;
```
