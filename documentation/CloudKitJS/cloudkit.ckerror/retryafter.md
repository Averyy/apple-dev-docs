# retryAfter

**Framework**: CloudKit JS  
**Kind**: property

The suggested time to wait before trying this operation again.

**Availability**:
- CloudKit JS 1.0+

## Declaration

```swift
readonly attribute Number retryAfter;
```

#### Discussion

If this key is `null`, the operation can’t be retried.
