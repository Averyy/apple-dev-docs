# getDefaultContainer

**Framework**: CloudKit JS  
**Kind**: method

Returns the default container.

**Availability**:
- CloudKit JS 1.0+

## Declaration

```swift
CloudKit.Container getDefaultContainer();
```

#### Return Value

The default container (the first container that appears in the configuration list of containers).
