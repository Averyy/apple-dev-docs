# CloudKit.ZoneID

**Framework**: CloudKit JS  
**Kind**: struct

An identifier for a zone, which is an area in a database for organizing related records.

**Availability**:
- CloudKit JS 1.0+

## Declaration

```swift
dictionary CloudKit.ZoneID {
	String zoneName;
};
```

#### Overview

The `CloudKit.ZoneID` dictionary is the same as the dictionary described in [`Zone Dictionary`](https://developer.apple.comhttps://developer.apple.com/library/archive/documentation/DataManagement/Conceptual/CloudKitWebServicesReference/Types.html#//apple_ref/doc/uid/TP40015240-CH3-SW10) in [`CloudKit Web Services Reference`](https://developer.apple.comhttps://developer.apple.com/library/archive/documentation/DataManagement/Conceptual/CloudKitWebServicesReference/index.html#//apple_ref/doc/uid/TP40015240).

## Topics

### Instance Properties
- [zoneName](cloudkit.zoneid/zonename.md)
  The name that identifies the record zone. The default value is `_defaultZone`, which indicates the default zone of the current database. This key is required.
