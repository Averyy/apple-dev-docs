# hasErrors

**Framework**: CloudKit JS  
**Kind**: property

A Boolean value indicating whether errors occurred in the request.

**Availability**:
- CloudKit JS 1.0+

## Declaration

```swift
readonly attribute Boolean hasErrors;
```

#### Discussion

`true` if errors occurred; otherwise, `false`.
