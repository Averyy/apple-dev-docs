# isResponse

**Framework**: CloudKit JS  
**Kind**: property

A Boolean value that indicates whether there is a response from the request.

**Availability**:
- CloudKit JS 1.0+

## Declaration

```swift
readonly attribute Boolean isResponse;
```

#### Discussion

This property is always `true`.
