# CloudKit.Response

**Framework**: CloudKit JS  
**Kind**: class

The [`CloudKit.Response`](cloudkit.response.md) class is an abstract superclass for subclasses that encapsulate the response from server requests. Donâ€™t create instances of this class. Instances of subclasses are returned by methods in the [`CloudKit.Container`](cloudkit.container.md) and [`CloudKit.Database`](cloudkit.database.md) classes. Most of these methods return a `Promise` object that resolves to a subclass of [`CloudKit.Response`](cloudkit.response.md) if the operation is successful.

**Availability**:
- CloudKit JS 1.0+

## Declaration

```swift
interface CloudKit.Response
```

#### Overview

For examples of methods that resolve to a [`CloudKit.Response`](cloudkit.response.md) subclass, read [`CloudKit.Database`](cloudkit.database.md).
