# id

**Framework**: Immersive Media Support  
**Kind**: property

An identifier string for this immersive camera. Must be unique for each immersive camera and cannot be blank.

**Availability**:
- macOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)
