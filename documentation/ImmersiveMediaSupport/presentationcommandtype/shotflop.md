# PresentationCommandType.shotFlop

**Framework**: Immersive Media Support  
**Kind**: case

A shotFlop command will cause the whole frame to be mirrored horizontally for the duration of the command.

**Availability**:
- macOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)
