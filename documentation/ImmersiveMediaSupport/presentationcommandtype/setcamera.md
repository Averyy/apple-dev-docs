# PresentationCommandType.setCamera

**Framework**: Immersive Media Support  
**Kind**: case

setCamera specifies the camera ID to be used for a specific frame during playback.

**Availability**:
- macOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)
