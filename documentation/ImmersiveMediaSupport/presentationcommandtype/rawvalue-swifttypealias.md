# PresentationCommandType.RawValue

**Framework**: Immersive Media Support  
**Kind**: typealias

The raw type that can be used to represent all values of the conforming type.

**Availability**:
- macOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
typealias RawValue = String
```
