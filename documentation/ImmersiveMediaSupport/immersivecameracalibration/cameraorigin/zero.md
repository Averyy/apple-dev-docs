# zero

**Framework**: Immersive Media Support  
**Kind**: property

Holds the default value of (0, 0, 0) for the immersive camera origin i.e zero vector for the left camera and zero vector for the right camera

**Availability**:
- macOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)
