# identity

**Framework**: Immersive Media Support  
**Kind**: property

Holds the default value of identity matrix for the immersive camera i.e identity matrix for the left camera and identity matrix for the right camera.

**Availability**:
- macOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)
