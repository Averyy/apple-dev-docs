# start()

**Framework**: Immersive Media Support  
**Kind**: method

Starts the sender.

**Availability**:
- macOS 26.0+

## Declaration

```swift
func start() async
```

#### Discussion

Call `start` before you start sending any frames, audio, or venue descriptor information to the receivers.
