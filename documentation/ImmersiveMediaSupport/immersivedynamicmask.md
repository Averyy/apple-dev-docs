# ImmersiveDynamicMask

**Framework**: Immersive Media Support  
**Kind**: struct

A type that holds the information required to dynamically generate an immersive media mask at load time.

**Availability**:
- macOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
struct ImmersiveDynamicMask
```

#### Overview

Immersive media dynamic masks are generated by interpolating a set of control points or sample points on the surface of a unit sphere.
