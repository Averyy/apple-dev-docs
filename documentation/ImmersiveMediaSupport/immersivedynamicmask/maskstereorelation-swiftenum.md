# ImmersiveDynamicMask.MaskStereoRelation

**Framework**: Immersive Media Support  
**Kind**: enum

An enumeration of the stereo relation of a mask. It determines how left mask relates to the right or vice versa.

**Availability**:
- macOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)
