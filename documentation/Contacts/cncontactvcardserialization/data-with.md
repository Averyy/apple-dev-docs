# data(with:)

**Framework**: Contacts  
**Kind**: method

Returns the vCard representation of the specified contacts.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
class func data(with contacts: [CNContact]) throws -> Data
```

#### Return Value

An [`NSData`](https://developer.apple.com/documentation/Foundation/NSData) object with the vCard representation of the contact.
