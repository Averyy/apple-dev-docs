# CNErrorUserInfoKeyPathsKey

**Framework**: Contacts  
**Kind**: var

An array of key paths associated with a given error.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
let CNErrorUserInfoKeyPathsKey: String
```

#### Discussion

For validation errors, this contains key paths to specific object properties.
