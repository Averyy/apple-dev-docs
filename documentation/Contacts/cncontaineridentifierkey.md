# CNContainerIdentifierKey

**Framework**: Contacts  
**Kind**: var

The identifier key of the container.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
let CNContainerIdentifierKey: String
```

#### Discussion

This key represents the container identifier property for KVC/KVO usage. This property is always fetched.
