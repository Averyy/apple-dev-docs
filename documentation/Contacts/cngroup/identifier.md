# identifier

**Framework**: Contacts  
**Kind**: property

The unique identifier for a group on the device.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
var identifier: String { get }
```

#### Discussion

It is recommended that you use the identifier when re-fetching the group. The identifier can be persisted between app launches.
