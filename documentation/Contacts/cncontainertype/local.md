# CNContainerType.local

**Framework**: Contacts  
**Kind**: case

A container for contacts only stored locally on the device.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
case local
```

#### Discussion

There is only one local container for a device.
