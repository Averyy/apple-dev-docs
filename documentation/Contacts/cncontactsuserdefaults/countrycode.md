# countryCode

**Framework**: Contacts  
**Kind**: property

An ISO country code.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
var countryCode: String { get }
```

#### Discussion

ISO is the default country code for phone numbers. This is determined by the device’s SIM card or the operating system’s configured language.
