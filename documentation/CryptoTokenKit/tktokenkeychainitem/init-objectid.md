# init(objectID:)

**Framework**: CryptoTokenKit  
**Kind**: init

Initializes a token keychain item with the specified object ID.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- macOS 10.12+
- tvOS 11.0+
- visionOS 1.0+
- watchOS 4.0+

## Declaration

```swift
init(objectID: TKToken.ObjectID)
```

#### Return Value

A new keychain item.

## Parameters

- `objectID`: The object ID.
