# isNFCSupported()

**Framework**: CryptoTokenKit  
**Kind**: method

Determines whether NFC (Near Field Communication) is supported on this device.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+

## Declaration

```swift
func isNFCSupported() -> Bool
```

#### Return Value

`YES` if NFC is supported and available for use, NO otherwise.
