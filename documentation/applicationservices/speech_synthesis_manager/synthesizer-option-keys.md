# Synthesizer Option Keys

**Framework**: Application Services

Keys used to specify synthesizer options.

## Topics

### Constants
- [let kSpeechNoEndingProsody: CFString](kspeechnoendingprosody.md)
  Disable prosody at the end of sentences.
- [let kSpeechNoSpeechInterrupt: CFString](kspeechnospeechinterrupt.md)
  Do not interrupt current speech.
- [let kSpeechPreflightThenPause: CFString](kspeechpreflightthenpause.md)
  Compute speech without generating it.
