# Global Variables

**Framework**: Application Services

## Topics

### Constants
- [var kAXAttachmentTextAttribute: Unmanaged<CFString>](kaxattachmenttextattribute.md)
  An `AXUIElementRef` value for the accessibility object representing the text attachment.
- [var kAXAutocorrectedTextAttribute: Unmanaged<CFString>](kaxautocorrectedtextattribute.md)
  A `CFBooleanRef` value that indicates whether the text was autocorrected.
- [var kAXBackgroundColorTextAttribute: Unmanaged<CFString>](kaxbackgroundcolortextattribute.md)
  A `CGColorRef` value that describes the background color.
- [var kAXFontFamilyKey: Unmanaged<CFString>](kaxfontfamilykey.md)
  Optional key of the [`kAXFontTextAttribute`](kaxfonttextattribute.md) dictionary for the font family name in a `CFStringRef`.
- [var kAXFontNameKey: Unmanaged<CFString>](kaxfontnamekey.md)
  Required key of the [`kAXFontTextAttribute`](kaxfonttextattribute.md) dictionary for the font name in a `CFStringRef`.
- [var kAXFontSizeKey: Unmanaged<CFString>](kaxfontsizekey.md)
  Required key of the [`kAXFontTextAttribute`](kaxfonttextattribute.md) dictionary for the font size in a `CFNumberRef`.
- [var kAXFontTextAttribute: Unmanaged<CFString>](kaxfonttextattribute.md)
  A dictionary (a `CFDictionaryRef`) of two or more font keys.
- [var kAXForegroundColorTextAttribute: Unmanaged<CFString>](kaxforegroundcolortextattribute.md)
  A `CGColorRef` value that describes the foreground color.
- [var kAXLinkTextAttribute: Unmanaged<CFString>](kaxlinktextattribute.md)
  An `AXUIElementRef` value used to access the object to which the specified range of text links.
- [var kAXMarkedMisspelledTextAttribute: Unmanaged<CFString>](kaxmarkedmisspelledtextattribute.md)
  A `CFBooleanRef` value that indicates whether the text is visibly shown as misspelled (for example, with a squiggle underneath the text).
- [var kAXMisspelledTextAttribute: Unmanaged<CFString>](kaxmisspelledtextattribute.md)
  A `CFBooleanRef` value that indicates whether the text is misspelled.
- [var kAXNaturalLanguageTextAttribute: Unmanaged<CFString>](kaxnaturallanguagetextattribute.md)
  A `CFStringRef` value that indicates the language in which the text will be spoken aloud by the system.
- [var kAXReplacementStringTextAttribute: Unmanaged<CFString>](kaxreplacementstringtextattribute.md)
  A `CFStringRef` value that indicates the string that will replace the text.
- [var kAXShadowTextAttribute: Unmanaged<CFString>](kaxshadowtextattribute.md)
  A `CFBooleanRef` value that indicates whether the text displays shadow.
- [var kAXStrikethroughColorTextAttribute: Unmanaged<CFString>](kaxstrikethroughcolortextattribute.md)
  A `CGColorRef` value that describes the color of the strikethrough.
- [var kAXStrikethroughTextAttribute: Unmanaged<CFString>](kaxstrikethroughtextattribute.md)
  A `CFBooleanRef` value that indicates whether the text displays strikethrough.
- [var kAXSuperscriptTextAttribute: Unmanaged<CFString>](kaxsuperscripttextattribute.md)
  A `CFNumberRef` value that indicates whether the text is superscript (value > 0) or subscript (value < 0).
- [var kAXUnderlineColorTextAttribute: Unmanaged<CFString>](kaxunderlinecolortextattribute.md)
  A `CGColorRef` value that describes the color of the underlining.
- [var kAXUnderlineTextAttribute: Unmanaged<CFString>](kaxunderlinetextattribute.md)
  A `CFNumberRef` value that describes the underline style (see [`AXUnderlineStyle`](axunderlinestyle.md) for possible values).
- [var kAXVisibleNameKey: Unmanaged<CFString>](kaxvisiblenamekey.md)
  Optional key of the [`kAXFontTextAttribute`](kaxfonttextattribute.md) dictionary for the font display name in a `CFStringRef`.
