# kAXDecrementAction

**Framework**: Application Services  
**Kind**: data

Decrements the value of the accessibility object. The amount the value is decremented by is determined by the value of the `kAXValueIncrementAttribute` attribute.

**Availability**:
- macOS 10.2+
