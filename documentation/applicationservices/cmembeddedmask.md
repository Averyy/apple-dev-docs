# cmEmbeddedMask

**Framework**: Application Services  
**Kind**: data

This mask provides access to bit 0 of the `flags` field, which specifies whether the profile is embedded. It has the value 1 if the profile is embedded, 0 if it is not.

**Availability**:
- macOS 10.0+
