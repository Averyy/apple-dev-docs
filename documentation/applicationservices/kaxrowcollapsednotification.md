# kAXRowCollapsedNotification

**Framework**: Application Services  
**Kind**: data

Notification that a row in an outline has been collapsed.

**Availability**:
- macOS 10.6+

## Declaration

```swift
var kAXRowCollapsedNotification: String { get }
```

#### Discussion

The value is the collapsed row UIElement.
