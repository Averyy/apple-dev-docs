# kAXShowDefaultUIAction

**Framework**: Application Services  
**Kind**: data

**Availability**:
- macOS 10.9+

## Declaration

```swift
var kAXShowDefaultUIAction: String { get }
```

#### Discussion

Show default UI. This is often used to trigger the same change that would occur when a mouse hover ends.
