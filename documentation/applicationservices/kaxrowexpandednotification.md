# kAXRowExpandedNotification

**Framework**: Application Services  
**Kind**: data

Notification that a row in an outline has been expanded.

**Availability**:
- macOS 10.6+

## Declaration

```swift
var kAXRowExpandedNotification: String { get }
```

#### Discussion

The value is the collapsed row UIElement.
