# kSpeechStatusOutputPaused

**Framework**: Application Services  
**Kind**: data

Indicates whether speech output in the speech channel has been paused by a call to the [`PauseSpeechAt(_:_:)`](1461174-pausespeechat.md) function.

**Availability**:
