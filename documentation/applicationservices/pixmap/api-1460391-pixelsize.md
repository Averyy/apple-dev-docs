# pixelSize

**Framework**: Application Services  
**Kind**: structp

The number of bits used to represent a pixel. Indexed pixels can have sizes of 1, 2, 4, and 8 bits; direct pixel sizes are 16 and 32 bits.

**Availability**:
- macOS 10.0+
