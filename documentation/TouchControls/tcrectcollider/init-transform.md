# init(transform:)

**Framework**: Touch Controls  
**Kind**: init

Creates a new rect collider with the specified transform.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)

## Declaration

```swift
init(transform: any TCTransform)
```

#### Return Value

A new `TCRectCollider` instance.

## Parameters

- `transform`: The transform to use for the collider.
