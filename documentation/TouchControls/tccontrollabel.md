# TCControlLabel

**Framework**: Touch Controls  
**Kind**: class

A label you associate with a touch control and provides a semantic description.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
