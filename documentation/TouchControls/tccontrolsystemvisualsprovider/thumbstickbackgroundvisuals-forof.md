# thumbstickBackgroundVisuals(for:of:)

**Framework**: Touch Controls  
**Kind**: method

The thumbstick background visuals for the specified label and size.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)

## Declaration

```swift
func thumbstickBackgroundVisuals(for label: TCControlLabel, of size: CGSize) -> TCControlVisuals
```

#### Return Value

The `TCControlVisuals` for the thumbstick background.

## Parameters

- `label`: The label for the thumbstick.
- `size`: The size of the thumbstick background in points.
