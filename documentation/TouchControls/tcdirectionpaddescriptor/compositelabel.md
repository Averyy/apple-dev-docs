# compositeLabel

**Framework**: Touch Controls  
**Kind**: property

A composite control label.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)

## Declaration

```swift
var compositeLabel: TCControlLabel? { get set }
```

#### Discussion

If set, the control will behave as a `GCControllerDirectionPad`. Otherwise, it will behave as separate buttons.
