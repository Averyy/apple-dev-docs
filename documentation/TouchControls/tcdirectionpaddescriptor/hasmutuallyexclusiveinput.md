# hasMutuallyExclusiveInput

**Framework**: Touch Controls  
**Kind**: property

A Boolean value that indicates whether the control has mutally exclusive input.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)

## Declaration

```swift
var hasMutuallyExclusiveInput: Bool { get set }
```

#### Discussion

If `YES`, only one element of the direction pad can be pressed at a time.
