# setEnabled(_:)

**Framework**: Touch Controls  
**Kind**: method  
**Required**: Yes

Sets whether the collider is enabled.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)

## Declaration

```swift
func setEnabled(_ enabled: Bool)
```

## Parameters

- `enabled`:   to enable the collider,   to disable it.
