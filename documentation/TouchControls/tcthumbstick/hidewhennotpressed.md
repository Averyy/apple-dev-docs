# hideWhenNotPressed

**Framework**: Touch Controls  
**Kind**: property

A Boolean value that indicates whether to hide the thumbstick when it is not being pressed.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
