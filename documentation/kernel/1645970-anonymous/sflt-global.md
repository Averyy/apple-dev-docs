# SFLT_GLOBAL

**Framework**: Kernel  
**Kind**: econst

**Availability**:
- macOS 10.12+

## Declaration

```swift
SFLT_GLOBAL = 0x01
```

#### Discussion

Indicates this socket filter should be attached to all new sockets when they're created.
