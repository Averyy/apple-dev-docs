# SFLT_PROG

**Framework**: Kernel  
**Kind**: econst

**Availability**:
- macOS 10.12+

## Declaration

```swift
SFLT_PROG = 0x02
```

#### Discussion

Indicates this socket filter should be attached only when request by the application using the SO_NKE socket option.
