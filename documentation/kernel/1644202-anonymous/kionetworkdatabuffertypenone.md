# kIONetworkDataBufferTypeNone

**Framework**: Kernel  
**Kind**: econst

**Availability**:
- macOS 10.12+

## Declaration

```swift
kIONetworkDataBufferTypeNone
```

#### Discussion

No data buffer. The only useful action perfomed by an IONetworkData object with this buffer type is to call the access notification handler.
