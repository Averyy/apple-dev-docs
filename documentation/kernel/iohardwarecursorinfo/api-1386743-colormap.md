# colorMap

**Framework**: Kernel  
**Kind**: structp

Pointer to array of IOColorEntry structures, with the number of elements set by the numColors field of the IOHardwareCursorDescriptor. Zero should be passed for direct pixel formats.

**Availability**:
- macOS 10.0+
