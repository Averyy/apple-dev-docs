# deviceStartedAudioEngine

**Framework**: Kernel  
**Kind**: pseudo

## Declaration

```swift
bool deviceStartedAudioEngine;
```

#### Overview

Used by the IOAudioDevice to determine responsibility for shutting the audio engine down when it is no longer needed.
