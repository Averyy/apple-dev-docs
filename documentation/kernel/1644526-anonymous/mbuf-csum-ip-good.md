# MBUF_CSUM_IP_GOOD

**Framework**: Kernel  
**Kind**: econst

**Availability**:
- macOS 10.12+

## Declaration

```swift
MBUF_CSUM_IP_GOOD = 0x0200
```

#### Discussion

Indicates whether or not the IP checksum was good or bad. Only valid when MBUF_CSUM_DID_IP is set.
