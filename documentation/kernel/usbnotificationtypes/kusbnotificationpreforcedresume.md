# kUSBNotificationPreForcedResume

**Framework**: Kernel  
**Kind**: econst

The system sends a notification before a resume, which occurs after a forced suspend.

**Availability**:
- macOS 10.9+

## Declaration

```swift
kUSBNotificationPreForcedResume = (1 << kUSBNotificationPreForcedResumeBit)
```
