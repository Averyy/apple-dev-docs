# enable

**Framework**: Kernel  
**Kind**: instm

Sets the enable state of the notification request.

## Declaration

```swift
virtual void enable(
 boolwas ) = 0;
```

#### Overview

Restores the enable state of the notification request, given the previous state passed in.

## Parameters

- `was`: The enable state of the notifier to restore.
