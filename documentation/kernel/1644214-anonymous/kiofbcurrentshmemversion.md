# kIOFBCurrentShmemVersion

**Framework**: Kernel  
**Kind**: econst

**Availability**:
- macOS 10.12+

## Declaration

```swift
kIOFBCurrentShmemVersion = 2
```

#### Discussion

The current version of the slice of shared memory that contains the cursor and window server state data in the StdFBShmem_t structure.
