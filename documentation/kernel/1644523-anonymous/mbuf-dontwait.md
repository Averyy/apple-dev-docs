# MBUF_DONTWAIT

**Framework**: Kernel  
**Kind**: econst

**Availability**:
- macOS 10.12+

## Declaration

```swift
MBUF_DONTWAIT = 1
```

#### Discussion

Don't allow the mbuf allocation call to block, if blocking is necessary fail and return immediately.
