# init

**Framework**: Kernel  
**Kind**: instm

Handles the basic initialization of the dispatch source.

**Availability**:
- DriverKit 19.0+
- macOS 10.15.2+

## Declaration

```swift
virtual bool init(void);
```

#### Return_value

`true` if initialization was successful, or `false` if an error occurred.

#### Discussion

Don't call this method directly. Call [`Create`](iodataqueuedispatchsource/3438177-create.md) when you want to create a new data-queue dispatch source.

## See Also

- [+ Create](iodataqueuedispatchsource/3438177-create.md)
  Creates a dispatch source that you use as a shared-memory data queue.
- [- free](iodataqueuedispatchsource/3438186-free.md)
  Performs any final cleanup for the data-queue dispatch source.
