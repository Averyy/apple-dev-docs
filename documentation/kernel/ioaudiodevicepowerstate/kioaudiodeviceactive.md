# kIOAudioDeviceActive

**Framework**: Kernel  
**Kind**: econst

**Availability**:
- macOS 10.1+

## Declaration

```swift
kIOAudioDeviceActive = 2
```

#### Discussion

State when one ore more IOAudioEngines are in use. This state transition must complete before the system will begin playing audio.
