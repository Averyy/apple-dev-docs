# vDSP_Length

**Framework**: Kernel  
**Kind**: tdef

Used for numbers of elements in arrays and indices of elements in arrays. It is also used for the base-two logarithm of numbers of elements.

**Availability**:
- macOS 10.13+

```swift
typedef unsigned long vDSP_Length;
```
