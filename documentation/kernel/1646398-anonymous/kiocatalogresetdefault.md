# kIOCatalogResetDefault

**Framework**: Kernel  
**Kind**: econst

**Availability**:
- macOS 10.12+

## Declaration

```swift
kIOCatalogResetDefault = 1
```

#### Discussion

Removes all entries from IOCatalogue except those used for booting the system.
