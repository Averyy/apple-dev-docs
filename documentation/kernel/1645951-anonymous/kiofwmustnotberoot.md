# kIOFWMustNotBeRoot

**Framework**: Kernel  
**Kind**: econst

**Availability**:
- macOS 10.12+

## Declaration

```swift
kIOFWMustNotBeRoot = (1 << 6)
```

#### Discussion

Attempt to prevent this device from being root, There is no guarentee Mac OS will succeed in preventing the device from being root.
