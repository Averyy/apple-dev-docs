# init

**Framework**: Kernel  
**Kind**: instm

Initializes the dispatch queue object.

**Availability**:
- DriverKit 19.0+
- macOS 10.15.2+

## Declaration

```swift
virtual bool init(void);
```

#### Return_value

`true` if initialization was successful, or `false` if it was unsuccessful.

#### Discussion

Do not call this method directly. Call [`Create`](iodispatchqueue/3438189-create.md) when you want to create a new dispatch queue.
