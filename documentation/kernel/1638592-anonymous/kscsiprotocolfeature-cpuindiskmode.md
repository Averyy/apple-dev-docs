# kSCSIProtocolFeature_CPUInDiskMode

**Framework**: Kernel  
**Kind**: econst

**Availability**:
- macOS 10.12+

## Declaration

```swift
kSCSIProtocolFeature_CPUInDiskMode = 2
```

#### Discussion

Used to determine if the SCSI Protocol Services Driver supports a CPU which is in target disk mode.
