# init

**Framework**: Kernel  
**Kind**: instm

Initializes the OSCollection object.

## Declaration

```swift
virtual bool init();
```

#### Return_value

`true` on success, `false` otherwise.

#### Overview

This function is used to initialize state within a newly created OSCollection object.
