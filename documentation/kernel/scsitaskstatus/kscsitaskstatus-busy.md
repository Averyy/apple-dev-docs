# kSCSITaskStatus_BUSY

**Framework**: Kernel  
**Kind**: econst

**Availability**:
- macOS 10.0+

## Declaration

```swift
kSCSITaskStatus_BUSY = 0x08
```

#### Discussion

The task completed with a status of BUSY. The device server might need time to process a request and a delay may be required.
