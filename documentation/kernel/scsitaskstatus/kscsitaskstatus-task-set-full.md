# kSCSITaskStatus_TASK_SET_FULL

**Framework**: Kernel  
**Kind**: econst

**Availability**:
- macOS 10.0+

## Declaration

```swift
kSCSITaskStatus_TASK_SET_FULL = 0x28
```

#### Discussion

The task completed with a status of TASK_SET_FULL. The device server may need to complete a task before the initiator sends another.
