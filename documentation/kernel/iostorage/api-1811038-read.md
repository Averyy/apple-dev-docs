**Kind**: instm

```swift
virtual IOReturn read(

#endif

```

#### Return_value

Returns the status of the data transfer.

#### Overview

Read data from the storage object at the specified byte offset into the specified buffer, synchronously. When the read completes, this method will return to the caller. The actual byte count field is optional.

## Parameters

- `client`: Client requesting the read.
- `byteStart`: Starting byte offset for the data transfer.
- `buffer`: Buffer for the data transfer. The size of the buffer implies the size of the data transfer.
- `attributes`: Attributes of the data transfer. See IOStorageAttributes.
- `actualByteCount`: Returns the actual number of bytes transferred in the data transfer.
