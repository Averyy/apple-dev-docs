# getBufferID

**Framework**: Kernel  
**Kind**: instm

Gets the buffer identifier for the IOStreamBuffer object.

## Declaration

```swift
virtual IOStreamBufferID getBufferID(
```

#### Overview

The buffer identifier is unique across all buffers in a stream.
