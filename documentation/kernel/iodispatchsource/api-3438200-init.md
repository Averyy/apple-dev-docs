# init

**Framework**: Kernel  
**Kind**: instm

Handles the basic initialization of the dispatch source.

**Availability**:
- DriverKit 19.0+
- macOS 10.15.2+

## Declaration

```swift
virtual bool init(void);
```

#### Return_value

`true` if initialization was successful, or `false` if an error occurred.
