# familyManagePower

**Framework**: Kernel  
**Kind**: pseudo

## Declaration

```swift
bool familyManagePower;
```

#### Overview

Set to true if the family is supposed to manage power - this is the default state. It can be changed early in the initialization process with a call to setFamilyManagePower().
