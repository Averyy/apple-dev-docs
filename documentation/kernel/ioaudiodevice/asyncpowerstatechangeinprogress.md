# asyncPowerStateChangeInProgress

**Framework**: Kernel  
**Kind**: pseudo

## Declaration

```swift
bool asyncPowerStateChangeInProgress;
```

#### Overview

Set to true while an asynchronous power change is pending and false all other times.
