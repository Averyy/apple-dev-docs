# kIONetworkFeatureTSOIPv4

**Framework**: Kernel  
**Kind**: econst

**Availability**:
- macOS 10.12+

## Declaration

```swift
kIONetworkFeatureTSOIPv4 = 0x010
```

#### Discussion

Set this bit to advertise support for TCP/IPv4 segmentation offload.
