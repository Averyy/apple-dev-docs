# kIONetworkFeatureTSOIPv6

**Framework**: Kernel  
**Kind**: econst

**Availability**:
- macOS 10.12+

## Declaration

```swift
kIONetworkFeatureTSOIPv6 = 0x020
```

#### Discussion

Set this bit to advertise support for TCP/IPv6 segmentation offload.
