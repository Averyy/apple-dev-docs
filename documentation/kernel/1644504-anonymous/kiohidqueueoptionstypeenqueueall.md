# kIOHIDQueueOptionsTypeEnqueueAll

**Framework**: Kernel  
**Kind**: econst

**Availability**:
- macOS 10.12+

## Declaration

```swift
kIOHIDQueueOptionsTypeEnqueueAll = 0x01
```

#### Discussion

Force the IOHIDQueue to enqueue all events, relative or absolute, regardless of change.
