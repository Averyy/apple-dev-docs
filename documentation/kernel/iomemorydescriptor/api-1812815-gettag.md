# getTag

**Framework**: Kernel  
**Kind**: instm

Accessor to the retrieve the tag for the memory descriptor.

## Declaration

```swift
virtual IOOptionBits getTag(
 void );
```

#### Return_value

The tag.

#### Overview

This method returns the tag for the memory descriptor. Tag bits are not interpreted by IOMemoryDescriptor.
