# shadow

**Framework**: RealityKit  
**Kind**: property

The shadow settings for a directional light.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 26.0+
- visionOS 2.0+

## Declaration

```swift
@MainActor
@preconcurrency var shadow: DirectionalLightComponent.Shadow? { get set }
```

#### Discussion

Set this value to `nil` to remove shadows.
