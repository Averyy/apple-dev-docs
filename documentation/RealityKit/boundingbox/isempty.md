# isEmpty

**Framework**: RealityKit  
**Kind**: property

A Boolean that indicates whether the bounding box is empty.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- visionOS ?+

## Declaration

```swift
var isEmpty: Bool { get }
```

#### Discussion

This value is `true` if the box is empty.
