# init(strength:)

**Framework**: RealityKit  
**Kind**: init

Creates a radial force effect with constant magnitude.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
- visionOS 2.0+

## Declaration

```swift
init(strength: Double)
```

## Parameters

- `strength`: The magnitude of the force.
