# isAutoCaptureEnabled

**Framework**: RealityKit  
**Kind**: property

Enables/disables auto-capture system.  If disabled, only manually triggered shots are taken.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+

## Declaration

```swift
@MainActor
var isAutoCaptureEnabled: Bool { get set }
```
