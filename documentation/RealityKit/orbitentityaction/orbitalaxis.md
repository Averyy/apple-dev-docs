# orbitalAxis

**Framework**: RealityKit  
**Kind**: property

A vector that describes the axis of rotation (in world space).

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
- visionOS 2.0+

## Declaration

```swift
var orbitalAxis: SIMD3<Float>
```

#### Discussion

The default value of this property is `[0, 1, 0]`, meaning the rotation occurs around the y-axis.
