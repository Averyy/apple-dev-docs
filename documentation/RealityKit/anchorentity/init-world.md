# init(world:)

**Framework**: RealityKit  
**Kind**: init

Creates an anchor entity with a target fixed at the given position in the scene.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 26.0+
- visionOS ?+

## Declaration

```swift
@MainActor
@preconcurrency convenience init(world position: SIMD3<Float>)
```

## Parameters

- `position`: The position with which to initialize the world target.
