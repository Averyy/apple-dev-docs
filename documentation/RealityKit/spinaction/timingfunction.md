# timingFunction

**Framework**: RealityKit  
**Kind**: property

A timing function that controls the progress of the animation.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
- visionOS 2.0+

## Declaration

```swift
var timingFunction: AnimationTimingFunction
```

#### Discussion

The default value of this property is [`default`](animationtimingfunction/default.md). This produces an ease transition.
