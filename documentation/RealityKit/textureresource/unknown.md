# ==(_:_:)

**Framework**: RealityKit  
**Kind**: op

Indicates whether two texture resources are equal.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
@MainActor
@preconcurrency static func == (lhs: TextureResource, rhs: TextureResource) -> Bool
```

#### Return Value

A Boolean value set to `true` if the two texture resources are equal.

## Parameters

- `lhs`: The first texture resource to compare.
- `rhs`: The second texture resource to compare.
