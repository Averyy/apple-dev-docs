# checksForInternalCollisions

**Framework**: RealityKit  
**Kind**: property

A Boolean that indicates whether the joint checks and reports collisions between the two entity instances.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- visionOS 2.0+

## Declaration

```swift
var checksForInternalCollisions: Bool
```

#### Discussion

> **Note**: This does not affect how collisions of the two entities referenced in [`pin0`](physicsdistancejoint/pin0.md) and [`pin1`](physicsdistancejoint/pin1.md) interact with other entities.
