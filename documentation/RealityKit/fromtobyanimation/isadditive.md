# isAdditive

**Framework**: RealityKit  
**Kind**: property

A Boolean value that indicates whether the animation blends additively with concurrent animations.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 26.0+
