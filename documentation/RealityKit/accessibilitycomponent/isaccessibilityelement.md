# isAccessibilityElement

**Framework**: RealityKit  
**Kind**: property

A Boolean value indicating whether the receiver is an accessibility entity that an assistive application can access.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 26.0+
- visionOS ?+
