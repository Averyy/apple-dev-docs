# intensityExponent

**Framework**: RealityKit  
**Kind**: property

The intensity value of the probe An intensityExponent of 0 means using the diffuse/specular intensities as-is Otherwise the intensity is multiplied by 2^intensityExponent

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- visionOS 1.0+
