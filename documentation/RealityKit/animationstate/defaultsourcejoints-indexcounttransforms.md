# defaultSourceJoints(index:count:transforms:)

**Framework**: RealityKit  
**Kind**: method

Retrieves a subset of default source joints, and stores them in the output transform array.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
