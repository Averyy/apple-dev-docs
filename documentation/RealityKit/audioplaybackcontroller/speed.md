# speed

**Framework**: RealityKit  
**Kind**: property

The rate of playback of the audio resource, with a range of `[.25, 4]`

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- visionOS ?+

## Declaration

```swift
@MainActor
@preconcurrency var speed: Double { get set }
```

#### Discussion

Set the speed to `1` for a normal playback rate.
