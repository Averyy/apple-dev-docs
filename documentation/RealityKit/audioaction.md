# AudioAction

**Framework**: RealityKit

An action that plays audio.

#### Overview

This prim implements one of two ways to play audio in a scene; to play audio without [`AudioAction`](audioaction.md), use [`SpatialAudio`](https://developer.apple.comhttps://graphics.pixar.com/usd/docs/UsdAudio-Proposal.html).

##### Declaration

```other
class Preliminary_Action "AudioAction"
```
