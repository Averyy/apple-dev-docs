**Availability**:

- Mac Catalyst 18.0+
- macOS 15.0+
- visionOS 2.0+

## Declaration

```swift
static func == (lhs: GeometricPin, rhs: GeometricPin) -> Bool
```

## Parameters

- `lhs`: A pin to compare.
- `rhs`: Another pin to compare.
