# name

**Framework**: RealityKit  
**Kind**: property

The name of the pin.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+
- visionOS 2.0+

## Declaration

```swift
var name: String
```

#### Discussion

Each `GeometricPins` for an[`Entity`](entity.md) have unique names. Adding a pin with the same name as another overrides the duplicate.
