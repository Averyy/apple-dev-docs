# init(environmentLightingWeight:)

**Framework**: RealityKit  
**Kind**: init

Creates an environment-lighting configuration component.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+
- visionOS 2.0+

## Declaration

```swift
init(environmentLightingWeight: Float = 1.0)
```

## Parameters

- `environmentLightingWeight`: A floating-point value in the range  ,   where   indicates no environment-lighting contribution and   indicates full environment-lighting   contribution.
