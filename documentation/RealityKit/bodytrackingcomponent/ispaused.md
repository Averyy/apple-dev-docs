# isPaused

**Framework**: RealityKit  
**Kind**: property

A Boolean that you can set to temporarily stop applying body tracking to the model and freeze the model in its current pose.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 14.0+

## Declaration

```swift
var isPaused: Bool
```
