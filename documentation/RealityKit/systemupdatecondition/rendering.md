# rendering

**Framework**: RealityKit  
**Kind**: property

A condition that is active whenever an update for rendering may be needed, usually matching the refresh rate of the display.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+
- visionOS 1.0+
