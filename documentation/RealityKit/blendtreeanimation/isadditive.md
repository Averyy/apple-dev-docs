# isAdditive

**Framework**: RealityKit  
**Kind**: property

A Boolean value that indicates whether the animation builds on the current state of the target entity or resets the state before running.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 26.0+
