# init(gain:)

**Framework**: RealityKit  
**Kind**: init

Configure the behavior of a channel audio source.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+
- visionOS 1.0+

## Declaration

```swift
init(gain: Audio.Decibel = .zero)
```

## Parameters

- `gain`: The overall level for all sounds emitted from an entity.
