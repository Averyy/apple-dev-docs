# receivesLighting

**Framework**: RealityKit  
**Kind**: property

The `.receivesLighting` option means that the virtual lights will interact with real world surfaces causing them to shine. The properties  of the mesh will be set to a default material.

**Availability**:
- iOS 13.4+
- iPadOS 13.4+
- Mac Catalyst 14.0+
- macOS 10.15+
