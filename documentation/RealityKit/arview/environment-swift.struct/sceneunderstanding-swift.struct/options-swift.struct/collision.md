# collision

**Framework**: RealityKit  
**Kind**: property

The `.collision` option means that the reconstructed geometry can be used for collision queries (i.e. raycasting)

**Availability**:
- iOS 13.4+
- iPadOS 13.4+
- Mac Catalyst 14.0+
