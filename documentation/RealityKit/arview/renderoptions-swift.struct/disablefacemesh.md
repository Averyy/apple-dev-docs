# disableFaceMesh

**Framework**: RealityKit  
**Kind**: property

Disable generation of the face entity with the default occlusion material.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 14.0+

## Declaration

```swift
static let disableFaceMesh: ARView.RenderOptions
```

#### Discussion

By default, [`ARKit`](https://developer.apple.com/documentation/ARKit) detects people and hides virtual objects behind their faces.
