# lensDistortionData

**Framework**: RealityKit  
**Kind**: property

If available, the estimated lens distortion data for the associated input image.  This can be used to rectify the image for computer vision applications.  If not available, this is nil.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
