# store(in:)

**Framework**: RealityKit  
**Kind**: method

Adds the animation to an entity without playing it.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 26.0+ (Beta)

## Declaration

```swift
@MainActor
@preconcurrency func store(in: Entity)
```

## Parameters

- `in`: The entity to which to attach the animation.
