# init(_:)

**Framework**: RealityKit  
**Kind**: init

Creates a custom texture by copying values from a material parameters texture.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 26.0+ (Beta)

## Declaration

```swift
init(_ texture: MaterialParameters.Texture)
```

## Parameters

- `texture`: The material parameters texture object.
