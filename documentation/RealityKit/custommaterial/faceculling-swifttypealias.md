# CustomMaterial.FaceCulling

**Framework**: RealityKit  
**Kind**: typealias

The type of object used to control the removal of polygons that aren’t visible to the user.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 26.0+
