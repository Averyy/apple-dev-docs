# EnvironmentResource.CreateOptions.SamplingQuality

**Framework**: RealityKit  
**Kind**: enum

An object for controlling the skybox sampling quality for lighting textures.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
