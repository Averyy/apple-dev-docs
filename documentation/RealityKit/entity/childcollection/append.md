# append(_:)

**Framework**: RealityKit  
**Kind**: method

Adds the specified entity to the end of this collection.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- visionOS 1.0+
