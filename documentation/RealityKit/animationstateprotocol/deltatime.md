# deltaTime

**Framework**: RealityKit  
**Kind**: property  
**Required**: Yes

The time that has elapsed since the most recent evaluation, or 0 if animation is paused.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+
- visionOS 2.0+
