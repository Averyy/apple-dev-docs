# init(_:)

**Framework**: RealityKit  
**Kind**: init

Creates an opacity object using a custom material’s opacity property.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 26.0+ (Beta)

## Declaration

```swift
init(_ value: CustomMaterial.Opacity)
```

## Parameters

- `value`: The custom material’s opacity property.
