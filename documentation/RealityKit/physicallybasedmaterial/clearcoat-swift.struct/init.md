# init(_:)

**Framework**: RealityKit  
**Kind**: init

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 26.0+ (Beta)

## Declaration


## Parameters

- `value`: The custom material’s clearcoat property.
