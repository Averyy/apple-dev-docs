# init(_:)

**Framework**: RealityKit  
**Kind**: init

Creates a base color object from a custom material’s base color property.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+

## Declaration

```swift
init(_ value: CustomMaterial.BaseColor)
```

## Parameters

- `value`: The custom material’s base color property.
