# ARViewBase

**Framework**: RealityKit  
**Kind**: typealias

The platform-specific base class for the view into which RealityKit renders content.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
