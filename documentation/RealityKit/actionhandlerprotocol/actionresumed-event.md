# actionResumed(event:)

**Framework**: RealityKit  
**Kind**: method  
**Required**: Yes

The function used to respond to action resumed events.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- visionOS 2.0+

## Declaration

```swift
mutating func actionResumed(event: Self.EventType)
```

#### Discussion

Action resumed is raised when animation playback is resumed after being paused.
