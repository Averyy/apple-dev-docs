# weight

**Framework**: RealityKit  
**Kind**: property

The weight of the joint rotation limit demand.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+
- visionOS 2.0+

## Declaration

```swift
var weight: Float
```

#### Discussion

The value is in the closed range `[0, 1]`, where `0` means no influence, and `1` is maximum correction.
