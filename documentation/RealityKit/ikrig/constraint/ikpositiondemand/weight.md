# weight

**Framework**: RealityKit  
**Kind**: property

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+
- visionOS 2.0+

## Declaration

```swift
var weight: SIMD3<Float>
```

#### Discussion

> **Note**: Values of `0` or less remove the demand influence.
