# subscript(_:)

**Framework**: Realitykit  
**Kind**: subscript

Accesses the element with the specified name.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- visionOS 2.0+

## Declaration

```swift
subscript(name: String) -> IKRig.ConstraintsCollection.Element? { get set }
```

#### Overview

> **Note**: Set `nil` to remove the element with matching name.

## Parameters

- `name`: The name of the requested element.
