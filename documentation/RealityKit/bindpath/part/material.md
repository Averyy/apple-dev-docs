# BindPath.Part.material(_:)

**Framework**: RealityKit  
**Kind**: case

A path component to animate a material property.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
- visionOS 1.0+

## Declaration

```swift
case material(Int)
```

#### Discussion

This path component indicates that another component follows, which either contains or identifies the material property that animates.
