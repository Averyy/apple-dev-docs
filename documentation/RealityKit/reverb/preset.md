# preset(_:)

**Framework**: RealityKit  
**Kind**: method

Returns a reverb instance that you can set on a reverb component.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- visionOS 2.0+

## Declaration

```swift
static func preset(_ preset: Reverb.Preset) -> Reverb
```

#### Return Value

Reverb that can be set on a [`ReverbComponent`](reverbcomponent.md).

## Parameters

- `preset`: A reverberation preset.
