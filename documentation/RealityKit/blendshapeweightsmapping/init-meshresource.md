# init(meshResource:)

**Framework**: RealityKit  
**Kind**: init

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+
- visionOS 2.0+

## Declaration

```swift
init(meshResource: MeshResource)
```

## Parameters

- `meshResource`: The definition of the mesh from which to build the mapping.
