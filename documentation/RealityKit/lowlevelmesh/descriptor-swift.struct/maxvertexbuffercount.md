# maxVertexBufferCount

**Framework**: RealityKit  
**Kind**: property

The maximum number of separate buffers the system supports.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
- visionOS 2.0+

## Declaration

```swift
static let maxVertexBufferCount: Int
```

#### Discussion

For optimal performance, most applications use one buffer.
