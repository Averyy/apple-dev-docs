# offset

**Framework**: RealityKit  
**Kind**: property

The location of an attribute in vertex data, determined by the byte offset from the start of the vertex data.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+
- visionOS 2.0+
