# speed

**Framework**: RealityKit  
**Kind**: property

The initial speed, in meters per second, for newly spawned particles. Defaults to 0.5.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
