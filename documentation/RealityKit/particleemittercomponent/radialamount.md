# radialAmount

**Framework**: RealityKit  
**Kind**: property

Radial sweep angle for sphere, cylinder, cone, and torus emitter shapes. Defaults to 2 * pi.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
