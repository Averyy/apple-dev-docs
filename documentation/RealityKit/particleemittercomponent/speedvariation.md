# speedVariation

**Framework**: RealityKit  
**Kind**: property

Defines a plus/minus range (in meters per second) from which a value is randomly selected to offset particle speed.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
- visionOS 1.0+
