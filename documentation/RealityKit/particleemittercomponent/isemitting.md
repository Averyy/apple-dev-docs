# isEmitting

**Framework**: RealityKit  
**Kind**: property

Disables/enables particle emission, independent of `simulationState`. Existing particles will not be affected.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+
- visionOS 1.0+

## Declaration

```swift
var isEmitting: Bool
```
