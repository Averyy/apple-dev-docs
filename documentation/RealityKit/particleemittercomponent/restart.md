# restart()

**Framework**: RealityKit  
**Kind**: method

Restarts the emission of particles. Requires the component to be re-assigned to the entity to take effect.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
