# simulationState

**Framework**: RealityKit  
**Kind**: property

Controls particle simulation state: playing, paused or stopped. Defaults to `play`.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- visionOS 1.0+
