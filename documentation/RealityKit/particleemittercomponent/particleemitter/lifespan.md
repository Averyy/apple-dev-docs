# lifeSpan

**Framework**: RealityKit  
**Kind**: property

The duration, in seconds, for which each particle is rendered before being removed from the scene. Defaults to 1.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- visionOS 1.0+
