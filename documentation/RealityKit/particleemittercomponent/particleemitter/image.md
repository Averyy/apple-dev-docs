# image

**Framework**: RealityKit  
**Kind**: property

The image that each particle will use (requires pre-multiplied RGB values). Defaults to a white circular texture.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+
- visionOS 1.0+
