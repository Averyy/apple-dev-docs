# angularSpeedVariation

**Framework**: RealityKit  
**Kind**: property

Defines a plus/minus range (in radians per second) from which a value is randomly selected to offset `angularSpeed`.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
