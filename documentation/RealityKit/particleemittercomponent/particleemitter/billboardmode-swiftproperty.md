# billboardMode

**Framework**: RealityKit  
**Kind**: property

The mode defining whether and how particles orient towards the camera. Defaults to `billboardYAligned`.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+
- visionOS 1.0+
