# massVariation

**Framework**: RealityKit  
**Kind**: property

Defines a plus/minus range (in grams) from which a value is randomly selected to offset `mass`.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
