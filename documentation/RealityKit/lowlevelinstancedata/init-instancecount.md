# init(instanceCount:)

**Framework**: RealityKit  
**Kind**: init

Creates a low-level instance data object with an instance count.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+

## Declaration

```swift
convenience init(instanceCount: Int) throws
```

#### Discussion

This initializer also applies the instance count to [`instanceCapacity`](lowlevelinstancedata/instancecapacity.md).

## Parameters

- `instanceCount`: The number of instances in this instance data object.
