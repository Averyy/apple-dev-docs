# init(_:physicalSize:)

**Framework**: RealityKit  
**Kind**: init

Creates the image anchoring source from image file URL.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
- visionOS 2.0+

## Declaration

```swift
init(_ url: URL, physicalSize: SIMD2<Float>)
```

## Parameters

- `url`: The image file URL.
- `physicalSize`: The physical size of tracking image
