# init(group:name:)

**Framework**: RealityKit  
**Kind**: init

Creates the object anchoring source by group and name in AR Resources.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)

## Declaration

```swift
init(group: String, name: String)
```

## Parameters

- `group`: The group name of the object in AR Resources.
- `name`: The name of the object in AR Resources.
