# init(_:)

**Framework**: RealityKit  
**Kind**: init

Creates an anchoring component with the given AR anchor.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 14.0+
- macOS 10.15+

## Declaration

```swift
init(_ anchor: ARAnchor)
```

## Parameters

- `anchor`: An existing AR anchor to use.
