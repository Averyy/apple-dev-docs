# ==(_:_:)

**Framework**: RealityKit  
**Kind**: op

Indicates whether two targets are equal.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 14.0+
- macOS 10.15+
- tvOS 26.0+ (Beta)
- visionOS ?+

## Declaration

```swift
static func == (lhs: AnchoringComponent.Target, rhs: AnchoringComponent.Target) -> Bool
```

#### Return Value

A Boolean value set to `true` if the two targets are equal.

## Parameters

- `lhs`: The first target to compare.
- `rhs`: The second target to compare.
