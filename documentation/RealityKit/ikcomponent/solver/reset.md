# reset()

**Framework**: RealityKit  
**Kind**: method

Enqueues a solver reset call that executes before the next solve.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- visionOS 2.0+

## Declaration

```swift
func reset()
```

#### Discussion

Resets the solver to its rest pose. Useful if the source animation or constraint demands change significantly.
