# init(resource:)

**Framework**: RealityKit  
**Kind**: init

Creates a new component value referencing the specified resource.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
- visionOS 2.0+

## Declaration

```swift
init(resource: IKResource?)
```

## Parameters

- `resource`: The resource with the desired IK rig for the new solver.
