# contains(_:)

**Framework**: RealityKit  
**Kind**: method

Returns a Boolean value that indicates whether the collection contains an element with a specific identifier.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+
- visionOS 2.0+

## Declaration

```swift
func contains(_ id: IKComponent.ConstraintCollection.Element.ID) -> Bool
```

## Parameters

- `id`: The identifier of the element to look up.
