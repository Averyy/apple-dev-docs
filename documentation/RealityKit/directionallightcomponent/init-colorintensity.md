# init(color:intensity:)

**Framework**: RealityKit  
**Kind**: init

Creates a directional light with a configuration.

**Availability**:
- visionOS 2.0+

## Declaration

```swift
init(color: DirectionalLightComponent.Color = .white, intensity: Float = 2145.7078)
```

## Parameters

- `color`: A color for the light.
- `intensity`: The intensity of the light, measured in lumen per square   meter.
