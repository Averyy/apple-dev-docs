# depthBias

**Framework**: RealityKit  
**Kind**: property

A constant value that RealityKit applies as a bias to its shadow calculations.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 26.0+
- visionOS 2.0+
