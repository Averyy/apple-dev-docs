# isStopped

**Framework**: RealityKit  
**Kind**: property

A Boolean value that indicates whether the animation stopped.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 26.0+ (Beta)

## Declaration

```swift
@MainActor
@preconcurrency var isStopped: Bool { get }
```

#### Discussion

This function returns `true` for stopped animations.
