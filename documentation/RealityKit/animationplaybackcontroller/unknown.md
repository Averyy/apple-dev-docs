# ==(_:_:)

**Framework**: RealityKit  
**Kind**: op

Indicates whether two animation playback controllers are equal.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 26.0+ (Beta)
- visionOS ?+

## Declaration

```swift
nonisolated
static func == (lhs: AnimationPlaybackController, rhs: AnimationPlaybackController) -> Bool
```

#### Return Value

A Boolean value set to `true` if the two controllers are equal.

## Parameters

- `lhs`: The first controller to compare.
- `rhs`: The second controller to compare.
