# identity

**Framework**: RealityKit  
**Kind**: property

The identity transform.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- visionOS ?+

## Declaration

```swift
static let identity: Transform
```

#### Discussion

The identity transform is defined as `scale = (1, 1, 1)`, `rotation = (0, 0, 0, 1)`, and `translation = (0, 0, 0)`.
