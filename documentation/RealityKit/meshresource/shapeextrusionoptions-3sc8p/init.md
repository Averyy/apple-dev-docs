# init()

**Framework**: RealityKit  
**Kind**: init

Creates the shape extrusion options with default values.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
- visionOS 2.0+

## Declaration

```swift
init()
```

#### Discussion

By default, the extrusion method is [`MeshResource.ShapeExtrusionOptions.ExtrusionMethod.linear(depth:)`](meshresource/shapeextrusionoptions-3sc8p/extrusionmethod-swift.enum/linear(depth:).md), with a depth of 0.25m, and all sides are assigned the first material.
