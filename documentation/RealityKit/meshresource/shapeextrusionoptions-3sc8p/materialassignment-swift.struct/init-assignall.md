# init(assignAll:)

**Framework**: RealityKit  
**Kind**: init

Creates a material assignment structure that assigns the same material to all faces.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
- visionOS 2.0+

## Declaration

```swift
init(assignAll all: UInt32)
```

## Parameters

- `all`: The material index for all mesh faces.
