# init(shape:)

**Framework**: RealityKit  
**Kind**: init

Generates a MeshResource from a ShapeResource.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
- visionOS 1.0+

## Declaration

```swift
@MainActor
@preconcurrency convenience init(shape resource: ShapeResource)
```

## Parameters

- `resource`: The ShapeResource which will be used for generating the mesh.
