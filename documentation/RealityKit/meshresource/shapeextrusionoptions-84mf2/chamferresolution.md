# chamferResolution

**Kind**: property

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
- visionOS 2.0+

## Declaration

```swift
var chamferResolution: MeshResource.ShapeExtrusionOptions.CurveStrokeResolution
```

#### Discussion

The default value is [`MeshResource.ShapeExtrusionOptions.CurveStrokeResolution.uniformSegmentsPerSpan(segmentCount:)`](meshresource/shapeextrusionoptions-84mf2/curvestrokeresolution/uniformsegmentsperspan(segmentcount:).md) with a segment count of 8.
