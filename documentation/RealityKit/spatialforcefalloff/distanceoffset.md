# distanceOffset

**Framework**: RealityKit  
**Kind**: property

**Availability**:

- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+
- visionOS 2.0+

## Declaration


#### Discussion

Use an offset value that is less than [`bounds`](spatialforcefalloff/bounds.md).
