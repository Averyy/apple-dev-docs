# subscript(_:)

**Framework**: RealityKit  
**Kind**: subscript

**Availability**:

- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)

## Declaration

```swift

```

## Parameters

- `name`: The name of the pin to look up.
