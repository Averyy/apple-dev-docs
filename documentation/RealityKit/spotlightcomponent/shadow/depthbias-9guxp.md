# depthBias

**Framework**: RealityKit  
**Kind**: property

A constant value that RealityKit applies as a bias to its shadow calculations.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
- visionOS 2.0+
