# angularMotionAroundZ

**Framework**: RealityKit  
**Kind**: property

The angular motion limits around the z-axis.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+
- visionOS 2.0+

## Declaration

```swift
var angularMotionAroundZ: PhysicsCustomJoint.MotionLimit
```

#### Discussion

This property sets the limits of `pin1` around the z-axis of `pin0`.
