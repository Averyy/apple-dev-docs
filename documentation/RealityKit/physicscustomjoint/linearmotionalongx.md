# linearMotionAlongX

**Framework**: RealityKit  
**Kind**: property

The linear motion limits along the x-axis.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+
- visionOS 2.0+

## Declaration

```swift
var linearMotionAlongX: PhysicsCustomJoint.MotionLimit
```

#### Discussion

This property sets the limits of `pin1` along the x-axis of `pin0`.
