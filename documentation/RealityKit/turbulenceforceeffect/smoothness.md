# smoothness

**Framework**: RealityKit  
**Kind**: property

A smoothing factor that applies to the force, reducing randomness.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
- visionOS 2.0+

## Declaration

```swift
let smoothness: Float
```

#### Discussion

A value of `0.0` specifies maximum noise, and a value of `1.0` specifies no noise at all.
