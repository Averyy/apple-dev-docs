# isReversible

**Framework**: RealityKit  
**Kind**: property  
**Required**: Yes

A Boolean value that determines whether the action reverses prior operations when playback is reverses.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- visionOS 2.0+

## Declaration

```swift
var isReversible: Bool { get }
```

#### Discussion

The default value is `true`.
