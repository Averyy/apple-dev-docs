# positionIterations

**Framework**: RealityKit  
**Kind**: property

The number of position iterations the solver performs.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
- visionOS 2.0+

## Declaration

```swift
var positionIterations: Int
```

#### Discussion

Use a value in the inclusive range `[1, 255]`. The default value is `6`.
