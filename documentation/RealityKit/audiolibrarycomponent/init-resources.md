# init(resources:)

**Framework**: RealityKit  
**Kind**: init

Creates a new audio library from a dictionary.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
- visionOS 2.0+

## Declaration

```swift
init(resources: [String : AudioResource] = [:])
```

## Parameters

- `resources`: A dictionary of audio resources that you key by name.
