# fadeOutDuration

**Framework**: RealityKit  
**Kind**: property

The time it takes in seconds for the effect to fade out.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- visionOS 2.0+

## Declaration

```swift
var fadeOutDuration: TimeInterval { get set }
```

#### Discussion

Maximum value allowed is 10 seconds.
