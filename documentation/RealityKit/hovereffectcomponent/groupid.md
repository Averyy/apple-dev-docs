# HoverEffectComponent.GroupID

**Framework**: RealityKit  
**Kind**: struct

A struct for grouping hover effects together.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- visionOS 26.0+

## Declaration

```swift
struct GroupID
```

#### Overview

Use this struct to group hover effects together and activate them at the same time from a single input. Hover effects in a group will not respond to the default hierarchical system behaviors.
