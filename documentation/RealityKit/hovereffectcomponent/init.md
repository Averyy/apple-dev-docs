# init(_:)

**Framework**: RealityKit  
**Kind**: init

Creates a hover effect component with a particular visualization effect.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- visionOS 2.0+

## Declaration

```swift
init(_ hoverEffect: HoverEffectComponent.HoverEffect)
```

## Parameters

- `hoverEffect`: A hover effect that defines   the visualization of this component.
