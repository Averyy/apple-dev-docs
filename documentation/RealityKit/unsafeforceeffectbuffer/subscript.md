# subscript(_:)

**Framework**: RealityKit  
**Kind**: subscript

Returns an element by index.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
- visionOS 2.0+

## Declaration

```swift
subscript(index: Int) -> T { get }
```

#### Overview

The index should be in the range [0, [`physicsBodyCount`](forceeffectparameters/physicsbodycount.md)].
