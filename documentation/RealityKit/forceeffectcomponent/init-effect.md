# init(effect:)

**Framework**: RealityKit  
**Kind**: init

Creates a force effect component with a single force effect, and automatically plays it.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
- visionOS 2.0+

## Declaration

```swift
init(effect: any ForceEffectBase)
```

## Parameters

- `effect`: The  effect to simulate forces with.
