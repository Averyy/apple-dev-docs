# BindTarget.path(_:)

**Framework**: RealityKit  
**Kind**: case

Provides a complex bind path capable of animating additional entities other than the current entity.

**Availability**:

- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
