# to

**Framework**: RealityKit  
**Kind**: property

The state of the animated property after the animation ends.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- visionOS 2.0+

## Declaration

```swift
let to: Value?
```

#### Discussion

When the value is `nil`, the system uses the default source value.
