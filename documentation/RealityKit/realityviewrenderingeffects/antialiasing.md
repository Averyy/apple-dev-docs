# antialiasing

**Framework**: RealityKit  
**Kind**: property

Enables or disables an antialiasing effect which smooths the edges of virtual content.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+

## Declaration

```swift
var antialiasing: AntialiasingMode
```
