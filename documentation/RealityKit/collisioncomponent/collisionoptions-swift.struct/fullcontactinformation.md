# fullContactInformation

**Framework**: RealityKit  
**Kind**: property

Reports full contact information for collision events.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+
- visionOS 1.0+

## Declaration

```swift
static let fullContactInformation: CollisionComponent.CollisionOptions
```

#### Discussion

> **Note**: This does not work with triggers because triggers do not have contact information.
