# receivesShadow

**Framework**: RealityKit  
**Kind**: property

A Boolean value that indicates whether an entity with the grounding shadow component receives grounding shadows from other model entities.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
- visionOS 2.0+
