# intrinsics

**Framework**: RealityKit  
**Kind**: property

A matrix that converts between the 2D camera plane and the 3D world coordinate space defined by the `sessionID`.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+

## Declaration

```swift
var intrinsics: simd_float3x3
```
