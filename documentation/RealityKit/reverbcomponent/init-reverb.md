# init(reverb:)

**Framework**: RealityKit  
**Kind**: init

Creates a component from a reverberation setting.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
- visionOS 2.0+

## Declaration

```swift
init(reverb: Reverb)
```

## Parameters

- `reverb`: A setting the component uses to reverberate spatial audio.
