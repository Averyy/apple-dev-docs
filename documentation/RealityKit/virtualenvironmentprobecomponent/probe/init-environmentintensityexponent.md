# init(environment:intensityExponent:)

**Framework**: RealityKit  
**Kind**: init

Creates a virtual-environment probe from an environment resource and intensity value.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 26.0+ (Beta)
- visionOS 2.0+

## Declaration

```swift
init(environment: EnvironmentResource, intensityExponent: Float = 0)
```

## Parameters

- `environment`: An environment resource.
- `intensityExponent`: An intensity value for the resource.
