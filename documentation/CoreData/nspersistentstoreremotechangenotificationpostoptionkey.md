# NSPersistentStoreRemoteChangeNotificationPostOptionKey

**Framework**: Core Data  
**Kind**: var

A key that indicates a persistent store posts a remote change notification for every write to the store, including writes by other processes.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+
