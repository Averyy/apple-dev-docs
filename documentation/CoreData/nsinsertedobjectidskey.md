# NSInsertedObjectIDsKey

**Framework**: Core Data  
**Kind**: var

A user info key to identify inserted object identifiers in notifications after saving a managed object context.

**Availability**:
- iOS 10.3+
- iPadOS 10.3+
- Mac Catalyst 13.1+
- macOS 10.12.4+
- tvOS 10.2+
- visionOS 1.0+
- watchOS 3.2+
