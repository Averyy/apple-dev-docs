# reset()

**Framework**: Core Data  
**Kind**: method

Resets the association tables for the migration.

**Availability**:
- iOS 3.0+
- iPadOS 3.0+
- Mac Catalyst 13.1+
- macOS 10.5+
- tvOS ?+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
func reset()
```

#### Discussion

This method does not reset the source or destination contexts.
