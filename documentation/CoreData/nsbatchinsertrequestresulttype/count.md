# NSBatchInsertRequestResultType.count

**Framework**: Core Data  
**Kind**: case

A value that indicates that the return type is the number of inserted rows.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
case count
```
