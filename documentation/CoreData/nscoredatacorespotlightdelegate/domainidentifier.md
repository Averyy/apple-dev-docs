# domainIdentifier()

**Framework**: Core Data  
**Kind**: method

Returns the domain identifier.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.13+
- visionOS 1.0+

## Declaration

```swift
func domainIdentifier() -> String
```

#### Discussion

The default value is the persistent store’s identifier.
