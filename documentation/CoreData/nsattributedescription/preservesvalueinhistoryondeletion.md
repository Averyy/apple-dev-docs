# preservesValueInHistoryOnDeletion

**Framework**: Core Data  
**Kind**: property

A Boolean value that indicates whether the attribute records its value in the persistent history transaction for a managed object’s deletion.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
