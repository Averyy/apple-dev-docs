# devices

**Framework**: Matter  
**Kind**: property

Returns the list of MTRDevice instances that this controller has loaded into memory. Returns an empty array if no devices are in memory.

**Availability**:
- iOS 18.4+
- iPadOS 18.4+
- Mac Catalyst 18.4+
- macOS 15.4+
- tvOS 18.4+
- visionOS 2.4+
- watchOS 11.4+
