# forgetDevice(withNodeID:)

**Framework**: Matter  
**Kind**: method

Forget any information we have about the device with the given node ID.  That includes clearing any information we have stored about it.

**Availability**:
- iOS 18.4+
- iPadOS 18.4+
- Mac Catalyst 18.4+
- macOS 15.4+
- tvOS 18.4+
- visionOS 2.4+
- watchOS 11.4+
