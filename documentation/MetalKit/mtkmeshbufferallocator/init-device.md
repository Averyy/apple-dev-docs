# init(device:)

**Framework**: MetalKit  
**Kind**: init

Initializes a new allocator object.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
init(device: any MTLDevice)
```

#### Return Value

An initialized allocator object.

## Parameters

- `device`: The Metal device on which to create buffers.
