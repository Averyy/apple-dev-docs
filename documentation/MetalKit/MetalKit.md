# MetalKit

**Framework**: MetalKit  
**Kind**: module

Build Metal apps quicker and easier using a common set of utility classes.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.0+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+
