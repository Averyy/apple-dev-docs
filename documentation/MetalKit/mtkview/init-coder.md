# init(coder:)

**Framework**: MetalKit  
**Kind**: init

Initializes a view from data in a given unarchiver.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
init(coder: NSCoder)
```

#### Return Value

An initialized view object.

## Parameters

- `coder`: An unarchiver object.
