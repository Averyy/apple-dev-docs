# BADownloadDomainAllowList

**Framework**: Bundle Resources  
**Kind**: typealias

The permitted list of domains the extension can use when scheduling the initial set of asset downloads.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- macOS 13.0+
- tvOS 18.4+
- visionOS 2.4+

#### Discussion

This key is required to use Background Assets.
