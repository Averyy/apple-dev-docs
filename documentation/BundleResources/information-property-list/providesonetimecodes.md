# ProvidesOneTimeCodes

**Framework**: Bundle Resources  
**Kind**: typealias

Allows this credential provider to show up in one-time-code text fields, to allowing filling time-based verification codes.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- visionOS 2.0+
