# UIURLToLaunchScreenAssociations

**Framework**: Bundle Resources  
**Kind**: dictionary

The mapping of URL schemes to launch screen configurations.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+

#### Discussion

Set the keys of this dictionary to the URL schemes that your app supports. Provide a value for each key that is the identifier, stored in the [`UILaunchScreenIdentifier`](information-property-list/uilaunchscreens/uilaunchscreendefinitions/uilaunchscreenidentifier.md) key, of one of the launch screen definitions in your [`UILaunchScreenDefinitions`](information-property-list/uilaunchscreens/uilaunchscreendefinitions.md) array.
