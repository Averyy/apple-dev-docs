# CFBundleURLTypes

**Framework**: Bundle Resources  
**Kind**: dictionary

A list of URL schemes (http, ftp, and so on) supported by the app.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- macOS 10.0+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+
