# BGTaskSchedulerPermittedIdentifiers

**Framework**: Bundle Resources  
**Kind**: typealias

An array of strings containing developer-specified task identifiers in reverse URL notation.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- tvOS 13.0+
- visionOS 1.0+
