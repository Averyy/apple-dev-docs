# NSHomeKitUsageDescription

**Framework**: Bundle Resources  
**Kind**: typealias

A message that tells the user why the app is requesting access to the user’s HomeKit configuration data.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- visionOS 1.0+
- watchOS 2.0+

#### Discussion

> ❗ **Important**:  This key is required if your app uses APIs that access the user’s HomeKit configuration data.
