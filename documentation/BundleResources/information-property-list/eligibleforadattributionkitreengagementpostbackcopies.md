# EligibleForAdAttributionKitReengagementPostbackCopies

**Framework**: Bundle Resources  
**Kind**: typealias

A Boolean value that indicates whether the developer receives copies of AdAttributionKit reengagement postbacks.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+

#### Discussion

Use this key to opt in to receive copies of reengagement postbacks. For more information, see [`Configuring an advertised app`](https://developer.apple.com/documentation/AdAttributionKit/configuring-an-advertised-app).
