# NSDockTilePlugIn

**Framework**: Bundle Resources  
**Kind**: typealias

The name of the app’s plug-in bundle.

**Availability**:
- macOS 10.0+

#### Discussion

Set this key to the name of the plug-in with the `.docktileplugin` filename extension that resides in the app’s `Contents/PlugIns` folder.
