# UILaunchImageOrientation

**Framework**: Bundle Resources  
**Kind**: typealias

A string containing the orientation of the image

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- tvOS 9.0+

#### Discussion

If it is not specified, the default orientation is `Portrait`.
