# BAAppGroupID

**Framework**: Bundle Resources  
**Kind**: typealias

The app group identifier that you share between your app and the extension that uses asset packs.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 16.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+
