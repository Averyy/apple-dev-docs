# UIApplicationExitsOnSuspend

**Framework**: Bundle Resources  
**Kind**: typealias

A Boolean value indicating whether the app terminates, rather than moves to the background, when the app quits.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+
