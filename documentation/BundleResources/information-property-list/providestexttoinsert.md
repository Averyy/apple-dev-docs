# ProvidesTextToInsert

**Framework**: Bundle Resources  
**Kind**: typealias

Allows this credential provider to show up in the system AutoFill context menu, so that it can fill text in any text field.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
