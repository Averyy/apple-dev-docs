# CoreSpotlightActions

**Framework**: Bundle Resources  
**Kind**: dictionary

A dictionary that contains details about actions available to users for Spotlight search results.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- visionOS 1.0+
