# OSLogPreferences

**Framework**: Bundle Resources  
**Kind**: dictionary

A dictionary that defines the logging configuration profile for the app.

**Availability**:
- macOS 10.15+

#### Discussion

The keys and values you use in this dictionary are the same as those you use in a logging configuration profile property list file, described in [`Customizing Logging Behavior While Debugging`](https://developer.apple.com/documentation/os/customizing-logging-behavior-while-debugging).
