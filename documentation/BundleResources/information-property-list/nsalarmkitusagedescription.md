# NSAlarmKitUsageDescription

**Framework**: Bundle Resources  
**Kind**: typealias

A message that tells people why the app is requesting access to schedule alarms.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)

#### Discussion

Include a descriptive string explaining why your app schedules alarms. This string appears in the system prompt when requesting authorization. If the `NSAlarmKitUsageDescription` key is missing or its value is an empty string, your app canâ€™t schedule alarms with AlarmKit.
