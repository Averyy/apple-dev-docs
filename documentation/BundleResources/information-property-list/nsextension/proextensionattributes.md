# ProExtensionAttributes

**Framework**: Bundle Resources  
**Kind**: dictionary

A dictionary that specifies the minimum size of the floating window in which Final Cut Pro hosts the extension view.

**Availability**:
- ProVideo Workflow Extensions 1.0+

#### Discussion

See [`Designing Workflow Extensions`](https://developer.apple.com/documentation/Professional-Video-Applications/designing-workflow-extensions).

## Topics

### Attributes
- [ContentViewMinimumHeight](information-property-list/nsextension/proextensionattributes/contentviewminimumheight.md)
  An integer that specifies the minimum height of the floating window.
- [ContentViewMinimumWidth](information-property-list/nsextension/proextensionattributes/contentviewminimumwidth.md)
  An integer that specifies the minimum width of the floating window.
