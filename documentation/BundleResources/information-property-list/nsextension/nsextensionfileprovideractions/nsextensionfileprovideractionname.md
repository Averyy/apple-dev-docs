# NSExtensionFileProviderActionName

**Framework**: Bundle Resources  
**Kind**: typealias

The localized name for a File Provider extension action that appears in the context menu.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- visionOS 1.0+
