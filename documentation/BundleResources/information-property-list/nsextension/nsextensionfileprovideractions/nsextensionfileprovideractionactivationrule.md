# NSExtensionFileProviderActionActivationRule

**Framework**: Bundle Resources  
**Kind**: typealias

A predicate that determines whether a File Provider extension action appears in the context menu.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- visionOS 1.0+
