# ASAccountAuthenticationModificationPasswordGenerationRequirements

**Framework**: Bundle Resources  
**Kind**: typealias

The rules the system satisfies when generating a strong password for your extension during an automatic upgrade.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- visionOS 1.0+

#### Discussion

For more information about the format and content of this value, see [`Customizing Password AutoFill rules`](https://developer.apple.com/documentation/Security/customizing-password-autofill-rules).
