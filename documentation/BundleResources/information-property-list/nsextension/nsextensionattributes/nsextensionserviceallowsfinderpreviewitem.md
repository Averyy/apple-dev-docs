# NSExtensionServiceAllowsFinderPreviewItem

**Framework**: Bundle Resources  
**Kind**: typealias

A Boolean value indicating whether the extension appears in the Finder Preview pane and Quick Actions menu.

**Availability**:
- macOS 10.14+
