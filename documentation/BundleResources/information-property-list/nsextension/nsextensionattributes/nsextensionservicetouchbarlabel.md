# NSExtensionServiceTouchBarLabel

**Framework**: Bundle Resources  
**Kind**: typealias

A name for display when the extension appears as a Quick Action in the Touch Bar.

**Availability**:
- macOS 10.14+

#### Discussion

This key is used in conjunction with the [`NSExtensionServiceAllowsTouchBarItem`](information-property-list/nsextension/nsextensionattributes/nsextensionserviceallowstouchbaritem.md) key.

If the [`NSExtensionServiceTouchBarLabel`](information-property-list/nsextension/nsextensionattributes/nsextensionservicetouchbarlabel.md) key isn’t provided, the extension’s display name is used.
