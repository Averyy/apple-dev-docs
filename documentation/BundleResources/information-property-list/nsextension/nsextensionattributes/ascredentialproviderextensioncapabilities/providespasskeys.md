# ProvidesPasskeys

**Framework**: Bundle Resources  
**Kind**: typealias

A Boolean value that indicates whether your credential provider extension provides passkeys.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- macOS 14.0+

#### Discussion

The operating system includes your extensionâ€™s passkeys in AutoFill and sign in sheets if this value is `true`.
