# ProvidesPasswords

**Framework**: Bundle Resources  
**Kind**: typealias

A Boolean value that indicates whether your credential provider extension provides passwords.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- macOS 14.0+
