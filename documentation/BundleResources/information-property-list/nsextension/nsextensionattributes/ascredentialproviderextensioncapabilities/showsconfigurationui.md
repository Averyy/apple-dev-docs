# ShowsConfigurationUI

**Framework**: Bundle Resources  
**Kind**: typealias

A Boolean value that indicates whether your extension presents a user interface when someone enables it in Passwords settings.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- macOS 14.0+
