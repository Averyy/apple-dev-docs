# NSExtensionActivationRule

**Framework**: Bundle Resources  
**Kind**: dictionary

The semantic data types that a Share or Action extension supports.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- macOS 10.10+
- visionOS 1.0+
