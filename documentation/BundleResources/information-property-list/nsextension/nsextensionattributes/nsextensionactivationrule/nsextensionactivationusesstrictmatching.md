# NSExtensionActivationUsesStrictMatching

**Framework**: Bundle Resources  
**Kind**: typealias

A Boolean value indicating whether strict or fuzzy matching is used when determining the asset types an app extension handles.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- macOS 10.11+
- visionOS 1.0+
