# NSExtensionActivationSupportsText

**Framework**: Bundle Resources  
**Kind**: typealias

A Boolean value indicating whether the app extension supports text.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- macOS 10.10+
- visionOS 1.0+
