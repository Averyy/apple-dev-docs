# NSExtensionFileProviderSupportsEnumeration

**Framework**: Bundle Resources  
**Kind**: typealias

A Boolean value that indicates whether a File Provider extension enumerates its content.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- visionOS 1.0+
