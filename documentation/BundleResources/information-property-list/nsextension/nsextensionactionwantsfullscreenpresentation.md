# NSExtensionActionWantsFullScreenPresentation

**Framework**: Bundle Resources  
**Kind**: typealias

A Boolean value indicating whether the Action extension is presented in full screen.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- visionOS 1.0+
