# UIPrerenderedIcon

**Framework**: Bundle Resources  
**Kind**: typealias

A Boolean value indicating whether the app’s icon files already incorporate a shine effect.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+
