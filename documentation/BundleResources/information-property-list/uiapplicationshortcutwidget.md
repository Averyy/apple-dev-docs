# UIApplicationShortcutWidget

**Framework**: Bundle Resources  
**Kind**: typealias

The bundle ID of the widget that’s available as a Home screen quick action in apps that have more than one widget.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- tvOS 9.0+
- watchOS 2.0+
