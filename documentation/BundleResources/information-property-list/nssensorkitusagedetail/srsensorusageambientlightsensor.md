# SRSensorUsageAmbientLightSensor

**Framework**: Bundle Resources  
**Kind**: dictionary

A collection of properties that explain your app’s need to observe light levels in the user’s environment.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+

#### Discussion

This key is a possible member of the [`NSSensorKitUsageDetail`](information-property-list/nssensorkitusagedetail.md) dictionary.

## Topics

### Sensor usage keys
- [Description](information-property-list/nssensorkitusagedetail/srsensorusageambientlightsensor/description.md)
  An explanatory string that details the manner in which your study uses the sensor’s data.
- [Required](information-property-list/nssensorkitusagedetail/srsensorusageambientlightsensor/required.md)
  A Boolean value that indicates whether your app’s study relies on this sensor.
