# SRSensorUsageOdometer

**Framework**: Bundle Resources  
**Kind**: dictionary

A collection of properties that explains your app’s need to observe the user’s odometer data.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+

#### Discussion

This key is a possible member of the [`NSSensorKitUsageDetail`](information-property-list/nssensorkitusagedetail.md) dictionary.

## Topics

### Sensor usage keys
- [Required](information-property-list/nssensorkitusagedetail/srsensorusageodometer/required.md)
  A Boolean value that indicates whether your app’s study relies on this sensor.
- [Description](information-property-list/nssensorkitusagedetail/srsensorusageodometer/description.md)
  An explanatory string that details the manner in which your study uses the sensor’s data.
