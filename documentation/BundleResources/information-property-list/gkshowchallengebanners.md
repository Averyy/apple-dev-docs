# GKShowChallengeBanners

**Framework**: Bundle Resources  
**Kind**: typealias

A Boolean value that indicates whether GameKit can display challenge banners in a game.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- visionOS 1.0+
