# LSBundleContainsCoreMLmlmodelc

**Framework**: Bundle Resources  
**Kind**: typealias

A Boolean value indicating whether the app contains a Core ML model to optimize loading the model.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- macOS 10.0+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+
