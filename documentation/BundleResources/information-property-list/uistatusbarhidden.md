# UIStatusBarHidden

**Framework**: Bundle Resources  
**Kind**: typealias

A Boolean value that indicates whether the system initially hides the status bar when the app launches.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+

#### Discussion

If `YES`, the system hides the status bar. If `NO`, it displays the status bar.
