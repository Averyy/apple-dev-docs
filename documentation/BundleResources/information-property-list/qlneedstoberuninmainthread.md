# QLNeedsToBeRunInMainThread

**Framework**: Bundle Resources  
**Kind**: typealias

A Boolean value indicating whether a Quick Look app’s generator can be run in threads other than the main thread.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- macOS 10.0+
- visionOS 1.0+
