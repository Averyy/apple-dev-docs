# LSSupportsOpeningDocumentsInPlace

**Framework**: Bundle Resources  
**Kind**: typealias

A Boolean value indicating whether the app may open the original document from a file provider, rather than a copy of the document.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- visionOS 1.0+
