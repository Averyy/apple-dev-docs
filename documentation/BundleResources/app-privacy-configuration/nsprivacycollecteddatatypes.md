# NSPrivacyCollectedDataTypes

**Framework**: Bundle Resources  
**Kind**: dictionary

A list of dictionaries that report the categories of private data your app or third-party SDK collects.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 14.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+
- watchOS 10.0+
