# NSPrivacyAccessedAPIType

**Framework**: Bundle Resources  
**Kind**: typealias

A string that describes a category of required reason API your app or third-party SDK uses.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 14.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+
- watchOS 10.0+
