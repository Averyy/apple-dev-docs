# NSPrivacyCollectedDataTypePurposes

**Framework**: Bundle Resources  
**Kind**: typealias

An array of strings that identify reasons your app or third-party SDK collects private data.

- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 14.0+
- macOS 14.0+
- tvOS 17.0+
