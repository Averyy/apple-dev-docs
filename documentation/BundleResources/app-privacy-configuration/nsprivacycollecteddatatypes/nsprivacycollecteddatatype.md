# NSPrivacyCollectedDataType

**Framework**: Bundle Resources  
**Kind**: typealias

A string that identifies the type of data your app or third-party SDK collects.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 14.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+
- watchOS 10.0+

## Mentions

- [Describing data use in privacy manifests](describing-data-use-in-privacy-manifests.md)

#### Discussion

Choose the value that matches the data your app or third-party SDK collects.
