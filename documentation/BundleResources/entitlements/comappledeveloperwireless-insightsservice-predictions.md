# Wireless Insights Service Predictions

**Framework**: Bundle Resources  
**Kind**: typealias

A Boolean value that indicates whether the app can use the WirelessInsights framework to obtain wireless service predictions.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
