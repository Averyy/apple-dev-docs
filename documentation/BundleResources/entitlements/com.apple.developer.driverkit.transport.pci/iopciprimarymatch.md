# IOPCIPrimaryMatch

**Framework**: Bundle Resources  
**Kind**: typealias

A key to match PCI devices by vendor and device ID registers.

**Availability**:
- macOS 10.15.4+

#### Discussion

This value of this key matches the vendor and device ID (`0x00`) register.
