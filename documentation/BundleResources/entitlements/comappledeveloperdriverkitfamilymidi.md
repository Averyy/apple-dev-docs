# com.apple.developer.driverkit.family.midi

**Framework**: Bundle Resources  
**Kind**: typealias

A Boolean value that indicates whether to match the driver against devices that support MIDI.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- macOS 15.0+
