# iCloud Container Identifiers Entitlement

**Framework**: Bundle Resources  
**Kind**: typealias

The container identifiers for the iCloud production environment.

**Availability**:
- iOS 3.0+
- iPadOS 3.0+
- macOS 10.7+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

#### Discussion

To add this entitlement to your app, enable the iCloud capability in Xcode.
