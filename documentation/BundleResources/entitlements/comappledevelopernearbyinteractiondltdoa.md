# Nearby Interaction NI DL-TDOA API (development)

**Framework**: Bundle Resources  
**Kind**: typealias

A Boolean value that indicates whether your app can enable Downlink Time-Difference-of-Arrival (DL-DTOA) ranging.

**Availability**:
- iOS 26.1+
- iPadOS 26.1+

#### Discussion

To add this entitlement to your app, search for and add Time-Difference-of-Arrival ranging in your target’s Signing & Capabilities tab in Xcode.

> **Note**: The Nearby Interaction DL-TDoA entitlement only supports development builds and Ad Hoc testing. App Store and Test Flight distribution using this capability isn’t currently available.
