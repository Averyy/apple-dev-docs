# Managed App Installation UI

**Framework**: Bundle Resources  
**Kind**: typealias

An entitlement you enable so your app can use Managed App Distribution.

**Availability**:
- iOS 17.2+
- iPadOS 17.2+

#### Discussion

To add this entitlement to your app, enable the Managed App Installation UI capability in Xcode.
