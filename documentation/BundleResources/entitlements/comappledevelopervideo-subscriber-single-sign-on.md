# com.apple.developer.video-subscriber-single-sign-on

**Framework**: Bundle Resources  
**Kind**: typealias

A Boolean value that indicates whether your app can use the TV Provider Authentication service.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- macOS 10.14+
- tvOS 10.0+

#### Discussion

This entitlement lets you use the [`Video Subscriber Account`](https://developer.apple.com/documentation/VideoSubscriberAccount) framework.
