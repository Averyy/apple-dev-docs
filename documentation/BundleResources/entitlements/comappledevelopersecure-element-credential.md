# com.apple.developer.secure-element-credential

**Framework**: Bundle Resources  
**Kind**: typealias

A Boolean value that indicates whether your app can use the SecureElementCredential framework.

**Availability**:
- iOS 18.1+
- iPadOS 18.1+
