# mediaItems

**Framework**: ShazamKit  
**Kind**: property

An array of the media items in the catalog that match the query signature, in order of the quality of the match.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+
