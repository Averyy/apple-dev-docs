# init(catalog:)

**Framework**: ShazamKit  
**Kind**: init

Creates a new managed session object for matching audio in a custom catalog.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+
- watchOS 10.0+

## Declaration

```swift
init(catalog: SHCatalog)
```

## Parameters

- `catalog`: The catalog that contains the reference audio signatures and their associated metadata.
