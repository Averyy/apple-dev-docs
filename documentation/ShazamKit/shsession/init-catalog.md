# init(catalog:)

**Framework**: ShazamKit  
**Kind**: init

Creates a new session object for matching audio in a custom catalog.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
init(catalog: SHCatalog)
```

## Parameters

- `catalog`: The catalog that contains the reference audio signatures and their associated metadata.
