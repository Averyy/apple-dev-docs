# init(properties:)

**Framework**: ShazamKit  
**Kind**: init

Creates a media item object with a dictionary of properties and their associated values.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
convenience init(properties: [SHMediaItemProperty : Any])
```

## Parameters

- `properties`: A dictionary that contains the media item properties and their associated values.
