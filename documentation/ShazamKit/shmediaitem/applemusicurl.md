# appleMusicURL

**Framework**: ShazamKit  
**Kind**: property

A link to the Apple Music page that contains the full information for the song.

**Availability**:

- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
