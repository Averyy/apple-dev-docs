# Playing and editing Cinematic mode video

**Framework**: Cinematic

Play and edit Cinematic mode video with an adjustable depth of field and focus points.

**Availability**:
- macOS 14.0+
- Xcode 15.0+

> **Note**: This sample code project is associated with WWDC23 session [`10137: Support Cinematic mode videos in your app`](https://developer.apple.comhttps://developer.apple.com/wwdc23/10137/).
