# init(style:)

**Framework**: UIKit  
**Kind**: init

Creates a blur effect with the designated style.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
@MainActor
init(style: UIBlurEffect.Style)
```

#### Return Value

The blur effect to be used by a UIVisualEffectView object.

## Parameters

- `style`: The intensity of the blur effect. See   for valid options.
