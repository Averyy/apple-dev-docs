# isAbsoluteOffset

**Framework**: UIKit  
**Kind**: property

A Boolean value that indicates whether the anchor’s offset is expressed as an absolute value.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
