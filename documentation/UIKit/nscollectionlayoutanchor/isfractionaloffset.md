# isFractionalOffset

**Framework**: UIKit  
**Kind**: property

A Boolean value that indicates whether the anchor’s offset is expressed as a fraction of its supplementary item’s dimension.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- tvOS 13.0+
- visionOS 1.0+
