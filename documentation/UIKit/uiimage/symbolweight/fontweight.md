# fontWeight()

**Framework**: UIKit  
**Kind**: method

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
func fontWeight() -> UIFont.Weight
```

#### Return Value

The [`UIFont.Weight`](uifont/weight.md) value that matches the specified symbol weight.
