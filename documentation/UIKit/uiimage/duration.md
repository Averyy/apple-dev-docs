# duration

**Framework**: UIKit  
**Kind**: property

The time interval for displaying an animated image.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
var duration: TimeInterval { get }
```

#### Discussion

For a non-animated image, the value of this property is `0.0`.
