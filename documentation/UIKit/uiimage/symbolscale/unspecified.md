# UIImage.SymbolScale.unspecified

**Framework**: UIKit  
**Kind**: case

An unspecified scale.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
case unspecified
```

#### Discussion

When you specify this value, the image inherits the scale value from its current environment.
