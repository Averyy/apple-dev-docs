# UIImage.SymbolScale.default

**Framework**: UIKit  
**Kind**: case

The default scale variant that matches the system usage.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
case `default`
```

#### Discussion

UIKit chooses the same scale used for similar content in other parts of the system.
