# systemGray6

**Framework**: UIKit  
**Kind**: property

A sixth-level shade of gray that adapts to the environment.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- visionOS 1.0+

## Declaration

```swift
class var systemGray6: UIColor { get }
```

#### Discussion

This color adapts to the current environment, and is close in color to [`systemBackground`](uicolor/systembackground.md). In light environments, this gray is slightly lighter than [`systemGray5`](uicolor/systemgray5.md). In dark environments, this gray is slightly darker than [`systemGray5`](uicolor/systemgray5.md).
