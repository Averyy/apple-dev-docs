# init(_:)

**Framework**: UIKit  
**Kind**: init

Creates a color object that encapsulates a SwiftUI color.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst ?+
- tvOS 14.0+
- visionOS ?+
- watchOS 7.0+

## Declaration

```swift
convenience init(_ color: Color)
```

## Parameters

- `color`: The initial color value, which can belong to any available color space.
