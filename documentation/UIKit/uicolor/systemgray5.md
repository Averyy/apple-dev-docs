# systemGray5

**Framework**: UIKit  
**Kind**: property

A fifth-level shade of gray that adapts to the environment.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- visionOS 1.0+

## Declaration

```swift
class var systemGray5: UIColor { get }
```

#### Discussion

This color adapts to the current environment. In light environments, this gray is slightly lighter than [`systemGray4`](uicolor/systemgray4.md). In dark environments, this gray is slightly darker than [`systemGray4`](uicolor/systemgray4.md).
