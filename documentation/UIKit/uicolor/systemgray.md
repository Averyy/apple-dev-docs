# systemGray

**Framework**: UIKit  
**Kind**: property

The standard base gray color that adapts to the environment.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
class var systemGray: UIColor { get }
```

#### Discussion

This color represents the standard system gray. It adapts to the current environment.
