# init(ciColor:)

**Framework**: UIKit  
**Kind**: init

Creates a color object that encapsulates a Core Image color.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
init(ciColor: CIColor)
```

#### Return Value

An initialized color object.

## Parameters

- `ciColor`: The Core Image color to convert.
