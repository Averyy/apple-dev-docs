# opaqueSeparator

**Framework**: UIKit  
**Kind**: property

The color for borders or divider lines that hides any underlying content.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
class var opaqueSeparator: UIColor { get }
```

#### Discussion

This color is always opaque. It adapts to the underlying trait environment.
