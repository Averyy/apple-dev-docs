# UISemanticContentAttribute.unspecified

**Framework**: UIKit  
**Kind**: case

The default value for views.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
case unspecified
```

#### Discussion

The view is flipped when switching between left-to-right and right-to-left layouts.
