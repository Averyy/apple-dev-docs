# hasActiveDrag

**Framework**: UIKit  
**Kind**: property

A Boolean value that indicates whether items were lifted from the collection view and have not yet been dropped.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- visionOS 1.0+
