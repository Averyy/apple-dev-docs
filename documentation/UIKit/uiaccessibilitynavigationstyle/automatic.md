# UIAccessibilityNavigationStyle.automatic

**Framework**: UIKit  
**Kind**: case

The assistive technology automatically determines how the receiver’s elements should be navigated.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
case automatic
```

#### Discussion

This is the default value.
