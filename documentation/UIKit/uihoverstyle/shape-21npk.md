# shape

**Framework**: UIKit  
**Kind**: property

The shape to use for the hover effect.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst ?+
- visionOS 1.0+

## Declaration

```swift
@MainActor
@preconcurrency var shape: UIShape? { get set }
```

#### Discussion

If the value of this property is `nil`, the system chooses a default shape based on the view.
