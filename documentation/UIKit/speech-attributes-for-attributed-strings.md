# Speech attributes for attributed strings

**Framework**: UIKit

Apply attributes to text in an attributed string to modify the pronunciation of that text.

- [static let accessibilitySpeechPitch: NSAttributedString.Key](../Foundation/NSAttributedString/Key/accessibilitySpeechPitch.md)
