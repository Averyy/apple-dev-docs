# UIKeyboardHIDUsage

**Framework**: UIKit  
**Kind**: enum

A set of HID usage codes that identify the keys of a USB keyboard.

**Availability**:
- iOS 13.4+
- iPadOS 13.4+
- Mac Catalyst 13.4+
- tvOS 13.4+
- visionOS 1.0+

## Declaration

```swift
enum UIKeyboardHIDUsage
```
