# pixelAspectRatio

**Framework**: UIKit  
**Kind**: property

The aspect ratio of a single pixel.

**Availability**:
- iOS 3.2+
- iPadOS 3.2+
- Mac Catalyst 13.1+
- tvOS ?+

## Declaration

```swift
var pixelAspectRatio: CGFloat { get }
```

#### Discussion

The aspect ratio is defined as `x/y`, where `x` is the width of the pixel and `y` is the height of the pixel.
