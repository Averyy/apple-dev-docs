# removeItem(_:)

**Framework**: UIKit  
**Kind**: method

Removes a specific dynamic item from the dynamic item behavior.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+

```swift
@MainActor
```

#### Discussion

## Parameters

- `item`: The dynamic item that you want to remove.
