# isPinned

**Framework**: UIKit  
**Kind**: property

A Boolean value that indicates whether the view is in a pinned state.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst ?+
- tvOS 15.0+
- visionOS ?+

## Declaration

```swift
var isPinned: Bool { get set }
```

#### Discussion

This state applies to header and footer views.
