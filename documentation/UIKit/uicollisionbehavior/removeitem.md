# removeItem(_:)

**Framework**: UIKit  
**Kind**: method

Removes a specific dynamic item from the collision behavior.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
@MainActor
func removeItem(_ item: any UIDynamicItem)
```

## Parameters

- `item`: The dynamic item that you want to remove.
