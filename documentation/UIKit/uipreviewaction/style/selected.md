# UIPreviewAction.Style.selected

**Framework**: UIKit  
**Kind**: case

The style for a selected peek quick action.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- tvOS 9.0+

## Declaration

```swift
case selected
```

#### Discussion

Use this style for the initially-selected item within a disclosed peek quick action group.
