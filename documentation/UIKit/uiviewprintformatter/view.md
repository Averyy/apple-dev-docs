# view

**Framework**: UIKit  
**Kind**: property

The view from which the view print formatter was derived.

**Availability**:
- iOS 4.2+
- iPadOS 4.2+
- Mac Catalyst 13.1+
- visionOS 1.0+

## Declaration

```swift
var view: UIView { get }
```

#### Discussion

You obtain a view print formatter by calling [`viewPrintFormatter()`](uiview/viewprintformatter().md) on a [`UIView`](uiview.md) object.
