# propertyList

**Framework**: UIKit  
**Kind**: property

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var propertyList: Any? { get }
```

#### Discussion

Use [`propertyList`](uikeycommand/propertylist.md) to associate a small amount of data to the command.
