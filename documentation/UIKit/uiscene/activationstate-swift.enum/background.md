# UIScene.ActivationState.background

**Framework**: UIKit  
**Kind**: case

A state that indicates that the scene is running in the background and is not onscreen.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
case background
```

#### Discussion

A background scene does not have a visible interface.
