# universalLinksOnly

**Framework**: UIKit  
**Kind**: property

A Boolean value that indicates whether URLs must be universal links and have a configured app to open them.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- tvOS 13.0+
- visionOS 1.0+
