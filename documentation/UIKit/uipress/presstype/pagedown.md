# UIPress.PressType.pageDown

**Framework**: UIKit  
**Kind**: case

A constant that represents the page down button.

**Availability**:
- tvOS 14.3+

## Declaration

```swift
case pageDown
```

#### Discussion

Add gesture recognizers for [`UIPress.PressType.pageDown`](uipress/presstype/pagedown.md) and [`UIPress.PressType.pageUp`](uipress/presstype/pageup.md) to navigate content in your app.

In an electronic program guide, select the item below the current one in response to [`UIPress.PressType.downArrow`](uipress/presstype/downarrow.md), and display the next screen of items in response to [`UIPress.PressType.pageDown`](uipress/presstype/pagedown.md). While your content plays, change the channel when your app receives [`UIPress.PressType.pageUp`](uipress/presstype/pageup.md) or [`UIPress.PressType.pageDown`](uipress/presstype/pagedown.md).

For more details on browsing multichannel content, see [`Providing Channel Navigation`](https://developer.apple.com/documentation/TVServices/providing-channel-navigation).
