# focused

**Framework**: UIKit  
**Kind**: property

The appearance data to apply to the tab bar item when it’s focused.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var focused: UITabBarItemStateAppearance { get }
```

#### Discussion

To set the attributes for this state, get the object in this property and modify it.
