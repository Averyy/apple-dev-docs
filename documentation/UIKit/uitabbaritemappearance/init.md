# init()

**Framework**: UIKit  
**Kind**: init

Creates an appearance object with default values for a stacked tab bar item.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
convenience init()
```

#### Return Value

A new appearance object containing appropriate default values for a stacked tab bar item.
