# backgroundEffect

**Framework**: UIKit  
**Kind**: property

The visual effect to apply to the button.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+

## Declaration

```swift
@NSCopying
@MainActor var backgroundEffect: UIVisualEffect? { get set }
```

#### Discussion

Assigning a visual effect object to this property adds that effect to the background of the action button.
