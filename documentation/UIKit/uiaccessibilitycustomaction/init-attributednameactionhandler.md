# init(attributedName:actionHandler:)

**Framework**: UIKit  
**Kind**: init

Creates a custom action object with the specified attributed name and action handler.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
```
