# translation

**Framework**: UIKit  
**Kind**: property

A vector to apply to a transform to match system interaction hinting.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.1+
- tvOS 12.0+
- visionOS 1.0+

## Declaration

```swift
var translation: CGVector { get }
```

#### Discussion

This property represents an X,Y-axis translation expressed in points.
