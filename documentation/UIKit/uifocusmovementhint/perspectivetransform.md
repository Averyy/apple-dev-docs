# perspectiveTransform

**Framework**: UIKit  
**Kind**: property

A 3D transform that represents a perspective matrix to be applied to match UIKit interaction hinting.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.1+
- tvOS 12.0+
- visionOS 1.0+
