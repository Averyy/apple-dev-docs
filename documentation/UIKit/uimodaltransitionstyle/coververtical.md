# UIModalTransitionStyle.coverVertical

**Framework**: UIKit  
**Kind**: case

**Availability**:

## Declaration

```swift
case coverVertical
```

#### Discussion

When the view controller is presented, its view slides up from the bottom of the screen. On dismissal, the view slides back down. This is the default transition style.
