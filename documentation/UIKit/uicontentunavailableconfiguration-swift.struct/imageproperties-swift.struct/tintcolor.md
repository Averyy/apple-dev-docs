# tintColor

**Framework**: UIKit  
**Kind**: property

The tint color applied to the image view.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst ?+
- tvOS 17.0+
- visionOS ?+

## Declaration

```swift
var tintColor: UIColor? { get set }
```

#### Discussion

Setting the value to `nil` makes the image view use its inherited tint color.
