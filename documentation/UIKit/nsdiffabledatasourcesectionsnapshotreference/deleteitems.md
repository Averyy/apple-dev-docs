# deleteItems(_:)

**Framework**: UIKit  
**Kind**: method

Deletes the items with the specified identifiers, and any of their child items, from the section snapshot.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- tvOS 14.0+
- visionOS 1.0+

## Declaration

```swift
func deleteItems(_ items: [Any])
```
