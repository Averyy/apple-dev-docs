# insert(_:afterItem:)

**Framework**: UIKit  
**Kind**: method

Inserts the provided section snapshot immediately after the item with the specified identifier in the section snapshot.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- tvOS 14.0+
- visionOS 1.0+
