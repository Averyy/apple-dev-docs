# isExpanded(_:)

**Framework**: UIKit  
**Kind**: method

Indicates whether the item with the specified identifier is in an expanded state.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- tvOS 14.0+
- visionOS 1.0+

## Declaration

```swift
func isExpanded(_ item: Any) -> Bool
```

#### Discussion

This expansion state persists along with the section snapshot.
