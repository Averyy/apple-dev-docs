# layoutQueue

**Framework**: UIKit  
**Kind**: property

The queue on which the framework dispatches layout operations.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- tvOS 15.0+
- visionOS 1.0+

## Declaration

```swift
var layoutQueue: OperationQueue? { get set }
```

#### Discussion

If non-`nil`, the queue the framework uses for layout operations.
