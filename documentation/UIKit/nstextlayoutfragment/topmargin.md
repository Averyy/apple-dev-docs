# topMargin

**Framework**: UIKit  
**Kind**: property

The amount of space reserved during paragraph layout between the top of the text layout fragment and the top of the first line in the paragraph.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- tvOS 15.0+
- visionOS 1.0+
