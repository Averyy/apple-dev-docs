# init(rawValue:)

**Framework**: UIKit  
**Kind**: init

Creates an instance of the enumeration with the provided unsigned integer value.

**Availability**:
- iOS 15.0+

- Mac Catalyst 15.0+
- tvOS 15.0+
- visionOS 1.0+

## Declaration

## Parameters
