# init(coder:)

**Framework**: UIKit  
**Kind**: init

Creates a segmented control with data from an unarchiver.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
@MainActor
init?(coder: NSCoder)
```

## Parameters

- `coder`: The unarchiver to read data from.
