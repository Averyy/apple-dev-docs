# init(frame:)

**Framework**: UIKit  
**Kind**: init

Creates an empty segmented control with the frame you specify.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
@MainActor
init(frame: CGRect)
```

## Parameters

- `frame`: A rectangle that specifies the segmented control’s frame in a superview’s coordinate system.
