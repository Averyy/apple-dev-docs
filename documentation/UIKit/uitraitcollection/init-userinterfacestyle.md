# init(userInterfaceStyle:)

**Framework**: UIKit  
**Kind**: init

Creates a trait collection that contains only the specified user interface style trait.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.1+
- tvOS 10.0+
- visionOS 1.0+

## Declaration

```swift
init(userInterfaceStyle: UIUserInterfaceStyle)
```

#### Return Value

A new trait collection containing only the interface style trait.

## Parameters

- `userInterfaceStyle`: The interface style for the trait collection. For a list of possible values, see  .
