# tabAccessoryEnvironment

**Framework**: UIKit  
**Kind**: property

The tab accessory environment represents whether a given trait collection is from a view in a `UITabAccessory` content view.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
