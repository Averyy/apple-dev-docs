# init()

**Framework**: UIKit  
**Kind**: init

Creates a new bar appearance object containing default values.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
convenience init()
```

#### Return Value

A new bar appearance object containing default values for the current device idiom.
