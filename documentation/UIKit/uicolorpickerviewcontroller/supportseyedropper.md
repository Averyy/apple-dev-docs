# supportsEyedropper

**Framework**: UIKit  
**Kind**: property

If set to `NO` the eyedropper functionality is not supported for this color picker.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- visionOS 26.0+

## Declaration

```swift
@MainActor
var supportsEyedropper: Bool { get set }
```
