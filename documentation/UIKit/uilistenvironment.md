# UIListEnvironment

**Framework**: UIKit  
**Kind**: enum

Constants that indicate the style of the containing list in a collection view or table view.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- tvOS 18.0+
- visionOS 2.0+
