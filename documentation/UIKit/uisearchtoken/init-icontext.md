# init(icon:text:)

**Framework**: UIKit  
**Kind**: init

Creates a search token with the specified text and icon (if any).

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- visionOS 1.0+

## Declaration

```swift
@MainActor
init(icon: UIImage?, text: String)
```

#### Return Value

A new search token.

## Parameters

- `icon`: An image to display with the text.
- `text`: The text to display on the search token.
