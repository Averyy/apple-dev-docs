# default

**Framework**: UIKit  
**Kind**: property

A default-priority announcement that interrupts existing speech, but is interruptible if a new speech utterance starts.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- tvOS 17.0+
- visionOS 1.0+
- watchOS 10.0+
