# init(text:)

**Framework**: UIKit  
**Kind**: init

Returns a simple-text print formatter initialized with plain text.

**Availability**:
- iOS 4.2+
- iPadOS 4.2+
- Mac Catalyst 13.1+
- visionOS 1.0+

## Declaration

```swift
init(text: String)
```

#### Return Value

An initialized instance of `UISimpleTextPrintFormatter` or `nil` if the object could not be created.

## Parameters

- `text`: A string of plain text or   if you intend to assign the text later.
