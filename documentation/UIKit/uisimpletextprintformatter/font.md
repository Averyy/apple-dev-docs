# font

**Framework**: UIKit  
**Kind**: property

The font of the printed text.

**Availability**:
- iOS 4.2+
- iPadOS 4.2+
- Mac Catalyst 13.1+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var font: UIFont? { get set }
```

#### Discussion

If the value of this property is `nil` (the default), UIKit uses the standard system font, 12 points.
