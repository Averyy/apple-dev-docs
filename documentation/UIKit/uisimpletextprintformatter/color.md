# color

**Framework**: UIKit  
**Kind**: property

The color of the printed text.

**Availability**:
- iOS 4.2+
- iPadOS 4.2+
- Mac Catalyst 13.1+
- visionOS 1.0+

## Declaration

```swift
var color: UIColor? { get set }
```

#### Discussion

If the value of this property is `nil` (the default), UIKit uses a black color when printing.
