# parent

**Framework**: UIKit  
**Kind**: property

A value that represents the parent element if this text element is a child of an enclosing element.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- tvOS 16.0+
- visionOS 1.0+
