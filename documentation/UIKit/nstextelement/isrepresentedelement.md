# isRepresentedElement

**Framework**: UIKit  
**Kind**: property

A Boolean value that indicates whether this element is in the text layout.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- tvOS 16.0+
- visionOS 1.0+
