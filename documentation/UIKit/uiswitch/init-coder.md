# init(coder:)

**Framework**: UIKit  
**Kind**: init

Creates a switch control from data in an unarchiver.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- visionOS 1.0+

## Declaration

```swift
@MainActor
init?(coder: NSCoder)
```

## Parameters

- `coder`: An object containing data used to initialize the switch control.
