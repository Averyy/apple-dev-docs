# UIReturnKeyType.continue

**Framework**: UIKit  
**Kind**: case

Specifies that the Return key displays an indication to continue what the user is doing.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- tvOS 9.0+
- visionOS 1.0+
