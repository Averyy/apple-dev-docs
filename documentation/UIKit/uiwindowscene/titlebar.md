# titlebar

**Framework**: UIKit  
**Kind**: property

The title bar displayed in a window of a Mac app.

**Availability**:
- Mac Catalyst 13.0+

## Declaration

```swift
@MainActor
var titlebar: UITitlebar? { get }
```

#### Discussion

Use this property to configure the title bar displayed across the top of a window in your Mac app built with Mac Catalyst.
