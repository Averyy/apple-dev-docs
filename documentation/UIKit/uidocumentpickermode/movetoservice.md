# UIDocumentPickerMode.moveToService

**Framework**: UIKit  
**Kind**: case

The document picker moves a local file outside the app’s sandbox and provides access to it as an external file.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
