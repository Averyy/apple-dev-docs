# init(delegate:)

**Framework**: UIKit  
**Kind**: init

Creates an object that tracks a week a person selects from a calendar view, with an optional delegate to manage selectable weeks and selection changes.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- visionOS 2.0+
