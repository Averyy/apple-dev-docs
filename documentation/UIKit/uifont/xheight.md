# xHeight

**Framework**: UIKit  
**Kind**: property

The x-height of the font.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
var xHeight: CGFloat { get }
```

#### Discussion

This value measures (in points) the height of the lowercase character “x”.
