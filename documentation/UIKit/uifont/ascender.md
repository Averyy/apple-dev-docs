# ascender

**Framework**: UIKit  
**Kind**: property

The top y-coordinate, offset from the baseline, of the font’s longest ascender.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
var ascender: CGFloat { get }
```

#### Discussion

The ascender value is measured in points.
