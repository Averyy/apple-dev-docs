# init(style:)

**Framework**: UIKit  
**Kind**: init

Creates an impact feedback generator with the specified style.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+

## Declaration

```swift
@MainActor
init(style: UIImpactFeedbackGenerator.FeedbackStyle)
```

#### Return Value

A newly initialized feedback generator.

#### Discussion

For more information on using feedback generators, see `Using feedback generators`.

## Parameters

- `style`: A value representing the mass of the colliding objects. For a list of valid feedback styles, see the   enumeration.
