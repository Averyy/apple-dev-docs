# fixedSpace()

**Framework**: UIKit  
**Kind**: method

Creates a new fixed space item of zero width.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- tvOS 26.0+
- visionOS 26.0+

## Declaration

```swift
@MainActor
class func fixedSpace() -> Self
```

#### Discussion

A fixed space of 0 width separates the shared background used in navigation bars and toolbars to visually group items.
