# UIViewAnimatingPosition.current

**Framework**: UIKit  
**Kind**: case

The current position. Use this constant when you want the most recent value set by an animator object.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- tvOS 10.0+
- visionOS 1.0+
