# init(defaultToolTip:)

**Framework**: UIKit  
**Kind**: init

Creates a tooltip interaction object and sets the default tooltip text.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
init(defaultToolTip: String)
```

#### Return Value

A tooltip interaction object.

## Parameters

- `defaultToolTip`: The text that appears in the tooltip by default.
