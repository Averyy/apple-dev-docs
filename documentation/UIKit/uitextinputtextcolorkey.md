# UITextInputTextColorKey

**Framework**: UIKit  
**Kind**: var

**Availability**:
- iOS 3.2+
- iPadOS 3.2+
- Mac Catalyst 13.1+

## Declaration

```swift

```

#### Discussion

The value of this key is a [`UIColor`](uicolor.md) object.
