# UITouch.TouchType.pencil

**Framework**: UIKit  
**Kind**: case

A touch from Apple Pencil.

**Availability**:
- iOS 9.1+
- iPadOS 9.1+
- Mac Catalyst 13.1+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
case pencil
```

#### Discussion

A pencil touch occurs when Apple Pencil interacts with the device’s screen.
