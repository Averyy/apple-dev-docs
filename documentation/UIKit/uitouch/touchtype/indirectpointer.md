# UITouch.TouchType.indirectPointer

**Framework**: UIKit  
**Kind**: case

A touch resulting from a button-based, indirect input device that describes the input sequence from button press to button release.

**Availability**:
- iOS 13.4+
- iPadOS 13.4+
- Mac Catalyst 13.4+
