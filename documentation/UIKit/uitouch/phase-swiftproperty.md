# phase

**Framework**: UIKit  
**Kind**: property

The phase of the touch.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var phase: UITouch.Phase { get }
```

#### Discussion

The property value is a constant that indicates whether the touch began, moved, ended, or was canceled. For descriptions of the possible values of this property, see [`UITouch.Phase`](uitouch/phase-swift.enum.md).
