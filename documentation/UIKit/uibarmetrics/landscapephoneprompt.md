# landscapePhonePrompt

**Framework**: UIKit  
**Kind**: property

Specifies metrics for landscape orientation using the phone idiom for bars with the prompt property, such as [`UINavigationBar`](uinavigationbar.md) and [`UISearchBar`](uisearchbar.md).

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- tvOS ?+
