# UILocalNotificationDefaultSoundName

**Framework**: UIKit  
**Kind**: var

Identifies the default system sound to play when a notification alert is displayed.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.1+
- watchOS 2.0+

## Declaration

```swift
let UILocalNotificationDefaultSoundName: String
```

#### Discussion

You assign this value to the [`soundName`](uilocalnotification/soundname.md) property.
