# isUniformAcrossSiblings

**Framework**: UIKit  
**Kind**: property

A Boolean value that indicates whether the dimension grows to match the dimension of its largest sibling.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- tvOS 17.0+
- visionOS 1.0+
