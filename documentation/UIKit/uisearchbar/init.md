# init()

**Framework**: UIKit  
**Kind**: init

Initializes the search bar to its default state.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
@MainActor
convenience init()
```

#### Return Value

An initialized search bar or `nil` if the view could not be initialized.
