# wantsIconRepresentation

**Framework**: UIKit  
**Kind**: property

A Boolean value that determines whether to render an icon of the document in the navigation bar.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- visionOS 1.0+
