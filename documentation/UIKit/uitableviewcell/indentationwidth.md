# indentationWidth

**Framework**: UIKit  
**Kind**: property

The width for each level of indentation of a cell’s content.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var indentationWidth: CGFloat { get set }
```

#### Discussion

The default indentation width is `10.0` points.
