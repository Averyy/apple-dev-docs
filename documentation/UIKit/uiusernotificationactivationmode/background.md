# UIUserNotificationActivationMode.background

**Framework**: UIKit  
**Kind**: case

Activate the app and put it in the background. If the app is already in the foreground, it remains in the foreground.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
