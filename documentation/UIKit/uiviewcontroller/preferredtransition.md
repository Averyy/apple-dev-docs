# preferredTransition

**Framework**: UIKit  
**Kind**: property

An object that defines the transition animation when switching to the view controller.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- tvOS 18.0+
- visionOS 2.0+
