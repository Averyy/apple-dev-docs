# init(wrappedValue:)

**Framework**: UIKit  
**Kind**: init

Creates a property wrapper that loads the view controller’s view before accessing the property.

**Availability**:
- iOS 16.4+
- iPadOS 16.4+
- Mac Catalyst ?+
- tvOS 16.4+
- visionOS ?+
- Swift 5.1+

## Declaration

```swift
init(wrappedValue: Value)
```

## Parameters

- `wrappedValue`: The underlying value tied to the loading of the view.
