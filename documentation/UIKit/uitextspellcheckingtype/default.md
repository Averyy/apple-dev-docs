# UITextSpellCheckingType.default

**Framework**: UIKit  
**Kind**: case

Specifies the default spell-checking behavior.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
case `default`
```

#### Discussion

The default behavior is to enable spell-checking whenever autocorrection is also enabled.
