# UITextDropProposal.Action.replaceAll

**Framework**: UIKit  
**Kind**: case

A text drop action style specifying that the dropped text replaces all text in the target text view.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- visionOS 1.0+
