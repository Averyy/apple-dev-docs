# usesDefaultHyphenation

**Framework**: UIKit  
**Kind**: property

A Boolean value that indicates whether the paragraph style uses the system hyphenation settings.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+
