# hasActiveDrag

**Framework**: UIKit  
**Kind**: property

A Boolean value that indicates whether the table view is currently tracking a drag session.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- visionOS 1.0+
