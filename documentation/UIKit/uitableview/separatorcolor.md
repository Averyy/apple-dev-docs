# separatorColor

**Framework**: UIKit  
**Kind**: property

The color of separator rows in the table view.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var separatorColor: UIColor? { get set }
```

#### Discussion

The default color is gray.
