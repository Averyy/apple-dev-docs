# cellLayoutMarginsFollowReadableWidth

**Framework**: UIKit  
**Kind**: property

A Boolean value that indicates whether the cell margins derive from the width of the readable content guide.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- tvOS 9.0+
- visionOS 1.0+
