# sectionHeaderTopPadding

**Framework**: UIKit  
**Kind**: property

The amount of padding above each section header.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- tvOS 15.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var sectionHeaderTopPadding: CGFloat { get set }
```

#### Discussion

The default value is [`automaticDimension`](uitableview/automaticdimension.md).
