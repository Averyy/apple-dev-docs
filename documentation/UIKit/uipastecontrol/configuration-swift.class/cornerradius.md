# cornerRadius

**Framework**: UIKit  
**Kind**: property

A value that rounds the edges of a paste button.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var cornerRadius: CGFloat { get set }
```

#### Discussion

Set this property to `0` for square edges. A nonzero value indicates rounded edges.
