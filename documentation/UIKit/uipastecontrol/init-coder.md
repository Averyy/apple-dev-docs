# init(coder:)

**Framework**: UIKit  
**Kind**: init

Creates a paste button by deserializing the specified coder.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
init?(coder: NSCoder)
```

## Parameters

- `coder`: An object that contains a serialized paste button.
