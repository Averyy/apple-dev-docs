# init(frame:)

**Framework**: UIKit  
**Kind**: init

Creates a paste button with the specified size and position.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
init(frame: CGRect)
```

## Parameters

- `frame`: A structure that determines the paste button’s size and position within the parent view’s frame.
