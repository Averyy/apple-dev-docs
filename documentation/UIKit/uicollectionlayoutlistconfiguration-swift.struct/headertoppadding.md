# headerTopPadding

**Framework**: UIKit  
**Kind**: property

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst ?+
- tvOS 15.0+
- visionOS ?+

## Declaration

```swift
var headerTopPadding: CGFloat? { get set }
```

#### Discussion

The default value is `nil`, which indicates automatic padding.
