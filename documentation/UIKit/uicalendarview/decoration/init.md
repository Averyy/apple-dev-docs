# init()

**Framework**: UIKit  
**Kind**: init

Creates a default calendar view decoration with a filled circle image, using the system fill color and medium size.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- visionOS 1.0+
