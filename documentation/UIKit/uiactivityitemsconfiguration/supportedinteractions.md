# supportedInteractions

**Framework**: UIKit  
**Kind**: property

The types of interactions that the configuration supports.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var supportedInteractions: [UIActivityItemsConfigurationInteraction] { get set }
```

#### Discussion

The default value is all known interactions.
