# reloadAllComponents()

**Framework**: UIKit  
**Kind**: method

Reloads all components of the picker view.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- visionOS 1.0+

## Declaration

```swift
@MainActor
func reloadAllComponents()
```

#### Discussion

Calling this method causes the picker view to query the delegate for new data for all components.
