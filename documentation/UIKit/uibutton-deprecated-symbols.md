# Deprecated symbols

Symbols that buttons no longer support.

### Button presentation
- [var adjustsImageWhenHighlighted: Bool](uibutton/adjustsimagewhenhighlighted.md)

  A Boolean value that determines whether the image changes when the button is disabled.
- [var showsTouchWhenHighlighted: Bool](uibutton/showstouchwhenhighlighted.md)
  A Boolean value that determines whether tapping the button causes it to glow.

- [var contentEdgeInsets: UIEdgeInsets](uibutton/contentedgeinsets.md)

- [var imageEdgeInsets: UIEdgeInsets](uibutton/imageedgeinsets.md)
  The inset or outset margins for the rectangle around the button’s image.
