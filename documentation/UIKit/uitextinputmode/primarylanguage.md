# primaryLanguage

**Framework**: UIKit  
**Kind**: property

The primary language, if any, of the input mode.

**Availability**:
- iOS 4.2+
- iPadOS 4.2+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var primaryLanguage: String? { get }
```

#### Discussion

The value of this property is a BCP 47 language code such as “es”, “en-US”, or “fr-CA”.
