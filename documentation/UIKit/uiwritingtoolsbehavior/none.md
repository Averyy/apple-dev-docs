# UIWritingToolsBehavior.none

**Framework**: UIKit  
**Kind**: case

An option to prevent the writing tools from modifying the text in the view.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- visionOS 2.4+
