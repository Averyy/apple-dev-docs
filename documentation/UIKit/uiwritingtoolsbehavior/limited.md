# UIWritingToolsBehavior.limited

**Framework**: UIKit  
**Kind**: case

An option to provide a limited, overlay-panel experience for the text view.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- visionOS 2.4+
