# NSRectAlignment

**Framework**: UIKit  
**Kind**: enum

Constants that specify alignment to an edge or a set of edges depending on the user interface layout direction.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
enum NSRectAlignment
```
