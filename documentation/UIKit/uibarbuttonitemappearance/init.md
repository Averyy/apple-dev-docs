# init()

**Framework**: UIKit  
**Kind**: init

Creates an appearance object with default values that are appropriate for a plain button.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
convenience init()
```

#### Return Value

A new bar button item appearance object containing the default appearances for a plain button.
