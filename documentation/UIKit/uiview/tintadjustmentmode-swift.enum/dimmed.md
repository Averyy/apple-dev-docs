# UIView.TintAdjustmentMode.dimmed

**Framework**: UIKit  
**Kind**: case

The view’s tint color property returns a desaturated, dimmed version of the view’s original tint color.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+
