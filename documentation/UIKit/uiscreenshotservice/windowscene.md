# windowScene

**Framework**: UIKit  
**Kind**: property

The window scene that contains the windows to capture in your PDF data.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- tvOS 13.0+

## Declaration

```swift
@MainActor
weak var windowScene: UIWindowScene? { get }
```

#### Discussion

Use this property to access the window scene containing the relevant content.
