# default

**Framework**: UIKit  
**Kind**: property

The default font metrics object for content.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- tvOS 11.0+
- visionOS 1.0+
- watchOS 4.0+

## Declaration

```swift
class var `default`: UIFontMetrics { get }
```

#### Discussion

The font metrics object in this property uses the [`body`](uifont/textstyle/body.md) style.
