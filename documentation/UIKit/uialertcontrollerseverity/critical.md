# UIAlertControllerSeverity.critical

**Framework**: UIKit  
**Kind**: case

Indicates that the system should present the alert using the critical, or caution, style.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- tvOS 16.0+
- visionOS 1.0+
