# UIDragPreviewTarget

**Framework**: UIKit  
**Kind**: class

A geometric specification for the source or destination of a drag item preview, used by the system when a user drops items or cancels a drag activity.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- visionOS 1.0+
