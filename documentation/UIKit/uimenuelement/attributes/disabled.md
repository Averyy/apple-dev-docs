# disabled

**Framework**: UIKit  
**Kind**: property

An attribute indicating the disabled style.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
static var disabled: UIMenuElement.Attributes { get }
```

#### Discussion

When you use this attribute, the user can’t select the menu element.
