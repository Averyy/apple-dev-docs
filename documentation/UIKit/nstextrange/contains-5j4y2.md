# contains(_:)

**Framework**: UIKit  
**Kind**: method

Determines if the text range you specify is in the current text range.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
func contains(_ textRange: NSTextRange) -> Bool
```

#### Return Value

Returns `true` if the range you provide is in the current range; otherwise `false`.
