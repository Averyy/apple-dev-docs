# announcementWasSuccessfulUserInfoKey

**Framework**: UIKit  
**Kind**: property

A Boolean value that indicates whether the announcement is successful.

**Availability**:
- iOS 6.0+
- iPadOS 6.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
nonisolated
static let announcementWasSuccessfulUserInfoKey: String
```

#### Discussion

The value of this key is an [`NSNumber`](https://developer.apple.com/documentation/Foundation/NSNumber) object that the system interprets as a Boolean value.
