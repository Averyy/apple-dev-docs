# buttonShapesEnabled

**Framework**: Uikit  
**Kind**: property

A Boolean value that indicates whether the Button Shapes setting is in an enabled state.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- tvOS 14.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
static var buttonShapesEnabled: Bool { get }
```
