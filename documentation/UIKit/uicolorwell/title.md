# title

**Framework**: UIKit  
**Kind**: property

The title for the color picker.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var title: String? { get set }
```

#### Discussion

Use this property to explain what kind of color to pick, like “Stroke Color” or “Fill Color”.
