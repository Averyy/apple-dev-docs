# supportsEyedropper

**Framework**: UIKit  
**Kind**: property

If set to `NO` the eyedropper functionality is not supported for this color well.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- visionOS 26.0+ (Beta)
