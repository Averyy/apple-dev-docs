# UIModalPresentationStyle.fullScreen

**Framework**: UIKit  
**Kind**: case

A presentation style in which the presented view covers the screen.

**Availability**:
- iOS ?+
- iPadOS ?+
- Mac Catalyst ?+
- tvOS ?+
- visionOS ?+

## Declaration

```swift
case fullScreen
```

#### Discussion

The views belonging to the presenting view controller are removed after the presentation completes.
