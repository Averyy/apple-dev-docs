# NSLayoutConstraint.Attribute.topMargin

**Framework**: UIKit  
**Kind**: case

The object’s top margin.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
case topMargin
```

#### Discussion

For [`UIView`](uiview.md) objects, the margins are defined by their [`layoutMargins`](uiview/layoutmargins.md) property.
