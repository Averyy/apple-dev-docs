# tintColor

**Framework**: UIKit  
**Kind**: property

The tint color to apply to the accessory.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst ?+
- tvOS 14.0+
- visionOS ?+

## Declaration

```swift
var tintColor: UIColor?
```

#### Discussion

The default value of this property is `nil`, which means that the accessory uses the system default tint color.
