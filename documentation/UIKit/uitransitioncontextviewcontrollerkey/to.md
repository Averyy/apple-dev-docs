# to

**Framework**: UIKit  
**Kind**: property

A key that identifies the view controller that’s visible at the end of a completed transition.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
static let to: UITransitionContextViewControllerKey
```

#### Discussion

This view controller is the one being presented.
