# NSWritingDirection.natural

**Framework**: UIKit  
**Kind**: case

The writing direction of the current script that the system determines using the Unicode Bidi Algorithm rules P2 and P3.

**Availability**:
- iOS 6.0+
- iPadOS 6.0+
- Mac Catalyst 13.1+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
case natural
```

#### Discussion

This is the default writing direction.
