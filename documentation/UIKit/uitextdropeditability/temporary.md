# UITextDropEditability.temporary

**Framework**: UIKit  
**Kind**: case

A text-drop editability specifier indicating that a noneditable text view does accept drops but reverts to its noneditable status immediately afterward.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- visionOS 1.0+

```swift
case temporary
```
