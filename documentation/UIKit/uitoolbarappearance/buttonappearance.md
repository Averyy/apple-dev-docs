# buttonAppearance

**Framework**: UIKit  
**Kind**: property

The appearance attributes for plain bar button items in the toolbar.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
@NSCopying
@MainActor var buttonAppearance: UIBarButtonItemAppearance { get set }
```

#### Discussion

To configure the appearance of a done button item, use the [`doneButtonAppearance`](uitoolbarappearance/donebuttonappearance.md) property instead.
