# bouncesVertically

**Framework**: UIKit  
**Kind**: property

A Boolean value that determines whether the scroll view bounces when it reaches the ends of its vertical axis.

**Availability**:
- iOS 17.4+
- iPadOS 17.4+
- Mac Catalyst 17.4+
- tvOS 17.4+
- visionOS 1.1+
