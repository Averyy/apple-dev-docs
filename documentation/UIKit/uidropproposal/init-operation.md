# init(operation:)

**Framework**: UIKit  
**Kind**: init

Initializes a new drop proposal with a drop operation type.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- visionOS 1.0+

## Declaration

```swift
@MainActor
init(operation: UIDropOperation)
```

#### Return Value

A drop proposal that has a drop operation.

## Parameters

- `operation`: The operation that the drop interaction proposes to perform.
