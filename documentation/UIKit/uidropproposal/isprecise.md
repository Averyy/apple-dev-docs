# isPrecise

**Framework**: UIKit  
**Kind**: property

A Boolean value that proposes that the drop interaction define the drop location precisely, such as at a specific point within existing text.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- visionOS 1.0+
