# UITextInputTextBackgroundColorKey

**Framework**: UIKit  
**Kind**: var

The background color of the text.

**Availability**:
- iOS 3.2+
- iPadOS 3.2+
- Mac Catalyst 13.1+

## Declaration

```swift
let UITextInputTextBackgroundColorKey: String
```

#### Discussion

The value of this key is a [`UIColor`](uicolor.md) object.
