# UITab.Placement.optional

**Framework**: UIKit  
**Kind**: case

The item doesn’t appear in the tab bar, but people can add it and move it.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- tvOS 18.0+
- visionOS 2.0+

## Declaration

```swift
case optional
```
