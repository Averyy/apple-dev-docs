# UIStackView.Alignment.leading

**Framework**: UIKit  
**Kind**: case

A layout for vertical stacks where the stack view aligns the leading edge of its arranged views along its leading edge.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
case leading
```

#### Discussion

This value is equivalent to the [`top`](uistackview/alignment-swift.enum/top.md) alignment for horizontal stacks.

The following image shows an example of a vertical stack view that uses the [`UIStackView.Alignment.leading`](uistackview/alignment-swift.enum/leading.md) alignment.

![A vertical stack view with four arranged subviews. The stack view aligns the subviews to its leading edge.](https://docs-assets.developer.apple.com/published/139d69d392b2ec2d73b8956be18f4c7a/media-2557458%402x.png)
