# UIStackView.Alignment.lastBaseline

**Framework**: UIKit  
**Kind**: case

A layout for horizontal stacks where the stack view aligns its arranged views based on their last baseline.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
case lastBaseline
```
