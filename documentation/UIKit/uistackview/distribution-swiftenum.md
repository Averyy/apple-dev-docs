# UIStackView.Distribution

**Framework**: UIKit  
**Kind**: enum

The layout that defines the size and position of the arranged views along the stack view’s axis.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- tvOS 9.0+
- visionOS 1.0+
