# UIBandSelectionInteraction.State.ended

**Framework**: UIKit  
**Kind**: case

A state that indicates the current interaction ended.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- visionOS 1.0+

## Declaration

```swift
case ended
```

#### Discussion

Use this state to finalize the interaction. For example, you might finalize the selection of items in your view.
