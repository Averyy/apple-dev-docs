# point

**Framework**: UIKit  
**Kind**: property

Returns the geometric point of interest for the accessibility location descriptor within its associated view and in the coordinate space of the view.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- visionOS 1.0+
