# physicalValue

**Framework**: UIKit  
**Kind**: property

An option to detect strings with the format of a physical value, such as length or temperature.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- visionOS 1.0+

## Declaration
