# localizedCreateDocumentActionTitle

**Framework**: UIKit  
**Kind**: property

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- visionOS 1.0+

## Declaration

```swift
@MainActor
```
