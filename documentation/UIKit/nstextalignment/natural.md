# NSTextAlignment.natural

**Framework**: UIKit  
**Kind**: case

Text uses the default alignment for the current localization of the app.

**Availability**:
- iOS 6.0+
- iPadOS 6.0+
- Mac Catalyst 13.1+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
case natural
```

#### Discussion

The default alignment for left-to-right scripts is [`NSTextAlignment.left`](nstextalignment/left.md), and the default alignment for right-to-left scripts is [`NSTextAlignment.right`](nstextalignment/right.md).
