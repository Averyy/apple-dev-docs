# largeSubtitle

**Framework**: UIKit  
**Kind**: property

String to be rendered below the large title.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)

## Declaration

```swift
@MainActor
var largeSubtitle: String? { get set }
```

#### Discussion

When `nil`, the navigation bar will fall back to the `subtitle`.
