# family

**Framework**: UIKit  
**Kind**: property

The font family attribute.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
static let family: UIFontDescriptor.AttributeName
```

#### Discussion

The value is an optional [`NSString`](https://developer.apple.com/documentation/Foundation/NSString) object that specifies the font family.
