# fixedSpace()

**Framework**: UIKit  
**Kind**: method

Returns a new group that contains a single zero-width fixed space item inside it.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)

## Declaration

```swift
@MainActor
class func fixedSpace() -> UIBarButtonItemGroup
```

#### Discussion

If you specify a group with a single zero-width fixed space, the navigation bar to visually separate the following groups.
