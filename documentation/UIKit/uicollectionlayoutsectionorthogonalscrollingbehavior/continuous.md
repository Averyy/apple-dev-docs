# UICollectionLayoutSectionOrthogonalScrollingBehavior.continuous

**Framework**: UIKit  
**Kind**: case

The section allows users to scroll its content orthogonally with continuous scrolling.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

