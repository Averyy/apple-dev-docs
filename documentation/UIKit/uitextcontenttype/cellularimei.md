# cellularIMEI

**Framework**: UIKit  
**Kind**: property

A property that defines the content in a text input area to contain an international mobile equipment identity number for an eSIM.

**Availability**:
- iOS 17.4+
- iPadOS 17.4+
- Mac Catalyst 17.4+
- tvOS 17.4+
- visionOS 1.1+

## Declaration

```swift
static let cellularIMEI: UITextContentType
```
