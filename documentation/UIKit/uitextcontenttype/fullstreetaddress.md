# fullStreetAddress

**Framework**: UIKit  
**Kind**: property

A property that defines the content in a text input area as a street address that fully identifies a location.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- tvOS 10.0+
- visionOS 1.0+

## Declaration

```swift
static let fullStreetAddress: UITextContentType
```
