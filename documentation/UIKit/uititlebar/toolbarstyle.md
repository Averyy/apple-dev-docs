# toolbarStyle

**Framework**: UIKit  
**Kind**: property

The style of the toolbar determining its appearance and location related to the title bar.

**Availability**:
- Mac Catalyst 14.0+

## Declaration

```swift
@MainActor
var toolbarStyle: UITitlebarToolbarStyle { get set }
```

#### Discussion

The default value is [`UITitlebarToolbarStyle.automatic`](uititlebartoolbarstyle/automatic.md).
