# isShowing(_:)

**Framework**: UIKit  
**Kind**: method

A Boolean value that indicates whether the split view interface is showing the specified column.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- tvOS 26.0+
- visionOS 26.0+

## Declaration

```swift
@MainActor
func isShowing(_ column: UISplitViewController.Column) -> Bool
```
