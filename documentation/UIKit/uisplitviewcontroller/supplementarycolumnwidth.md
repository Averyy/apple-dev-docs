# supplementaryColumnWidth

**Framework**: UIKit  
**Kind**: property

The width, in points, of the supplementary view controller’s content.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- tvOS 14.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var supplementaryColumnWidth: CGFloat { get }
```

#### Discussion

This property contains the actual width applied to the supplementary view controller’s view.
