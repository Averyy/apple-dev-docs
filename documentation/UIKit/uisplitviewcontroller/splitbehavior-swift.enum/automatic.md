# UISplitViewController.SplitBehavior.automatic

**Framework**: UIKit  
**Kind**: case

The split view controller automatically decides the most appropriate split behavior based on the device and the current app size.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- tvOS 14.0+
- visionOS 1.0+
