# init(for:)

**Framework**: UIKit  
**Kind**: init

Creates a text interaction with the specified mode.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- visionOS 1.0+

## Declaration

```swift
@MainActor
convenience init(for mode: UITextInteractionMode)
```

#### Return Value

A newly initialized text interaction object.

## Parameters

- `mode`: A mode determining the selection behaviors that the interaction will provide.
