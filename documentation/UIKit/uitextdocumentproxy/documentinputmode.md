# documentInputMode

**Framework**: UIKit  
**Kind**: property  
**Required**: Yes

The text-input mode for the keyboard.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- tvOS 10.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var documentInputMode: UITextInputMode? { get }
```

#### Discussion

Use the value of this key to configure the keyboard for the specified region and language.
