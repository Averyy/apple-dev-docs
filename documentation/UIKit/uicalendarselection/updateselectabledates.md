# updateSelectableDates()

**Framework**: UIKit  
**Kind**: method

Informs the calendar view to update the view for selectable dates.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
func updateSelectableDates()
```

#### Discussion

Call this function after you change the set of dates that are selectable in your calendar view.
