# isActivated

**Framework**: UIKit  
**Kind**: property

A Boolean value that indicates whether to display the system selection UI.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- tvOS 17.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var isActivated: Bool { get set }
```

#### Discussion

Set this property to [`true`](https://developer.apple.com/documentation/Swift/true) when you want the system to display the selection UI. Set the property to [`false`](https://developer.apple.com/documentation/Swift/false) to hide the selection UI again.
