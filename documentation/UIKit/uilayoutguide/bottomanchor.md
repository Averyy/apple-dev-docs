# bottomAnchor

**Framework**: UIKit  
**Kind**: property

A layout anchor representing the bottom edge of the layout guide’s frame.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var bottomAnchor: NSLayoutYAxisAnchor { get }
```

#### Discussion

Use this anchor to create constraints with the layout guide’s bottom edge. You can combine this anchor only with other [`NSLayoutYAxisAnchor`](nslayoutyaxisanchor.md) anchors. For more information, see [`NSLayoutAnchor`](nslayoutanchor.md).
