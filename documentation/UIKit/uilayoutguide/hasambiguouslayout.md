# hasAmbiguousLayout

**Framework**: UIKit  
**Kind**: property

A Boolean value indicating whether the constraints impacting the layout guide specify its location ambiguously.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
