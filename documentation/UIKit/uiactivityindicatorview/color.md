# color

**Framework**: UIKit

- iPadOS 5.0+

```swift

var color: UIColor! { get set }
```

#### Discussion

If you set a color for an activity indicator, it overrides the color provided by the [`style`](uiactivityindicatorview/style-swift.property.md) property.
