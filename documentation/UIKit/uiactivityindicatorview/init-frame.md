# init(frame:)

**Framework**: UIKit  
**Kind**: init

Creates an activity indicator with the specified frame rectangle.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
@MainActor
init(frame: CGRect)
```

## Parameters

- `frame`: The frame rectangle for the view, specified in points.
