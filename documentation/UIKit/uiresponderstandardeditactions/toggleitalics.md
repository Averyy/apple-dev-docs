# toggleItalics(_:)

**Framework**: UIKit  
**Kind**: method

Toggles the italic style information of the selected text.

**Availability**:
- iOS 6.0+
- iPadOS 6.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
@MainActor
optional func toggleItalics(_ sender: Any?)
```

#### Discussion

Use this method to apply or remove style information to the currently selected content.

## Parameters

- `sender`: The object calling this method.
