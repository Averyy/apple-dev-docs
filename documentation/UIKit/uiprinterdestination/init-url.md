# init(url:)

**Framework**: UIKit  
**Kind**: init

Creates a printer destination with the specified address.

**Availability**:
- iOS 14.5+
- iPadOS 14.5+
- Mac Catalyst 14.5+
- visionOS 1.0+

## Declaration

```swift
@MainActor
init(url: URL)
```

#### Return Value

An object representing the printer.

## Parameters

- `url`: A URL that identifies the location of the printer.
