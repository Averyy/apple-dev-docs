# url

**Framework**: UIKit  
**Kind**: property

The address of the printer.

**Availability**:
- iOS 14.5+
- iPadOS 14.5+
- Mac Catalyst 14.5+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var url: URL { get set }
```

#### Discussion

This property locates the IP address and resource path for the printer.
