# init(delegate:)

**Framework**: UIKit  
**Kind**: init

Creates a Scribble interaction that allows customizing the behavior of Scribble on text input views with the delegate you provide.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+

## Declaration

```swift
@MainActor
init(delegate: any UIScribbleInteractionDelegate)
```

## Parameters

- `delegate`: The object that acts as the delegate of this   instance.
