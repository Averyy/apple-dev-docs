# UITextField.DidEndEditingReason.committed

**Framework**: UIKit  
**Kind**: case

The text field changes were accepted by the user.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- tvOS 10.0+
- visionOS 1.0+

## Declaration

```swift
case committed
```

#### Discussion

On tvOS, text field changes become committed when the user presses the Done button.
