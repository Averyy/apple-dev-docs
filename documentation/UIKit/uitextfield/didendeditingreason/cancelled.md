# UITextField.DidEndEditingReason.cancelled

**Framework**: UIKit  
**Kind**: case

The text field changes were aborted.

**Availability**:
- tvOS 10.0+

## Declaration

```swift
case cancelled
```

#### Discussion

On tvOS, text field changes become cancelled when the user dismisses the text entry interface by pressing the Menu button.
