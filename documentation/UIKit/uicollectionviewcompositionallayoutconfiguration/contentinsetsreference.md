# contentInsetsReference

**Framework**: UIKit  
**Kind**: property

The boundary to reference when defining content insets.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- tvOS 14.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var contentInsetsReference: UIContentInsetsReference { get set }
```

#### Discussion
