# interSectionSpacing

**Framework**: UIKit  
**Kind**: property

The amount of space between the sections in the layout.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var interSectionSpacing: CGFloat { get set }
```

#### Discussion

The default value of this property is `0.0`.
