# removeTabStop(_:)

**Framework**: UIKit  
**Kind**: method

Removes the first text tab with a location and type equal to the specified tab stop.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+
