# init(markupText:)

**Framework**: UIKit  
**Kind**: init

Returns a markup-text print formatter initialized with an HTML string.

**Availability**:
- iOS 4.2+
- iPadOS 4.2+
- Mac Catalyst 13.1+
- visionOS 1.0+

## Declaration

```swift
@MainActor
init(markupText: String)
```

#### Return Value

An instance of `UIMarkupTextPrintFormatter` or `nil` if the object could not be created.

## Parameters

- `markupText`: A string of HTML markup text or   if you want to add the markup text later.
