# isSpringLoaded

**Framework**: UIKit  
**Kind**: property  
**Required**: Yes

A Boolean value that specifies whether the object is participating in spring-loaded interaction for a drag and drop activity.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- visionOS 1.0+
