# UITextSmartDashesType

**Framework**: UIKit  
**Kind**: enum

Constants that specify the automatic conversion behavior between hyphens and en or em dashes.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- tvOS 11.0+
- visionOS 1.0+
