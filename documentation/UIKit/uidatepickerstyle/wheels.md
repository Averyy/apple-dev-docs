# UIDatePickerStyle.wheels

**Framework**: UIKit  
**Kind**: case

A style indicating that the date picker displays as a wheel picker.

**Availability**:
- iOS 13.4+
- iPadOS 13.4+
- Mac Catalyst 13.4+
- visionOS 1.0+

## Declaration

```swift
case wheels
```

#### Discussion

Editing the date picker value occurs inline when using this style.
