# UIDatePickerStyle.automatic

**Framework**: UIKit  
**Kind**: case

A style indicating that the system picks the concrete style based on the current platform and date picker mode.

**Availability**:
- iOS 13.4+
- iPadOS 13.4+
- Mac Catalyst 13.4+
- visionOS 1.0+
