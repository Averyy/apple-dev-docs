# remove(_:)

**Framework**: UIKit  
**Kind**: method

Removes a placeholder object from the text input view.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
optional func remove(_ textPlaceholder: UITextPlaceholder)
```

## Parameters

- `textPlaceholder`: The placeholder object to remove.
