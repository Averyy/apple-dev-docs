# NSTextSelectionNavigation.Destination.paragraph

**Framework**: UIKit  
**Kind**: case

The selection moves to the next paragraph boundary, ignoring the end of line elastic characters and paragraph separators.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- tvOS 15.0+
