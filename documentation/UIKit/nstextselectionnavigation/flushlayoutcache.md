# flushLayoutCache()

**Framework**: UIKit  
**Kind**: method

Flushes cached layout information.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- tvOS 15.0+
- visionOS 1.0+

## Declaration

```swift
func flushLayoutCache()
```

#### Discussion

You call this method whenever the contents of the document change.
