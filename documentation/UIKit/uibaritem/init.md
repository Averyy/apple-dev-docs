# init()

**Framework**: UIKit  
**Kind**: init

Initializes the bar item to its default state.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
@MainActor
init()
```

#### Return Value

An initialized bar item or `nil` if the item could not be initialized.
