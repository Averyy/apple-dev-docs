# alpha

**Framework**: UIKit  
**Kind**: property

The transparency of the item.

**Availability**:
- iOS 6.0+
- iPadOS 6.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
@MainActor
var alpha: CGFloat { get set }
```

#### Discussion

Possible values are between 0.0 (transparent) and 1.0 (opaque). The default is 1.0.
