# propertyForGlyph(at:)

**Framework**: UIKit  
**Kind**: method

Returns the glyph property of the glyph at the specified index.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
func propertyForGlyph(at glyphIndex: Int) -> NSLayoutManager.GlyphProperty
```

#### Return Value

The glyph property associated with the specified glyph. [`NSLayoutManager.GlyphProperty`](nslayoutmanager/glyphproperty.md) lists the values that can be returned.

#### Discussion

If noncontiguous layout is not enabled, this method causes generation of all glyphs up to and including the one at `glyphIndex`.

## Parameters

- `glyphIndex`: The glyph whose glyph property is returned.
