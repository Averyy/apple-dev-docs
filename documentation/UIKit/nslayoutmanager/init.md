# init()

**Framework**: UIKit  
**Kind**: init

Initializes a newly created layout manager object.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
init()
```

#### Discussion

This method is the designated initializer for the `NSLayoutManager` class. Returns an initialized object.
