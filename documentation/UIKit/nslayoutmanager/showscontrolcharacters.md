# showsControlCharacters

**Framework**: UIKit  
**Kind**: property

A Boolean value that indicates whether the layout manager substitutes visible glyphs for control characters in the layout.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- tvOS ?+
