# addTextContainer(_:)

**Framework**: UIKit  
**Kind**: method

Appends the specified text container to the series of text containers where the layout manager arranges text.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift

```

- `container`: The text container to append.
