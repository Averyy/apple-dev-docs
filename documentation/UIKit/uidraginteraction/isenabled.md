# isEnabled

**Framework**: UIKit  
**Kind**: property

A Boolean value that specifies whether the drag interaction responds to touches and is allowed to participate in a drag activity.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- visionOS 1.0+
