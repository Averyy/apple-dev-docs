# connect()

**Framework**: Touch Controller  
**Kind**: method

Connects the touch controller to the app, allowing its controls to be drawn and an associated `GCController` to be created.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- visionOS 26.0+ (Beta)

```swift

```
