# contains(_:)

**Framework**: Touch Controller  
**Kind**: method  
**Required**: Yes

Determines whether the collider contains the specified point.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
func contains(_ point: CGPoint) -> Bool
```

#### Return Value

`YES` if the collider contains the point; otherwise, `NO`.

## Parameters

- `point`: The point to check.
