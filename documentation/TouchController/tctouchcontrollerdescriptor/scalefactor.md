# scaleFactor

**Framework**: Touch Controller  
**Kind**: property

The scale factor of the screen.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
var scaleFactor: CGFloat { get set }
```

#### Discussion

For example, 2.0 or 3.0 for Retina displays.
