# init(images:)

**Framework**: Touch Controller  
**Kind**: init

Creates a new instance with the specified images.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+

## Declaration

```swift
convenience init(images: [TCControlImage])
```

#### Return Value

A new `TCControlContents` instance.

## Parameters

- `images`: An array of   objects to use for the contents.
