# handleTouchMoved(at:)

**Framework**: Touch Controller  
**Kind**: method  
**Required**: Yes

Handles a touch moved event at the specified point.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+

## Declaration

```swift
func handleTouchMoved(at point: CGPoint)
```

## Parameters

- `point`: The point where the touch moved to.
