# toggleVisuals(for:of:of:)

**Framework**: Touch Controller  
**Kind**: method

The toggle visuals for the specified label, size, and shape.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
func toggleVisuals(for label: TCControlLabel, of size: CGSize, of shape: TCButtonVisualShape) -> TCControlVisuals
```

#### Return Value

The `TCControlVisuals` for the toggle button.

## Parameters

- `label`: The label for the toggle button.
- `size`: The size of the toggle button in points.
- `shape`: The shape of the toggle button.
