# buttonVisuals(for:of:of:)

**Framework**: Touch Controller  
**Kind**: method

The button visuals for the specified label, size, and shape.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
func buttonVisuals(for label: TCControlLabel, of size: CGSize, of shape: TCButtonVisualShape) -> TCControlVisuals
```

#### Return Value

The `TCControlVisuals` for the button.

## Parameters

- `label`: The label for the button.
- `size`: The size of the button in points.
- `shape`: The shape of the button.
