# init(touchController:)

**Framework**: Touch Controller  
**Kind**: init

Creates a new visuals provider with the specified touch controller.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
init(touchController: TCTouchController)
```

#### Return Value

A new `TCControlSystemVisualsProvider` instance.

## Parameters

- `touchController`: The touch controller to use for creating the visuals.
