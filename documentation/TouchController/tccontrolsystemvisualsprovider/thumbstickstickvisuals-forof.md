# thumbstickStickVisuals(for:of:)

**Framework**: Touch Controller  
**Kind**: method

The thumbstick stick visuals for the specified label and size.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
func thumbstickStickVisuals(for label: TCControlLabel, of size: CGSize) -> TCControlVisuals
```

#### Return Value

The `TCControlVisuals` for the thumbstick stick.

## Parameters

- `label`: The label for the thumbstick.
- `size`: The size of the thumbstick stick in points.
