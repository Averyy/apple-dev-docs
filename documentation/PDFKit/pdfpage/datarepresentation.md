# dataRepresentation

**Framework**: PDFKit  
**Kind**: property

Returns the PDF data (that is, a PDF document) representing this page. This method does not preserve external page links.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.4+
- tvOS 11.0+
- visionOS 1.0+
