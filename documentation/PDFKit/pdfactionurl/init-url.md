# init(url:)

**Framework**: PDFKit  
**Kind**: init

Initializes a URL action with the specified URL.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.5+
- tvOS 11.0+
- visionOS 1.0+

## Declaration

```swift
init(url: URL)
```

#### Return Value

An initialized `PDFActionURL` instance, or `NULL` if the object could not be initialized.

## Parameters

- `url`: The URL to set the action to.
