# setMaximumLength(_:)

**Framework**: PDFKit  
**Kind**: method

Sets the maximum number of characters allowed in the annotation string.

**Availability**:
- macOS 10.4+

## Declaration

```swift
func setMaximumLength(_ maxLen: Int)
```

## Parameters

- `maxLen`: The maximum number of characters allowed in the annotation string. Pass   to indicate that there is no specified maximum.
