# setFont(_:)

**Framework**: PDFKit  
**Kind**: method

Sets the font used in the text field of the annotation.

**Availability**:
- macOS 10.4+

## Declaration

```swift
func setFont(_ font: NSFont!)
```

## Parameters

- `font`: The font to be used in the annotation’s text field.
