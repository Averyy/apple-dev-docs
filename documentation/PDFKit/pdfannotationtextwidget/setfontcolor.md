# setFontColor(_:)

**Framework**: PDFKit  
**Kind**: method

Sets the font color used for the annotation’s text field.

**Availability**:
- macOS 10.4+

## Declaration

```swift
func setFontColor(_ color: NSColor!)
```

## Parameters

- `color`: The font color to be used in the annotation’s text field.
