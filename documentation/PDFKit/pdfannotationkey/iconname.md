# iconName

**Framework**: PDFKit  
**Kind**: property

A string value that specifies the name of an icon for a text or stamp annotation.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.13+
- tvOS 11.0+
- visionOS 1.0+
