# popup

**Framework**: PDFKit  
**Kind**: property

A dictionary or annotation object that specifies the annotation to pop up for text entry or editing.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.13+
- tvOS 11.0+
- visionOS 1.0+
