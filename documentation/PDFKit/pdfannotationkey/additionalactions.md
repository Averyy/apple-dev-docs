# additionalActions

**Framework**: PDFKit  
**Kind**: property

A dictionary or PDF action object that represents additional actions an annotation can perform, such as when it receives input focus.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.13+
- tvOS 11.0+
- visionOS 1.0+
