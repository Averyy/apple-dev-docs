# linePoints

**Framework**: PDFKit  
**Kind**: property

An array of floating point values that specifies the starting and ending points, in page-space coordinates, of a line.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.13+
- tvOS 11.0+
- visionOS 1.0+
