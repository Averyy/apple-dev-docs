# thumbnailSize

**Framework**: PDFKit  
**Kind**: property

**Availability**:

- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.5+
- tvOS 11.0+
- visionOS 1.0+

The maximum width and height of the thumbnails in the thumbnail view.
