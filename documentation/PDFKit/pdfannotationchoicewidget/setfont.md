# setFont(_:)

**Framework**: PDFKit  
**Kind**: method

Sets the font used to display the text in the widget annotation.

**Availability**:
- macOS 10.4+

## Declaration

```swift
func setFont(_ font: NSFont!)
```

## Parameters

- `font`: The font to be used for the text in the widget annotation.
