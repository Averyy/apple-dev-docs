# setFontColor(_:)

**Framework**: PDFKit  
**Kind**: method

Sets the font color used to display the text in the widget annotation.

**Availability**:
- macOS 10.4+

## Declaration

```swift
func setFontColor(_ color: NSColor!)
```

## Parameters

- `color`: The color of the font to be used for the text in the widget annotation.
