# setCaption(_:)

**Framework**: PDFKit  
**Kind**: method

Sets the text of the label on a push button control.

**Availability**:
- macOS 10.4+

## Declaration

```swift
func setCaption(_ name: String!)
```

## Parameters

- `name`: The text to be used as the label on a push button control.
