# isListChoice

**Framework**: PDFKit  
**Kind**: property

A Boolean value that indicates whether the choice widget annotation is a list or a pop-up menu.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 11.0+
- macOS 10.4+
- tvOS ?+
- visionOS 1.0+
