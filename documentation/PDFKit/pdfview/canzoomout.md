# canZoomOut

**Framework**: PDFKit  
**Kind**: property

Returns a Boolean value indicating whether the user can view an expanded area and zoom out.

**Availability**:

- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.4+
