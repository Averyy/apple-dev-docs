# canZoomIn

**Framework**: PDFKit  
**Kind**: property

Returns a Boolean value indicating whether the user can magnify the view and zoom in.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.4+
- tvOS 11.0+
- visionOS 1.0+
