# enableDataDetectors

**Framework**: PDFKit  
**Kind**: property

A Boolean value indicating whether to turns on or off data detection, which adds annotations for detected URLs in a page.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.6+
- tvOS 11.0+
- visionOS 1.0+
