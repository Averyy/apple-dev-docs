# go(to:)

**Framework**: PDFKit  
**Kind**: method

Scrolls to the first character of the specified selection.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.4+
- tvOS 11.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
func go(to selection: PDFSelection)
```

#### Discussion

PDF Kit records the move in its page history.
