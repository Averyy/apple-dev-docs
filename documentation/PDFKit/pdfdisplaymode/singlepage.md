# PDFDisplayMode.singlePage

**Framework**: PDFKit  
**Kind**: case

A display mode where the document displays one page at a time horizontally and vertically.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.4+
- tvOS 11.0+
- visionOS 1.0+

## Declaration

```swift
case singlePage
```

#### Discussion

Vertical and horizontal scrolling apply only to the current page.
