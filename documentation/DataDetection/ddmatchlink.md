# DDMatchLink

**Framework**: DataDetection  
**Kind**: class

An object that contains a web link that the data detection system matches.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
class DDMatchLink
```

#### Overview

The DataDetection framework returns a link match in a `DDMatchLink` object, which contains a [`URL`](https://developer.apple.com/documentation/Foundation/URL).

## Topics

### Getting link information
- [var url: URL](ddmatchlink/url.md)
  An address for a web resource, such as a webpage or image.
