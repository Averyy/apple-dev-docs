# Group Activities updates

**Framework**: Updates

Learn about important changes to Group Activities.

#### Overview

Browse notable changes in [`Group Activities`](https://developer.apple.com/documentation/GroupActivities).

#### June 2024

- Customize the placement of personas in a shared activity using the [`SpatialTemplate`](https://developer.apple.com/documentation/GroupActivities/SpatialTemplate) and [`SpatialTemplateElement`](https://developer.apple.com/documentation/GroupActivities/SpatialTemplateElement) protocols.
