# Core MIDI updates

**Framework**: Updates

Learn about important changes to Core MIDI.

#### Overview

Browse notable changes in [`Core MIDI`](https://developer.apple.com/documentation/CoreMIDI).

#### June 2024

##### General

- Add support for v1.2 MIDI-CI Specification that supports new compatible hardware.
- Add support for v1.1.1 UMP/MIDI 2.0 Protocol Specification that introduces new message types and features.

##### Mididriverkit

- Add support for DriverKit-based MIDI device drivers using the new [`MIDIDriverKit`](https://developer.apple.com/documentation/MIDIDriverKit) framework.
