# WeatherKit updates

**Framework**: Updates

Learn about important changes to WeatherKit.

#### Overview

Browse notable changes in [`WeatherKit`](https://developer.apple.comhttps://developer.apple.com/documentation/weatherkit).

#### June 2024

- Benefit from efficient transfers with FlatBuffers.
- Obtain hourly, daily, and monthly weather statistics for a particular location based on over 50 years of historical weather data.
- Obtain a daily summary of high temperature, low temperature, precipitation amount, and snowfall amount for a specific day since August 2021.
- Get a list of upcoming significant weather changes for the following elements: high temperature, low temperature, day precipitation amount, and night precipitation amount.
- Access a new property for the current weather: [`cloudCoverByAltitude`](https://developer.apple.com/documentation/WeatherKit/CurrentWeather/cloudCoverByAltitude).
- Access new elements for the current dayâ€™s weather: - [`highTemperatureTime`](https://developer.apple.com/documentation/WeatherKit/DayWeather/highTemperatureTime) and [`lowTemperatureTime`](https://developer.apple.com/documentation/WeatherKit/DayWeather/lowTemperatureTime)
- [`minimumHumidity`](https://developer.apple.com/documentation/WeatherKit/DayWeather/minimumHumidity) and [`maximumHumidity`](https://developer.apple.com/documentation/WeatherKit/DayWeather/maximumHumidity)
- [`minimumVisibility`](https://developer.apple.com/documentation/WeatherKit/DayWeather/minimumVisibility) and [`maximumVisibility`](https://developer.apple.com/documentation/WeatherKit/DayWeather/maximumVisibility)
- [`highWindSpeed`](https://developer.apple.com/documentation/WeatherKit/DayWeather/highWindSpeed)
- [`precipitationAmountByType`](https://developer.apple.com/documentation/WeatherKit/DayWeather/precipitationAmountByType)
- [`daytimeForecast`](https://developer.apple.com/documentation/WeatherKit/DayWeather/daytimeForecast), [`overnightForecast`](https://developer.apple.com/documentation/WeatherKit/DayWeather/overnightForecast), and [`restOfDayForecast`](https://developer.apple.com/documentation/WeatherKit/DayWeather/restOfDayForecast)
- Get forecasts for part of the day with [`DayPartForecast`](https://developer.apple.com/documentation/WeatherKit/DayPartForecast).
- Get hourly cloud cover by altitude with [`CloudCoverByAltitude`](https://developer.apple.com/documentation/WeatherKit/CloudCoverByAltitude).
