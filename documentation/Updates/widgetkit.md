# WidgetKit updates

**Framework**: Updates

Learn about important changes in WidgetKit.

#### Overview

Browse notable changes in [`WidgetKit`](https://developer.apple.com/documentation/WidgetKit).

#### June 2024

##### General

- Controls allow people to customize and configure Control Center, their Lock Screen, and the Action button with actions from your apps. Take advantage of [`ControlWidgetButton`](https://developer.apple.com/documentation/WidgetKit/ControlWidgetButton) to execute an action and [`ControlWidgetToggle`](https://developer.apple.com/documentation/WidgetKit/ControlWidgetToggle) to toggle some state in your app.
- Accent views of widgets on iOS and iPadOS using [`widgetAccentable(_:)`](https://developer.apple.com/documentation/SwiftUI/View/widgetAccentable(_:)).
- Provide context for when and where a widget is most relevant for surfacing in a Smart Stack with doc://com.apple.documentation/documentation/widgetkit/widgetrelevances. Define context using a collection of doc://com.apple.documentation/documentation/widgetkit/widgetrelevanceentry objects that work with existing [`TimelineProvider`](https://developer.apple.com/documentation/WidgetKit/TimelineProvider), [`AppIntentTimelineProvider`](https://developer.apple.com/documentation/WidgetKit/AppIntentTimelineProvider), and [`IntentTimelineProvider`](https://developer.apple.com/documentation/WidgetKit/IntentTimelineProvider) protocols.

##### Apple Watch

- Create interactive widgets on watchOS using a [`Button`](https://developer.apple.com/documentation/SwiftUI/Button) or [`Toggle`](https://developer.apple.com/documentation/SwiftUI/Toggle) to perform actions without launching your app.
- Use an [`AccessoryWidgetGroup`](https://developer.apple.com/documentation/WidgetKit/AccessoryWidgetGroup) container to provide up to three content views, optionally interactive, in a watchOS widget. Style [`AccessoryWidgetGroup`](https://developer.apple.com/documentation/WidgetKit/AccessoryWidgetGroup) content views using [`circular`](https://developer.apple.com/documentation/WidgetKit/AccessoryWidgetGroupStyle/circular) or [`roundedSquare`](https://developer.apple.com/documentation/WidgetKit/AccessoryWidgetGroupStyle/roundedSquare).
