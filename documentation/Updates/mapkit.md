# MapKit updates

**Framework**: Updates

Learn about important changes to MapKit.

#### Overview

Browse notable changes in [`MapKit`](https://developer.apple.comhttps://developer.apple.com/documentation/mapkit).

#### June 2024

- Obtain a place identifier to track a specific location, such as a business, park, physical feature, or landmark, over its lifetime.
- Show detailed information about a point of interest across [`MapKit`](https://developer.apple.comhttp://developer.apple.com/documentation/mapkit/mapkit_for_appkit_and_uikit), [`MapKit for SwiftUI`](https://developer.apple.comhttps://developer.apple.com/documentation/mapkit/mapkit_for_swiftui), and [`MapKit JS`](https://developer.apple.comhttps://developer.apple.com/documentation/mapkitjs).
- Filter location results using criteria such as neighborhoods, postal codes, and municipalities.
- Use pagination to obtain significantly more results over a successive set of calls when using the [`Apple Maps Server API`](https://developer.apple.comhttps://developer.apple.com/documentation/applemapsserverapi/).
