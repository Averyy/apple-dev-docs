# kCFStreamSocketSecurityLevelTLSv1

**Framework**: Core Foundation  
**Kind**: var

Specifies that TLS version 1 be set as the security protocol for a socket stream.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.2+
- tvOS ?+
- visionOS 1.0+
- watchOS 2.0+
