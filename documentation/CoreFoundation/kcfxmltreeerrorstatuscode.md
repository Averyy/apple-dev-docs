# kCFXMLTreeErrorStatusCode

**Framework**: Core Foundation  
**Kind**: var

Dictionary key whose value is a CFNumber containing the error status code. See [`CFXMLParser`](cfxmlparser.md) for possible status code values.

**Availability**:
- macOS ?+
