# weekday

**Framework**: Core Foundation  
**Kind**: property

Specifies the weekday unit.

**Availability**:
- iOS ?+
- iPadOS ?+
- Mac Catalyst ?+

## Declaration

```swift
static var weekday: CFCalendarUnit { get }
```

#### Discussion

The weekday units are the numbers `1`-`N` (where for the Gregorian calendar `N=7` and `1` is Sunday).
