# CFTimeZoneNameStyle.daylightSaving

**Framework**: Core Foundation  
**Kind**: case

Specifies the daylight saving name style; for example, “Central Daylight Time” for the Central time zone.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.5+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+
