# quotationEndDelimiterKey

**Framework**: Core Foundation  
**Kind**: property

Specifies the end quotation symbol associated with the locale.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.1+
- macOS 10.6+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
static let quotationEndDelimiterKey: CFLocaleKey!
```

#### Discussion

The corresponding value is a CFString object.
