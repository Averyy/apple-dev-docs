# kCFURLFileDirectoryContents

**Framework**: Core Foundation  
**Kind**: var

A `CFArray` object holding `CFURL` objects for the contents of a directory referred to by a URL.

**Availability**:
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
let kCFURLFileDirectoryContents: CFString!
```
