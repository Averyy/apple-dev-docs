# LocalizedError Implementations

**Framework**: TelephonyMessagingKit

## Topics

### Instance Properties
- [var errorDescription: String?](rcsservice/error/errordescription.md)
  A localized message describing what error occurred.
- [var failureReason: String?](rcsservice/error/failurereason.md)
  A localized message describing the reason for the failure.
- [var helpAnchor: String?](rcsservice/error/helpanchor.md)
  A localized message providing “help” text if the user requests help.
- [var recoverySuggestion: String?](rcsservice/error/recoverysuggestion.md)
  A localized message describing how one might recover from the failure.
