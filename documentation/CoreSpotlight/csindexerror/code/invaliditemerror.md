# CSIndexError.Code.invalidItemError

**Framework**: Core Spotlight  
**Kind**: case

The searchable item object is invalid.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- visionOS 1.0+

## Declaration

```swift
case invalidItemError
```

#### Discussion

Verify the information in your [`CSSearchableItem`](cssearchableitem.md) object is correct.
