# orientation

**Framework**: Core Spotlight  
**Kind**: property

The orientation of the data.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- visionOS 1.0+

## Declaration

```swift
var orientation: NSNumber? { get set }
```

#### Discussion

When the value of this property is 1, the orientation is portrait; when the value is 0, the orientation is landscape.
