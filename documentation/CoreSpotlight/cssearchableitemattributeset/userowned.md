# userOwned

**Framework**: Core Spotlight  
**Kind**: property

A value that indicates the user purchased or owns the item.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- visionOS 1.0+

## Declaration

```swift
var userOwned: NSNumber? { get set }
```

#### Discussion

Examples of items to set this property on include songs and movies the user purchases.
