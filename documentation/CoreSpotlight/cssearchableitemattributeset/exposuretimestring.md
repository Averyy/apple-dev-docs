# exposureTimeString

**Framework**: Core Spotlight  
**Kind**: property

The time that the lens was open during exposure, in a string, such as “1/250 seconds”.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- visionOS 1.0+

## Declaration

```swift
var exposureTimeString: String? { get set }
```
