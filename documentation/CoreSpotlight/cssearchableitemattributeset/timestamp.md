# timestamp

**Framework**: Core Spotlight  
**Kind**: property

The timestamp on the item.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- visionOS 1.0+

## Declaration

```swift
var timestamp: Date? { get set }
```

#### Discussion

The value of this property is generally used to specify the time at which the event captured by the item took place.
