# value(forCustomKey:)

**Framework**: Core Spotlight  
**Kind**: method

**Availability**:

- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- visionOS 1.0+

## Declaration

```swift
func value(forCustomKey key: CSCustomAttributeKey) -> (any NSSecureCoding)?
```

#### Return Value

The value associated with the custom attribute key.
