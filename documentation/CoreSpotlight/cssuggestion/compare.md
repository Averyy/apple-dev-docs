# compare(_:)

**Framework**: Core Spotlight  
**Kind**: method

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- visionOS 1.0+

## Declaration

```swift
func compare(_ other: CSSuggestion) -> ComparisonResult
```

#### Return Value

Returns an `NSComparisonResult`.

## Parameters

- `other`: The suggestion to compare to this suggestion.
