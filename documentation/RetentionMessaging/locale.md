# locale

**Framework**: Retention Messaging API  
**Kind**: typealias

A string that represents a locale short code.

**Availability**:
- Retention Messaging API 1.0+

## Declaration

```swift
string locale
```

#### Discussion

For a list of locale short codes the App Store supports, see [`Managing metadata in your app by using locale short codes`](https://developer.apple.comhttps://developer.apple.com/documentation/appstoreconnectapi/managing-metadata-in-your-app-by-using-locale-shortcodes).

The locale string is case-sensitive.

For a list of locales and languages the App Store supports, see [`App Store localizations`](https://developer.apple.comhttps://developer.apple.com/help/app-store-connect/reference/app-store-localizations/).
