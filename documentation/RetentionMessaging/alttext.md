# altText

**Framework**: Retention Messaging API  
**Kind**: typealias

The alternative text for a corresponding image.

**Availability**:
- Retention Messaging API 1.0+

## Declaration

```swift
string altText
```

#### Discussion

Provide the `altText` as a UTF-8-encoded string. The maximum string length for `altText` is `150`.
