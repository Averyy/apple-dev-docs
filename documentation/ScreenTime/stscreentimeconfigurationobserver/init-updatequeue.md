# init(updateQueue:)

**Framework**: Screen Time  
**Kind**: init

Creates a configuration observer that reports updates on the queue you specify.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+

## Declaration

```swift
init(updateQueue: dispatch_queue_t)
```

## Parameters

- `updateQueue`: The queue on which to report updates.
