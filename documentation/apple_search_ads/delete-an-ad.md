# Delete an Ad

**Framework**: Apple Search Ads  
**Kind**: httpRequest

Deletes an ad from an ad group.

**Availability**:
- Search Ads 5.0+

#### Discussion

Use this endpoint to delete an [`Ad`](ad.md) assignment from an ad group. Use your `adId` in the resource path.
