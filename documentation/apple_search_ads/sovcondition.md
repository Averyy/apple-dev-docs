# SovCondition

**Framework**: Apple Search Ads  
**Kind**: dictionary

The list of condition objects that allow users to filter a list of records.

**Availability**:
- Search Ads 4.6+

## Declaration

```swift
object SovCondition
```

#### Discussion

The `Condition` object functionality is similar to the `WHERE` clause in SQL.
