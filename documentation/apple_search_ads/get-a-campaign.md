# Get a Campaign

**Framework**: Apple Search Ads  
**Kind**: httpRequest

Fetches a specific campaign by campaign identifier.

**Availability**:
- Search Ads 5.0+

#### Discussion

Use this endpoint to return data for a specific campaign. You can also use a partial fetch as necessary. For more information, see the Use a Partial Fetch section of [`Using Apple Search Ads API Functionality`](using-apple-search-ads-api-functionality.md).

##### Payload Example Get a Campaign
