# Get All Ad Group Negative Keywords

**Framework**: Apple Search Ads  
**Kind**: httpRequest

Fetches all negative keywords in ad groups.

**Availability**:
- Search Ads 5.0+

#### Discussion

To return all ad group negative keywords, use the associated `campaignId` and `adgroupId` in the URI. You can also use a partial fetch. For more information, see the Use a Partial Fetch section of [`Using Apple Search Ads API Functionality`](using-apple-search-ads-api-functionality.md).

##### Payload Example Get All Ad Group Negative Keywords
