# init(_:)

**Framework**: LightweightCodeRequirements  
**Kind**: init

Matches if the process is running as the specified platform type or if the file supports the specified platform type.

**Availability**:
- iOS 17.4+
- iPadOS 17.4+
- Mac Catalyst 17.4+
- macOS 14.4+
- tvOS 17.4+
- visionOS 1.1+
- watchOS 10.4+
