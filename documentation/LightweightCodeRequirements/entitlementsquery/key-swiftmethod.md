# key(_:)

**Framework**: LightweightCodeRequirements  
**Kind**: method

## Declaration

```swift
func key(_ keyName: String) -> EntitlementsQuery
```

#### Discussion

Keys are matched byte for byte (i.e. no unicode normalization occurs).
