# init(script:)

**Framework**: Natural Language  
**Kind**: init

Creates a contextual embedding from a script.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+
- watchOS 10.0+

## Declaration

```swift
init?(script: NLScript)
```

## Parameters

- `script`: The script the framework uses to find the most recent embedding suitable for the value you specify.
