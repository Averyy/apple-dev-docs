# init(modelIdentifier:)

**Framework**: Natural Language  
**Kind**: init

Creates a contextual embedding from a model identifier.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+
- watchOS 10.0+

## Declaration

```swift

```

## Parameters

- `modelIdentifier`: The model identifier that identifies the embedding model.
