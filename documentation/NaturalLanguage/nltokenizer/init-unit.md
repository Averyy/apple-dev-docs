# init(unit:)

**Framework**: Natural Language  
**Kind**: init

Creates a tokenizer with the specified unit.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.1+
- macOS 10.14+
- tvOS 12.0+
- visionOS 1.0+
- watchOS 5.0+

## Declaration

```swift
init(unit: NLTokenUnit)
```

## Parameters

- `unit`: The level of granularity that the tokenizer operates at.
