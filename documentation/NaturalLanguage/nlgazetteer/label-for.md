# label(for:)

**Framework**: Natural Language  
**Kind**: method

Retrieves the label for the given term.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.1+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
func label(for string: String) -> String?
```

#### Return Value

A string if the term is in the vocabulary; otherwise `nil`.

## Parameters

- `string`: The term used to find a label.
