# appleProtectedMPEG4Audio

**Framework**: Uniform Type Identifiers  
**Kind**: property

A type that represents data in Apple-protected MPEG-4 format.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+
- watchOS 7.0+

## Declaration

```swift
static var appleProtectedMPEG4Audio: UTType { get }
```

#### Discussion

The identifier for this type is `com.apple.protected-mpeg-4-audio`.

This type conforms to [`UTTypeAudio`](uttypeaudio.md).
