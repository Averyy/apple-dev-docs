# phpScript

**Framework**: Uniform Type Identifiers  
**Kind**: property

A type that represents a PHP script.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+
- watchOS 7.0+

## Declaration

```swift
static var phpScript: UTType { get }
```

#### Discussion

The identifier for this type is `public.php-script`.

This type conforms to [`UTTypeShellScript`](uttypeshellscript.md).
