# bmp

**Framework**: Uniform Type Identifiers  
**Kind**: property

A type that represents a Windows bitmap image.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+
- watchOS 7.0+

## Declaration

```swift
static var bmp: UTType { get }
```

#### Discussion

The identifier for this type is `com.microsoft.bmp`.

This type conforms to [`UTTypeImage`](uttypeimage.md).
