# effectiveDenyExplicitContent

**Framework**: ManagedSettings  
**Kind**: property

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- tvOS 26.0+

## Declaration

```swift
@Published
```
