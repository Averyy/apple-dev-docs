# init(token:)

**Framework**: ManagedSettings  
**Kind**: init

Creates an object that represents the provided domain.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- visionOS 1.0+

## Declaration

```swift
init(token: WebDomainToken)
```

## Parameters

- `token`: A token to identify the web domain privately.
