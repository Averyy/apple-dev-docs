# lockAppCellularData

**Framework**: ManagedSettings  
**Kind**: property

The metadata associated with the constraint that locks the cellular data setting.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+

## Declaration

```swift
static let lockAppCellularData: SettingMetadata<Bool>
```

#### Discussion

The default value is `false`.
