# token

**Framework**: ManagedSettings  
**Kind**: property

An opaque representation of a specific web domain.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+

## Declaration

```swift
let token: ApplicationToken?
```

#### Discussion

Use  a token to share information about an app with other devices in the same Family Sharing group.
