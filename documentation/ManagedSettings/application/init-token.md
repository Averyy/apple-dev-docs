# init(token:)

**Framework**: ManagedSettings  
**Kind**: init

Creates an object that represents the app with the specified token.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+

## Declaration

```swift
init(token: ApplicationToken)
```

## Parameters

- `token`: A token that identifies an app.
