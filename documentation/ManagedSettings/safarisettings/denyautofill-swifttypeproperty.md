# denyAutoFill

**Framework**: ManagedSettings  
**Kind**: property

The metadata associated with the setting that deactivates Safari’s AutoFill feature.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+

## Declaration

```swift
static let denyAutoFill: SettingMetadata<Bool>
```

#### Discussion

Use `denyAutoFill` to access the metadata for [`denyAutoFill`](safarisettings/denyautofill-swift.property.md). The default value is `false`.
