# NSInteger

**Framework**: Objective-C Runtime  
**Kind**: typealias

Describes an integer.

## Declaration

```swift
typealias NSInteger = Int
```

#### Discussion

When building 32-bit applications, NSInteger is a 32-bit integer. A 64-bit application treats NSInteger as a 64-bit integer.
