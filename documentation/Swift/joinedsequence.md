# JoinedSequence

**Framework**: Swift  
**Kind**: struct

A sequence that presents the elements of a base sequence of sequences concatenated using a given separator.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.0+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+
