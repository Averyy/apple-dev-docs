# newlineSequence

**Framework**: Swift  
**Kind**: property

A character class that matches any newline sequence.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+
- watchOS 9.0+

## Declaration

```swift
static var newlineSequence: CharacterClass { get }
```

#### Discussion

This character class is equivalent to `\R` or `\n` in regex syntax.
