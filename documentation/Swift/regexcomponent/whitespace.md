# whitespace

**Framework**: Swift  
**Kind**: property

A character class that matches any element that is classified as whitespace.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+
- watchOS 9.0+

## Declaration

```swift
static var whitespace: CharacterClass { get }
```

#### Discussion

This character class is equivalent to `\s` in regex syntax.
