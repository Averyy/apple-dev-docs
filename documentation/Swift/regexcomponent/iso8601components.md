# iso8601Components

**Framework**: Swift  
**Kind**: property

Creates a regex component to match an ISO 8601 date and time, such as “2015-11-14’T’15:05:03’Z’”, and capture the string as a `DateComponents` using the time zone as specified in the string.

**Availability**:
