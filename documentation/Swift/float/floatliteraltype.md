# Float.FloatLiteralType

**Framework**: Swift  
**Kind**: typealias

A type that represents a floating-point literal.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.0+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
typealias FloatLiteralType = Float
```

#### Discussion

Valid types for `FloatLiteralType` are `Float`, `Double`, and `Float80` where available.
