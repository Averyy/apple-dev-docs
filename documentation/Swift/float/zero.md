# zero

**Framework**: Swift  
**Kind**: property

The zero value.

**Availability**:

- Mac Catalyst 13.0+
- macOS 10.10+
- tvOS 9.0+

## Declaration

```swift
static var zero: Self { get }
```

#### Discussion

Zero is the identity element for addition. For any value, `x + .zero == x` and `.zero + x == x`.
