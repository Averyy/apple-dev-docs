# usesIndexedCollation

**Framework**: Swift  
**Kind**: property

If set to true, presents the list of options with an alphabetical index on the right side of the screen (table view section index titles).

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+
- watchOS 9.0+
