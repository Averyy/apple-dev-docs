# Int128.IntegerLiteralType

**Framework**: Swift  
**Kind**: typealias

A type that represents an integer literal.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+

## Declaration

```swift
typealias IntegerLiteralType = Int128
```

#### Discussion

The standard library integer and floating-point types are all valid types for `IntegerLiteralType`.
