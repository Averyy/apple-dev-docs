# lowerBound

**Framework**: Swift  
**Kind**: property

The range’s lower bound.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.0+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
let lowerBound: Bound
```

#### Discussion

In an empty range, `lowerBound` is equal to `upperBound`.
