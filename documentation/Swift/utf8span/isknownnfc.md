# isKnownNFC

**Framework**: Swift  
**Kind**: property

Returns whether the contents are known to be NFC. This is not always checked at initialization time and is set by `checkForNFC`.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+
- watchOS 26.0+
