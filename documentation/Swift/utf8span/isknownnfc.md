# isKnownNFC

**Framework**: Swift  
**Kind**: property

Returns whether the contents are known to be NFC. This is not always checked at initialization time and is set by `checkForNFC`.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
- tvOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)
- watchOS 26.0+ (Beta)
