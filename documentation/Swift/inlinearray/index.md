# InlineArray.Index

**Framework**: Swift  
**Kind**: typealias

A type that represents a position in the array.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+
- watchOS 26.0+

## Declaration

```swift
typealias Index = Int
```

#### Discussion

Valid indices consist of the position of every element and a “past the end” position that’s not valid for use as a subscript argument.
