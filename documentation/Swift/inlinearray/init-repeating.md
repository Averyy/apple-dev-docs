# init(repeating:)

**Framework**: Swift  
**Kind**: init

Initializes every element in this array to a copy of the given value.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+
- watchOS 26.0+

## Declaration

```swift
init(repeating value: Element)
```

#### Discussion

> **Note**: O(), where  is the number of elements in the array.

## Parameters

- `value`: The instance to initialize this array with.
