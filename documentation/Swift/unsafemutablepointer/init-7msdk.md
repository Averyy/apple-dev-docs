# init(_:)

**Framework**: Swift  
**Kind**: init

Creates a mutable typed pointer referencing the same memory as the given mutable pointer.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.0+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
init(_ other: UnsafeMutablePointer<Pointee>)
```

## Parameters

- `other`: The pointer to convert.
