# init(_:)

**Framework**: Swift  
**Kind**: init

Creates a new buffer over the same memory as the given buffer.

**Availability**:

- iPadOS 8.0+
- Mac Catalyst 13.0+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
init(_ bytes: UnsafeMutableRawBufferPointer)
```

## Parameters

- `bytes`: The buffer to convert.
