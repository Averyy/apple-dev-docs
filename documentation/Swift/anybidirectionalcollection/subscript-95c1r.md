# subscript(_:)

**Framework**: Swift  
**Kind**: subscript

Accesses the element indicated by `position`.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.0+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
subscript(position: AnyBidirectionalCollection<Element>.Index) -> Element { get }
```

#### Overview

> **Note**: `position` indicates a valid position in `self` and `position != endIndex`.
