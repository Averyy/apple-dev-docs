# removeLast(_:)

**Framework**: Swift  
**Kind**: method

Remove the last N elements, returning the memory they occupy to the uninitialized state.

**Availability**:
- iOS 12.2+
- iPadOS 12.2+
- Mac Catalyst 12.2+
- macOS 10.14.4+
- tvOS 12.2+
- visionOS 1.0+
- watchOS 5.2+

## Declaration

```swift
mutating func removeLast(_ k: Int)
```

#### Discussion

`n` must not be greater than `count`
