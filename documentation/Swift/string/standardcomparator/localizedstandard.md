# localizedStandard

**Framework**: Swift  
**Kind**: property

**Availability**:
- iOS 15.0+

- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+

## Declaration

```swift
static let localizedStandard: String.StandardComparator
```

#### Discussion

Uses a localized, numeric comparison in the current locale.

The default `SortComparator` used in `String` comparisons.
