# isQuotationMark

**Framework**: Swift  
**Kind**: property

A Boolean value indicating whether the scalar is one that is used in writing to surround quoted text.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.0+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
var isQuotationMark: Bool { get }
```

#### Discussion

This property corresponds to the “Quotation_Mark” property in the [`Unicode Standard`](https://developer.apple.comhttp://www.unicode.org/versions/latest/).
