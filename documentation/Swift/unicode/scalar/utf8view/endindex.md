# endIndex

**Framework**: Swift  
**Kind**: property

The “past the end” position—that is, the position one greater than the last valid subscript argument.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
var endIndex: Int { get }
```

#### Discussion

If the collection is empty, `endIndex` is equal to `startIndex`.
