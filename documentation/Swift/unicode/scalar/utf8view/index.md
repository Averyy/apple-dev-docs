# Unicode.Scalar.UTF8View.Index

**Framework**: Swift  
**Kind**: typealias

A type that represents a position in the collection.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
typealias Index = Int
```

#### Discussion

Valid indices consist of the position of every element and a “past the end” position that’s not valid for use as a subscript argument.
