# subscript(_:)

**Framework**: Swift  
**Kind**: subscript

Accesses the code unit at the specified position.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
subscript(position: Int) -> UTF8.CodeUnit { get }
```

## Parameters

- `position`: The position of the element to access.    must be a valid index of the collection that is not equal to the    property.
