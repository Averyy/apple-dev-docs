# startIndex

**Framework**: Swift  
**Kind**: property

The position of the first element in a nonempty collection.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
var startIndex: CollectionDifference<ChangeElement>.Index { get }
```

#### Discussion

If the collection is empty, `startIndex` is equal to `endIndex`.
