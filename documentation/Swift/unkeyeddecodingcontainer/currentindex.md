# currentIndex

**Framework**: Swift  
**Kind**: property  
**Required**: Yes

The current decoding index of the container (i.e. the index of the next element to be decoded.) Incremented after every successful decode call.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.0+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+
