# UnicodeScalarLiteralType

**Kind**: associatedtype  
**Required**: Yes

## Declaration

```swift
associatedtype UnicodeScalarLiteralType : _ExpressibleByBuiltinUnicodeScalarLiteral
```

Valid types for `UnicodeScalarLiteralType` are `Unicode.Scalar`, `Character`, `String`, and `StaticString`.
