# ==(_:_:)

**Framework**: Swift  
**Kind**: op

Returns a Boolean value indicating whether two values are equal.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)

- macOS 26.0+ (Beta)
- tvOS 26.0+ (Beta)

## Declaration

```swift
static func == (lhs: AsyncStream<Element>.Continuation, rhs: AsyncStream<Element>.Continuation) -> Bool
```

#### Discussion

Equality is the inverse of inequality. For any values `a` and `b`, `a == b` implies that `a != b` is `false`.

## Parameters

- `lhs`: A value to compare.
- `rhs`: Another value to compare.
