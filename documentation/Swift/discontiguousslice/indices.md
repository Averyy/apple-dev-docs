# DiscontiguousSlice.Indices

**Framework**: Swift  
**Kind**: typealias

A type that represents the indices that are valid for subscripting the collection, in ascending order.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+

## Declaration

```swift
typealias Indices = DefaultIndices<DiscontiguousSlice<Base>>
```
