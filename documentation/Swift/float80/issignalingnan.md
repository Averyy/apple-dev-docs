# isSignalingNaN

**Framework**: Swift  
**Kind**: property

A Boolean value indicating whether the instance is a signaling NaN.

**Availability**:
- macOS 10.10+

## Declaration

```swift
var isSignalingNaN: Bool { get }
```

#### Discussion

Signaling NaNs typically raise the Invalid flag when used in general computing operations.
