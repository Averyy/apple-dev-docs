# +=(_:_:)

**Framework**: Swift  
**Kind**: op

Adds two values and stores the result in the left-hand-side variable.

**Availability**:
- macOS 10.10+

## Declaration

```swift
static func += (lhs: inout Self, rhs: Self)
```

## Parameters

- `lhs`: The first value to add.
- `rhs`: The second value to add.
