# init(_:)

**Framework**: Swift  
**Kind**: init

Creates a character containing the given Unicode scalar value.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.0+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
init(_ content: Unicode.Scalar)
```

## Parameters

- `content`: The Unicode scalar value to convert into a character.
