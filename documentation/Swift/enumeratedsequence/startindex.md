# startIndex

**Framework**: Swift  
**Kind**: property

The position of the first element in a nonempty collection.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+
- watchOS 26.0+

## Declaration

```swift
var startIndex: EnumeratedSequence<Base>.Index { get }
```

#### Discussion

If the collection is empty, `startIndex` is equal to `endIndex`.
