# init(_:)

**Framework**: Swift  
**Kind**: init  
**Required**: Yes

Creates a new instance from the given value, rounded to the closest possible representation.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.0+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
init(_ value: Float)
```

## Parameters

- `value`: A floating-point value to be converted.
