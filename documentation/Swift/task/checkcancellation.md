# checkCancellation()

**Framework**: Swift  
**Kind**: method

Throws an error if the task was canceled.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
static func checkCancellation() throws
```

#### Discussion

The error is always an instance of `CancellationError`.

> **Note**: `isCancelled()`
