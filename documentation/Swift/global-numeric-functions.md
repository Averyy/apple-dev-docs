# Global Numeric Functions

**Framework**: Swift

Use these functions with numeric values and other comparable types.

## Topics

### Choosing the Smallest and Largest Value
- [func min<T>(T, T) -> T](min(_:_:).md)
  Returns the lesser of two comparable values.
- [func min<T>(T, T, T, T...) -> T](min(_:_:_:_:).md)
  Returns the least argument passed.
- [func max<T>(T, T) -> T](max(_:_:).md)
  Returns the greater of two comparable values.
- [func max<T>(T, T, T, T...) -> T](max(_:_:_:_:).md)
  Returns the greatest argument passed.
### Finding the Absolute Value
- [func abs<T>(T) -> T](abs(_:).md)
  Returns the absolute value of the given number.
