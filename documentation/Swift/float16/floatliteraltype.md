# Float16.FloatLiteralType

**Framework**: Swift  
**Kind**: typealias

A type that represents a floating-point literal.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+
- watchOS 7.0+

## Declaration

```swift
typealias FloatLiteralType = Float16
```

#### Discussion

Valid types for `FloatLiteralType` are `Float`, `Double`, and `Float80` where available.
