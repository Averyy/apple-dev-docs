# isInfinite

**Framework**: Swift  
**Kind**: property

A Boolean value indicating whether the instance is infinite.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+
- watchOS 7.0+

## Declaration

```swift
var isInfinite: Bool { get }
```

#### Discussion

For NaN, both `isFinite` and `isInfinite` are false.
