# replacingPercentEscapes(using:)

**Framework**: Swift  
**Kind**: method

Returns a new string made by replacing in the `String` all percent escapes with the matching characters as determined by a given encoding.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 8.0+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
