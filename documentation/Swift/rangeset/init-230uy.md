# init(_:)

**Framework**: Swift  
**Kind**: init

Creates a range set containing the given range.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+

## Declaration

```swift
init(_ range: Range<Bound>)
```

## Parameters

- `range`: The range to use for the new range set.
