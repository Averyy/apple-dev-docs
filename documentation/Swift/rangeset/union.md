# union(_:)

**Framework**: Swift  
**Kind**: method

Returns a new range set containing the contents of both this set and the given set.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+

## Declaration

```swift
func union(_ other: RangeSet<Bound>) -> RangeSet<Bound>
```

#### Return Value

A new range set.

## Parameters

- `other`: The range set to merge with this one.
