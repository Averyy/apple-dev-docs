# subtracting(_:)

**Framework**: Swift  
**Kind**: method

Returns a new set containing the contents of this range set that are not also in the given range set.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+

## Declaration

```swift
func subtracting(_ other: RangeSet<Bound>) -> RangeSet<Bound>
```

#### Return Value

A new range set.

## Parameters

- `other`: The range set to subtract.
