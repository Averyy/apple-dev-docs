# init(stringLiteral:)

**Framework**: Swift  
**Kind**: init  
**Required**: Yes

Creates an instance initialized to the given string value.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.0+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
init(stringLiteral value: Self.StringLiteralType)
```

## Parameters

- `value`: The value of the new instance.
