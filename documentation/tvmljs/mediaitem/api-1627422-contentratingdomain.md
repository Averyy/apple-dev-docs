# contentRatingDomain

**Framework**: TVMLKit JS  
**Kind**: instp

The domain that the rating applies to.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
attribute String contentRatingDomain;
```

#### Discussion

There are three valid values for this property: `movie`, `music`, and `tvshow`.
