# MediaItem

**Framework**: TVMLKit JS  
**Kind**: instm

Creates a new `MediaItem` object from the information stored in the URL location.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
new MediaItem(
    in String type, 
    in optional String url
);
```

#### Return_value

Returns the `MediaItem` object found in the location specified by the URL.
