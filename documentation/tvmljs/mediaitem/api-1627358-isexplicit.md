# isExplicit

**Framework**: TVMLKit JS  
**Kind**: instp

A Boolean value indicating whether the item has explicit lyrics.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
attribute Boolean isExplicit;
```

#### Discussion

This property is ignored if the `MediaItem` object type is `video`.
