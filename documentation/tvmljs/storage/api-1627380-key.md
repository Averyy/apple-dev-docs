# key

**Framework**: TVMLKit JS  
**Kind**: instm

Returns the key located in the specified location.

**Availability**:
- tvOS 9.0+
- Safari Desktop 10.0+
- Safari Mobile 10.0+

## Declaration

```swift
DOMString? key(
    unsigned long index
);
```

#### Return_value

The found key.

#### Discussion

If the value in the `index` parameter is greater than the number of key-value pairs currently in the storage list, this function returns `null`.

## Parameters

- `index`: The location of the key to be retrieved.
