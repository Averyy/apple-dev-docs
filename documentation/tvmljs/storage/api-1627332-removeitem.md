# removeItem

**Framework**: TVMLKit JS  
**Kind**: instm

Deletes a key and all associated data from storage.

**Availability**:
- tvOS 9.0+
- Safari Desktop 4.0+
- Safari Mobile 3.0+

## Declaration

```swift
void removeItem(
    DOMString key
);
```

## Parameters

- `key`: A   object containing the key being searched for.
