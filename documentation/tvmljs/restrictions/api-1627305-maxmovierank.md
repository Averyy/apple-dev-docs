# maxMovieRank

**Framework**: TVMLKit JS  
**Kind**: instp

The maximum allowed ranking for a movie.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
readonly attribute int maxMovieRank;
```

#### Discussion

The rank for a movie is an integer ranging from `0` to `1000`.
