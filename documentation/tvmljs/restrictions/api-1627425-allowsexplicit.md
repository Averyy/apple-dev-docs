# allowsExplicit

**Framework**: TVMLKit JS  
**Kind**: instp

A boolean value that indicates whether any explicit media is allowed.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
readonly attribute Boolean allowsExplicit;
```

#### Discussion

When this property is `true`, explicit media items are allowed.
