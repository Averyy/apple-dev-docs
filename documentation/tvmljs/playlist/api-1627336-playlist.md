# Playlist

**Framework**: TVMLKit JS  
**Kind**: instm

Creates a new playlist object.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
new Playlist();
```

#### Return_value

An empty playlist array.

#### Discussion

After creating a new playlist, use the [`push`](playlist/1627433-push.md) method to add new media items to the end of the playlist.
