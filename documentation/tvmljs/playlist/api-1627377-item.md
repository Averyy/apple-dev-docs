# item

**Framework**: TVMLKit JS  
**Kind**: instm

Returns the media item located in the indicated array index.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
MediaItem item(
    in int index
);
```

#### Return_value

The [`MediaItem`](mediaitem.md) object located in the designated index location. This method returns `null` if the index is out of range.
