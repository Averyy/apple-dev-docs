# pop

**Framework**: TVMLKit JS  
**Kind**: instm

Removes a media item from the end of a playlist.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
MediaItem pop();
```

#### Return_value

The [`MediaItem`](mediaitem.md) object located at the end of the playlist. This method returns `null` if the playlist is empty.
