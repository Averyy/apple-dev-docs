# abort

**Framework**: TVMLKit JS  
**Kind**: instm

Cancels any network activity.

**Availability**:
- tvOS 9.0+
- Safari Desktop 4.0+
- Safari Mobile 3.0+

## Declaration

```swift
void abort();
```

#### Discussion

This method raises a `DOMException` if any arguments are passed into this function.
