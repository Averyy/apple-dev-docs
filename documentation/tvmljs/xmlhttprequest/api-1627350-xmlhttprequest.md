# XMLHttpRequest

**Framework**: TVMLKit JS  
**Kind**: instm

Creates a new XMLHttpRequest.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
new XMLHttpRequest();
```

#### Return_value

A new instance of an XMLHttpRequest.

#### Discussion

You create a new XMLHttpRequest instance to send and receive data between your app and a server.
