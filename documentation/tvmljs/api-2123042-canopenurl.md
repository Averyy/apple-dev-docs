# canOpenURL

**Framework**: TVMLKit JS  
**Kind**: func

Determines if a deep-link to another app can be opened.

**Availability**:
- tvOS 10.0+

## Declaration

```swift
Boolean canOpenURL(
    in String url
);
```

#### Return_value

Returns `YES` if the app can be opened.

## Parameters

- `url`: The deep-link URL being verified.
