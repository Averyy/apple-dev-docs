# Slideshow

**Framework**: TVMLKit JS  
**Kind**: cl

An object used to display images on Apple TV in a slideshow format.

**Availability**:
- tvOS 10.0+

## Declaration

```swift
interface Slideshow
```

#### Overview

There is no constructor for the `Slideshow` class. Use the [`start`](slideshow/1657265-start.md) method to create and begin a new slideshow.

## Topics

### Creating and Modifying a Slideshow
- [append](slideshow/1657228-append.md)
  Add additional images to a currently running slideshow.
- [dismiss](slideshow/1657241-dismiss.md)
  Dismisses a currently running slideshow.
- [start](slideshow/1657265-start.md)
  Starts a slideshow with the given set of images.
