# dismissModal

**Framework**: TVMLKit JS  
**Kind**: instm

Dismisses the document displayed in modal view.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
void dismissModal();
```

#### Discussion

When the document is dismissed, the previous document is brought into focus.
