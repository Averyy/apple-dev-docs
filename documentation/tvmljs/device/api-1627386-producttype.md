# productType

**Framework**: TVMLKit JS  
**Kind**: instp

The version of the product installed on the Apple TV.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
readonly attribute String productType;
```

#### Discussion

This property contains an identifier for the device, for example, `AppleTV 3,1`.
