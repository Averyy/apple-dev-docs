# vendorIdentifier

**Framework**: TVMLKit JS  
**Kind**: instp

The universally unique identifier (UUID) of the device.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
readonly attribute String vendorIdentifier;
```

#### Discussion

The UUID for a device stays the same across all apps from a vendor.
