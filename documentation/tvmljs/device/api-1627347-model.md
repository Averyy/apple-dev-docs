# model

**Framework**: TVMLKit JS  
**Kind**: instp

A string that identifies the device model.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
readonly attribute String model;
```

#### Discussion

The property contains a human readable string identifying the device, for example, AppleTV.
