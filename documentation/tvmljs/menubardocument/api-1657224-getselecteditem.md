# getSelectedItem

**Framework**: TVMLKit JS  
**Kind**: instm

Retrieves the menu item in the menu bar that is currently in focus.

**Availability**:
- tvOS 10.0+

## Declaration

```swift
Element getSelectedItem();
```

#### Return_value

The currently selected element.
