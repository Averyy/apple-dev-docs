# setSelectedItem

**Framework**: TVMLKit JS  
**Kind**: instm

Sets the focus in a menu bar to the specified menu item.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
void setSelectedItem(
    in Object menuItem
);
```

## Parameters

- `menuItem`: The menu item to be in focus.
