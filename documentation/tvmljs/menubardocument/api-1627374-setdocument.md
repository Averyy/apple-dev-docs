# setDocument

**Framework**: TVMLKit JS  
**Kind**: instm

Associates a document with a menu item.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
void setDocument(
    in Document document, 
    in Object menuItem
);
```

## Parameters

- `document`: The TVML file to be associated with the menu item.
- `menuItem`: The selected menu item.
