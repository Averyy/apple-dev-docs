# getDocument

**Framework**: TVMLKit JS  
**Kind**: instm

Retrieves the document associated with the specified menu item.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
Document getDocument(
    in Element menuItem
);
```

#### Return_value

The TVML page associated with the selected menu item.

## Parameters

- `menuItem`: The menu item currently in focus.
