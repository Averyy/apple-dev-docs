# setData

**Framework**: TVMLKit JS  
**Kind**: instm

Stores the data for a given key.

**Availability**:
- tvOS 10.0+

## Declaration

```swift
void setData(
    in String key, 
    in String data
);
```

#### Discussion

You can set the `data` parameter to `null` to remove the data associated with the key.

## Parameters

- `key`: The key for the data to be stored.
- `data`: The data to be stored.
