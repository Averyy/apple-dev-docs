# getData

**Framework**: TVMLKit JS  
**Kind**: instm

Returns the data associated with a particular key.

**Availability**:
- tvOS 10.0+

## Declaration

```swift
String getData(
    in String key
);
```

#### Return_value

The retrieved data.

## Parameters

- `key`: The key for data to be retrieved.
