# UserDefaults

**Framework**: TVMLKit JS  
**Kind**: cl

An object that contains the app's default preferences.

**Availability**:
- tvOS 10.0+

## Declaration

```swift
interface UserDefaults
```

#### Overview

The `UserDefaults` class is exposed to the JavaScript context as `userDefaults`.

## Topics

### Modifying User Defaults
- [getData](userdefaults/1682084-getdata.md)
  Returns the data associated with a particular key.
- [removeData](userdefaults/1682085-removedata.md)
  Removes the data associated with a particular key.
- [setData](userdefaults/1682089-setdata.md)
  Stores the data for a given key.
