# openURL

**Framework**: TVMLKit JS  
**Kind**: func

Opens a deep link into another app.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
void openURL(
    in String url
);
```

#### Discussion

Invoking this method with a valid URL launches the handling application.

## Parameters

- `url`: The URL of the app that is to be opened.
