# text

**Framework**: TVMLKit JS  
**Kind**: instp

The text inside a search or text field.

**Availability**:
- tvOS 9.0+

## Declaration

```swift
attribute String text;
```

#### Discussion

Use this attribute to set or get the text in a `searchField` or `textField` element.
