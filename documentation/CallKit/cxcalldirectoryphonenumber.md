# CXCallDirectoryPhoneNumber

**Framework**: CallKit  
**Kind**: typealias

A value that represents a phone number consisting of a country calling code followed by a sequence of digits.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.0+
- macOS 13.0+
- visionOS 1.0+
- watchOS 9.0+

## Declaration

```swift
typealias CXCallDirectoryPhoneNumber = Int64
```
