# overrideTitle

**Framework**: AVRouting  
**Kind**: property

A string to use to override the title of the item’s type.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- visionOS 1.0+

## Declaration

```swift
var overrideTitle: String? { get set }
```

#### Discussion

Use this value to dynamically override the title of the custom item.
