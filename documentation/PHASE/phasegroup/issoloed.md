# isSoloed

**Framework**: PHASE  
**Kind**: property

A Boolean value that indicates whether the app silences all groups other than this group.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 17.0+
- visionOS 1.0+
