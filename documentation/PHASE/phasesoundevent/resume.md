# resume()

**Framework**: PHASE  
**Kind**: method

Resumes the sound event.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 17.0+
- visionOS 1.0+

## Declaration

```swift
func resume()
```

#### Discussion

If the sound event plays audio, this function resumes audio playback.
