# pause()

**Framework**: PHASE  
**Kind**: method

Pauses the sound event.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 17.0+
- visionOS 1.0+

## Declaration

```swift
func pause()
```

#### Discussion

If the sound event plays audio, this function pauses audio playback.
