# init(value:)

**Framework**: PHASE  
**Kind**: init

Creates a specification for a metaparameter with the given numeric value.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 17.0+
- visionOS 1.0+

## Declaration

```swift
convenience init(value: Double)
```

## Parameters

- `value`: A default value for the metaparameter specification.
