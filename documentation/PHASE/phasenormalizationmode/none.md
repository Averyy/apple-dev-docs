# PHASENormalizationMode.none

**Framework**: PHASE  
**Kind**: case

A mode that instructs the framework not to adjust a sound’s volume according to the user’s output device.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
