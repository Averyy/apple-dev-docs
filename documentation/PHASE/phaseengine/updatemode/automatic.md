# PHASEEngine.UpdateMode.automatic

**Framework**: PHASE  
**Kind**: case

A mode that indicates PHASE sets the timing of state adjustments.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+

## Declaration

```swift
case automatic
```

#### Discussion

In this mode, the framework updates internal states at an optimized rate opaque to the app.
