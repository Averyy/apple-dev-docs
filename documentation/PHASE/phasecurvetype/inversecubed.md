# PHASECurveType.inverseCubed

**Framework**: PHASE  
**Kind**: case

A curve that increases at a rate of one divided by the input’s cube.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+

## Declaration

```swift
case inverseCubed
```

#### Discussion

The function `y` `=` `1` `/` `x^3` shapes this curve.
