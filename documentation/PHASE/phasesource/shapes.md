# shapes

**Framework**: PHASE  
**Kind**: property

An array of shapes that collectively define the audio-emitting surface area of a volumetric source.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 17.0+
- visionOS 1.0+

## Declaration

```swift
var shapes: [PHASEShape] { get }
```

#### Discussion

The framework sets the contents to the `shapes` argument of the [`init(engine:shapes:)`](phasesource/init(engine:shapes:).md) initializer.
