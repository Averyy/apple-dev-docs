# devices

**Framework**: Core Audio  
**Kind**: property

An array of AudioHardwareDevices that represent all the device objects that came out of the given box. Note that until a box is enabled, this list will be empty.

**Availability**:
- Mac Catalyst ?+
- macOS 15.0+
