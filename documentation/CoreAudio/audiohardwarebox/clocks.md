# clocks

**Framework**: Core Audio  
**Kind**: property

An array of AudioHardwareClocks that represent all the clock objects that came out of the given box. Note that until a box is enabled, this list will be empty.

**Availability**:
- Mac Catalyst ?+
- macOS 15.0+
