# unload()

**Framework**: Core Audio  
**Kind**: method

Terminates all IO on all devices within the process and releases all resources capable of being released. This essentially returns the HAL to its uninitialized state.

**Availability**:
- Mac Catalyst ?+
- macOS 15.0+
