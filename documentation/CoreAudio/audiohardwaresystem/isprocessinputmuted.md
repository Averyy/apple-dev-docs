# isProcessInputMuted

**Framework**: Core Audio  
**Kind**: property

A Bool where true indicates that all data coming into the process for all devices will be silent. A value of false indicates that input data will be received normally.

**Availability**:
- Mac Catalyst ?+
- macOS 15.0+
