# AUParameterNode

**Framework**: Audio Toolbox  
**Kind**: class

An object that represents a node in an audio unit’s parameter tree.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
class AUParameterNode
```

#### Overview

Nodes are instances of either an [`AUParameter`](auparameter.md) or [`AUParameterGroup`](auparametergroup.md) class.
