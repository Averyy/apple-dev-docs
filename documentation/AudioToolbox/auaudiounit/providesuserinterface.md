# providesUserInterface

**Framework**: Audio Toolbox  
**Kind**: property

A Boolean that indicates whether the audio unit provides a user interface, normally in the form of a view controller.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.13+
- tvOS 11.0+
- visionOS 1.0+
