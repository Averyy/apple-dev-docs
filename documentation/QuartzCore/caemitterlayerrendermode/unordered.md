# unordered

**Framework**: Core Animation  
**Kind**: property

Particles are rendered unordered. This mode uses source-over compositing.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- Mac Catalyst 13.1+
- macOS 10.6+
- tvOS 9.0+
- visionOS 1.0+

## Declaration
