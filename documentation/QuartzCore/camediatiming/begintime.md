# beginTime

**Framework**: Core Animation  
**Kind**: property  
**Required**: Yes

Specifies the begin time of the receiver in relation to its parent object, if applicable.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.5+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
var beginTime: CFTimeInterval { get set }
```

#### Discussion

Defaults to 0.
