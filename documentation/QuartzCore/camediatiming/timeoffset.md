# timeOffset

**Framework**: Core Animation  
**Kind**: property  
**Required**: Yes

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.5+
- tvOS 9.0+
- visionOS 1.0+
