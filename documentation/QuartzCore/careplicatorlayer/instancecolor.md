# instanceColor

**Framework**: Core Animation  
**Kind**: property

Defines the color used to multiply the source object. Animatable.

**Availability**:
- iOS 3.0+
- iPadOS 3.0+
- Mac Catalyst 13.1+
- macOS 10.6+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
var instanceColor: CGColor? { get set }
```

#### Discussion

Defaults to opaque white.
