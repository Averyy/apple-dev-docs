# keyPath

**Framework**: Core Animation  
**Kind**: property

Specifies the key path the receiver animates.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.5+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
var keyPath: String? { get set }
```

#### Discussion

The key path is relative to the layer the receiver is attached to.
