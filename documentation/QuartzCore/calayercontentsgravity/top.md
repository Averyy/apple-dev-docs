# top

**Framework**: Core Animation  
**Kind**: property

The content is horizontally centered at the top-edge of the bounds rectangle.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.5+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
static let top: CALayerContentsGravity
```
