# name

**Framework**: Core Animation  
**Kind**: property

The name of the receiver.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.5+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
var name: String? { get set }
```

#### Discussion

The layer name is used by some layout managers to identify a layer. The default value of this property is `nil`.
