# timingFunction

**Framework**: Core Animation  
**Kind**: property

An optional timing function defining the pacing of the animation.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.5+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
var timingFunction: CAMediaTimingFunction? { get set }
```

#### Discussion

Defaults to `nil`, indicating linear pacing.
