# scroll(to:)

**Framework**: Core Animation  
**Kind**: method

Scroll the contents of the receiver to ensure that the rectangle is visible.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.5+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
func scroll(to r: CGRect)
```

## Parameters

- `r`: The rectangle that should be visible.
