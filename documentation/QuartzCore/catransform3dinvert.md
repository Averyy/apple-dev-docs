# CATransform3DInvert(_:)

**Framework**: Core Animation  
**Kind**: func

Inverts `t` and returns the result.

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.5+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
func CATransform3DInvert(_ t: CATransform3D) -> CATransform3D
```

#### Discussion

Returns the original matrix if `t` has no inverse.
