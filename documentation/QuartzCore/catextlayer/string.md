# string

**Framework**: Core Animation  
**Kind**: property

**Availability**:
- iOS 2.0+
- iPadOS 2.0+
- Mac Catalyst 13.1+
- macOS 10.5+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
var string: Any? { get set }
```

#### Discussion

The text must be an instance of `NSString` or `NSAttributedString`. Defaults to `nil`.
