# SCCaptureResolutionType.best

**Framework**: ScreenCaptureKit  
**Kind**: case

Capture streaming content at the best available resolution.

**Availability**:
- Mac Catalyst 18.2+
- macOS 12.3+

## Declaration

```swift
case best
```

#### Discussion

This streaming resolution only applies to independent window capture.
