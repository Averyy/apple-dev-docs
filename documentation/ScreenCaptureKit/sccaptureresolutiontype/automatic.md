# SCCaptureResolutionType.automatic

**Framework**: ScreenCaptureKit  
**Kind**: case

Allow ScreenCaptureKit to automatically select the quality of content depending on factors such as network connection.

**Availability**:
- Mac Catalyst 18.2+
- macOS 12.3+
