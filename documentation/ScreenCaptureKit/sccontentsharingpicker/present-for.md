# present(for:)

**Framework**: ScreenCaptureKit  
**Kind**: method

Displays the picker with an already running capture stream.

**Availability**:
- Mac Catalyst 18.2+
- macOS 14.0+

## Declaration

```swift
func present(for stream: SCStream)
```

## Parameters

- `stream`: The capture stream to display in the picker.
