# isActive

**Framework**: ScreenCaptureKit  
**Kind**: property

A Boolean value that indicates if the window is currently streaming.

**Availability**:
- Mac Catalyst 16.2+
- macOS 13.1+

## Declaration

```swift
var isActive: Bool { get }
```

#### Discussion

When this value is `true`, the window is currently streaming, even if offscreen.
