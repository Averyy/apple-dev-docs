# preservesAspectRatio

**Framework**: ScreenCaptureKit  
**Kind**: property

A Boolean value that determines if the stream preserves aspect ratio.

**Availability**:
- Mac Catalyst 17.0+
- macOS 14.0+

## Declaration

```swift
var preservesAspectRatio: Bool { get set }
```

#### Discussion

The default value is `true`.
