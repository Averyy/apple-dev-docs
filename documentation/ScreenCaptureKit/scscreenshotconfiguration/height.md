# height

**Framework**: ScreenCaptureKit  
**Kind**: property

An integer value that specifies the output height, measured in pixels.

**Availability**:
- Mac Catalyst ?+
- macOS ?+

## Declaration

```swift
var height: Int { get set }
```

#### Discussion

The default value is the height of the captured content.
