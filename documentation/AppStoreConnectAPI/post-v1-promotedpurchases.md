# Promote a Purchase

**Framework**: App Store Connect API  
**Kind**: httpRequest

Add an existing in-app purchase or auto-renewable subscription to the promoted in-app purchases on an app listing in the App Store.

**Availability**:
- App Store Connect API 2.0+
