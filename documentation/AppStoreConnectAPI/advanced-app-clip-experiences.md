# Advanced App Clip Experiences

**Framework**: App Store Connect API

Create, read, update, and delete your advanced App Clip experiences.

#### Overview

The `appClipAdvancedExperiences` resource represents an advanced App Clip experience. You use it to access an advanced App Clip experience you already created, create a new advanced App Clip experience, or update an existing advanced App Clip experience. Additionally, you can use the resourceâ€™s `PATCH` operation to delete an unreleased advanced App Clip experience.
