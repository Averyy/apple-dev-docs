# App Store Connect API 1.6 release notes

**Framework**: App Store Connect API

Update your server-side code to use new features, and test your code against API changes.

#### Overview

App Store Connect API version 1.6 provides resources that enable you to automate actions you take in App Store Connect.

##### New Features

- The API adds [`App Clips and App Clip Experiences`](app-clips-and-app-clip-experiences.md).
- For TestFlight, a new attribute in [`BetaGroupUpdateRequest.Data.Attributes`](betagroupupdaterequest/data-data.dictionary/attributes-data.dictionary.md) helps developers manage their iOS apps on Apple silicon Mac computers.
- The [`Build.Relationships.BuildBundles`](build/relationships-data.dictionary/buildbundles-data.dictionary.md) and [`BuildBundle.Relationships.BuildBundleFileSizes`](buildbundle/relationships-data.dictionary/buildbundlefilesizes-data.dictionary.md) relationships provide developers more insight into their app binaries.
- The [`Apps`](apps.md) resource adds the ability to configure a subscription status URL for production and sandbox environments in [`App.Attributes`](app/attributes-data.dictionary.md) and [`AppUpdateRequest.Data.Attributes`](appupdaterequest/data-data.dictionary/attributes-data.dictionary.md).
