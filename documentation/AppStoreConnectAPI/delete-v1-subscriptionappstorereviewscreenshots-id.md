# Delete a Review Screenshot for an Auto-Renewable Subscription

**Framework**: App Store Connect API  
**Kind**: httpRequest

Delete an image that you uploaded for review of an auto-renewable subscription.

**Availability**:
- App Store Connect API 2.0+
