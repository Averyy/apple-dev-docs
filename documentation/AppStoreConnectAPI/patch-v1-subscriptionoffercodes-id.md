# Deactivate a Subscription Offer with Offer Codes

**Framework**: App Store Connect API  
**Kind**: httpRequest

Deactivate a subscription offer that has offer codes for an auto-renewable subscription.

**Availability**:
- App Store Connect API 2.0+
