# Modify a Beta App Localization

**Framework**: App Store Connect API  
**Kind**: httpRequest

Update the localized information for a specific beta app and locale.

**Availability**:
- App Store Connect API 1.0+

## Mentions

- [App Store Connect API 3.7 release notes](app-store-connect-api-3-7-release-notes.md)

#### Overview

> ‚ùó **Important**:  A description is required for all `betaAppLocalizations` before you can submit to beta app review. After you have added data to the fields for this resource, you can change that data, but you cannot remove data.
