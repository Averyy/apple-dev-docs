# App Store Connect API Release Notes

**Framework**: App Store Connect API

Learn about new features and updates in the App Store Connect API.

#### Overview

The App Store Connect API is a REST API that enables the automation of actions you take in App Store Connect. Download the latest [`OpenAPI specification`](https://developer.apple.comhttps://developer.apple.com/sample-code/app-store-connect/app-store-connect-openapi-specification.zip).
