# DELETE /v1/gameCenterAppVersions/{id}/relationships/compatibilityVersions

**Framework**: App Store Connect API  
**Kind**: httpRequest

Remove a compatible version relationship from an app version.

**Availability**:
- App Store Connect API 3.6+
