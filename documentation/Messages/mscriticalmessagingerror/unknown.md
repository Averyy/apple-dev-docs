# MSCriticalMessagingError.unknown

**Framework**: Messages  
**Kind**: case

The error code the framework returns after an unknown error occurs.

**Availability**:
- iOS 18.2+
- iPadOS 18.2+

## Declaration

```swift
case unknown
```

#### Discussion

The framework returned an unknown error.
