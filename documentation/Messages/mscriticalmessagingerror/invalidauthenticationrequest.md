# MSCriticalMessagingError.invalidAuthenticationRequest

**Framework**: Messages  
**Kind**: case

The authentication request isn’t valid.

**Availability**:
- iOS 18.2+
- iPadOS 18.2+

## Declaration

```swift
case invalidAuthenticationRequest
```

#### Discussion

The framework isn’t able to validate the authentication request.
