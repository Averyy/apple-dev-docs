# imageTitle

**Framework**: Messages  
**Kind**: property

The title for the image or media file.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+

## Declaration

```swift
var imageTitle: String? { get set }
```

#### Discussion

The title is drawn using white text on the bottom-left corner of the image or media file. This property defaults to `nil`.
