# imageSubtitle

**Framework**: Messages  
**Kind**: property

The subtitle for the image or media file.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+

## Declaration

```swift
var imageSubtitle: String? { get set }
```

#### Discussion

The subtitle is drawn using white text, just below the title. This property defaults to `nil`.
