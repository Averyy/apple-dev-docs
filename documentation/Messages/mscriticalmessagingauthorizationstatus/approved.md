# MSCriticalMessagingAuthorizationStatus.approved

**Framework**: Messages  
**Kind**: case

A person has approved a request to use the Critical Messaging API.

**Availability**:
- iOS 18.2+
- iPadOS 18.2+

## Declaration

```swift
case approved
```

#### Discussion

The framework returns this value when someone approves a request to use the API.
