# init(stickerSize:)

**Framework**: Messages  
**Kind**: init

Creates a new sticker browser view controller with stickers of the provided size.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+

## Declaration

```swift
@MainActor
init(stickerSize: MSStickerSize)
```

#### Return Value

A newly instantiated sticker browser controller.

## Parameters

- `stickerSize`: A constant that indicates the size of the stickers. For a list of possible values, see  .
