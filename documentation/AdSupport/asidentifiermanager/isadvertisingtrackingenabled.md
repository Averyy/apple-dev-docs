# isAdvertisingTrackingEnabled

**Framework**: AdSupport  
**Kind**: property

A Boolean value that indicates whether the user has limited ad tracking enabled.

**Availability**:
- iOS 6.0+
- iPadOS 6.0+
- Mac Catalyst 13.1+
- macOS 10.14+
- tvOS 6.0+

## Declaration

```swift
var isAdvertisingTrackingEnabled: Bool { get }
```

#### Discussion

> ⚠️ **Warning**: This property is deprecated. Functionality has been replaced by the [`App Tracking Transparency`](https://developer.apple.com/documentation/AppTrackingTransparency) framework.
