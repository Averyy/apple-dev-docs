# roughness(_:)

**Framework**: Swift Charts  
**Kind**: method

A value that controls the degree of surface roughness.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
nonisolated
func roughness(_ ratio: Double) -> some Chart3DContent
```

#### Discussion

Zero represents a smooth surface. One represents a completely rough surface.

## Parameters

- `ratio`: The degree of roughness.
