# plotSize

**Framework**: Swift Charts  
**Kind**: property

The size of the plot in the chart.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+
- watchOS 10.0+

## Declaration

```swift
var plotSize: CGSize { get }
```

#### Discussion

The plot is the area between the x and y axes, not including the axes themselves.
