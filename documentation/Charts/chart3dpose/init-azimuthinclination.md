# init(azimuth:inclination:)

**Framework**: Swift Charts  
**Kind**: init

Initialize a `Chart3DPose` with the provided azimuth and inclination angles.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- visionOS 26.0+

## Declaration

```swift
init(azimuth: Angle2D, inclination: Angle2D)
```

## Parameters

- `azimuth`: The horizontal angle of a Chart’s pose.
- `inclination`: The vertical angle of a Chart’s pose.
