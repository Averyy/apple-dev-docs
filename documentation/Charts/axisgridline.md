# AxisGridLine

**Framework**: Swift Charts  
**Kind**: struct

A line that a chart draws across its plot area to indicate a reference point along a particular axis.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+
