# linear

**Framework**: Swift Charts  
**Kind**: property

A number scale where each range value y can be expressed as a linear function of the domain value x, with `y = a * x + b`.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
