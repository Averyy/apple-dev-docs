# append

**Framework**: Metal Performance Shaders Graph  
**Kind**: property

Flag to append to an existing .mpsgraphpackage if found at provided url.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+

## Declaration

```swift
var append: Bool { get set }
```

#### Discussion

If false, the exisiting .mpsgraphpackage will be overwritten.
