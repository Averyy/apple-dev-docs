# deploymentPlatform

**Framework**: Metal Performance Shaders Graph  
**Kind**: property

The deployment platform used to serialize the executable.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+

## Declaration

```swift
var deploymentPlatform: MPSGraphDeploymentPlatform { get set }
```

#### Discussion

Defaults to the current platform.
