# reductionMode

**Framework**: Metal Performance Shaders Graph  
**Kind**: property

**Availability**:

- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
