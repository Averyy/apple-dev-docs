# boundaryMode

**Framework**: Metal Performance Shaders Graph  
**Kind**: property

The property that determines which values to use for padding the input tensor.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+

## Declaration

```swift
var boundaryMode: MPSGraphPaddingMode { get set }
```

#### Discussion

Default value: `MPSGraphPaddingModeZero`.
