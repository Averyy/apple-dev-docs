# paddingStyle

**Framework**: Metal Performance Shaders Graph  
**Kind**: property

The property that defines what kind of padding to apply to the stencil operation.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+

## Declaration

```swift
var paddingStyle: MPSGraphPaddingStyle { get set }
```

#### Discussion

Default value: `MPSGraphPaddingStyleExplicit`.
