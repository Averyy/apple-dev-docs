# init(mtlDevice:)

**Framework**: Metal Performance Shaders Graph  
**Kind**: init

Creates a device from a given Metal device.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+

## Declaration

```swift
convenience init(mtlDevice metalDevice: any MTLDevice)
```

#### Return Value

A valid device.

## Parameters

- `metalDevice`:   to create an MPSGraphDevice from.
