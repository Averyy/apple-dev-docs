# init(stringLiteral:)

**Framework**: System  
**Kind**: init

Creates a file path from a string literal.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+
- watchOS 7.0+

## Declaration

```swift
init(stringLiteral: String)
```

## Parameters

- `stringLiteral`: A string literal   whose Unicode encoded contents to use as the contents of the path.
