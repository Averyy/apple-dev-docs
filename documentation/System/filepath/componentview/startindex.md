# startIndex

**Framework**: System  
**Kind**: property

The position of the first element in a nonempty collection.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
var startIndex: FilePath.ComponentView.Index { get }
```

#### Discussion

If the collection is empty, `startIndex` is equal to `endIndex`.
