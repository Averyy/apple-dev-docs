# QLPreviewViewStyle.compact

**Framework**: Quick Look UI  
**Kind**: case

Provides a compact preview style suited for smaller previews or inspectors.

**Availability**:
- macOS 10.7+

## Declaration

```swift
case compact
```

#### Discussion

A more compact style for a [`QLPreviewView`](qlpreviewview.md) object. Use this style for smaller previews or inspectors.
