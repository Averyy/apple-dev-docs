# QLPreviewViewStyle.normal

**Framework**: Quick Look UI  
**Kind**: case

Provides the default preview style for larger, full-sized previews.

**Availability**:
- macOS 10.7+

## Declaration

```swift
case normal
```

#### Discussion

The default style for a [`QLPreviewView`](qlpreviewview.md) object. Use this style should for larger, full-size previews.
