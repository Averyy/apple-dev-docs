# language

**Framework**: MapKit JS  
**Kind**: property

The language code for a place lookup.

**Availability**:
- MapKit JS 5.78.1+

## Declaration

```swift
attribute string language;
```

#### Discussion

Supported values for this property are locale IDs, such as `en-GB` or `es-MX`.
