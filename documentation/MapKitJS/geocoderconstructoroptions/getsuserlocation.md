# getsUserLocation

**Framework**: MapKit JS  
**Kind**: property

An initial Boolean value that indicates whether the geocoder returns results near the user’s location.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
attribute boolean getsUserLocation;
```

#### Discussion

When you set this to `true`, the geocoder returns results that are near the user’s location. The default value is `false`.
