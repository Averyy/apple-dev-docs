# getsUserLocation

**Framework**: MapKit JS  
**Kind**: property

A Boolean value that indicates whether to limit the search results to the user’s location, according to the web browser.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
attribute boolean getsUserLocation;
```

#### Discussion

If you set this to `true`, the search queries the browser for the user’s location to deliver relevant local results. The default value is `false`.
