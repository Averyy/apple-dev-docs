# includeAddresses

**Framework**: MapKit JS  
**Kind**: property

A Boolean value that indicates whether the search results include addresses.

**Availability**:
- MapKit JS 5.32.2+

## Declaration

```swift
attribute boolean includeAddresses;
```

#### Discussion

The default value is `true`.
