# coordinate

**Framework**: MapKit JS  
**Kind**: property

A map coordinate that provides a hint for the geographic area to search.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
coordinate?: Coordinate;
```

#### Discussion

This property supplies coordinates as a reference for a search.

```javascript
{ coordinate: new mapkit.Coordinate(37.37, -122.04) }
```
