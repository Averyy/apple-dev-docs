# enabled

**Framework**: MapKit JS  
**Kind**: property

A Boolean value that determines whether the annotation responds to user interaction.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
attribute boolean enabled;
```

#### Discussion

The default value is `true`.
