# appearanceAnimation

**Framework**: MapKit JS  
**Kind**: property

A CSS animation that runs when the annotation appears on the map.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
attribute string appearanceAnimation;
```

#### Discussion

CSS animation running when the annotation appears. The default value is `""` (empty string).
