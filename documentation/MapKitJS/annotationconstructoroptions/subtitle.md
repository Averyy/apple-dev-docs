# subtitle

**Framework**: MapKit JS  
**Kind**: property

The text to display as a subtitle on the second line of an annotation’s callout.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
attribute string subtitle;
```

#### Discussion

The default value is `""` (empty string).
