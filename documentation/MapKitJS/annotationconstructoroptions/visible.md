# visible

**Framework**: MapKit JS  
**Kind**: property

A Boolean value that determines whether the annotation is visible or hidden.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
visible?: boolean;
```

#### Discussion

The default value is `true`.
