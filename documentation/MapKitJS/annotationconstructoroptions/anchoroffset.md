# anchorOffset

**Framework**: MapKit JS  
**Kind**: property

The offset, in CSS pixels, of the element from the bottom center.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
attribute DOMPoint anchorOffset;
```

#### Discussion

The default value is `DOMPoint(0, 0)`.
