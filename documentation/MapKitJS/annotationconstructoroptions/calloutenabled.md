# calloutEnabled

**Framework**: MapKit JS  
**Kind**: property

A Boolean value that determines whether the map shows a callout.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
attribute boolean calloutEnabled;
```

#### Discussion

The default value is `true`.
