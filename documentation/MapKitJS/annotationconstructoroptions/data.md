# data

**Framework**: MapKit JS  
**Kind**: property

Data that you define and assign to the annotation.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
data?: object;
```

#### Discussion

Custom data to associate with this annotation. The default value is `{}.`
