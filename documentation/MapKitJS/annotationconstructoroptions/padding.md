# padding

**Framework**: MapKit JS  
**Kind**: property

Spacing to add around the annotation when showing items.

**Availability**:
- MapKit JS 5.16+

## Declaration

```swift
padding?: Padding;
```

#### Discussion

Padding prevents any items from touching the edges of the map. Set `padding` to a [`Padding`](padding.md) object. The default is no padding on all sides.
