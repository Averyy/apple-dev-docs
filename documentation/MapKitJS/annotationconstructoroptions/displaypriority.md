# displayPriority

**Framework**: MapKit JS  
**Kind**: property

A hint the map uses to prioritize displaying the annotation.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
attribute number displayPriority;
```

## Mentions

- [MapKit JS 5](mapkit-js-5.md)

#### Discussion

The default value is `Annotation.DisplayPriority.Required.`
