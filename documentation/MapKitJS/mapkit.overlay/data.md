# data

**Framework**: MapKit JS  
**Kind**: property

Custom data to associate with the overlay.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
attribute Object data;
```

#### Discussion

You choose the data to assign to the overlay. It can be any JavaScript value.
