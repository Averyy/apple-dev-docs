# selected

**Framework**: MapKit JS  
**Kind**: property

A Boolean value that indicates whether the user selects the overlay.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
attribute boolean selected;
```

## Mentions

- [Adding interactivity to overlays](adding-interactivity-to-overlays.md)

#### Discussion

The default value is `false`.
