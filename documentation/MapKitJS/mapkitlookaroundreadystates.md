# mapkit.LookAround.ReadyStates

**Framework**: MapKit JS  
**Kind**: enum

Values that indicate the state of the Look Around object in the browser.

**Availability**:
- MapKit JS 5.79+

## Declaration

```swift
interface mapkit.LookAround.ReadyStates {
	const string Loading;

	const string Destroyed;
	const string Error;
};
```
