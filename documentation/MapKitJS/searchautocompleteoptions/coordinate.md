# coordinate

**Framework**: MapKit JS  
**Kind**: property

A map coordinate that provides a hint for the geographic area to search.

**Availability**:
- MapKit JS 5.32.2+

## Declaration

```swift
attribute mapkit.Coordinate coordinate;
```

#### Discussion

This overrides the [`coordinate`](searchconstructoroptions/coordinate.md) provided to the [`mapkit.Search`](mapkit.search/mapkit.search.md) constructor.
