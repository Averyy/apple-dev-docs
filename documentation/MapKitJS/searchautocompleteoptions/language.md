# language

**Framework**: MapKit JS  
**Kind**: property

A language ID that determines the language for the search result text.

**Availability**:
- MapKit JS 5.32.2+

## Declaration

```swift
attribute string language;
```

#### Discussion

This overrides the [`language`](searchconstructoroptions/language.md) provided to the [`mapkit.Search`](mapkit.search/mapkit.search.md) constructor.
