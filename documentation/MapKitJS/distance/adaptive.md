# Adaptive

**Framework**: MapKit JS  
**Kind**: case

A constant indicating the measurement system is adaptive, and determined based on the map’s language.

**Availability**:
- MapKit JS 5.13+

## Declaration

```swift
readonly Adaptive: "adaptive";
```

#### Discussion

The measurement system used to display distances, either metric or imperial, is chosen dynamically based on the map’s set [`language`](mapkit/language.md).
