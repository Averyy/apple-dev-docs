# coordinate

**Framework**: MapKit JS  
**Kind**: property

The coordinate of the result when it corresponds to a single place.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
coordinate?: Coordinate;
```

#### Discussion

The coordinate may not be available for all results, and can return `undefined`. The coordinate is an approximate location.
