# subLocality

**Framework**: MapKit JS  
**Kind**: property

The name of the area within the locality.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
subLocality?: string;
```

#### Discussion

For example, the [`subLocality`](searchautocompleteresult/sublocality.md) for the Golden Gate Park is `"Golden Gate Park"`.
