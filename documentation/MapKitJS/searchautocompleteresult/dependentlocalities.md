# dependentLocalities

**Framework**: MapKit JS  
**Kind**: property

An array of common names for the local area or neighborhood of the place.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
dependentLocalities?: string[];
```

#### Discussion

For example, the dependent locality for Golden Gate Park in San Francisco, California, is `["Golden Gate Park"]`.
