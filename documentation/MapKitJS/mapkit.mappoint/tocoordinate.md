# toCoordinate

**Framework**: MapKit JS  
**Kind**: method

Converts a map point into a coordinate with latitude and longitude.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
mapkit.Coordinate toCoordinate();
```

#### Return Value

A coordinate that contains the latitude and longitude corresponding to the location.
