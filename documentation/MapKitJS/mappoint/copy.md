# copy()

**Framework**: MapKit JS  
**Kind**: method

Returns a copy of the location.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
copy(): MapPoint;
```

#### Return Value

A MapPoint object that is a copy of the map location.
