# TileOverlayConstructorOptions

**Framework**: MapKit JS  
**Kind**: struct

Attributes for initializing a tile overlay, including minimum and maximum zoom, opacity, and custom data.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
dictionary TileOverlayConstructorOptions {
	number minimumZ;
	number maximumZ;
	number opacity;
	Object data;
};
```
