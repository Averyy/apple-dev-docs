# latitude

**Framework**: MapKit JS  
**Kind**: property

The latitude, in degrees.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
attribute number latitude;
```

#### Discussion

Positive values indicate latitudes north of the equator. Negative values indicate latitudes south of the equator.
