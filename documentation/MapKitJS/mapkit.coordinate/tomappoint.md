# toMapPoint

**Framework**: MapKit JS  
**Kind**: method

Returns the map point that corresponds to the coordinate.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
mapkit.MapPoint toMapPoint();
```

#### Return Value

The map point value that corresponds to the coordinate on a two-dimensional map projection.
