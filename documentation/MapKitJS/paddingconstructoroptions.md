# PaddingConstructorOptions

**Framework**: MapKit JS  
**Kind**: struct

Initial values of the edge insets for padding.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
dictionary PaddingConstructorOptions {
	number top;
	number right;
	number bottom;
	number left;
};
```

## Topics

### Padding properties

- [left](paddingconstructoroptions/left.md)
  The amount of padding, in CSS pixels, to inset the map from the left edge.
- [right](paddingconstructoroptions/right.md)
  The amount of padding, in CSS pixels, to inset the map from the right edge.
- [top](paddingconstructoroptions/top.md)
  The amount of padding, in CSS pixels, to inset the map from the top edge.
