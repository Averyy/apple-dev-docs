# toCoordinateRegion

**Framework**: MapKit JS  
**Kind**: method

Returns the coordinate region that corresponds to the calling bounding region.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
mapkit.CoordinateRegion toCoordinateRegion();
```

#### Return Value

A [`mapkit.CoordinateRegion`](mapkit.coordinateregion.md) that corresponds to the calling [`mapkit.BoundingRegion`](mapkit.boundingregion/mapkit.boundingregion.md).
