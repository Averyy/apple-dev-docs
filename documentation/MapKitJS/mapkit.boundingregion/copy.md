# copy

**Framework**: MapKit JS  
**Kind**: method

Returns a copy of the calling bounding region.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
mapkit.BoundingRegion copy();
```

#### Return Value

A [`mapkit.BoundingRegion`](mapkit.boundingregion/mapkit.boundingregion.md) that is a copy of the calling bounding region.
