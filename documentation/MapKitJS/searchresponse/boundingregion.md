# boundingRegion

**Framework**: MapKit JS  
**Kind**: property

The region that encloses the places from the search results.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
boundingRegion?: CoordinateRegion;
```

#### Discussion

This property isn’t present if there aren’t any results.
