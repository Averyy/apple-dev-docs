# query

**Framework**: MapKit JS  
**Kind**: property

The query string for performing the search.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
query?: string | SearchAutocompleteResult;
```

#### Discussion

This property is empty when you use a [`SearchAutocompleteResult`](searchautocompleteresult.md) object to perform the search.
