# places

**Framework**: MapKit JS  
**Kind**: property

A list of places that match the search query.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
attribute Place[] places;
```

#### Discussion

The [`places`](searchresponse/places.md) array is empty if there is no match.
