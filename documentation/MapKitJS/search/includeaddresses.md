# includeAddresses

**Framework**: MapKit JS  
**Kind**: property

A Boolean value that indicates whether the search results include addresses.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
get includeAddresses(): boolean;
set includeAddresses(value: boolean);
```

#### Discussion

The default value is `true`.
