# includeQueries

**Framework**: MapKit JS  
**Kind**: property

A Boolean value that indicates whether the search results include queries.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
get includeQueries(): boolean;
set includeQueries(value: boolean);
```

#### Discussion

The default value is `true`.
