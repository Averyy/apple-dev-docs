# mapkit.AddressFilter

**Framework**: MapKit JS  
**Kind**: class

An object that filters which address options to include or exclude in search results.

**Availability**:
- MapKit JS 5.78.1+

## Declaration

```swift
interface mapkit.AddressFilter
```

#### Overview

Use this object to filter search results by criteria, such as country, region, and municipality. See [`mapkit.AddressCategory`](mapkit.addresscategory.md) for more information.

## Topics

### Configuring the filter
- [excluding](mapkit.addressfilter/excluding.md)
  A list of categories to exclude from a search.
- [including](mapkit.addressfilter/including.md)
  A list of categories to include in a search.
- [excludesCategory](mapkit.addressfilter/excludescategory.md)
  A Boolean value that indicates whether to exclude a category from a search.
- [includesCategory](mapkit.addressfilter/includescategory.md)
  A Boolean value that indicates whether to include a category from a search.
- [excludingAllCategories](mapkit.addressfilter/excludingallcategories.md)
  A value that excludes all address categories.
- [includingAllCategories](mapkit.addressfilter/includingallcategories.md)
  A value that includes all address categories.
