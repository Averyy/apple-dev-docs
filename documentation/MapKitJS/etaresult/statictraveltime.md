# staticTravelTime

**Framework**: MapKit JS  
**Kind**: property

The estimated travel time in seconds, excluding delays for traffic.

**Availability**:
- MapKit JS 5.46+

## Declaration

```swift
staticTravelTime: number;
```

#### Discussion

This static travel time reflects the time it takes to arrive at the destination under ideal conditions.
