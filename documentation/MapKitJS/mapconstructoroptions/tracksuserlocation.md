# tracksUserLocation

**Framework**: MapKit JS  
**Kind**: property

A Boolean value that determines whether to center the map on the user’s location.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
tracksUserLocation?: boolean;
```

#### Discussion

The default value is `false`.
