# showsUserLocation

**Framework**: MapKit JS  
**Kind**: property

A Boolean value that determines whether to show the user’s location on the map.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
attribute boolean showsUserLocation;
```

#### Discussion

The default value is `false`.
