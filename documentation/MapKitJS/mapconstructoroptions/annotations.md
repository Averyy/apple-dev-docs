# annotations

**Framework**: MapKit JS  
**Kind**: property

An array holding all the annotations on the map.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
annotations?: Annotation[];
```

#### Discussion

The default value is `[]` (empty array).
