# isRotationEnabled

**Framework**: MapKit JS  
**Kind**: property

A Boolean value that determines whether the user may rotate the map using the compass control or a rotate gesture.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
isRotationEnabled?: boolean;
```

#### Discussion

The default value is `true`.
