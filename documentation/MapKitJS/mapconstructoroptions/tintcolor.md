# tintColor

**Framework**: MapKit JS  
**Kind**: property

The CSS color that MapKit JS uses for the user interface controls on the map.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
attribute string tintColor;
```

#### Discussion

Use [`tintColor`](mapconstructoroptions/tintcolor.md) to color the outline of the controls, text, and glyphs. Use any valid [`CSS color value`](https://developer.apple.comhttps://www.w3.org/TR/css-color-3/).
