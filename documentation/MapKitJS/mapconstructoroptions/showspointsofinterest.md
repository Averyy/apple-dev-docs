# showsPointsOfInterest

**Framework**: MapKit JS  
**Kind**: property

A Boolean value that determines whether the map displays points of interest.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
attribute boolean showsPointsOfInterest;
```

#### Discussion

The default value is `true.`
