# map

**Framework**: MapKit JS  
**Kind**: property

The map that the framework adds the annotation to.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
readonly attribute mapkit.Map map;
```

#### Discussion

This property is `null` before MapKit JS adds the annotation to a [`mapkit.Map`](mapkit.map.md), and after MapKit JS removes the annotation from a `mapkit.Map`.
