# data

**Framework**: MapKit JS  
**Kind**: property

Data that you define that’s specific to an annotation.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
get data(): {};
set data(data: {});
```

#### Discussion

This is data of your choosing that you can assign to the annotation. It can be any JavaScript value, and defaults to an empty object.
