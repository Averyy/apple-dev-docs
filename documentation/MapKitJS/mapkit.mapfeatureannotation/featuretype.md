# featureType

**Framework**: MapKit JS  
**Kind**: property

A value that describes the type of place the feature represents.

**Availability**:
- MapKit JS 5.74+

## Declaration

```swift
attribute mapkit.MapFeatureType featureType;
```

#### Discussion

This is one of the possible [`mapkit.MapFeatureType`](mapkit.mapfeaturetype.md) values.
