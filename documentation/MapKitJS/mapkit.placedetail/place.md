# place

**Framework**: MapKit JS  
**Kind**: property

The place that the place detail displays.

**Availability**:
- MapKit JS 5.78.1+

## Declaration

```swift
attribute Place|null place;
```

#### Discussion

if this attribute is `null`, the place detail renders in a collapsed state and doesn’t show any information.
