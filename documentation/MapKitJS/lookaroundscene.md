# LookAroundScene

**Framework**: MapKit JS  
**Kind**: class

Object that represents the current location of the view.

**Availability**:
- MapKit JS 5.79+

## Declaration


#### Discussion

You can retain the object and use it on a different [`LookAround`](lookaround.md) view or [`LookAroundPreview`](lookaroundpreview.md) object
