# mapkit.LineGradient

**Framework**: MapKit JS  
**Kind**: class

A line that displays with a gradient along the length of the line.

**Availability**:
- MapKit JS 5.45+

## Declaration

```swift
interface mapkit.LineGradient
```

#### Overview

Use [`mapkit.LineGradient`](mapkit.linegradient/mapkit.linegradient.md) with a [`mapkit.PolylineOverlay`](mapkit.polylineoverlay.md) to show a gradient along a route or path. You can also use it to provide additional information about locations along the path, for example, the running pace along a route.
