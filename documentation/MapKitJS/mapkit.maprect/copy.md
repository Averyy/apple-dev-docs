# copy

**Framework**: MapKit JS  
**Kind**: method

Returns a copy of a map rectangle.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
mapkit.MapRect copy();
```

#### Return Value

A MapRect object that is a copy of the original.
