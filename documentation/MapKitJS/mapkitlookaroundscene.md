# mapkit.LookAroundScene

**Framework**: MapKit JS  
**Kind**: class

Object that represents the current location of the view.

**Availability**:
- MapKit JS 5.79+

## Declaration

```swift
interface mapkit.LookAroundScene
```

#### Discussion

You can retain the object and use it on a different [`mapkit.LookAround`](mapkit.lookaround.md) view, or [`mapkit.LookAroundPreview`](mapkit.lookaroundpreview.md) object.
