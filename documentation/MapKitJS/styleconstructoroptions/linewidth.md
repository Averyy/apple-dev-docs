# lineWidth

**Framework**: MapKit JS  
**Kind**: property

The width of a line’s stroke, in CSS pixels.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
attribute number lineWidth;
```

#### Discussion

The default line width is `1`.
