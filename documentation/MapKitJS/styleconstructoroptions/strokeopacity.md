# strokeOpacity

**Framework**: MapKit JS  
**Kind**: property

The opacity of the stroke color.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
strokeOpacity?: number;
```

#### Discussion

The value of this property can be in the range of `0` (fully transparent) to `1` (fully opaque). The default stroke opacity is `1`.
