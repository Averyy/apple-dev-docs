# mapkit.Map.MapTypes

**Framework**: MapKit JS  
**Kind**: enum

Constants representing the type of map to display.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
interface mapkit.Map.MapTypes {
	const string Standard;
	const string MutedStandard;
	const string Hybrid;
	const string Satellite;
};
```

## Mentions

- [Handling map events](handling-map-events.md)

## Topics

### Map type values
- [Hybrid](mapkit.map.maptypes/hybrid.md)
  A satellite image of the area with road and road name layers on top.
- [MutedStandard](mapkit.map.maptypes/mutedstandard.md)
  A street map that emphasizes your data over the underlying map details.
- [Satellite](mapkit.map.maptypes/satellite.md)
  A satellite image of the area.
- [Standard](mapkit.map.maptypes/standard.md)
  A street map that shows the position of all roads and some road names.
