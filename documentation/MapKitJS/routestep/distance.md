# distance

**Framework**: MapKit JS  
**Kind**: property

The step distance, in meters.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
attribute number distance;
```

#### Discussion

This property reflects the distance that the user covers while traveling along this segment of the route.
