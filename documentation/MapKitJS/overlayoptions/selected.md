# selected

**Framework**: MapKit JS  
**Kind**: property

A Boolean value that indicates whether the overlay is in a selected state.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
selected?: boolean;
```

#### Discussion

The default value is `false`.
