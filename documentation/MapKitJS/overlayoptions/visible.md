# visible

**Framework**: MapKit JS  
**Kind**: property

A Boolean value that determines if an overlay is visible.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
attribute boolean visible;
```

#### Discussion

The default value is `true`.
