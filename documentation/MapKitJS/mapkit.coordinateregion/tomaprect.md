# toMapRect

**Framework**: MapKit JS  
**Kind**: method

Returns the map rectangle that corresponds to the calling coordinate region.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
mapkit.MapRect toMapRect();
```

#### Return Value

A [`mapkit.MapRect`](mapkit.maprect/mapkit.maprect.md) that corresponds to the calling [`mapkit.CoordinateRegion`](mapkit.coordinateregion/mapkit.coordinateregion.md).
