# copy

**Framework**: MapKit JS  
**Kind**: method

Returns a copy of the calling coordinate region.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
void copy();
```

#### Return Value

A [`mapkit.CoordinateRegion`](mapkit.coordinateregion/mapkit.coordinateregion.md) that is a copy of the calling coordinate region.
