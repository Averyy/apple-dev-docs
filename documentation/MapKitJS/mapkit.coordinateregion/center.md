# center

**Framework**: MapKit JS  
**Kind**: property

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift

```

#### Discussion

Create a [`mapkit.Coordinate`](mapkit.coordinate/mapkit.coordinate.md) with a latitude and longitude pair:

```javascript
var center = new mapkit.Coordinate(37.415, -122.048333); // latitude, longitude
```
