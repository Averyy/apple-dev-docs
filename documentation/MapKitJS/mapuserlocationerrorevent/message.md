# message

**Framework**: MapKit JS  
**Kind**: property

The human-readable error message of the user location error for the developer.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
readonly message: string;
```

#### Discussion

This framework doesn’t display this message to a person; it’s printed to the console.
