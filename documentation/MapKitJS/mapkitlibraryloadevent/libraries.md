# libraries

**Framework**: MapKit JS  
**Kind**: property

The libraries that the framework loaded or failed to load.

**Availability**:
- MapKit JS 5.75+

## Declaration

```swift
readonly libraries: string[];
```

#### Discussion

The event type can be `load` or `load-error`, which indicates if the loading operation succeeded or failed.
