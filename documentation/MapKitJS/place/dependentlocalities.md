# dependentLocalities

**Framework**: MapKit JS  
**Kind**: property

Common names for the local area or neighborhood of the place.

**Availability**:
- MapKit JS 5.41.1+

## Declaration

```swift
dependentLocalities?: string[];
```

#### Discussion

For example, dependent localities for Golden Gate Park in San Francisco, CA are `["Golden Gate Park"].`
