# dependentLocalities

**Framework**: MapKit JS  
**Kind**: property

Common names for the local area or neighborhood of the place.

**Availability**:
- MapKit JS 5.41.1+

## Declaration

```swift
attribute string[]? dependentLocalities;
```

#### Discussion

For example, dependent localities for the Golden Gate Park are `[”Golden Gate Park”].`
