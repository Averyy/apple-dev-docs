# areasOfInterest

**Framework**: MapKit JS  
**Kind**: property

Common names of the area in which the place resides.

**Availability**:
- MapKit JS 5.41.1+

## Declaration

```swift
attribute string[]? areasOfInterest;
```

#### Discussion

For example, the [`areasOfInterest`](place/areasofinterest.md) for the Golden Gate Park are `[”Lindley Meadow”, “Golden Gate Park”]`.
