# thoroughfare

**Framework**: MapKit JS  
**Kind**: property

The street name at the place.

**Availability**:
- MapKit JS 5.41.1+

## Declaration

```swift
attribute string? thoroughfare;
```

#### Discussion

For example, the [`thoroughfare`](place/thoroughfare.md) for the Golden Gate Park is `“John F Kennedy Dr”`.
