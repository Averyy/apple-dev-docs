# fullThoroughfare

**Framework**: MapKit JS  
**Kind**: property

A combination of thoroughfare and subthoroughfare.

**Availability**:
- MapKit JS 5.41.1+

## Declaration

```swift
attribute string? fullThoroughfare;
```

#### Discussion

For example, the [`fullThoroughfare`](place/fullthoroughfare.md) for the Golden Gate Park is `“1054 John F Kennedy Dr”`.
