# subLocality

**Framework**: MapKit JS  
**Kind**: property

The name of the area within the locality.

**Availability**:
- MapKit JS 5.41.1+

## Declaration

```swift
subLocality?: string;
```

#### Discussion

For example, the [`subLocality`](place/sublocality.md) for Golden Gate Park in San Francisco, California, is `"Golden Gate Park"`.
