# locality

**Framework**: MapKit JS  
**Kind**: property

The city of the place.

**Availability**:
- MapKit JS 5.41.1+

## Declaration

```swift
locality?: string;
```

#### Discussion

For example, the [`locality`](place/locality.md) for Golden Gate Park in San Francisco, California, is `"San Francisco"`.
