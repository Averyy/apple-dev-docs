# administrativeArea

**Framework**: MapKit JS  
**Kind**: property

**Availability**:
- MapKit JS 5.41.1+

## Declaration

```swift

```

#### Discussion

For example, the [`administrativeArea`](place/administrativearea.md) for the Golden Gate Park is “`California”`.
