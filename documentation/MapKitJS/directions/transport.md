# Transport

**Framework**: MapKit JS  
**Kind**: var

A static property that refers to an object that describes the available transport type values.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
static Transport: typeof TransportType;
```

#### Discussion

This value allows you to access the [`TransportType`](transporttype.md) enumeration with `mapkit.Directions.Transport`.
