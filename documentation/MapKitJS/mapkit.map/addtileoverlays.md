# addTileOverlays

**Framework**: MapKit JS  
**Kind**: method

Adds an array of tile overlays to the map.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
mapkit.TileOverlay[] addTileOverlays(
	mapkit.TileOverlay[] tileOverlays
);
```

#### Return Value

Returns the array of tile overlays.

## Parameters

- `tileOverlays`: An array of tile overlays to add.
