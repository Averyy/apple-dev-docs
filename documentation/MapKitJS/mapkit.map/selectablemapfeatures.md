# selectableMapFeatures

**Framework**: MapKit JS  
**Kind**: property

An array of map features that users can select from the map.

**Availability**:
- MapKit JS 5.74+

## Declaration

```swift
attribute mapkit.MapFeatureType[] selectableMapFeatures;
```

#### Discussion

This is an array of the selectable map features the map displays from the types available in [`mapkit.MapFeatureType`](mapkit.mapfeaturetype.md).
