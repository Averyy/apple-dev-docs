# mapType

**Framework**: MapKit JS  
**Kind**: property

The type of data that the map displays.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
attribute string mapType;
```

## Mentions

- [MapKit JS 5](mapkit-js-5.md)

#### Discussion

The map type determines whether the map shows a standard view, a satellite view, or a hybrid view. The map type is a value of [`mapkit.Map.MapTypes`](mapkit.map.maptypes.md).
