# MutedStandard

**Framework**: MapKit JS  
**Kind**: case

A street map that emphasizes your data over the underlying map details.

**Availability**:
- MapKit JS 5.12+

## Declaration

```swift
const string MutedStandard;
```

## Mentions

- [MapKit JS 5](mapkit-js-5.md)

#### Discussion

In this style, map features are less prominent, which allows user features, such as annotations and overlays, to stand out by comparison.
