# anchorOffset

**Framework**: MapKit JS  
**Kind**: property

An offset that changes the annotation’s default anchor point.

**Availability**:
- MapKit JS 5.78.1+

## Declaration

```swift
get anchorOffset(): DOMPoint;
```

#### Discussion

You can’t change the anchor offset for a [`PlaceAnnotation`](placeannotation.md).
