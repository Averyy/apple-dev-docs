# includePointsOfInterest

**Framework**: MapKit JS  
**Kind**: property

A Boolean value that indicates whether the search results should include points of interest.

**Availability**:
- MapKit JS 5.32.2+

## Declaration

```swift
includePointsOfInterest?: boolean;
```

#### Discussion

The default value is `true`.
