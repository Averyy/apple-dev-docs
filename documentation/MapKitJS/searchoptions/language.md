# language

**Framework**: MapKit JS  
**Kind**: property

A language ID that determines the language for the search result text.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
language?: string;
```

#### Discussion

This overrides the [`language`](serviceconstructoroptions/language.md) provided to the [`Search`](search.md) constructor.
