# copy()

**Framework**: MapKit JS  
**Kind**: method

Returns a copy of the calling bounding region.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
copy(): BoundingRegion;
```

#### Return Value

A [`BoundingRegion`](boundingregion.md) that is a copy of the calling bounding region.
