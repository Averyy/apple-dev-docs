# copy()

**Framework**: MapKit JS  
**Kind**: method

Returns a copy of the calling coordinate region.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
copy(): CoordinateRegion;
```

#### Return Value

A [`CoordinateRegion`](coordinateregion.md) that is a copy of the calling coordinate region.
