# toMapRect()

**Framework**: MapKit JS  
**Kind**: method

Returns the map rectangle that corresponds to the calling coordinate region.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
toMapRect(): MapRect;
```

#### Return Value

A [`MapRect`](maprect.md) that corresponds to the calling [`CoordinateRegion`](coordinateregion.md).
