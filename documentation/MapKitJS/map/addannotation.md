# addAnnotation(annotation)

**Framework**: MapKit JS  
**Kind**: method

Adds an annotation to the map.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
addAnnotation(annotation: Annotation): Annotation | undefined;
```

#### Return Value

Returns the annotation.

## Parameters

- `annotation`: The annotation to add.
