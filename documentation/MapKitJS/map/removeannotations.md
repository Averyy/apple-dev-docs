# removeAnnotations(annotations)

**Framework**: MapKit JS  
**Kind**: method

Removes multiple annotations from the map.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
removeAnnotations(annotations: Annotation[]): Annotation[];
```

#### Return Value

Returns the array of annotations.

## Parameters

- `annotations`: An array of annotations to remove.
