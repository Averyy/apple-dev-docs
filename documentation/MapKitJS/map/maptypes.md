# MapTypes

**Framework**: MapKit JS  
**Kind**: var

A static property that allows you to access the map type enumeration.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
static MapTypes: typeof MapType;
```

#### Discssion

Use this object reference to access the [`MapType`](maptype.md) enumeration with `mapkit.Map.MapTypes`.
