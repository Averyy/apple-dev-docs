# Distances

**Framework**: MapKit JS  
**Kind**: var

A static property that allows you to access the distance enumeration.

**Availability**:
- MapKit JS 5.13+

## Declaration

```swift
static Distances: typeof Distance;
```

#### Discussion

Use this object to access the [`Distance`](distance.md) enumeration with `mapkit.Map.Distances`.
