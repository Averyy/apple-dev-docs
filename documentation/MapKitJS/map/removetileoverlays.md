# removeTileOverlays(tileOverlays)

**Framework**: MapKit JS  
**Kind**: method

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
removeTileOverlays(tileOverlays: TileOverlay[]): TileOverlay[];
```

#### Return Value

Returns the array of tile overlays.

## Parameters
