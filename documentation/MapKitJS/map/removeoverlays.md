# removeOverlays(overlays)

**Framework**: MapKit JS  
**Kind**: method

Removes multiple overlays from the map.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
removeOverlays(overlays: Overlay[]): Overlay[];
```

#### Return Value

Returns the array of overlays.

## Parameters

- `overlays`: An array of overlays to remove.
