# LoadPriorities

**Framework**: MapKit JS  
**Kind**: var

A static property that allows you to access the load priority enumeration.

**Availability**:
- MapKit JS 5.73+

## Declaration

```swift
static LoadPriorities: typeof LoadPriority;
```

#### Discussion

Use this object to access the [`LoadPriority`](loadpriority.md) enumeration with `mapkit.Map.LoadPriorities`.
