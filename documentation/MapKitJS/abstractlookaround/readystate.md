# readyState

**Framework**: MapKit JS  
**Kind**: property

A value that represents the loading state of the Look Around object.

**Availability**:
- MapKit JS 5.79+

## Declaration

```swift
get readyState(): LookAroundReadyState;
```

#### Discussion

The Look Around view dispatches a `readystatechange` event when the loading state changes.
