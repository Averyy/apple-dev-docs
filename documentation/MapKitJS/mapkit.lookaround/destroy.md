# destroy

**Framework**: MapKit JS  
**Kind**: method

A method you call to destroy the Look Around view.

**Availability**:
- MapKit JS 5.79+

## Declaration

```swift
void destroy();
```

#### Discussion

You should destroy the view when it is no longer in use. You must destroy the view in order for the next view to initialize.
