# readyState

**Framework**: MapKit JS  
**Kind**: property

A value that represents the loading state of the Look Around object.

**Availability**:
- MapKit JS 5.79+

## Declaration

```swift
attribute mapkit.LookAround.ReadyStates readyState;
```

#### Discussion

The Look Around view dispatches `readystatechange` event when the loading state changes.
