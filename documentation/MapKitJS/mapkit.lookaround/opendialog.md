# openDialog

**Framework**: MapKit JS  
**Kind**: property

A Boolean value that indicates whether to expand the Look Around view.

**Availability**:
- MapKit JS 5.79+

## Declaration

```swift
attribute boolean openDialog;
```

#### Discussion

Look Around view dispatches `enter-dialog` event when the view expands, and `leave-dialog` event when the view contracts. These events can be cancelled to stop the view from expanding or contracting.
