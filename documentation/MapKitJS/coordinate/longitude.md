# longitude

**Framework**: MapKit JS  
**Kind**: property

The longitude, in degrees.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
longitude: number;
```

#### Discussion

Measurements are relative to the zero meridian, with positive values extending east of the meridian and negative values extending west of the meridian.
