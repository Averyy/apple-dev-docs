# place

**Framework**: MapKit JS  
**Kind**: property

The place that the place detail displays.

**Availability**:
- MapKit JS 5.78.1+

## Declaration

```swift
get place(): Place | null;
set place(value: Place | null);
```

#### Discussion

if this property is `null`, the place detail renders in a collapsed state and doesn’t show any information.
