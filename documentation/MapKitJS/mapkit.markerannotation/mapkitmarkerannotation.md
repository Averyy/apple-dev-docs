# mapkit.MarkerAnnotation

**Framework**: MapKit JS  
**Kind**: init

Creates a marker annotation at the coordinate location with provided options.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
new mapkit.MarkerAnnotation(
	mapkit.Coordinate|Place location,
	optional MarkerAnnotationConstructorOptions options
);
```

## Parameters

- `location`: The coordinate where the annotation appears.
- `options`: A hash of properties to create the annotation with.
