# strokeColor

**Framework**: MapKit JS  
**Kind**: property

The stroke color of a line.

**Availability**:
- MapKit JS 5.0+

## Declaration

```swift
get strokeColor(): string;
set strokeColor(strokeColor: string);
```

#### Discussion

This can be any valid CSS color value. Set to `null` for no stroke. The default stroke color is `rgb(0, 122, 255)`.
