# PointsOfInterest

**Framework**: MapKit JS  
**Kind**: case

Prioritizes loading of the full standard map, with rendered POIs.

**Availability**:
- MapKit JS 5.73+

## Declaration

```swift
readonly PointsOfInterest: "PointsOfInterest";
```
