# subtitleVisibility

**Framework**: MapKit JS  
**Kind**: property

A value that determines the subtitle’s visibility.

**Availability**:
- MapKit JS 5.74.1+

## Declaration

```swift
get subtitleVisibility(): FeatureVisibility;
```

#### Discussion

You can’t set the subtitle visibility property of the map feature annotation.
