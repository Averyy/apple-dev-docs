# featureType

**Framework**: MapKit JS  
**Kind**: property

A value that describes the type of place the feature represents.

**Availability**:
- MapKit JS 5.74.1+

## Declaration

```swift
get featureType(): "" | MapFeatureType;
```

#### Discussion
