# all

**Framework**: Core Transferable  
**Kind**: property

The visibility level that specifies that any app or process can access the item.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+
- watchOS 9.0+

## Declaration

```swift

```
