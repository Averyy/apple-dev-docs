# Read a location

**Framework**: Roster API  
**Kind**: httpRequest

Returns a specific location in an Apple School Manager organization.

**Availability**:
- Roster API 1.0.0+

#### Discussion

Access to the `locations` resource requires authorization to either the `edu.users.read` or `edu.classes.read` scope.
