# compile()

**Framework**: Foundation Models  
**Kind**: method

Prepares an adapter before being used with a [`LanguageModelSession`](languagemodelsession.md). You should call this if your adapter has a draft model.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- visionOS 26.0+
