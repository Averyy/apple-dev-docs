# init(shareURLs:)

**Framework**: CloudKit  
**Kind**: init

Creates a share request access operation configured with specified share URLs.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+
- watchOS 26.0+

## Declaration

```swift
convenience init(shareURLs: [URL])
```

#### Return Value

A configured [`CKShareRequestAccessOperation`](cksharerequestaccessoperation.md) instance.

## Parameters

- `shareURLs`: An array of   objects representing the shares to request access to.
