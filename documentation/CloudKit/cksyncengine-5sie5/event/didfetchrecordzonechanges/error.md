# error

**Framework**: CloudKit  
**Kind**: property

An error that describes the cause of a failed fetch operation.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS ?+
- watchOS 10.0+

## Declaration

```swift
let error: CKError?
```

#### Discussion

A `nil` value indicates a successful fetch.
