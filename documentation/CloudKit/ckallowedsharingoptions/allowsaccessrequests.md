# allowsAccessRequests

**Framework**: CloudKit  
**Kind**: property

Default value is `NO`. If set, the system sharing UI will allow the user to configure whether access requests are enabled on the share.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- visionOS 26.0+
