# init()

**Framework**: CloudKit  
**Kind**: init

Creates an empty fetch operation.

**Availability**:
- iOS 9.2+
- iPadOS 9.2+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 9.1+
- visionOS 1.0+
- watchOS 3.0+

## Declaration

```swift
init()
```

#### Discussion

You must set [`apiToken`](ckfetchwebauthtokenoperation/apitoken.md) before you execute the operation or add it to a queue.
