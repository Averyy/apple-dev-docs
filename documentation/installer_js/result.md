# Result

**Framework**: Installer JS  
**Kind**: uid

An object that provides methods to obtain information about the result of an Installation Check or Volume Check script. 

#### Overview

This object is accessed using the `my.result` property.

## Topics

### Working with Result Properties
- [message](result/1811962-message.md)
  A String describing the cause of a warning or fatal result.
- [type](result/1811969-type.md)
  The type of the result.
- [title](result/1811977-title.md)
  The title of the warning.
