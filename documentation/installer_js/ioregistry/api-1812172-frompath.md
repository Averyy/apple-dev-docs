# fromPath

**Framework**: Installer JS  
**Kind**: uid

Returns a dictionary with the properties of a given IOKit object.

## Declaration

```swift
fromPath(filePath)
```

## Parameters

- `filePath`: String with the path of the desired IOKit object.
