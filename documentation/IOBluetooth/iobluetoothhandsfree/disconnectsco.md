# disconnectSCO()

**Framework**: IOBluetooth  
**Kind**: method

Disconnect the SCO connection with the device

**Availability**:
- macOS 10.7+

## Declaration

```swift
func disconnectSCO()
```

#### Discussion

Disconnects the SCO connection with the device (if one exists). Delegate methods will be called once the disconnection is complete.
