# inputVolume

**Framework**: IOBluetooth  
**Kind**: property

Return the input volume

**Availability**:
- macOS 10.7+

## Declaration

```swift
var inputVolume: Float { get set }
```

#### Return Value

The input volume

#### Discussion

Returns the input volume between 0 and 1. 0 is the same as mute.
