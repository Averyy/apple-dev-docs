# outputVolume

**Framework**: IOBluetooth  
**Kind**: property

Return the output volume

**Availability**:
- macOS 10.7+

## Declaration

```swift
var outputVolume: Float { get set }
```

#### Return Value

The output volume

#### Discussion

Returns the output volume between 0 and 1. 0 is the same as mute.
