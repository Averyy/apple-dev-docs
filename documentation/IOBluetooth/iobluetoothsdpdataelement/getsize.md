# getSize()

**Framework**: IOBluetooth  
**Kind**: method

Returns the size in bytes of the target data element.

**Availability**:
- macOS ?+

## Declaration

```swift
func getSize() -> UInt32
```

#### Return Value

Returns the size in bytes of the target data element.

#### Discussion

The size is valid whether the data element has a fixed or variable size descriptor.
