# placeAllOthers(onHold:)

**Framework**: IOBluetooth  
**Kind**: method

Places all calls except the call with the specified index on hold.

**Availability**:
- macOS 10.7+

## Declaration

```swift
func placeAllOthers(onHold index: Int32)
```

## Parameters

- `index`: The index of the call that remains active.
