# isPaired()

**Framework**: IOBluetooth  
**Kind**: method

Returns whether the target device is paired.

**Availability**:
- macOS ?+

## Declaration

```swift
func isPaired() -> Bool
```

#### Return Value

Returns TRUE if the target device is paired, FALSE if not.

#### Discussion

NOTE: This method is only available in macOS 10.2.5 (Bluetooth v1.2) or later.
