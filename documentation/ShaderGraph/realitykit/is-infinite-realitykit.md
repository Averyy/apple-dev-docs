# Is Infinite (RealityKit)

**Framework**: ShaderGraph  
**Kind**: subscript

Returns true if the incoming value is infinite.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 26.0+ (Beta)
- visionOS 1.0+
