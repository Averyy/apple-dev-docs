# trackingAreaCode

**Framework**: Network Extension  
**Kind**: property

The Tracking Area Code of the private LTE network.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- visionOS 1.0+

## Declaration

```swift
var trackingAreaCode: String? { get set }
```

#### Discussion

This property is only applicable for band 48 private LTE networks.
