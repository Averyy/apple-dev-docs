# isConstrained

**Framework**: Network Extension  
**Kind**: property

A Boolean that indicates whether or not the path uses a constrained interface, such as when using low-data mode.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 17.0+
- visionOS 1.0+
