# NEVPNConnectionError.serverDisconnected

**Framework**: Network Extension  
**Kind**: case

An error code that indicates the VPN connection failed because the VPN server terminated the connection.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 17.0+
- visionOS 1.0+
