# cancel()

**Framework**: Network Extension  
**Kind**: method

Cancel the connection.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 17.0+
- visionOS 1.0+

## Declaration

```swift
func cancel()
```

#### Discussion

This will clean up the resources associated with this object and transition this object to the `NWTCPConnectionStateCancelled` state.
