# Relays

**Framework**: Network Extension

Create and manage a system-wide network relay configuration that uses built-in proxying for TCP and UDP traffic over HTTP/3 and HTTP/2.

## Topics

### Essentials
- [Network Extensions Entitlement](../BundleResources/Entitlements/com.apple.developer.networking.networkextension.md)
  The APIs an app can use to customize networking features.
### Relay configuration
- [class NERelayManager](nerelaymanager.md)
  An object you use to create and manage a network relay configuration.
- [class NERelay](nerelay.md)
  A single relay server configuration that you can chain together with other relays.
