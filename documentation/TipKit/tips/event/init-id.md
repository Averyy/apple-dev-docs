# init(id:)

**Framework**: TipKit  
**Kind**: init

- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+

## Declaration

```swift

```

## Parameters

- `id`: The unique identifier for persisting the event and its donations.
