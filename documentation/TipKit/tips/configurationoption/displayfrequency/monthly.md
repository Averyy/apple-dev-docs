# monthly

**Framework**: TipKit  
**Kind**: property

A monthly display frequency.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+
- watchOS 10.0+

## Declaration

```swift
static var monthly: Tips.ConfigurationOption.DisplayFrequency { get }
```

#### Discussion

When you set this, the system shows no more than one tip per month.
