# immediate

**Framework**: TipKit  
**Kind**: property

An immediate display frequency.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+
- watchOS 10.0+

## Declaration

```swift
static var immediate: Tips.ConfigurationOption.DisplayFrequency { get }
```

#### Discussion

When you set this, the system shows tips with no limitations on frequency.
