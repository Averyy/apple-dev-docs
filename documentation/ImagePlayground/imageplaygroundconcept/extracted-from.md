# extracted(from:)

**Framework**: Image Playground  
**Kind**: method

Builds a playground concept by extracting multiple concepts from a long text that describes the contents of the image.

**Availability**:
- iOS 18.1+
- iPadOS 18.1+
- Mac Catalyst 18.1+
- macOS 15.1+
- visionOS 2.4+
- Unknown ?+ - Deprecated

## Declaration

```swift
static func extracted(from text: String) -> ImagePlaygroundConcept
```

## Parameters

- `text`: However, if the string does not meet the minimum length required for concept extraction, it might decide to use the string as-is as input prompt.
