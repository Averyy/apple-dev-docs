# periodic

**Framework**: Symbols  
**Kind**: property

A repeat behavior that prefers to repeat indefinitely using periodic animations. Periodic animations play the effect at regular intervals starting and stopping each time.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst ?+

- tvOS 18.0+

- watchOS 11.0+

