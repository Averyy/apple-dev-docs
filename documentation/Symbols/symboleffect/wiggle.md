# wiggle

**Framework**: Symbols  
**Kind**: property

A symbol effect that applies the Wiggle animation to symbol images.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst ?+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+

## Declaration

```swift
static var wiggle: WiggleSymbolEffect { get }
```

#### Discussion

The Wiggle animation applies a transitory translation or rotation effect to the symbol.
