# upUp

**Framework**: Symbols  
**Kind**: property

Returns an effect requesting the Up-Up variant of the Replace animation.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst ?+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+

## Declaration

```swift
static var upUp: ReplaceSymbolEffect { get }
```

#### Discussion

The initial symbol scales up as it is removed, and the new symbol scales up as it is added.
