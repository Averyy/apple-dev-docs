# cornerRadius

**Framework**: TVUIKit  
**Kind**: property

The radius to use when drawing rounded corners for the cell.

**Availability**:
- tvOS 13.0+

## Declaration

```swift
@MainActor
var cornerRadius: CGFloat { get }
```

#### Discussion

Setting the radius to a value greater than 0.0 points applies increasingly rounded corners to the cell.
