# image

**Framework**: TVUIKit  
**Kind**: property

The custom image for the monogram.

**Availability**:
- tvOS 12.0+

## Declaration

```swift
@MainActor
var image: UIImage? { get set }
```

#### Discussion

If you supply an image, the system uses this image instead of a generic image or an image created from the name components.
