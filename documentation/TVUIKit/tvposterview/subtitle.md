# subtitle

**Framework**: TVUIKit  
**Kind**: property

The subtitle for the poster view.

**Availability**:
- tvOS 12.0+

## Declaration

```swift
@MainActor
var subtitle: String? { get set }
```

#### Discussion

A subtitle for a poster view is optional. If it exists, it appears beneath the [`title`](tvposterview/title.md).
