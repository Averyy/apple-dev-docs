# image

**Framework**: TVUIKit  
**Kind**: property

The image for the poster view.

**Availability**:
- tvOS 12.0+

## Declaration

```swift
@MainActor
var image: UIImage? { get set }
```

#### Discussion

The size of the poster view is determined by the natural size of the supplied image when no frame or content size is explicitly set.
