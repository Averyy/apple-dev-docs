# isSecureDigitEntry

**Framework**: TVUIKit  
**Kind**: property

A Boolean value that indicates whether an entered digit is immediately obscured.

**Availability**:
- tvOS 12.0+

## Declaration

```swift
@MainActor
var isSecureDigitEntry: Bool { get set }
```

#### Discussion

The defualt value is `NO`.
