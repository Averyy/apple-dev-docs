# CVOpenGLTexture

**Framework**: Core Video  
**Kind**: typealias

A reference to an OpenGL texture-based image buffer object.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.4+

## Declaration

```swift
typealias CVOpenGLTexture = CVImageBuffer
```

#### Discussion

The Core Video OpenGL texture is a wrapper around the standard OpenGL texture.
