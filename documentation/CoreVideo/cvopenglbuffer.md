# CVOpenGLBuffer

**Framework**: Core Video  
**Kind**: typealias

A reference to a Core Video OpenGL buffer object.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.4+

## Declaration

```swift
typealias CVOpenGLBuffer = CVImageBuffer
```

#### Discussion

The Core Video OpenGL buffer is a wrapper around the standard OpenGL pbuffer.
