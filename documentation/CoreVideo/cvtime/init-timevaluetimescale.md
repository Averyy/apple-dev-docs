# init(timeValue:timeScale:)

**Framework**: Core Video  
**Kind**: init

Initialize a valid CVTime value. Note: When `timeValue` has nonzero value, `timeScale` must be greater than 0.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 4.0+
- macOS 10.4+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 4.0+
