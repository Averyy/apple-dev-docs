# kCVImageBufferChromaLocation_Bottom

**Framework**: Core Video  
**Kind**: var

A key that indicates the chroma sample is horizontally centered, but is co-sited with the bottom row of luma samples.

**Availability**:
- iOS 4.0+
- iPadOS 4.0+
- Mac Catalyst 13.1+
- macOS 10.5+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+
