# resetVisibility()

**Framework**: SwiftUI  
**Kind**: method

Resets all tab sidebar visibilities back to the default, preserving the section customizations.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
