# resetSectionOrder(for:)

**Framework**: SwiftUI  
**Kind**: method

Resets ordering back to the default for the section with `sectionID`, preserving any customized tab visibilities.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- visionOS 2.0+

## Declaration

```swift
mutating func resetSectionOrder(for sectionID: String)
```
