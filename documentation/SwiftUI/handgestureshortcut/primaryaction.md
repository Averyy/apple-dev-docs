# primaryAction

**Framework**: SwiftUI  
**Kind**: property

The hand gesture shortcut for the primary action.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- watchOS 11.0+

## Declaration

```swift
static let primaryAction: HandGestureShortcut
```

#### Discussion

On watchOS, this is performed by double-tapping the thumb and index finger together.
