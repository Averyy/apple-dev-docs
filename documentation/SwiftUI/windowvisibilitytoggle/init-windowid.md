# init(windowID:)

**Framework**: SwiftUI  
**Kind**: init

Create a window visibility toggle to alter the visibility of a specific window.

**Availability**:
- macOS 15.0+

## Declaration

```swift
init(windowID: String) where Label == DefaultWindowVisibilityToggleLabel
```

## Parameters

- `windowID`: The   of the singleton window type that   should be toggled. If this is not a valid id, the toggle will be   disabled and non-functional.
