# windowArrangement

**Framework**: SwiftUI  
**Kind**: property

Placement for commands that arrange all of an app’s windows.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- visionOS 1.0+

## Declaration

```swift
static let windowArrangement: CommandGroupPlacement
```

#### Discussion

By default, this group includes the following command in macOS:

- Bring All to Front
