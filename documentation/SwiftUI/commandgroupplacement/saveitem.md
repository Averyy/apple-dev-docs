# saveItem

**Framework**: SwiftUI  
**Kind**: property

Placement for commands that save open documents and close windows.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- visionOS 1.0+

## Declaration

```swift
static let saveItem: CommandGroupPlacement
```

#### Discussion

By default, this group includes the following commands in macOS:

- Close
- Save
- Save As/Duplicate
- Revert to Saved
