# toolbar

**Framework**: SwiftUI  
**Kind**: property

Placement for commands that manipulate the toolbar.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- visionOS 1.0+

## Declaration

```swift
static let toolbar: CommandGroupPlacement
```

#### Discussion

By default, this group includes the following commands in macOS:

- Show/Hide Toolbar
- Customize Toolbar
