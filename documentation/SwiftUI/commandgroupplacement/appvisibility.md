# appVisibility

**Framework**: SwiftUI  
**Kind**: property

Placement for commands that control the visibility of running apps.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- visionOS 1.0+

## Declaration

```swift
static let appVisibility: CommandGroupPlacement
```

#### Discussion

By default, this group includes the following commands in macOS:

- Hide App
- Hide Others
- Show All
