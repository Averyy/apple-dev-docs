# printItem

**Framework**: SwiftUI  
**Kind**: property

Placement for commands related to printing app content.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- visionOS 1.0+

## Declaration

```swift
static let printItem: CommandGroupPlacement
```

#### Discussion

By default, this group includes the following commands in macOS:

- Page Setup
- Print
