# wheel

**Framework**: SwiftUI  
**Kind**: property

A date picker style that displays each component as columns in a scrollable wheel.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- visionOS 1.0+
- watchOS 10.0+
