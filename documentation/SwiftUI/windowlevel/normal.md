# normal

**Framework**: SwiftUI  
**Kind**: property

Normal window level.

**Availability**:
- macOS 15.0+

## Declaration

```swift
static var normal: WindowLevel { get }
```

#### Discussion

Use this if you want your scene to have the normal window level instead of the level computed by [`automatic`](windowlevel/automatic.md).
