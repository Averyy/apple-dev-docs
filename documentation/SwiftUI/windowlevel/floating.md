# floating

**Framework**: SwiftUI  
**Kind**: property

Floating window level.

**Availability**:
- macOS 15.0+

## Declaration

```swift
static var floating: WindowLevel { get }
```

#### Discussion

Use this if your window needs to function as a utility window and stay on top of other windows, for example, a PIP (picture-in-picture) window.
