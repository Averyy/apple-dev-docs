# init()

**Framework**: SwiftUI  
**Kind**: init

Creates a navigation view style represented by a view stack that only shows a single top view at a time.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 7.0+

## Declaration

```swift
init()
```

#### Discussion

Use [`stack`](navigationviewstyle/stack.md) to construct this style.
