# automatic

**Framework**: SwiftUI  
**Kind**: property

An effect  that attempts to determine the effect automatically. This is the default effect.

**Availability**:
- iOS 13.4+
- iPadOS 13.4+
- Mac Catalyst 13.4+
- tvOS 16.0+
- visionOS 1.0+
