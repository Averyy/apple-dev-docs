# supported

**Framework**: SwiftUI  
**Kind**: property

The action should only be run when the new viewpoint is equivalent to one of the values provided through [`supportedVolumeViewpoints(_:)`](view/supportedvolumeviewpoints(_:).md).

**Availability**:
- visionOS 2.0+

## Declaration

```swift
static let supported: VolumeViewpointUpdateStrategy
```

#### Discussion

The viewpoint will be equivalent to where the window bar and ornaments are presented for a volume.
