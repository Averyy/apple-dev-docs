# init(transform:)

**Framework**: SwiftUI  
**Kind**: init

Creates a new manipulation gesture state with the given transform.

**Availability**:
- visionOS 26.0+

## Declaration

```swift
init(transform: AffineTransform3D = .identity)
```

## Parameters

- `transform`: The initial transform of a manipulable view.
