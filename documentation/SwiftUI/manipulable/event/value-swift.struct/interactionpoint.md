# interactionPoint

**Framework**: SwiftUI  
**Kind**: property

The point at which a person interacted with a view to begin manipulating it.

**Availability**:
- visionOS 26.0+

## Declaration

```swift
let interactionPoint: Point3D
```

#### Discussion

This point is in the coordinate space configured in the view modifier.
