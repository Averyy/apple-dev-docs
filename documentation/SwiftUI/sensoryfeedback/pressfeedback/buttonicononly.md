# buttonIconOnly

**Framework**: SwiftUI  
**Kind**: property

Indicates that an icon-only button, without text, has been pressed (touch down).

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
- tvOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)
- watchOS 26.0+ (Beta)

## Declaration

```swift
static let buttonIconOnly: SensoryFeedback.PressFeedback
```

#### Discussion

Only plays feedback on visionOS.
