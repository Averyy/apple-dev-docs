# button

**Framework**: SwiftUI  
**Kind**: property

Indicates that a button has been pressed (touch down).

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+
- watchOS 26.0+

## Declaration

```swift
static let button: SensoryFeedback.PressFeedback
```

#### Discussion

Only plays feedback on visionOS.
