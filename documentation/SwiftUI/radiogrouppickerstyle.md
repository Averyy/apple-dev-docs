# RadioGroupPickerStyle

**Framework**: SwiftUI  
**Kind**: struct

A picker style that presents the options as a group of radio buttons.

**Availability**:
- macOS 10.15+

## Declaration

```swift
struct RadioGroupPickerStyle
```

#### Overview

You can also use [`radioGroup`](pickerstyle/radiogroup.md) to construct this style.
