# prominentDetail

**Framework**: SwiftUI  
**Kind**: property

A navigation split style that attempts to maintain the size of the detail content when hiding or showing the leading columns.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+
