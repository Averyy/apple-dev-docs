# onScrollDown

**Framework**: SwiftUI  
**Kind**: property

Minimize the tab bar when downwards scrolling starts. Minimizing is supported for tab bars on only iPhone.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)

## Declaration

```swift
static let onScrollDown: TabBarMinimizeBehavior
```
