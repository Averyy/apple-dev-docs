# Previews

**Kind**: associatedtype  
**Required**: Yes

**Availability**:
- iOS 13.0+

- Mac Catalyst 13.0+

## Declaration

```swift
associatedtype Previews : View
```

#### Discussion

When you create a preview, Swift infers this type from your implementation of the required [`previews`](previewprovider/previews-swift.type.property.md) property.
