# showsIndicators

**Framework**: SwiftUI  
**Kind**: property

A value that indicates whether the scroll view displays the scrollable component of the content offset, in a way that’s suitable for the platform.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
var showsIndicators: Bool { get set }
```

#### Discussion

The default is `true`.
