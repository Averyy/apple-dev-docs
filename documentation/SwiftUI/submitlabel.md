# SubmitLabel

**Framework**: SwiftUI  
**Kind**: struct

A semantic label describing the label of submission within a view hierarchy.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
struct SubmitLabel
```

#### Overview

A submit label is a description of a submission action provided to a view hierarchy using the [`onSubmit(of:_:)`](view/onsubmit(of:_:).md) modifier.
