# UnifiedWindowToolbarStyle

**Framework**: SwiftUI  
**Kind**: struct

A window toolbar style which displays its toolbar and title bar inline.

**Availability**:
- macOS 11.0+

## Declaration

```swift
struct UnifiedWindowToolbarStyle
```

#### Overview

You can also use [`unified`](windowtoolbarstyle/unified.md) or [`unified(showsTitle:)`](windowtoolbarstyle/unified(showstitle:).md) to construct this style.
