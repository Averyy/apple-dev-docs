# SelectionShapeStyle

**Framework**: SwiftUI  
**Kind**: struct

A style used to visually indicate selection following platform conventional colors and behaviors.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 10.15+
- visionOS 1.0+
