# Body

**Framework**: SwiftUI  
**Kind**: associatedtype  
**Required**: Yes

The type of configuration representing the content of this control.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 26.0+ (Beta)
- watchOS 26.0+ (Beta)

## Declaration

```swift
associatedtype Body : ControlWidgetConfiguration
```

#### Discussion

When you create a custom control, Swift infers this type from your implementation of the required [`body`](controlwidget/body-swift.property.md) property.
