# isScrolling

**Framework**: SwiftUI  
**Kind**: property

Whether the scroll view is actively scrolling.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+

## Declaration

```swift
var isScrolling: Bool { get }
```

#### Discussion

This convenience is equivalent to `phase != .idle`.
