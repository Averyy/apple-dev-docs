# offset(z:)

**Framework**: SwiftUI  
**Kind**: method

Brings a view forward in Z by the provided distance in points.

**Availability**:
- visionOS 1.0+

## Declaration

```swift
func offset(z: CGFloat) -> some VisualEffect
```

#### Return Value

An effect that is extruded forward in Z by `distance`.
