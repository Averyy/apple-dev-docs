# fontDesign(_:)

**Framework**: SwiftUI  
**Kind**: method

**Availability**:

- iPadOS 16.1+
- Mac Catalyst 16.1+
- macOS 13.0+
- tvOS 16.1+
- visionOS 1.0+

## Declaration

```swift
nonisolated
func fontDesign(_ design: Font.Design?) -> Text
```

#### Return Value

## Parameters

- `design`: One of the available font designs.
