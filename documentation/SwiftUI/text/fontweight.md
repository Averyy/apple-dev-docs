# fontWeight(_:)

**Framework**: SwiftUI  
**Kind**: method

Sets the font weight of the text.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
nonisolated
func fontWeight(_ weight: Font.Weight?) -> Text
```

#### Return Value

Text that uses the font weight you specify.

## Parameters

- `weight`: One of the available font weights.
