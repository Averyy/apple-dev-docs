# Text.Case.lowercase

**Framework**: Swiftui  
**Kind**: case

Displays text in all lowercase characters.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+
- watchOS 7.0+

## Declaration

```swift
case lowercase
```

#### Discussion

For example, “Hello” would be displayed as “hello”.

> **Note**: `StringProtocol.lowercased(with:)`
