# reorderable

**Framework**: SwiftUI  
**Kind**: property

The reorderable customization behavior.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- visionOS 2.0+

## Declaration

```swift
static var reorderable: TabCustomizationBehavior { get }
```

#### Discussion

Items with this behavior can be moved by the user, but cannot be removed.
