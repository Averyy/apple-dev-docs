# init(integerLiteral:)

**Framework**: SwiftUI  
**Kind**: init

Create a fixed corner style with an int value as its corner radius. This is a shorthand of writing `.fixed(value)`.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+
- watchOS 26.0+

## Declaration

```swift
init(integerLiteral value: Edge.Corner.Style.IntegerLiteralType)
```

## Parameters

- `value`: The fixed corner radius.
