# isIdentity

**Framework**: SwiftUI  
**Kind**: property

A Boolean that indicates whether the transition should have an identity effect, i.e. not change the appearance of its view.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+
- watchOS 10.0+

## Declaration

```swift
var isIdentity: Bool { get }
```

#### Discussion

This is true in the `identity` phase.
