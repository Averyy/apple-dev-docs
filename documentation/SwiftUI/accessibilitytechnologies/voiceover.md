# voiceOver

**Framework**: SwiftUI  
**Kind**: property

The value that represents the VoiceOver screen reader, allowing use of the system without seeing the screen visually.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+
