# init(_:)

**Framework**: SwiftUI  
**Kind**: init

Creates a new proposed size from a specified size.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+
- watchOS 9.0+

## Declaration

```swift
init(_ size: CGSize)
```

## Parameters

- `size`: A proposed size with dimensions measured in points.
