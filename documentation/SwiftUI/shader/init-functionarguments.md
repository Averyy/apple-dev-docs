# init(function:arguments:)

**Framework**: SwiftUI  
**Kind**: init

Creates a new shader from a function and the uniform argument values to bind to the function.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+
