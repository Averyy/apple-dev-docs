# look

**Framework**: SwiftUI  
**Kind**: property

On visionOS, by looking at the edge of a scroll view the content can automatically scroll. The axes will be determined automatically.

**Availability**:
- visionOS 26.0+

## Declaration

```swift
static let look: ScrollInputKind
```

#### Discussion

This is an opt-in behavior.
