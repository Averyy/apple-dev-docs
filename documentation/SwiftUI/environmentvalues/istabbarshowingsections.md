# isTabBarShowingSections

**Framework**: SwiftUI  
**Kind**: property

A Boolean value that determines whether a tab view shows the expanded contents of a tab section.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+
