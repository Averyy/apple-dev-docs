# accessibilityEnabled

**Framework**: SwiftUI  
**Kind**: property

A Boolean value that indicates whether the user has enabled an assistive technology.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+
