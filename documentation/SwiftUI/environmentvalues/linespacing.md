# lineSpacing

**Framework**: SwiftUI  
**Kind**: property

The distance in points between the bottom of one line fragment and the top of the next.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
var lineSpacing: CGFloat { get set }
```

#### Discussion

This value is always nonnegative.
