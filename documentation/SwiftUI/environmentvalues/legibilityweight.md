# legibilityWeight

**Framework**: SwiftUI  
**Kind**: property

The font weight to apply to text.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
var legibilityWeight: LegibilityWeight? { get set }
```

#### Discussion

This value reflects the value of the Bold Text display setting found in the Accessibility settings.
