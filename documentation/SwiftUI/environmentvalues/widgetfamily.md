# widgetFamily

**Framework**: SwiftUI  
**Kind**: property

The template of the widget — small, medium, or large.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- visionOS 26.0+ (Beta)
- watchOS 9.0+

## Declaration

```swift
var widgetFamily: WidgetFamily { get }
```

#### Discussion

Use this value to retrieve the widget size that the user chose for a widget.
