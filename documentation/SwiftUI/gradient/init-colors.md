# init(colors:)

**Framework**: SwiftUI  
**Kind**: init

Creates a gradient from an array of colors.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
init(colors: [Color])
```

#### Discussion

The gradient synthesizes its location values to evenly space the colors along the gradient.
