# InsetListStyle

**Framework**: SwiftUI  
**Kind**: struct

The list style that describes the behavior and appearance of an inset list.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- visionOS 1.0+
