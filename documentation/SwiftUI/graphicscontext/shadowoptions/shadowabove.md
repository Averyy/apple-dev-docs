# shadowAbove

**Framework**: SwiftUI  
**Kind**: property

An option that causes the filter to draw the shadow above the object, rather than below it.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
static var shadowAbove: GraphicsContext.ShadowOptions { get }
```
