# color(_:)

**Framework**: SwiftUI  
**Kind**: method

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+

## Declaration

```swift
static func color(_ color: Color) -> GraphicsContext.Shading
```

#### Return Value

A shading instance filled with a color.

## Parameters

- `color`: A   instance that defines the color   of the shading.
