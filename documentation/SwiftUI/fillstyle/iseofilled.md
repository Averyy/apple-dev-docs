# isEOFilled

**Framework**: SwiftUI  
**Kind**: property

A Boolean value that indicates whether to use the even-odd rule when rendering a shape.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
var isEOFilled: Bool
```

#### Discussion

When `isOEFilled` is `false`, the style uses the non-zero winding number rule.
