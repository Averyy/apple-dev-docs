# PalettePickerStyle

**Framework**: SwiftUI  
**Kind**: struct

A picker style that presents the options as a row of compact elements.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- visionOS 1.0+
