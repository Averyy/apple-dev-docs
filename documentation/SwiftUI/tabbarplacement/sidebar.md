# sidebar

**Framework**: SwiftUI  
**Kind**: property

The sidebar of a tab view.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+

## Declaration

```swift
static let sidebar: TabBarPlacement
```

#### Discussion

A tab bar can appear as a sidebar in [`sidebarAdaptable`](tabviewstyle/sidebaradaptable.md) on iPad, macOS, visionOS, and tvOS.
