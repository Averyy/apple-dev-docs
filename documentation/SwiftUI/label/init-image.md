# init(_:image:)

**Framework**: SwiftUI  
**Kind**: init

Creates a label with an icon image and a title generated from a localized string.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+
- watchOS 7.0+

## Declaration

```swift
nonisolated
init(_ titleKey: LocalizedStringKey, image name: String)
```

## Parameters

- `titleKey`: A title generated from a localized string.
