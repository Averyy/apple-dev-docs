# alignment

**Framework**: SwiftUI  
**Kind**: property

The alignment to use when placing each view.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+
- watchOS 7.0+

## Declaration

```swift
var alignment: Alignment?
```

#### Discussion

Use this property to anchor the view’s relative position to the same relative position in the view’s assigned grid space.
