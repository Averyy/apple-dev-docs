# spacing

**Framework**: SwiftUI  
**Kind**: property

The spacing to the next item.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+
- watchOS 7.0+

## Declaration

```swift
var spacing: CGFloat?
```

#### Discussion

If this value is `nil`, the item uses a reasonable default for the current platform.
