# isEditing

**Framework**: SwiftUI  
**Kind**: property

Indicates whether a view is being edited.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
var isEditing: Bool { get }
```

#### Discussion

This property returns `true` if the mode is something other than inactive.
