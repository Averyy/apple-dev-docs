**Framework**: SwiftUI  
**Kind**: property

**Availability**:
- iOS 18.0+
- iPadOS 18.0+

- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+

## Declaration

```swift
static var alwaysByOne: ViewAlignedScrollTargetBehavior.LimitBehavior { get }
```

#### Discussion

Limit the number of views that can be scrolled by a single interaction to a single view.
