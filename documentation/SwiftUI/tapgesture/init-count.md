# init(count:)

**Framework**: SwiftUI  
**Kind**: init

Creates a tap gesture with the number of required taps.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 16.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
init(count: Int = 1)
```

## Parameters

- `count`: The required number of taps to complete the tap   gesture.
