# fileURL

**Framework**: SwiftUI  
**Kind**: property

A URL of an open document.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- visionOS 1.0+

## Declaration

```swift
var fileURL: URL? { get }
```

#### Discussion

If the document has never been saved, returns `nil`.
