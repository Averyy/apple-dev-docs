# automatic

**Framework**: SwiftUI  
**Kind**: property

The automatic role.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+
- watchOS 9.0+

## Declaration

```swift
static var automatic: ToolbarRole { get }
```

#### Discussion

In iOS, tvOS, and watchOS this resolves to the [`navigationStack`](toolbarrole/navigationstack.md) role. In macOS, this resolves to the [`editor`](toolbarrole/editor.md) role.
