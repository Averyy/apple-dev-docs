# window

**Framework**: SwiftUI  
**Kind**: property

A menu bar extra style that renders its contents in a popover-like window.

**Availability**:
- macOS 13.0+

## Declaration

```swift
static var window: WindowMenuBarExtraStyle { get }
```

#### Discussion

The styling and layout of controls is similar to that when contained in a normal window, compared to the menu-like layout that the [`menu`](menubarextrastyle/menu.md) style provides.
