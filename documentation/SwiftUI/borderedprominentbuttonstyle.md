# BorderedProminentButtonStyle

**Framework**: SwiftUI  
**Kind**: struct

A button style that applies standard border prominent artwork based on the button’s context.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+
