# minimumDuration

**Framework**: SwiftUI  
**Kind**: property

The minimum duration of the long press that must elapse before the gesture succeeds.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 14.0+
- visionOS 1.0+
- watchOS 6.0+
