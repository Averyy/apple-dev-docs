# onTextEntry

**Framework**: SwiftUI  
**Kind**: property

An activation where the system shows search scopes when typing begins in the search field and hides search scopes after search cancellation.

**Availability**:
- iOS 16.4+
- iPadOS 16.4+
- Mac Catalyst 16.4+
- macOS 13.3+
- visionOS 1.0+
- watchOS 9.4+
