# automatic

**Framework**: SwiftUI  
**Kind**: property

Remap shortcuts to their international counterparts, mirrored for right-to-left usage if appropriate.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- visionOS 1.0+

## Declaration

```swift
static let automatic: KeyboardShortcut.Localization
```

#### Discussion

This is the default configuration.
