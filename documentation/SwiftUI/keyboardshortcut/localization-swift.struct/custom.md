# custom

**Framework**: SwiftUI  
**Kind**: property

Don’t use automatic shortcut remapping.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- visionOS 1.0+

## Declaration

```swift
static let custom: KeyboardShortcut.Localization
```

#### Discussion

When you use this mode, you have to take care of international use-cases separately.
