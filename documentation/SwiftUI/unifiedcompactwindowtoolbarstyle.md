# UnifiedCompactWindowToolbarStyle

**Framework**: SwiftUI  
**Kind**: struct

A window toolbar style similar to [`unified`](windowtoolbarstyle/unified.md), but with a more compact vertical sizing.

**Availability**:
- macOS 11.0+

## Declaration

```swift
struct UnifiedCompactWindowToolbarStyle
```

#### Overview

You can also use [`unifiedCompact`](windowtoolbarstyle/unifiedcompact.md) or [`unifiedCompact(showsTitle:)`](windowtoolbarstyle/unifiedcompact(showstitle:).md) to construct this style.
