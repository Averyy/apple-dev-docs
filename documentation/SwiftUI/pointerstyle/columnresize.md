# columnResize

**Framework**: SwiftUI  
**Kind**: property

The pointer style for resizing a column, or vertical division, in either direction.

**Availability**:
- macOS 15.0+

## Declaration

```swift
static let columnResize: PointerStyle
```

#### Discussion

You may apply this pointer style to a single view or a view hierarchy using the [`pointerStyle(_:)`](view/pointerstyle(_:).md) modifier.
