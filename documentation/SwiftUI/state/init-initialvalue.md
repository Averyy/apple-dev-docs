# init(initialValue:)

**Framework**: SwiftUI  
**Kind**: init

Creates a state property that stores an initial value.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
init(initialValue value: Value)
```

#### Discussion

This initializer has the same behavior as the [`init(wrappedValue:)`](state/init(wrappedvalue:).md) initializer. See that initializer for more information.

## Parameters

- `value`: An initial value to store in the state   property.
