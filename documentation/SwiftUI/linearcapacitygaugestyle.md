# LinearCapacityGaugeStyle

**Framework**: SwiftUI  
**Kind**: struct

A gauge style that displays bar that fills from leading to trailing edges as the gauge’s current value increases.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- visionOS 1.0+
- watchOS 9.0+
