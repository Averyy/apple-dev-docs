# ControlSize.extraLarge

**Framework**: SwiftUI  
**Kind**: case

A control version that is substantially sized. The largest control size. Resolves to [`ControlSize.large`](controlsize/large.md) on platforms other than visionOS.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+
- watchOS 10.0+
