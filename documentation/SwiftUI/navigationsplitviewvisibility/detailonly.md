# detailOnly

**Framework**: SwiftUI  
**Kind**: property

Hide the leading two columns of a three-column navigation split view, so that just the detail area shows.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+
- watchOS 9.0+

## Declaration

```swift
static var detailOnly: NavigationSplitViewVisibility { get }
```
