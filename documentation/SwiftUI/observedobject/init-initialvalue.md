# init(initialValue:)

**Framework**: SwiftUI  
**Kind**: init

Creates an observed object with an initial value.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
@MainActor
@preconcurrency init(initialValue: ObjectType)
```

#### Discussion

This initializer has the same behavior as the [`init(wrappedValue:)`](observedobject/init(wrappedvalue:).md) initializer. See that initializer for more information.

## Parameters

- `initialValue`: An initial value.
