# init(uniformLeadingCorners:uniformTrailingCorners:)

**Framework**: SwiftUI  
**Kind**: init

Create a rectangle with a corner style set on the leading two corners uniformly, and another style set on the trailing two corners uniformly.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+
- watchOS 26.0+

## Declaration

```swift
init(uniformLeadingCorners: Edge.Corner.Style = .concentric, uniformTrailingCorners: Edge.Corner.Style = .concentric)
```

## Parameters

- `uniformLeadingCorners`: The corner style to be applied to the leading   two corners uniformly.
- `uniformTrailingCorners`: The corner style to be applied to the   trailing two corners uniformly.
