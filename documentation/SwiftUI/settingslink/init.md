# init()

**Framework**: SwiftUI  
**Kind**: init

Creates a settings link with the default system label.

**Availability**:
- macOS 14.0+

## Declaration

```swift
init() where Label == DefaultSettingsLinkLabel
```

#### Discussion

The display of the label may be customized using the `labelStyle(_:)` modifier.
