# tabViewSidebar

**Framework**: SwiftUI  
**Kind**: property

The tab view sidebar. This is present on certain platforms when using the `.sidebarAdaptable` tab view style.

**Availability**:
- iOS 18.4+
- iPadOS 18.4+
- Mac Catalyst 18.4+
- visionOS 2.4+
