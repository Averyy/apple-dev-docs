# bounds

**Framework**: SwiftUI  
**Kind**: property

The bounds rect of the scroll view.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+

## Declaration

```swift
var bounds: CGRect { get }
```

#### Discussion

Unlike the visible rect, this value is within the content insets of the scroll view.
