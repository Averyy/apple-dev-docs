# contentInsets

**Framework**: SwiftUI  
**Kind**: property

The content insets of the scroll view.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+

## Declaration

```swift
var contentInsets: EdgeInsets
```

#### Discussion

Adding these insets to the content size of the scroll view will give you the total scrollable space of the scroll view.
