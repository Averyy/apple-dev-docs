# init(rootView:)

**Framework**: SwiftUI  
**Kind**: init

Creates a hosting menu object that wraps the specified SwiftUI view.

**Availability**:
- macOS 14.4+

## Declaration

```swift
init(rootView: Content)
```

## Parameters

- `rootView`: The root view of the SwiftUI view hierarchy that   you want to present using this menu.
