# defaultDisplay

**Framework**: SwiftUI  
**Kind**: property

The display on which new windows will be presented by default.

**Availability**:
- macOS 15.0+

## Declaration

```swift
var defaultDisplay: DisplayProxy { get }
```

#### Discussion

On macOS, this is typically the display which currently has focus.
