# center

**Framework**: SwiftUI  
**Kind**: property

A point that’s centered in a view.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
static let center: UnitPoint
```

#### Discussion

This point occupies the position where the horizontal and vertical alignment guides intersect for [`center`](alignment/center.md) alignment.
