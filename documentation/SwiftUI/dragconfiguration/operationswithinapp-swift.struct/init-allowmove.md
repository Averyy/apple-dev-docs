# init(allowMove:)

**Framework**: SwiftUI  
**Kind**: init

Creates a value that describes the operations allowed for drags that end within the application. Copy operation is always allowed.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
init(allowMove: Bool = false)
```
