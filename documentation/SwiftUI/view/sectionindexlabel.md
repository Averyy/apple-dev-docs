# sectionIndexLabel(_:)

**Framework**: SwiftUI  
**Kind**: method

Sets the label that is used in a section index to point to this section, typically only a single character long.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+
- watchOS 26.0+

```swift
nonisolated
func sectionIndexLabel(_ label: Text?) -> some View
```

- See also `listSectionIndexVisibility(_:)`

## Parameters

- `label`: The label to display in the section index, or    to display no label for this section.
