# minLength

**Framework**: SwiftUI  
**Kind**: property

The minimum length this spacer can be shrunk to, along the axis or axes of expansion.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
@MainActor
@preconcurrency var minLength: CGFloat?
```

#### Discussion

If `nil`, the system default spacing between views is used.
