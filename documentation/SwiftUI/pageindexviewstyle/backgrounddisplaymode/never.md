# never

**Framework**: SwiftUI  
**Kind**: property

Background is never displayed behind the page index view.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- tvOS 14.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
static let never: PageIndexViewStyle.BackgroundDisplayMode
```
