# sidebar

**Framework**: SwiftUI  
**Kind**: property

The sidebar of a tab view.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- visionOS 2.0+

## Declaration

```swift
static let sidebar: AdaptableTabBarPlacement
```

#### Discussion

A tab view can appear as a sidebar in [`sidebarAdaptable`](tabviewstyle/sidebaradaptable.md). Sidebar tabs represent all of the tabs in a tab view.
