# init(uiImage:)

**Framework**: SwiftUI  
**Kind**: init

Creates a SwiftUI image from a UIKit image instance.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
init(uiImage: UIImage)
```

## Parameters

- `uiImage`: The UIKit image to wrap with a SwiftUI    instance.
