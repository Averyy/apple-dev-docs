# CompactDatePickerStyle

**Framework**: SwiftUI  
**Kind**: struct

A date picker style that displays the components in a compact, textual format.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 13.4+
- macOS 10.15.4+
- visionOS 1.0+
