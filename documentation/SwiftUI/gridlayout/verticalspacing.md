# verticalSpacing

**Framework**: SwiftUI  
**Kind**: property

The vertical distance between adjacent subviews.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+
- watchOS 9.0+

## Declaration

```swift
var verticalSpacing: CGFloat?
```

#### Discussion

Set this value to `nil` to use default vertical distances between subviews.
