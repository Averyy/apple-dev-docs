# GraphicalDatePickerStyle

**Framework**: SwiftUI  
**Kind**: struct

A date picker style that displays an interactive calendar or clock.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 10.15+
- visionOS 1.0+
