# EmptyHoverEffect

**Framework**: SwiftUI  
**Kind**: struct

A base hover effect used to build additional effects.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- tvOS 18.0+
- visionOS 2.0+

## Declaration

```swift
struct EmptyHoverEffect
```

#### Overview

`EmptyHoverEffect` does not change a view on hover.
