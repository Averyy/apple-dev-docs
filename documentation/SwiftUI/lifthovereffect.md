# LiftHoverEffect

**Framework**: SwiftUI  
**Kind**: struct

A hover effect that slides the pointer under the view and disappears as the view scales up and gains a shadow.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- tvOS 18.0+
- visionOS 2.0+

## Declaration

```swift
struct LiftHoverEffect
```

#### Overview

You can also use [`lift`](customhovereffect/lift.md) to construct this hover effect.
