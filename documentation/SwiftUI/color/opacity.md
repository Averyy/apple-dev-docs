# opacity(_:)

**Framework**: SwiftUI  
**Kind**: method

Multiplies the opacity of the color by the given amount.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+
- watchOS 6.0+

## Declaration

```swift
func opacity(_ opacity: Double) -> Color
```

#### Return Value

A view with modified opacity.

## Parameters

- `opacity`: The amount by which to multiply the opacity of the   color.
