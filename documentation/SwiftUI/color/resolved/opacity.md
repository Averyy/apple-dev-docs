# opacity

**Framework**: SwiftUI  
**Kind**: property

The degree of opacity in the color, given in the range `0` to `1`.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+
- watchOS 10.0+

## Declaration

```swift
var opacity: Float
```

#### Discussion

A value of `0` means 100% transparency, while a value of `1` means 100% opacity.
