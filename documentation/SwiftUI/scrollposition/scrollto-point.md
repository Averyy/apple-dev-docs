# scrollTo(point:)

**Framework**: SwiftUI  
**Kind**: method

Scrolls the position of the scroll view to the point you provide.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+

## Declaration

```swift
mutating func scrollTo(point: CGPoint)
```

#### Discussion

The scroll view will clamp this value to only scroll to the size of its actual content.
