# Body

**Framework**: SwiftUI  
**Kind**: associatedtype  
**Required**: Yes

The type of scene that represents the body of this scene.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+
- watchOS 7.0+

## Declaration

```swift
associatedtype Body : Scene
```

#### Discussion

When you create a custom scene, Swift infers this type from your implementation of the required [`body`](scene/body-swift.property.md) property.
