# hasMotion

**Framework**: SwiftUI  
**Kind**: property

Whether the transition includes motion.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+
- watchOS 10.0+

## Declaration

```swift
var hasMotion: Bool
```

#### Discussion

When this behavior is included in a transition, that transition will be replaced by opacity when Reduce Motion is enabled.

Defaults to `true`.
