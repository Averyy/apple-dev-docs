# environment

**Framework**: SwiftUI  
**Kind**: property

Environment values that describe the current state of the system.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
@MainActor
@preconcurrency var environment: EnvironmentValues { get }
```

#### Discussion

Use the environment values to configure the state of your UIKit view controller when creating or updating it.
