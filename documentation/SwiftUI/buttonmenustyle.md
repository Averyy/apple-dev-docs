# ButtonMenuStyle

**Framework**: SwiftUI  
**Kind**: struct

A menu style that displays a button that toggles the display of the menu’s contents when pressed.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
