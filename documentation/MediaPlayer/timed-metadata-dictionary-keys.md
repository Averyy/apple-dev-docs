# Timed metadata dictionary keys

**Framework**: Media Player

Dictionary keys for use with the [`allMetadata`](mptimedmetadata/allmetadata.md) property. All keys are optional.

## Topics

### Constants
- [let MPMoviePlayerTimedMetadataKeyName: String](mpmovieplayertimedmetadatakeyname.md)
  The name of the timed metadata key.
- [let MPMoviePlayerTimedMetadataKeyInfo: String](mpmovieplayertimedmetadatakeyinfo.md)
  Arbitrary information about the timed metadata.
- [let MPMoviePlayerTimedMetadataKeyMIMEType: String](mpmovieplayertimedmetadatakeymimetype.md)
  The MIME type for the timed metadata.
- [let MPMoviePlayerTimedMetadataKeyDataType: String](mpmovieplayertimedmetadatakeydatatype.md)
  The data type of the timed metadata.
- [let MPMoviePlayerTimedMetadataKeyLanguageCode: String](mpmovieplayertimedmetadatakeylanguagecode.md)
  The metadata language, expressed using ISO 639-2, in a string object.
