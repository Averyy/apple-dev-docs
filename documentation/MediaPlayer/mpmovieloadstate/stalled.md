# stalled

**Framework**: Media Player  
**Kind**: property

**Availability**:
- iOS 3.2+
- iPadOS 3.2+
- Mac Catalyst 13.1+

## Declaration

```swift
static var stalled: MPMovieLoadState { get }
```

#### Discussion

The buffering of data has stalled. If started now, playback may pause automatically if the player runs out of buffered data.
