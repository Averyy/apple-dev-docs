# init(_:)

**Framework**: Media Player  
**Kind**: init

Creates a Media Player time range that indicates where an ad break exists in the current player.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- tvOS 16.0+
- visionOS 1.0+

## Declaration

```swift
init(_ timeRange: CMTimeRange)
```

## Parameters

- `timeRange`: A Core Media time range.
