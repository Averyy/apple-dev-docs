# init(name:)

**Framework**: Media Player  
**Kind**: init

Creates a new playlist metadata object with the designated name.

**Availability**:
- iOS 9.3+
- iPadOS 9.3+
- Mac Catalyst 13.1+
- visionOS 1.0+

## Declaration

```swift
init(name: String)
```

## Parameters

- `name`: The name to assign to the new playlist.
