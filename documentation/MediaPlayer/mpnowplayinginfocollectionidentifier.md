# MPNowPlayingInfoCollectionIdentifier

**Framework**: Media Player  
**Kind**: var

The identifier of the collection the Now Playing item belongs to.

**Availability**:
- iOS 9.3+
- iPadOS 9.3+
- Mac Catalyst 13.1+
- macOS 10.12.2+
- tvOS 9.2+
- visionOS 1.0+
- watchOS 5.0+

## Declaration

```swift
let MPNowPlayingInfoCollectionIdentifier: String
```

#### Discussion

The identifier can be an album, artist, playlist, etc.
