# MPError.Code.unknown

**Framework**: Media Player  
**Kind**: case

An error that indicates the requested operation can’t complete due to an unknown error.

**Availability**:
- iOS 9.3+
- iPadOS 9.3+
- Mac Catalyst 13.1+
- macOS 10.14.2+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 5.0+
