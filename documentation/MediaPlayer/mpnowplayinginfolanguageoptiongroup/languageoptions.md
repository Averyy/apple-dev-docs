# languageOptions

**Framework**: Media Player  
**Kind**: property

The available language options for the group.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.12.2+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 5.0+

## Declaration

```swift
var languageOptions: [MPNowPlayingInfoLanguageOption] { get }
```

#### Discussion

This property contains an array of available language options for the group.
