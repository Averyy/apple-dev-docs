# MPMediaItemPropertyPlaybackStoreID

**Framework**: Media Player  
**Kind**: var

The identifier for enqueueing store tracks.

**Availability**:
- iOS 10.3+
- iPadOS 10.3+
- Mac Catalyst 13.1+
- macOS 10.13+
- tvOS 10.3+
- visionOS 1.0+
- watchOS 5.0+

## Declaration

```swift
let MPMediaItemPropertyPlaybackStoreID: String
```

#### Discussion

Matches the identifier used by [`MPMusicPlayerController`](mpmusicplayercontroller.md) to enqueue store tracks.
