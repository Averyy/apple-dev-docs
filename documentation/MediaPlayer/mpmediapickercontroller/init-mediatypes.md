# init(mediaTypes:)

**Framework**: Media Player  
**Kind**: init

Initializes a media item picker for specified media types.

**Availability**:
- iOS 3.0+
- iPadOS 3.0+
- Mac Catalyst 13.1+

## Declaration

```swift
@MainActor
init(mediaTypes: MPMediaType)
```

#### Return Value

A new media item picker.

## Parameters

- `mediaTypes`: A set representing the media types for the media item picker. See the   enumeration.
