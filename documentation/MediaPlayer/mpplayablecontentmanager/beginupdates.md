# beginUpdates()

**Framework**: Media Player  
**Kind**: method

Updates several Media Player content items at once.

**Availability**:
- iOS 7.1+
- iPadOS 7.1+
- Mac Catalyst 13.1+
- visionOS 1.0+

## Declaration

```swift
func beginUpdates()
```

#### Discussion

Call this method to start a synchronized batch update of `MPContentItems` objects.
