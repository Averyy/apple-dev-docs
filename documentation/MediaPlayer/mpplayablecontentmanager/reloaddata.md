# reloadData()

**Framework**: Media Player  
**Kind**: method

Reloads the data from the data source.

**Availability**:
- iOS 7.1+
- iPadOS 7.1+
- Mac Catalyst 13.1+
- visionOS 1.0+

## Declaration

```swift
func reloadData()
```

#### Discussion

Call this method when the data source changed, which notifies the content manager to reload the data from the data source.
