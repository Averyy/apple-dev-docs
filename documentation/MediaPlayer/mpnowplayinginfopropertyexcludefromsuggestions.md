# MPNowPlayingInfoPropertyExcludeFromSuggestions

**Framework**: Media Player  
**Kind**: var

A number that denotes whether to exclude the Now Playing item from content suggestions.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+

## Declaration

```swift
let MPNowPlayingInfoPropertyExcludeFromSuggestions: String
```

#### Discussion

The value is an [`NSNumber`](https://developer.apple.com/documentation/Foundation/NSNumber) object configured as a `boolean`.
