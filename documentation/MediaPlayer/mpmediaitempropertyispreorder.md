# MPMediaItemPropertyIsPreorder

**Framework**: Media Player  
**Kind**: var

A Boolean value that indicates whether the media item is a preorder.

**Availability**:
- iOS 14.5+
- iPadOS 14.5+
- Mac Catalyst 14.5+
- macOS 11.3+
- tvOS 14.5+
- visionOS 1.0+
- watchOS 7.3+

## Declaration

```swift
let MPMediaItemPropertyIsPreorder: String
```

#### Discussion

This value is an [`NSNumber`](https://developer.apple.com/documentation/Foundation/NSNumber) object that represents a `BOOL` data type.
