# init(players:)

**Framework**: Media Player  
**Kind**: init

Creates a Now Playing session object.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- tvOS 14.0+
- visionOS 1.0+

## Declaration

```swift
init(players: [AVPlayer])
```

## Parameters

- `players`: An array of player objects. Passing   or an empty array raises an error.
