# MPMoviePlaybackState.paused

**Framework**: Media Player  
**Kind**: case

**Availability**:
- iOS 3.2+
- iPadOS 3.2+
- Mac Catalyst 13.1+

## Declaration

```swift
case paused
```

#### Discussion
