# MPMoviePlaybackState.stopped

**Framework**: Media Player  
**Kind**: case

**Availability**:
- iOS 3.2+
- iPadOS 3.2+
- Mac Catalyst 13.1+

## Declaration

```swift
case stopped
```

#### Discussion

Playback is currently stopped. Playback will commence from the beginning of the movie.
