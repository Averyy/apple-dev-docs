# MPChangeLanguageOptionSetting.nowPlayingItemOnly

**Framework**: Media Player  
**Kind**: case

The language option change is applied to the now playing item only.

**Availability**:
- iOS 9.3+
- iPadOS 9.3+
- Mac Catalyst 13.1+
- macOS 10.12.2+
- tvOS 9.2+
- visionOS 1.0+
- watchOS 5.0+
