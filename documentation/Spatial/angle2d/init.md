# init(_:)

**Framework**: Spatial  
**Kind**: init

Creates an angle from a SwiftUI angle.

**Availability**:
- visionOS 1.0+

## Declaration

```swift
init(_ angle: Angle)
```

#### Discussion

This function creates a new `Angle2D` object from an existing [`Angle`](https://developer.apple.com/documentation/SwiftUI/Angle) instance.
