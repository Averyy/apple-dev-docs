# -(_:_:)

**Framework**: Spatial  
**Kind**: op

Returns a point that’s the element-wise difference of a point’s `x`, `y`, and `z` and a size’s `width`, `height`, and `depth`.

- `lhs`: The first value.
- `rhs`: The second value.
