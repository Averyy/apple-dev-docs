# !=(_:_:)

**Framework**: RoomPlan  
**Kind**: op

Determines whether two instructions aren’t equal.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+

## Declaration

```swift
static func != (lhs: Self, rhs: Self) -> Bool
```

#### Return Value

`true` if the instructions aren’t equal. Otherwise, returns `false`.

## Parameters

- `lhs`: The instruction argument on the left-hand side of the operator.
- `rhs`: The instruction argument on the right-hand side of the operator.
