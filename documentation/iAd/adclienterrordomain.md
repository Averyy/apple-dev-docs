# ADClientErrorDomain

**Framework**: iAd  
**Kind**: var

The error domain that passes to the completion handler.

## Declaration

```swift
let ADClientErrorDomain: String
```

#### Discussion

After [`ADClient`](adclient.md), `NSErrors` from the `ADClientErrorDomain` are passed to the completion handler.
