# enqueueBarrier()

**Framework**: Metal  
**Kind**: method  
**Required**: Yes

Appends a barrier that tells the input/output command queue to finish running all in-flight command buffers before running any new command buffers.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+
