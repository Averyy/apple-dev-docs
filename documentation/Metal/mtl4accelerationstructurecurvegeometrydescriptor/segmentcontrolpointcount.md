# segmentControlPointCount

**Framework**: Metal  
**Kind**: property

Declares the number of control points per curve segment.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
- tvOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
var segmentControlPointCount: Int { get set }
```

#### Discussion

Valid values for this property are `2`, `3`, or `4`.
