# indexType

**Framework**: Metal  
**Kind**: property

Configures the size of the indices the `indexBuffer` contains, which is typically either 16 or 32-bits for each index.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
