# init(rawValue:)

**Framework**: Metal  
**Kind**: init

Creates a set of resource options from a raw value.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.13+
- tvOS 11.0+
- visionOS 1.0+

## Declaration

```swift
init(rawValue: UInt)
```

## Parameters

- `rawValue`: The raw value to convert into a set of resource options.
