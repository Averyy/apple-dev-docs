# libraryType

**Framework**: Metal  
**Kind**: property

The kind of library to create.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+

## Declaration

```swift
var libraryType: MTLLibraryType { get set }
```

#### Discussion

The default value is [`MTLLibraryType.executable`](mtllibrarytype/executable.md).
