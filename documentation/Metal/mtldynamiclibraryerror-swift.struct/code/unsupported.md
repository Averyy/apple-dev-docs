# MTLDynamicLibraryError.Code.unsupported

**Framework**: Metal  
**Kind**: case

An error code that indicates the GPU device doesn’t support dynamic libraries.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.0+
- visionOS 1.0+
