# MTLLibraryError.Code.unsupported

**Framework**: Metal  
**Kind**: case

Metal couldn’t support the requested action.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
case unsupported
```

#### Discussion

For example, the requested library file has improper formatting, or the requested library is not accessible.
