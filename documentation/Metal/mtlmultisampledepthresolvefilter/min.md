# MTLMultisampleDepthResolveFilter.min

**Framework**: Metal  
**Kind**: case

The GPU compares all depth samples in the pixel and selects the sample with the smallest value.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
