# samplePositions

**Framework**: Metal  
**Kind**: property

Configures the custom sample positions to use in MSAA rendering.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst ?+
- macOS 26.0+ (Beta)
- tvOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
var samplePositions: [MTLSamplePosition] { get set }
```

#### Discussion

Set to an empty array to disable custom sample positions.
