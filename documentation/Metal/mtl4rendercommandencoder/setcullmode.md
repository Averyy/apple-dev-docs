# setCullMode(_:)

**Framework**: Metal  
**Kind**: method  
**Required**: Yes

Controls whether Metal culls front facing primitives, back facing primitives, or culls no primitives at all.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
- tvOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
func setCullMode(_ cullMode: MTLCullMode)
```

## Parameters

- `cullMode`:   to set.
