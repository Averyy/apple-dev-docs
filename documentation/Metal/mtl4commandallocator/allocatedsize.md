# allocatedSize()

**Framework**: Metal  
**Kind**: method  
**Required**: Yes

Queries the size of the internal memory heaps of this command allocator that support encoding commands into command buffers.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
- tvOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
func allocatedSize() -> UInt64
```

#### Return Value

A size in bytes.
