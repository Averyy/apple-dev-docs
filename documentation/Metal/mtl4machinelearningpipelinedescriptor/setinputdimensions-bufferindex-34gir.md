# setInputDimensions(_:bufferIndex:)

**Framework**: Metal  
**Kind**: method

Sets the dimension of an input tensor at a buffer index.

**Availability**:

- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
- tvOS 26.0+ (Beta)

## Declaration

```swift

```

## Parameters

- `dimensions`: The dimensions of the tensor.
- `bufferIndex`: Index of the tensor to modify.
