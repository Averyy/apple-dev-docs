# setInputDimensions(_:bufferIndex:)

**Framework**: Metal  
**Kind**: method

Sets the dimension of an input tensor at a buffer index.

**Availability**:

- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
