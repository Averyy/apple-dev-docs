# count

**Framework**: Metal  
**Kind**: property

Assigns the number of entries in the heap.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+

## Declaration

```swift
var count: Int { get set }
```

#### Discussion

Each entry represents one item in the heap. The size of the individual entries depends on the heap type.
