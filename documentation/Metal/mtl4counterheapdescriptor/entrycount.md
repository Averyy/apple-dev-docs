# entryCount

**Framework**: Metal  
**Kind**: property

Assigns the number of entries in the heap.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
- tvOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
var entryCount: Int { get set }
```

#### Discussion

Each entry represents one item in the heap. The size of the individual entries depends on the heap type.
