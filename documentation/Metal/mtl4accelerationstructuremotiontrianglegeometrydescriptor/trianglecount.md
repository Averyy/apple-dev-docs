# triangleCount

**Framework**: Metal  
**Kind**: property

Declares the number of triangles in the vertex buffers that the buffer in the vertex buffers property references.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+

## Declaration

```swift
var triangleCount: Int { get set }
```

#### Discussion

All keyframes share the same triangle count.
