# boundingBoxCount

**Framework**: Metal  
**Kind**: property

Declares the number of bounding boxes in each buffer that `boundingBoxBuffer` references.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+

## Declaration

```swift
var boundingBoxCount: Int { get set }
```

#### Discussion

All keyframes share the same bounding box count.
