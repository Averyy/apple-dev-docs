# boundingBoxCount

**Framework**: Metal  
**Kind**: property

Declares the number of bounding boxes in each buffer that `boundingBoxBuffer` references.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
- tvOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
var boundingBoxCount: Int { get set }
```

#### Discussion

All keyframes share the same bounding box count.
