# Creating a Metal Dynamic Library

**Framework**: Metal

Compile a library of shaders and write it to a file as a dynamically linked library.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- macOS 11.5+
- Xcode 12.0+
