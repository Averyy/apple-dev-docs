# compileToBinary

**Framework**: Metal  
**Kind**: property

An option that tells Metal to compile the function to a binary format for dynamic linking.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 16.0+
- visionOS 1.0+
