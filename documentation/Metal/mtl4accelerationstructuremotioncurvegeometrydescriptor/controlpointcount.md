# controlPointCount

**Framework**: Metal  
**Kind**: property

Specifies the number of control points in the buffers the control point buffers reference.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
- tvOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
var controlPointCount: Int { get set }
```

#### Discussion

All keyframes have the same number of control points.
