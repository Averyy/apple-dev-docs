# curveEndCaps

**Framework**: Metal  
**Kind**: property

Configures the type of curve end caps.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+

## Declaration

```swift
var curveEndCaps: MTLCurveEndCaps { get set }
```

#### Discussion

Defaults to `MTLCurveEndCapsNone`. All keyframes share the same end cap type.
