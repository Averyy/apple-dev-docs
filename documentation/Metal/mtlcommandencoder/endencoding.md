# endEncoding()

**Framework**: Metal  
**Kind**: method  
**Required**: Yes

Declares that all command generation from the encoder is completed.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
func endEncoding()
```

#### Discussion

After `endEncoding` is called, the command encoder has no further use. You cannot encode any other commands with this encoder.
