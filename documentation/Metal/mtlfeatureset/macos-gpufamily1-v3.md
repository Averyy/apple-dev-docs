# MTLFeatureSet.macOS_GPUFamily1_v3

**Framework**: Metal  
**Kind**: case

**Availability**:
- macOS 10.13+

## Declaration


#### Discussion

This feature set corresponds with the first family of Mac GPUs that support Metal and run macOS 10.13.
