# Rendering reflections in real time using ray tracing

**Framework**: Metal

Implement realistic real-time lighting by dynamically generating reflection maps by encoding a ray-tracing compute pass.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- macOS 13.0+
- Xcode 15.3+

#### Overview

This sample code project relates to multiple WWDC sessions, including:

- [`10089: Bring your advanced games to Apple platforms`](https://developer.apple.comhttps://developer.apple.com/wwdc24/10089/)
- [`10101: Go bindless with Metal 3`](https://developer.apple.comhttps://developer.apple.com/wwdc22/10101/)
- [`10286: Explore bindless rendering in Metal`](https://developer.apple.comhttps://developer.apple.com/wwdc21/10286/)
- [`10150: Explore hybrid rendering with Metal ray tracing`](https://developer.apple.comhttps://developer.apple.com/wwdc21/10150/)

##### Configure the Sample Code Project

To run this sample app, you need the following:

- A Mac with macOS 13 or later, and Xcode 15.3 or later
- An iOS device with iOS 16 or later

> **Note**: This sample doesn’t support running in Simulator.
