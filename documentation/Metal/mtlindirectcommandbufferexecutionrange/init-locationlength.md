# init(location:length:)

**Framework**: Metal  
**Kind**: init

Initializes an command execution range.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.14+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
init(location: UInt32, length: UInt32)
```

## Parameters

- `location`: The start index of the range.
- `length`: The number of items in the range.
