# maxSamplerStateBindCount

**Framework**: Metal  
**Kind**: property

Determines the number of sampler state-binding slots for the argument table.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+

## Declaration

```swift
var maxSamplerStateBindCount: Int { get set }
```

#### Discussion

The maximum value of this parameter is 16.
