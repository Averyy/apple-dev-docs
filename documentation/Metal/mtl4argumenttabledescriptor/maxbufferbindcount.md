# maxBufferBindCount

**Framework**: Metal  
**Kind**: property

Determines the number of buffer-binding slots for the argument table.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
- tvOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)
