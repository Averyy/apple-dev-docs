# reset()

**Framework**: Metal  
**Kind**: method

Deletes all previously set constant values.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- macOS 10.12+
- tvOS 10.0+
- visionOS 1.0+

## Declaration

```swift
func reset()
```

#### Discussion

You don’t need to call this method if you don’t set any constant values.
