# bufferOffset

**Framework**: Metal  
**Kind**: property  
**Required**: Yes

An offset, in bytes, into the buffer instance this tensor shares its storage with, or zero if this tensor does not wrap an underlying buffer.

**Availability**:

- iPadOS 26.0+ (Beta)
