# drawPatches(_:patchStart:patchCount:patchIndexBuffer:patchIndexBufferOffset:instanceCount:baseInstance:tessellationFactorBuffer:tessellationFactorBufferOffset:tessellationFactorBufferInstanceStride:)

**Framework**: Metal  
**Kind**: method  
**Required**: Yes

Encodes a command to render a number of instances of tessellated patches.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.1+
- macOS 10.14+
- tvOS 14.5+
- visionOS 1.0+

## Declaration

```swift
func drawPatches(_ numberOfPatchControlPoints: Int, patchStart: Int, patchCount: Int, patchIndexBuffer: (any MTLBuffer)?, patchIndexBufferOffset: Int, instanceCount: Int, baseInstance: Int, tessellationFactorBuffer buffer: any MTLBuffer, tessellationFactorBufferOffset offset: Int, tessellationFactorBufferInstanceStride instanceStride: Int)
```

#### Discussion

The command generated by this method is equivalent to calling [`setTessellationFactorBuffer(_:offset:instanceStride:)`](mtlrendercommandencoder/settessellationfactorbuffer(_:offset:instancestride:).md) followed by [`drawPatches(numberOfPatchControlPoints:patchStart:patchCount:patchIndexBuffer:patchIndexBufferOffset:instanceCount:baseInstance:)`](mtlrendercommandencoder/drawpatches(numberofpatchcontrolpoints:patchstart:patchcount:patchindexbuffer:patchindexbufferoffset:instancecount:baseinstance:).md).

## Parameters

- `numberOfPatchControlPoints`: The number of control points per patch.   This value must be between   and  , inclusive.
- `patchStart`: The first patch to draw.
- `patchCount`: The number of patches in each instance.
- `patchIndexBuffer`: A buffer that contains indices to patches.
- `patchIndexBufferOffset`: The byte offset within   to start reading indices from.
- `instanceCount`: The number of instances to draw.
- `baseInstance`: The first instance to draw.
- `buffer`: The per-patch tessellation factors buffer.
- `offset`: The distance, in bytes, between the start of the data and the start of the buffer. This value must be a multiple of   bytes.
- `instanceStride`: The distance, in bytes, between two instances of data in the buffer.
