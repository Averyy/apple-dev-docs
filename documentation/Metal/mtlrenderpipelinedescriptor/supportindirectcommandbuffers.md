# supportIndirectCommandBuffers

**Framework**: Metal  
**Kind**: property

A Boolean value that determines whether you can encode commands into an indirect command buffer using the render pipeline.

**Availability**:
- iOS 12.0+
- iPadOS 12.0+
- Mac Catalyst 13.1+
- macOS 10.14+
- tvOS 12.0+
- visionOS 1.0+
