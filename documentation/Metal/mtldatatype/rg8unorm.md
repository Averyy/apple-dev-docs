# MTLDataType.rg8Unorm

**Framework**: Metal  
**Kind**: case

An ordinary pixel with two components, each of which is an 8-bit, normalized, unsigned integer value.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.5+
- visionOS 1.0+

## Declaration

```swift
case rg8Unorm
```

#### Discussion

The color components are in RG order, which stands for red and green.
