# MTLDataType.rgba16Snorm

**Framework**: Metal  
**Kind**: case

An ordinary pixel with four components, each of which is a 16-bit, normalized, signed integer value.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 14.0+
- macOS 11.0+
- tvOS 14.5+
- visionOS 1.0+

## Declaration

```swift
case rgba16Snorm
```

#### Discussion

The color components are in RGBA order, which stands for red, green, blue, and alpha.
