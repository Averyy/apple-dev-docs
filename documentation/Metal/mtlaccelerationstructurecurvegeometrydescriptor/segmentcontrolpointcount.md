# segmentControlPointCount

**Framework**: Metal  
**Kind**: property

The number of control points in each curve segment.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+

## Declaration

```swift
var segmentControlPointCount: Int { get set }
```

#### Discussion

This value can be `2`, `3`, or `4`.
