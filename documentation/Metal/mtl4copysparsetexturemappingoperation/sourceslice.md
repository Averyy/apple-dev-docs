# sourceSlice

**Framework**: Metal  
**Kind**: property

The index of the array slice in the texture source of the copy operation.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+

## Declaration

```swift
var sourceSlice: Int
```

#### Discussion

Provide `0` in this member if the texture type is not an array.
