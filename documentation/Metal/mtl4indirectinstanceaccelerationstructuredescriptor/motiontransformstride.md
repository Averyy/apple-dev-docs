# motionTransformStride

**Framework**: Metal  
**Kind**: property

Sets the stride for motion transform.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+

## Declaration

```swift
var motionTransformStride: Int { get set }
```

#### Discussion

Defaults to `0`, indicating that transforms are tightly packed according to the motion transform type.
