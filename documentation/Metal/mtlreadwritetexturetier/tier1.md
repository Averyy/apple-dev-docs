# MTLReadWriteTextureTier.tier1

**Framework**: Metal  
**Kind**: case

Tier 1 read/write textures are supported.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.13+
- tvOS 11.0+
- visionOS 1.0+

## Declaration

```swift
case tier1
```

#### Discussion

Read/write texture tier 1 supports the following pixel formats:

- [`MTLPixelFormat.r32Float`](mtlpixelformat/r32float.md)
- [`MTLPixelFormat.r32Uint`](mtlpixelformat/r32uint.md)
- [`MTLPixelFormat.r32Sint`](mtlpixelformat/r32sint.md)
