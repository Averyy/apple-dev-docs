# hasDividerLines

**Framework**: Notification Center  
**Kind**: property

**Availability**:
- macOS 10.10+

## Declaration

```swift
@MainActor
var hasDividerLines: Bool { get set }
```

#### Discussion

The default value of this property is [`true`](https://developer.apple.com/documentation/Swift/true).
