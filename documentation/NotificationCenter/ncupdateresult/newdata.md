# NCUpdateResult.newData

**Framework**: Notification Center  
**Kind**: case

The update resulted in new data to display.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 8.0+
- macOS 10.10+

## Declaration

```swift
case newData
```

#### Discussion

When new data is available, the widget might have to update its layout.
