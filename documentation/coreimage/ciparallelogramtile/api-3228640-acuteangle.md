# acuteAngle

**Framework**: Core Image  
**Kind**: intfp  
**Required**: Yes

The primary angle for the repeating parallelogram tile.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
var acuteAngle: Float { get set }
```

#### Discussion

Small values create thin diamond tiles, and higher values create fatter parallelogram tiles.
