# width

**Framework**: Core Image  
**Kind**: intfp  
**Required**: Yes

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- macOS 10.15+
- tvOS 13.0+

## Declaration


#### Discussion
