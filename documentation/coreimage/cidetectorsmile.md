# CIDetectorSmile

**Framework**: Core Image  
**Kind**: data

An option for whether Core Image will perform additional processing to recognize smiles in detected faces.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- macOS 10.9+
- tvOS 9.0+
- visionOS 1.0+
