# init(image:)

**Framework**: Core Image  
**Kind**: init

Initializes a sampler with an image object.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.4+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
convenience init(image im: CIImage)
```

## Parameters

- `im`: The image object to initialize the sampler with.
