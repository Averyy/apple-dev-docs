# initWithImage:

**Framework**: Core Image  
**Kind**: instm

**Availability**:

- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.4+
- tvOS 9.0+

## Declaration


## Parameters

- `im`: The image object to initialize the sampler with.
