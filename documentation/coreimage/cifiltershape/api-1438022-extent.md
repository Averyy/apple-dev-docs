# extent

**Framework**: Core Image  
**Kind**: instp

The extent of the filter shape.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.4+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
var extent: CGRect { get }
```

#### Discussion

Extent is a rectangle that describes the filter shape in the working coordinate space with a fixed area.
