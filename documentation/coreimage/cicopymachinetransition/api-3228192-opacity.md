# opacity

**Framework**: Core Image  
**Kind**: intfp  
**Required**: Yes

The opacity of the copier light.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
var opacity: Float { get set }
```

#### Discussion

A value of 0.0 is transparent. A value of 1.0 is opaque.
