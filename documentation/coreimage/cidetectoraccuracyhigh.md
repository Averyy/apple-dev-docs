# CIDetectorAccuracyHigh

**Framework**: Core Image  
**Kind**: data

Indicates that the detector should choose techniques that are higher in accuracy, even if it requires more processing time.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- Mac Catalyst 13.1+
