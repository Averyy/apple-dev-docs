# CIDetectorEyeBlink

**Framework**: Core Image  
**Kind**: var

An option for whether Core Image will perform additional processing to recognize closed eyes in detected faces.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- macOS 10.9+
- tvOS ?+
- visionOS 1.0+
