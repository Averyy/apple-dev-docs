# kCIInputBiasVectorKey

**Framework**: Core Image  
**Kind**: var

A key to get or set the vector bias value of a Core Image filter.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+

## Declaration

```swift
let kCIInputBiasVectorKey: String
```

#### Discussion

The value for this key needs to be a [`CIVector`](civector.md) instance.
