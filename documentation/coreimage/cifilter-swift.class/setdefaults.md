# setDefaults()

**Framework**: Core Image  
**Kind**: method

Sets all input values for a filter to default values.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- Mac Catalyst 13.1+
- macOS 10.4+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
func setDefaults()
```

#### Discussion

Input values whose default values are not defined are left unchanged.
