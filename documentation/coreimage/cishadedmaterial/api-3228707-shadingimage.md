# shadingImage

**Framework**: Core Image  
**Kind**: intfp  
**Required**: Yes

The image to use as the height field.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
var shadingImage: CIImage? { get set }
```

#### Discussion

The resulting image has greater heights with lighter shades, and lesser heights (lower areas) with darker shades.
