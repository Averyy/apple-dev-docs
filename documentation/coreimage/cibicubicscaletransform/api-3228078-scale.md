# scale

**Framework**: Core Image  
**Kind**: intfp  
**Required**: Yes

The scaling factor to use on the image.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
var scale: Float { get set }
```

#### Discussion

Values less than 1.0 scale down the images. Values greater than 1.0 scale up the image.
