# backsideImage

**Framework**: Core Image  
**Kind**: property  
**Required**: Yes

The image that appears on the back of the source image as the page curls to reveal the target image.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- Mac Catalyst 13.1+
- macOS 10.4+
- tvOS ?+
