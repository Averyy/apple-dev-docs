# CIDetectorAccuracy

**Framework**: Core Image  
**Kind**: data

A key used to specify the desired accuracy for the detector.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- Mac Catalyst 13.1+
- macOS 10.7+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
let CIDetectorAccuracy: String
```

#### Discussion

The value associated with the key should be one of the values found in [`Detector Accuracy Options`](cidetector/detector_accuracy_options.md).
