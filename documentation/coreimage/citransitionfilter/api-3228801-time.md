# time

**Framework**: Core Image  
**Kind**: intfp  
**Required**: Yes

The parametric time of the transition.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
var time: Float { get set }
```

#### Discussion

This value drives the transition from start, at time 0, to end, at time 1.
