# extrapolate

**Framework**: Core Image  
**Kind**: property  
**Required**: Yes

If true, then the color effect will be extrapolated if the input image contains RGB component values outside the range 0.0 to 1.0.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- tvOS 26.0+
- visionOS 26.0+
