# CIFilterProtocol

**Framework**: Core Image  
**Kind**: intf

The properties you use to configure a Core Image filter.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- Mac Catalyst 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
protocol CIFilterProtocol
```

## Topics

### Instance Properties
- [var outputImage: CIImage?](cifilterprotocol/3228048-outputimage.md)
  A [`CIImage`](ciimage.md) object that encapsulates the operations configured in the filter.
### Type Methods
- [static func customAttributes() -> [String : Any]?](cifilterprotocol/3228047-customattributes.md)
  Returns a dictionary that contains key-value pairs that describe the filter.
