# image()

**Framework**: Core Image  
**Kind**: method

Returns the current contents of the image accumulator.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.4+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
func image() -> CIImage
```

#### Return Value

The  image object that represents the current contents of the image accumulator.
