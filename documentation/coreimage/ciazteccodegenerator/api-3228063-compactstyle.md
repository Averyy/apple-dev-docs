# compactStyle

**Framework**: Core Image  
**Kind**: intfp  
**Required**: Yes

A Boolean that specifies whether to force a compact style Aztec code.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
var compactStyle: Float { get set }
```

#### Discussion

Set to `nil` for automatic.
