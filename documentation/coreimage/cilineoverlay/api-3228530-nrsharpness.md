# nrSharpness

**Framework**: Core Image  
**Kind**: intfp  
**Required**: Yes

The amount of sharpening done when removing noise in the image before tracing the edges of the image.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
var nrSharpness: Float { get set }
```

#### Discussion

This improves the edge acquisition.
