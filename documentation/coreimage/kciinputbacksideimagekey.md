# kCIInputBacksideImageKey

**Framework**: Core Image  
**Kind**: var

A key to get or set the backside image for a transition Core Image filter.

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
- tvOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)

## Declaration

```swift
let kCIInputBacksideImageKey: String
```

#### Discussion

The value for this key needs to be a [`CIImage`](ciimage.md) instance.
