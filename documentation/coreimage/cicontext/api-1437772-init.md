# init(forOfflineGPUAt:)

**Framework**: Core Image  
**Kind**: instctr

Creates an OpenGL-based Core Image context using a GPU that is not currently driving a display.

**Availability**:
- macOS 10.10+ - Deprecated in 10.14

## Declaration

```swift
init?(forOfflineGPUAt index: UInt32)
```

#### Return_value

A Core Image context.

#### Discussion

GPU devices that are not currently being used to drive a display can be used for Core Image rendering. Use the [`offlineGPUCount()`](cicontext/1437817-offlinegpucount.md) method to determine whether any such GPUs are available.

To create a Metal-based Core Image context using an offline GPU, use the [`MTLCopyAllDevices()`](https://developer.apple.com/documentation/metal/mtlcopyalldevices()) function to list Metal devices, then choose a device without a display to pass to the [`init(mtlDevice:)`](cicontext/1437609-init.md) method.

## Parameters

- `index`: The index of the offline GPU with which to create the context; a number between zero and the value returned by the   method.
