# init(mtlDevice:)

**Framework**: Core Image  
**Kind**: instctr

Creates a Core Image context using the specified Metal device.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
init(mtlDevice device: any MTLDevice)
```

#### Return_value

A Core Image context.

#### Discussion

Use this method to choose a specific Metal device for rendering when a system contains multiple Metal devices. To create a Metal-based context using the systemâ€™s default Metal device, use the [`init(options:)`](cicontext/1438261-init.md) method.

## Parameters

- `device`: The Metal device object to use for rendering.
