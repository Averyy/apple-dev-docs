# init(imageURL:)

**Framework**: Core Image  
**Kind**: instctr

Creates a RAW filter from the image at the URL location that you specify.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- tvOS 15.0+
- visionOS 1.0+

## Declaration

```swift
convenience init?(imageURL url: URL)
```

## Parameters

- `url`: The URL location of the image.
