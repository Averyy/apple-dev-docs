# kCIInputWeightsKey

**Framework**: Core Image  
**Kind**: data

A key for a [`CIVector`](civector.md) object that describes a weight matrix for use with a convolution filter.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+

- tvOS 9.0+
