# kCIInputWeightsKey

**Framework**: Core Image  
**Kind**: var

A key for a [`CIVector`](civector.md) object that describes a weight matrix for use with a convolution filter.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
let kCIInputWeightsKey: String
```
