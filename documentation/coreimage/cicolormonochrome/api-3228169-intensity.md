# intensity

**Framework**: Core Image  
**Kind**: intfp  
**Required**: Yes

The intensity of the monochrome effect. 

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
var intensity: Float { get set }
```

#### Discussion

A value of 1.0 creates a monochrome image using the supplied color. A value of 0.0 has no effect on the image.
