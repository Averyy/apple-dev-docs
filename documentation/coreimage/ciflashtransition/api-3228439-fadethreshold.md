# fadeThreshold

**Framework**: Core Image  
**Kind**: intfp  
**Required**: Yes

The amount of fade between the flash and the target image.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
var fadeThreshold: Float { get set }
```

#### Discussion

The higher the value, the more flash time and the less fade time.
