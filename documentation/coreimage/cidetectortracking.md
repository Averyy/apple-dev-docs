# CIDetectorTracking

**Framework**: Core Image  
**Kind**: data

A key used to enable or disable face tracking for the detector. Use this option when you want to track faces across frames in a video. 

**Availability**:
- iOS 6.0+
- iPadOS 6.0+
- Mac Catalyst 13.1+
- macOS 10.8+
- tvOS 9.0+
- visionOS 1.0+

```swift
let CIDetectorTracking: String
```
