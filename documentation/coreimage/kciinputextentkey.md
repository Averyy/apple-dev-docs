# kCIInputExtentKey

**Framework**: Core Image  
**Kind**: data

A key for a [`CIVector`](civector.md) object that specifies a rectangle that defines the extent of the effect.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- macOS 10.5+
- tvOS 9.0+
- visionOS 1.0+
