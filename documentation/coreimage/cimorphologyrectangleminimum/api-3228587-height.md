# height

**Framework**: Core Image  
**Kind**: intfp  
**Required**: Yes

The height, in pixels, of the morphological structuring element. 

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
var height: Float { get set }
```

#### Discussion

The value is rounded to the nearest odd integer.
