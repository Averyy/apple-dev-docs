- [let kCIAttributeTypeBoolean: String](kciattributetypeboolean.md)

- [let kCIAttributeTypeInteger: String](kciattributetypeinteger.md)
  An integer value.
- [let kCIAttributeTypeCount: String](kciattributetypecount.md)
