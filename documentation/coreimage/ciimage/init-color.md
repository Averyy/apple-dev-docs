# init(color:)

**Framework**: Core Image  
**Kind**: init

Initializes an image of infinite extent whose entire content is the specified color.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- Mac Catalyst 13.1+
- macOS 10.4+
- tvOS ?+
- visionOS 1.0+

## Declaration

```swift
init(color: CIColor)
```

#### Return Value

The initialized image object.

## Parameters

- `color`: A color object.
