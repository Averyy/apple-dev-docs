# init(color:)

**Framework**: Core Image  
**Kind**: instctr

Initializes an image of infinite extent whose entire content is the specified color.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- Mac Catalyst 13.1+
- macOS 10.4+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
init(color: CIColor)
```

#### Return_value

The initialized image object.

## Parameters

- `color`: A color object.
