# samplingLinear()

**Framework**: Core Image  
**Kind**: method

Create an image by changing the receiver’s sample mode to bilinear interpolation.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.13+
- tvOS 11.0+
- visionOS 1.0+

## Declaration

```swift
func samplingLinear() -> CIImage
```

#### Return Value

 An autoreleased [`CIImage`](ciimage.md) instance with a bilinear sampling.
