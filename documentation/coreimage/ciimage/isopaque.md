# isOpaque

**Framework**: Core Image  
**Kind**: property

Returns YES if the image is known to have and alpha value of `1.0` over the entire image extent.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- Mac Catalyst 13.1+
- macOS 10.4+
- tvOS ?+
