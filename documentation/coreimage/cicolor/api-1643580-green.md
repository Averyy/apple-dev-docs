# green

**Framework**: Core Image  
**Kind**: cldata

Returns a color object whose RGB values are `0.0`, `1.0`, and `0.0` and whose alpha value is `1.0`.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- macOS 10.12+
- tvOS 10.0+
- visionOS 1.0+

## Declaration

```swift
class var green: CIColor { get }
```

#### Return_value

The color object in the sRGB colorspace.
