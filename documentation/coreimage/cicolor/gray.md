# gray

**Framework**: Core Image  
**Kind**: property

Returns a color object whose RGB values are all `0.5` and whose alpha value is `1.0`.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- macOS 10.12+
- tvOS 10.0+
- visionOS 1.0+

## Declaration

```swift
class var gray: CIColor { get }
```

#### Return Value

The color object in the sRGB colorspace.
