# green

**Framework**: Core Image  
**Kind**: property

Returns a singleton Core Image color instance in the sRGB color space with RGB values `0,1,0` and alpha value `1`.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+

- tvOS 10.0+
- visionOS 1.0+
