# properties

**Framework**: Core Image  
**Kind**: structdata

The key for image metadata properties.

**Availability**:
- iOS 5.0+
- iPadOS 5.0+
- Mac Catalyst 13.1+
- macOS 10.8+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
static let properties: CIImageOption
```

#### Discussion

To ensure that an image has no metadata properties, set the value of this key to `[NSNull null]`.
