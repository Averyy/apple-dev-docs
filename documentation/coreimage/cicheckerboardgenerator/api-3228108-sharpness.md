# sharpness

**Framework**: Core Image  
**Kind**: intfp  
**Required**: Yes

The sharpness of the edges in the pattern.

**Availability**:
- iOS 13.0+
- iPadOS 13.0+
- macOS 10.15+
- tvOS 13.0+
- visionOS 1.0+

## Declaration

```swift
var sharpness: Float { get set }
```

#### Discussion

The smaller the value, the more blurry the pattern. Values range from 0.0 to 1.0.
