# MDLMaterialMipMapFilterMode.nearest

**Framework**: Model I/O  
**Kind**: case

Sampling a texture at a size between mipmap levels should return a texel value from the nearest mipmap level.

**Availability**:

- Mac Catalyst 13.0+
