# remove(_:)

**Framework**: Model I/O  
**Kind**: method  
**Required**: Yes

Removes the specified object from this object’s array of children.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
func remove(_ object: MDLObject)
```

## Parameters

- `object`: The object to remove.
