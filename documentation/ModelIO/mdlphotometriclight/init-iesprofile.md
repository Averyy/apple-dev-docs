# init(iesProfile:)

**Framework**: Model I/O  
**Kind**: init

Initializes a light from photometry data in the file at the specified URL.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
init?(iesProfile URL: URL)
```

#### Return Value

A new light object.

## Parameters

- `URL`: The URL to a file containing photometric data.
