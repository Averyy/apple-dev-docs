# MDLVertexAttributeAnisotropy

**Framework**: Model I/O  
**Kind**: var

The attribute data describes the degree to which a surface’s appearance changes in appearance when rotated about its normal vector.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.0+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+
