# Asset File Types

**Framework**: Model I/O

Uniform Type Identifiers for file formats supported by the Model I/O framework.

## Topics

### Constants
- [let kUTTypeAlembic: String](kuttypealembic.md)
  The Alembic file format (common extension `.abc`).
- [let kUTType3dObject: String](kuttype3dobject.md)
  The Object file format (common extension `.obj`).
- [let kUTTypePolygon: String](kuttypepolygon.md)
  The Polygon file format (common extension `.ply`).
- [let kUTTypeStereolithography: String](kuttypestereolithography.md)
  The Stereolithography file format (common extension `.stl`), also known as the Standard Tessellated Geometry format.
- [let kUTTypeUniversalSceneDescription: String](kuttypeuniversalscenedescription.md)
  The Universal Scene Description file format (common extension `.usd`).
