# MDLProbePlacement.uniformGrid

**Framework**: Model I/O  
**Kind**: case

An option to place light probes at each unit coordinate in a three-dimensional grid that evenly divides the region being evaluated.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.0+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+
