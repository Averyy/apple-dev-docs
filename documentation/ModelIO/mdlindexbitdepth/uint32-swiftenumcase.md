# MDLIndexBitDepth.uInt32

**Framework**: Model I/O  
**Kind**: case

Each index in the submesh’s index buffer is a 32-bit integer.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.0+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
case uInt32
```

#### Discussion

An index buffer with this data type can contain a maximum of 4,294,967,296 indices.
