# NSPreferencePaneUnselectReply.unselectLater

**Framework**: Preference Panes  
**Kind**: case

Delay the deselection until the preference pane invokes [`reply(toShouldUnselect:)`](nspreferencepane/reply(toshouldunselect:).md).

**Availability**:
- Mac Catalyst 14.0+
- macOS 10.1+
