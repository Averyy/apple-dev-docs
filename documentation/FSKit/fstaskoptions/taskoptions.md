# taskOptions

**Framework**: FSKit  
**Kind**: property

An array of strings that represent command-line options for the task.

**Availability**:
- macOS 15.4+

## Declaration

```swift
var taskOptions: [String] { get }
```

#### Discussion

This property is equivalent to the `argv` array of C strings passed to a command-line tool.
