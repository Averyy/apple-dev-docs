# blockSize

**Framework**: FSKit  
**Kind**: property

A property for the volume’s block size, in bytes.

**Availability**:
- macOS 15.4+

## Declaration

```swift
var blockSize: Int { get set }
```

#### Discussion

This value defaults to `4096`. Zero isn’t a valid block size.
