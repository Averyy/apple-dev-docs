# transport

**Framework**: Core HID  
**Kind**: property

The data transport for the device.

**Availability**:
- macOS 15.0+

## Declaration

```swift
var transport: HIDDeviceTransport?
```

#### Discussion

A[`HIDDeviceTransport`](hiddevicetransport.md) that specifies the type of transport that the data sent to and from the device  uses.
