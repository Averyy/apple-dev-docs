# reportID

**Framework**: Core HID  
**Kind**: property

The report ID for the report that contains this element.

**Availability**:
- macOS 15.0+

## Declaration

```swift
var reportID: HIDReportID?
```

#### Discussion

This may not be populated if the descriptor contains only one report.
