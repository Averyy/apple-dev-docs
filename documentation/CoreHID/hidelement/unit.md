# unit

**Framework**: Core HID  
**Kind**: property

The HID specification compliant unit code for this element.

**Availability**:
- macOS 15.0+

## Declaration

```swift
var unit: UInt32?
```

#### Discussion

The unit code specifies the physical units for the physical value, such as Seconds or Amperes.

See the HID specification for more details: [`https://www.usb.org/hid`](https://developer.apple.comhttps://www.usb.org/hid).
