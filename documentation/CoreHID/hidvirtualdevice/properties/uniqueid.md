# uniqueID

**Framework**: Core HID  
**Kind**: property

A unique ID for the device.

**Availability**:
- macOS 15.0+

## Declaration

```swift
let uniqueID: String?
```

#### Discussion

Use `uniqueID` to facilitate driver matching, or client matching using [`HIDDeviceManager`](hiddevicemanager.md).
