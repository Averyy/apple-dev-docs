# ssid

**Framework**: Wi-Fi Infrastructure  
**Kind**: property

The SSID of the network.

**Availability**:
- iOS 26.2+
- iPadOS 26.2+

## Declaration

```swift
let ssid: WISSID?
```

#### Discussion

Contains the network identifier when available, or `nil` when the SSID is unknown or not broadcast by the access point.
