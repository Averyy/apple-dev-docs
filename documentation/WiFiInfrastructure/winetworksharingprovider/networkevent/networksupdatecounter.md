# networksUpdateCounter

**Framework**: Wi-Fi Infrastructure  
**Kind**: property

A counter that increments whenever the network data changes.

**Availability**:
- iOS 26.2+
- iPadOS 26.2+

## Declaration

```swift
let networksUpdateCounter: UInt64
```

#### Discussion

The counter increments by `1`.
