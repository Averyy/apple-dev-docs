# FFERR_INVALIDPARAM

**Framework**: Force Feedback  
**Kind**: var

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.2+

## Declaration

```swift
var FFERR_INVALIDPARAM: HRESULT { get }
```

#### Discussion

An invalid parameter was passed to the returning function, or the object was not in a state that admitted the function to be called.
