# dwSize

**Framework**: Force Feedback  
**Kind**: property

Specifies the size of the structure in bytes. This member must be initialized before the structure is used.

**Availability**:
- Mac Catalyst 13.0+
- macOS 10.2+

## Declaration

```swift
UInt32 dwSize;
```
