# init(rawValue:)

**Framework**: Xcodekit  
**Kind**: init

Creates the key for a source-editor command by using the string value you specify.

**Availability**:
- macOS 10.12+

## Declaration

```swift
init(rawValue: String)
```

## Parameters

- `rawValue`: The raw string value of the key.
