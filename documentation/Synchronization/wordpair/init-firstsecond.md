# init(first:second:)

**Framework**: Synchronization  
**Kind**: init

Initialize a new `WordPair` value given both individual words.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+

## Declaration

```swift
init(first: UInt, second: UInt)
```

## Parameters

- `first`: The first word to use in the pair.
- `second`: The second word to use in the pair.
