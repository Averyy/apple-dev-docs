# init(_:)

**Framework**: Synchronization  
**Kind**: init

Initializes a value of this mutex with the given initial state.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+

## Declaration

```swift
init(_ initialValue: consuming sending Value)
```

## Parameters

- `initialValue`: The initial value to give to the mutex.
