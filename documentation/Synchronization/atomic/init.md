# init(_:)

**Framework**: Synchronization  
**Kind**: init

Initializes a value of this atomic with the given initial value.

**Availability**:
- iOS 18.0+
- iPadOS 18.0+
- Mac Catalyst 18.0+
- macOS 15.0+
- tvOS 18.0+
- visionOS 2.0+
- watchOS 11.0+

## Declaration

```swift
init(_ initialValue: consuming Value)
```

## Parameters

- `initialValue`: The initial value to set this atomic.
