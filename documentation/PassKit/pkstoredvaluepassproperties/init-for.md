# init(for:)

**Framework**: PassKit (Apple Pay and Wallet)  
**Kind**: init

Creates a stored-value pass properties object for the specified pass.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS ?+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
convenience init?(for pass: PKPass)
```

## Parameters

- `pass`: A pass that contains a value used for transactions.
