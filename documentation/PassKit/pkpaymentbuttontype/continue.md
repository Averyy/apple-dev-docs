# PKPaymentButtonType.continue

**Framework**: PassKit (Apple Pay and Wallet)  
**Kind**: case

An Apple Pay button useful for general purchases.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- macOS 12.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
case `continue`
```
