# multiTokenContexts

**Framework**: PassKit (Apple Pay and Wallet)  
**Kind**: property

An optional array of payment token contexts to request multiple payment tokens with one payment token per context.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- visionOS 1.0+
