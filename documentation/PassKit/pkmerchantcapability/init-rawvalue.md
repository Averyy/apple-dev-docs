# init(rawValue:)

**Framework**: PassKit (Apple Pay and Wallet)  
**Kind**: init

Creates a merchant capability using the raw value you provide.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 11.0+
- visionOS 1.0+
- watchOS 3.0+

## Declaration

```swift
init(rawValue: UInt)
```

## Parameters

- `rawValue`: An unsigned integer that represents the merchant capability.
