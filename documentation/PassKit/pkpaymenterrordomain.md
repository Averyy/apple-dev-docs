# PKPaymentErrorDomain

**Framework**: PassKit (Apple Pay and Wallet)  
**Kind**: var

The error domain for specific errors associated with Apple Pay in-app or web payments.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.13+
- visionOS 1.0+
- watchOS 4.0+
