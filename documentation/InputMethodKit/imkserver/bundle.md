# bundle()

**Framework**: InputMethodKit  
**Kind**: method

Returns an `NSBundle` object for the input method.

**Availability**:
- macOS 10.5+

## Declaration

```swift
func bundle() -> Bundle!
```

#### Return Value

An `NSBundle` object that is either created from the bundle identifier contained in the server object, or from the main bundle.
