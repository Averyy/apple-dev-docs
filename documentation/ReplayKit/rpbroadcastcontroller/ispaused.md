# isPaused

**Framework**: ReplayKit  
**Kind**: property

A Boolean value indicating whether the broadcast is paused.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- macOS 11.0+
- tvOS 10.0+
- visionOS 1.0+

## Declaration

```swift
var isPaused: Bool { get }
```

#### Discussion

When set to [`true`](https://developer.apple.com/documentation/swift/true), the broadcast is paused.
