# clipDuration

**Framework**: ReplayKit  
**Kind**: property

The duration of movie clips sent the to the movie clip handler extension.

**Availability**:
- iOS 10.0+
- iPadOS 10.0+
- Mac Catalyst 13.1+
- tvOS 10.0+
- visionOS 1.0+

## Declaration

```swift
var clipDuration: TimeInterval { get set }
```

#### Discussion

The default value for a movie clip is 5 seconds.
