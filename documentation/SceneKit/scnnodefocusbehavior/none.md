# SCNNodeFocusBehavior.none

**Framework**: SceneKit  
**Kind**: case

Node is not focusable.

**Availability**:
- iOS 11.0+
- iPadOS 11.0+
- Mac Catalyst 13.1+
- macOS 10.13+
- tvOS 11.0+
- visionOS 1.0+
- watchOS 4.0+

## Declaration

```swift
case none
```

#### Discussion

This behavior is the default for a node.

## See Also

- [SCNNodeFocusBehavior.focusable](scnnodefocusbehavior/focusable.md)
  Node is focusable and prevents nodes that it visually obscures from becoming focusable.
