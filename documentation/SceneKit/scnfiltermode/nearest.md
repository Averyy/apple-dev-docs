# SCNFilterMode.nearest

**Framework**: SceneKit  
**Kind**: case

Texture filtering returns the color from only one texel, whose location is nearest to the coordinates being sampled.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.8+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 3.0+
