# SCNBlendMode.replace

**Framework**: SceneKit  
**Kind**: case

Blend by replacing the destination color with the source color, ignoring alpha.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.8+
