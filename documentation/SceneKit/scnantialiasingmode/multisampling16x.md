# SCNAntialiasingMode.multisampling16X

**Framework**: SceneKit  
**Kind**: case

Enables multisample antialiasing, with sixteen samples per screen pixel.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.8+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 3.0+
