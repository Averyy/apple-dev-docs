# SCNLightSpotOuterAngleKey

**Framework**: SceneKit  
**Kind**: var

The angle, in degrees, of the area partially lit by a spotlight.

**Availability**:
- macOS 10.8+

## Declaration

```swift
let SCNLightSpotOuterAngleKey: String
```

#### Discussion

This attribute’s default value is `45.0`.
