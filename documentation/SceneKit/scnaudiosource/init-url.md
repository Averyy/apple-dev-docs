# init(url:)

**Framework**: SceneKit  
**Kind**: init

Initializes an audio source from the specified audio file.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
init?(url: URL)
```

#### Return Value

A new audio source object.

## Parameters

- `url`: A URL locating an audio file.
