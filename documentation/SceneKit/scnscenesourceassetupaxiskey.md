# SCNSceneSourceAssetUpAxisKey

**Framework**: SceneKit  
**Kind**: var

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.8+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 3.0+

## Declaration

```swift
let SCNSceneSourceAssetUpAxisKey: String
```

#### Discussion

The up-axis for the scene. For example, if the scene is oriented y-up, the corresponding value is the string `"0.0 1.0 0.0"`.
