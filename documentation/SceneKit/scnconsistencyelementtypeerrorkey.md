# SCNConsistencyElementTypeErrorKey

**Framework**: SceneKit  
**Kind**: var

The type of scene file element in which the error occurred.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.8+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 3.0+

## Declaration

```swift
let SCNConsistencyElementTypeErrorKey: String
```

#### Discussion

The value for this key is a [`NSString`](https://developer.apple.com/documentation/Foundation/NSString) object naming a scene file element type.
