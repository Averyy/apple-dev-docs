# Conversation.Capabilities.Element

**Framework**: LiveCommunicationKit  
**Kind**: typealias

The element type of the option set.

**Availability**:
- iOS 17.4+
- iPadOS 17.4+
- Mac Catalyst 17.4+
- visionOS 1.1+
- watchOS 10.4+

## Declaration

```swift
typealias Element = Conversation.Capabilities
```

#### Discussion

To inherit all the default implementations from the `OptionSet` protocol, the `Element` type must be `Self`, the default.
