# invalidate()

**Framework**: LiveCommunicationKit  
**Kind**: method

Invalidates the conversation manager, ends all conversations, and fails all pending concersation actions.

**Availability**:
- iOS 17.4+
- iPadOS 17.4+
- Mac Catalyst 17.4+
- visionOS 1.1+
- watchOS 10.4+
