# init(configuration:)

**Framework**: LiveCommunicationKit  
**Kind**: init

Creates a new conversation manager with a given conversation.

**Availability**:
- iOS 17.4+
- iPadOS 17.4+
- Mac Catalyst 17.4+
- visionOS 1.1+
- watchOS 10.4+

## Declaration

```swift
convenience init(configuration: ConversationManager.Configuration)
```

## Parameters

- `configuration`: A collection of options for configuring the conversation manager.
