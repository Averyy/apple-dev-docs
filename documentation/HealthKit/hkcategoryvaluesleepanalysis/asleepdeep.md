# HKCategoryValueSleepAnalysis.asleepDeep

**Framework**: HealthKit  
**Kind**: case

The user is in deep sleep.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- visionOS 1.0+
- watchOS 9.0+

## Declaration

```swift
case asleepDeep
```

#### Discussion

This value corresponds to stage N3 of the American Academy of Sleep Medicine’s scoring model.
