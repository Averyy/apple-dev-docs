# options

**Framework**: HealthKit  
**Kind**: property

A list of options that define the type of statistical calculations performed and the way in which HealthKit merges data from multiple sources.

**Availability**:
- iOS 15.4+
- iPadOS 15.4+
- Mac Catalyst 15.4+
- macOS 13.0+
- visionOS ?+
- watchOS 8.5+
