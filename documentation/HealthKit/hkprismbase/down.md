# HKPrismBase.down

**Framework**: HealthKit  
**Kind**: case

The prism’s base is at the bottom of the lens.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- visionOS 1.0+
- watchOS 9.0+

## Declaration

```swift
case down
```

#### Discussion

You can only use this value for the prism’s [`verticalBase`](hkvisionprism/verticalbase.md).
