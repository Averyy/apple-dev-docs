# coverageRecord

**Framework**: HealthKit  
**Kind**: property

A type identifier for records containing information about the user’s insurance coverage.

**Availability**:
- iOS 14.0+
- iPadOS 14.0+
- Mac Catalyst 14.0+
- macOS 13.0+
- visionOS 1.0+
- watchOS 7.0+

## Declaration

```swift
static let coverageRecord: HKClinicalTypeIdentifier
```

#### Discussion

Use this identifier for records containing information that might typically appear on the user’s insurance card.
