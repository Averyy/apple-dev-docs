# HKBloodType.notSet

**Framework**: HealthKit  
**Kind**: case

Either the user’s blood type is not set, or the user has not granted your app permission to read the blood type.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.0+
- macOS 13.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
case notSet
```

#### Discussion

For more information on permissions, see `HealthKit`.
