# internationalUnit()

**Framework**: HealthKit  
**Kind**: method

**Availability**:

- iPadOS 11.0+
- Mac Catalyst 13.0+
- macOS 13.0+
- visionOS 1.0+
- watchOS 4.0+

## Declaration

```swift
class func internationalUnit() -> Self
```
