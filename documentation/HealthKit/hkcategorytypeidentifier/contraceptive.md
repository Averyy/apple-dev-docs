# contraceptive

**Framework**: HealthKit  
**Kind**: property

A category sample type that records the use of contraceptives.

**Availability**:
- iOS 14.3+
- iPadOS 14.3+
- Mac Catalyst 14.3+
- macOS 13.0+
- visionOS 1.0+
- watchOS 7.2+

## Declaration

```swift
static let contraceptive: HKCategoryTypeIdentifier
```

#### Discussion

Samples of this type use values from the [`HKCategoryValueContraceptive`](hkcategoryvaluecontraceptive.md) enumeration.
