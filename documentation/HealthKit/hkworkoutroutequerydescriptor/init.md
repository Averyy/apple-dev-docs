# init(_:)

**Framework**: HealthKit  
**Kind**: init

Creates a query descriptor that reads locations from the provided workout route sample.

**Availability**:

- iPadOS 15.4+
- Mac Catalyst 15.4+
- macOS 13.0+
- visionOS ?+

## Parameters

- `workoutRoute`: A workout route sample that contains locations.
