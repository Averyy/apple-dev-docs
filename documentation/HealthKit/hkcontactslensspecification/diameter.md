# diameter

**Framework**: HealthKit  
**Kind**: property

Part of the contact’s fit, it measures the diameter of the lens, measured in mm.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- visionOS 1.0+
- watchOS 9.0+

## Declaration

```swift
@NSCopying
var diameter: HKQuantity? { get }
```

#### Discussion

The property’s range is 9 to 15 mm.
