# baseCurve

**Framework**: HealthKit  
**Kind**: property

Part of the contact’s fit, it measures the curve of the back side of the contact, measured in mm.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- visionOS 1.0+
- watchOS 9.0+

## Declaration

```swift
@NSCopying
var baseCurve: HKQuantity? { get }
```

#### Discussion

The property’s range is 7.8 to 9.2 mm.
