# init(healthStore:)

**Framework**: HealthKit  
**Kind**: init

Creates an attachment store for the provided HealthKit store.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- visionOS 1.0+
- watchOS 9.0+

## Declaration

```swift
init(healthStore: HKHealthStore)
```

## Parameters

- `healthStore`: The interface to the HealthKit store.
