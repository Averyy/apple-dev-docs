# HKMetadataKeyAppleFitnessPlusCatalogIdentifier

**Framework**: HealthKit  
**Kind**: var

**Availability**:
- iOS 18.2+
- iPadOS 18.2+
- Mac Catalyst 18.2+
- macOS 15.2+
- visionOS 2.2+
- watchOS 11.2+

## Declaration

```swift
let HKMetadataKeyAppleFitnessPlusCatalogIdentifier: String
```

#### Discussion

Contains the catalog identifier of the Fitness+ workout.

This key only exists if the workout was completed in Fitness+. The expected value type is an NSString containing the Fitness+ catalog identifier.
