# local()

**Framework**: HealthKit  
**Kind**: method

returns a device object that represents the current device.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.0+
- macOS 13.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
class func local() -> HKDevice
```

#### Return Value

A device object representing the hardware (iPhone, iPod Touch, or Apple Watch) that is running the app.
