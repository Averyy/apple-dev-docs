# isArchived

**Framework**: HealthKit  
**Kind**: property

**Availability**:
- iOS 26.0+ (Beta)
- iPadOS 26.0+ (Beta)
- Mac Catalyst 26.0+ (Beta)
- macOS 26.0+ (Beta)
- visionOS 26.0+ (Beta)
- watchOS 26.0+ (Beta)

## Declaration

```swift
var isArchived: Bool { get }
```

#### Discussion

True if a user moves a medication to the archived section in the medication tracking experience. False if a medication is not in the archived section.
