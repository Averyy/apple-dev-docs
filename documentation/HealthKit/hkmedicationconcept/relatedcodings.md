# relatedCodings

**Framework**: HealthKit  
**Kind**: property

The set of related clinical codings for the medication.

**Availability**:
- iOS 26.0+
- iPadOS 26.0+
- Mac Catalyst 26.0+
- macOS 26.0+
- visionOS 26.0+
- watchOS 26.0+

## Declaration

```swift
var relatedCodings: Set<HKClinicalCoding> { get }
```

#### Discussion

Each coding links the medication to an external medical terminology system, such as RxNorm.
