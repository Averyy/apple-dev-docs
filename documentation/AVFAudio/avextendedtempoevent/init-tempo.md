# init(tempo:)

**Framework**: AVFAudio  
**Kind**: init

Creates an extended tempo event.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+

## Declaration

```swift
init(tempo: Double)
```

#### Discussion

The new tempo begins at the timestamp for this event.

## Parameters

- `tempo`: The tempo in beats per minute as a positive value.
