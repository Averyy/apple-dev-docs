# init()

**Framework**: AVFAudio  
**Kind**: init

Creates an audio sequencer object.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- visionOS 1.0+

## Declaration

```swift
init()
```

#### Discussion

Use this initializer to create a sequencer with tracks that only send events to external MIDI endpoints.
