# init(rawValue:)

**Framework**: AVFAudio  
**Kind**: init

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+

## Declaration

```swift

```

## Parameters

- `rawValue`: The raw value to use for the new instance.
