# init(audioEngine:)

**Framework**: AVFAudio  
**Kind**: init

Creates an audio sequencer that the framework attaches to an audio engine instance.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.11+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
init(audioEngine engine: AVAudioEngine)
```

## Parameters

- `engine`: The engine to attach to.
