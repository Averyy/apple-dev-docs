# enableAdvancedDucking

**Framework**: AVFAudio  
**Kind**: property

Enables advanced ducking which ducks other audio based on the presence of voice activity from local and remote chat participants.

**Availability**:
- iOS 17.0+
- iPadOS 17.0+
- Mac Catalyst 17.0+
- macOS 14.0+
- visionOS 1.0+

## Declaration

```swift
var enableAdvancedDucking: ObjCBool
```
