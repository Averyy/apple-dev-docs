# AVSampleRateConverterAlgorithmKey

**Framework**: AVFAudio  
**Kind**: var

A string value that represents the sample rate converter algorithm to use.

**Availability**:
- iOS 7.0+
- iPadOS 7.0+
- Mac Catalyst 13.1+
- macOS 10.9+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+
