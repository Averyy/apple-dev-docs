# supportsMultichannelContent

**Framework**: AVFAudio  
**Kind**: property

A Boolean value that indicates whether your app supplies multichannel audio content.

**Availability**:
- iOS 15.0+
- iPadOS 15.0+
- Mac Catalyst 15.0+
- tvOS 15.0+
- visionOS 1.0+
- watchOS 8.0+

## Declaration

```swift
var supportsMultichannelContent: Bool { get }
```

#### Discussion

The default value is `false`.
