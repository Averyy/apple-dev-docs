# bandwidth

**Framework**: AVFAudio  
**Kind**: property

The bandwidth of the equalizer filter, in octaves.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
var bandwidth: Float { get set }
```

#### Discussion

The value range of values is `0.05` to `5.0` octaves.
