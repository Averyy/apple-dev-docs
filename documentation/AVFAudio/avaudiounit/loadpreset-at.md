# loadPreset(at:)

**Framework**: AVFAudio  
**Kind**: method

Loads an audio unit using a specified preset.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.10+
- tvOS 9.0+
- visionOS 1.0+

## Declaration

```swift
func loadPreset(at url: URL) throws
```

## Parameters

- `url`: The URL of an audio unit preset file.
