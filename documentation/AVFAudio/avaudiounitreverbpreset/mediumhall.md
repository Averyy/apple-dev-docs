# AVAudioUnitReverbPreset.mediumHall

**Framework**: AVFAudio  
**Kind**: case

A preset that represents a reverb with the acoustic characteristics of a medium-sized hall environment.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.10+
- tvOS ?+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
case mediumHall
```

#### Discussion

This is the default value for [`AVAudioUnitReverb`](avaudiounitreverb.md).
