# AVSpeechSynthesisVoiceIdentifierAlex

**Framework**: AVFAudio  
**Kind**: var

The voice that the system identifies as Alex.

**Availability**:
- iOS 9.0+
- iPadOS 9.0+
- Mac Catalyst 13.1+
- macOS 10.14+
- tvOS 9.0+
- visionOS 1.0+
- watchOS 2.0+

## Declaration

```swift
let AVSpeechSynthesisVoiceIdentifierAlex: String
```

#### Discussion

The Alex voice is only available for the `en-US` language code.
