# defaultInstrument

**Framework**: AVFAudio  
**Kind**: property

A constant that represents the default instrument identifier.

**Availability**:
- iOS 16.0+
- iPadOS 16.0+
- Mac Catalyst 16.0+
- macOS 13.0+
- tvOS 16.0+
- visionOS 1.0+

## Declaration

```swift
class let defaultInstrument: UInt32
```

#### Discussion

The system uses the instrument loaded on the channel [`groupID`](avextendednoteonevent/groupid.md).
