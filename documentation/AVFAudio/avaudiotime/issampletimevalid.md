# isSampleTimeValid

**Framework**: AVFAudio  
**Kind**: property

A Boolean value that indicates whether the sample time and sample rate properties are in a valid state.

**Availability**:
- iOS 8.0+
- iPadOS 8.0+
- Mac Catalyst 13.1+
- macOS 10.10+

- visionOS 1.0+
