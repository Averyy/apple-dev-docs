# storefrontId

**Framework**: App Store Server Notifications  
**Kind**: typealias

An Apple-defined value that uniquely identifies an App Store storefront.

**Availability**:
- App Store Server Notifications 2.8+

## Declaration

```swift
string storefrontId
```

## Mentions

- [App Store Server Notifications changelog](app-store-server-notifications-changelog.md)

#### Discussion

This value is the same as the [`id`](https://developer.apple.com/documentation/storekit/storefront/id-n42) value in StoreKit.
