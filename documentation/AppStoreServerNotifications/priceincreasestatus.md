# priceIncreaseStatus

**Framework**: App Store Server Notifications  
**Kind**: typealias

The status that indicates whether an auto-renewable subscription is subject to a price increase.

**Availability**:
- App Store Server Notifications 2.0+

## Declaration

```swift
int32 priceIncreaseStatus
```
